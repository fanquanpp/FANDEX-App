# 数据类型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

## 整数类型

**整数类型：存储整数值**
`TINYINT | SMALLINT | MEDIUMINT | INT | BIGINT`
```sql
-- MySQL 整数类型
TINYINT          -- 1字节，-128~127
SMALLINT         -- 2字节，-32768~32767
MEDIUMINT        -- 3字节，-8388608~8388607
INT              -- 4字节，-2147483648~2147483647
BIGINT           -- 8字节，-2^63~2^63-1

-- 带显示宽度（MySQL 8.0 已废弃显示宽度）
INT(11)          -- 显示宽度11位（不影响存储范围）

-- 无符号（仅 MySQL）
TINYINT UNSIGNED -- 0~255
INT UNSIGNED     -- 0~4294967295
```

**PostgreSQL 整数类型**
`SMALLINT | INTEGER | BIGINT | SERIAL | BIGSERIAL`
```sql
-- PostgreSQL 整数类型
SMALLINT         -- 2字节
INTEGER          -- 4字节（INT 是别名）
BIGINT           -- 8字节

-- 自增序列
SERIAL           -- 等价于 INTEGER + 自增
BIGSERIAL        -- 等价于 BIGINT + 自增
```

## 浮点与定点类型

**浮点类型：近似值存储**
`FLOAT | DOUBLE | REAL`
```sql
-- MySQL 浮点类型
FLOAT            -- 4字节，单精度
DOUBLE           -- 8字节，双精度
FLOAT(10, 2)     -- 总10位，小数2位（不推荐，精度问题）

-- PostgreSQL 浮点类型
REAL             -- 4字节，单精度
DOUBLE PRECISION -- 8字节，双精度
```

**定点类型：精确值存储**
`DECIMAL(<精度>, <标度>) | NUMERIC(<精度>, <标度>)`
```sql
-- 定点类型（推荐用于金额）
DECIMAL(10, 2)   -- 总10位，小数2位
NUMERIC(10, 2)   -- 等价于 DECIMAL

-- 金额字段
salary DECIMAL(10, 2)     -- -99999999.99 ~ 99999999.99
price  DECIMAL(10, 4)     -- 精确到4位小数
```

## 字符串类型

**定长字符串**
`CHAR(<长度>)`
```sql
-- CHAR: 定长字符串，不足补空格
CHAR(10)         -- 固定10字符，不足补空格
CHAR(255)        -- 最大255字符
```

**变长字符串**
`VARCHAR(<长度>)`
```sql
-- VARCHAR: 变长字符串
VARCHAR(50)      -- 最多50字符
VARCHAR(255)     -- 最多255字符

-- MySQL 大文本
TINYTEXT         -- 最多255字节
TEXT             -- 最多65535字节
MEDIUMTEXT       -- 最多16MB
LONGTEXT         -- 最多4GB
```

**PostgreSQL 字符串类型**
`CHAR(<n>) | VARCHAR(<n>) | TEXT`
```sql
-- PostgreSQL 字符串类型
CHAR(n)          -- 定长，不足补空格
VARCHAR(n)       -- 变长，最大 n 字符
TEXT             -- 变长，无长度限制
```

## 日期时间类型

**MySQL 日期时间类型**
`DATE | TIME | DATETIME | TIMESTAMP | YEAR`
```sql
-- MySQL 日期时间类型
DATE             -- '2026-06-14'，3字节
TIME             -- '14:30:00'，3字节
DATETIME         -- '2026-06-14 14:30:00'，8字节
TIMESTAMP        -- '2026-06-14 14:30:00'，4字节（1970~2038）
YEAR             -- 2026，1字节

-- 带毫秒
DATETIME(3)      -- 精确到毫秒
TIMESTAMP(6)     -- 精确到微秒
```

**PostgreSQL 日期时间类型**
`DATE | TIME | TIMESTAMP | TIMESTAMPTZ | INTERVAL`
```sql
-- PostgreSQL 日期时间类型
DATE             -- '2026-06-14'
TIME             -- '14:30:00'
TIMESTAMP        -- '2026-06-14 14:30:00'
TIMESTAMPTZ      -- 带时区的时间戳
INTERVAL         -- 时间间隔
```

## 布尔类型

**布尔类型：存储真/假值**
`BOOLEAN | BOOL | TINYINT(1)`
```sql
-- PostgreSQL: 原生 BOOLEAN
is_active BOOLEAN  -- true / false

-- MySQL: 用 TINYINT(1) 模拟
is_active TINYINT(1)  -- 1=true, 0=false

-- SQL Server: 用 BIT
is_active BIT  -- 1=true, 0=false
```

## 二进制类型

**二进制类型：存储二进制数据**
`BINARY(<n>) | VARBINARY(<n>) | BLOB`
```sql
-- MySQL 二进制类型
BINARY(n)        -- 定长二进制
VARBINARY(n)     -- 变长二进制
TINYBLOB         -- 最多255字节
BLOB             -- 最多65535字节
MEDIUMBLOB       -- 最多16MB
LONGBLOB         -- 最多4GB

-- PostgreSQL 二进制类型
BYTEA            -- 变长二进制
```

## JSON 类型

**JSON 类型：存储 JSON 数据**
`JSON | JSONB`
```sql
-- MySQL: JSON
metadata JSON

-- PostgreSQL: JSON（文本存储）vs JSONB（二进制存储）
metadata JSON    -- 原始文本，保留空格和键顺序
metadata JSONB   -- 二进制，更快查询，不保留空格

-- 插入 JSON 数据
INSERT INTO products (name, metadata)
VALUES ('Phone', '{"color": "black", "storage": 128}');

-- 查询 JSON 字段
SELECT metadata->>'color' AS color FROM products;
```

## 枚举与集合类型

**ENUM：枚举类型**
`ENUM('<值1>', '<值2>', ...)`
```sql
-- MySQL: ENUM
status ENUM('active', 'inactive', 'deleted') DEFAULT 'active'

-- PostgreSQL: ENUM
CREATE TYPE status_type AS ENUM ('active', 'inactive', 'deleted');
CREATE TABLE users (
  status status_type DEFAULT 'active'
);
```

**SET：集合类型**
`SET('<值1>', '<值2>', ...)`
```sql
-- MySQL: SET（可多选）
tags SET('tech', 'design', 'marketing')
```

## UUID 类型

**UUID 类型：存储唯一标识符**
`UUID | CHAR(36) | BINARY(16)`
```sql
-- PostgreSQL: 原生 UUID
id UUID DEFAULT gen_random_uuid()

-- MySQL: 用 CHAR(36) 或 BINARY(16)
id CHAR(36)
id BINARY(16)

-- 插入 UUID
INSERT INTO users (id, name) VALUES (UUID(), '张三');
```

## 数组类型

**数组类型：存储数组**
`<类型>[] | ARRAY`
```sql
-- PostgreSQL: 数组类型
tags TEXT[]
scores INTEGER[]

-- 插入数组
INSERT INTO posts (title, tags)
VALUES ('Hello', '{"tech", "redis"}');

-- 查询数组
SELECT * FROM posts WHERE 'redis' = ANY(tags);
```

## 货币类型

**货币类型：存储金额**
`MONEY | DECIMAL`
```sql
-- PostgreSQL: MONEY
salary MONEY

-- 通用: DECIMAL（推荐）
salary DECIMAL(10, 2)
```
