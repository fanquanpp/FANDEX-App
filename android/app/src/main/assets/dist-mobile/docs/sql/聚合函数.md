# 聚合函数

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本聚合函数

**COUNT：统计行数**
`COUNT([DISTINCT] <列|*>)`
```sql
-- COUNT(*): 统计所有行（包括 NULL）
SELECT COUNT(*) FROM employees;

-- COUNT(列): 统计非 NULL 行数
SELECT COUNT(phone) FROM employees;

-- COUNT(DISTINCT 列): 统计去重后的行数
SELECT COUNT(DISTINCT department) FROM employees;
```

**SUM：求和**
`SUM([DISTINCT] <列>)`
```sql
-- SUM: 求和（自动忽略 NULL）
SELECT SUM(salary) AS total_salary FROM employees;

-- SUM(DISTINCT): 去重后求和
SELECT SUM(DISTINCT salary) FROM employees;
```

**AVG：平均值**
`AVG([DISTINCT] <列>)`
```sql
-- AVG: 平均值（自动忽略 NULL）
SELECT AVG(salary) AS avg_salary FROM employees;

-- AVG(DISTINCT): 去重后求平均
SELECT AVG(DISTINCT salary) FROM employees;
```

**MAX / MIN：最大值 / 最小值**
`MAX(<列>) | MIN(<列>)`
```sql
-- MAX / MIN: 最大值 / 最小值（自动忽略 NULL）
SELECT MAX(salary) AS max_salary, MIN(salary) AS min_salary
FROM employees;

-- 日期最值
SELECT MAX(hire_date) AS latest_hire, MIN(hire_date) AS earliest_hire
FROM employees;
```

## GROUP BY 分组

**GROUP BY：按列分组聚合**
`GROUP BY <分组列> [, ...]`
```sql
-- 基本分组
SELECT department, COUNT(*) AS emp_count, AVG(salary) AS avg_salary
FROM employees
GROUP BY department;

-- 多列分组
SELECT department, job_title, COUNT(*) AS cnt, AVG(salary) AS avg_salary
FROM employees
GROUP BY department, job_title;
```

**GROUP BY 与 ORDER BY**
`GROUP BY <列> ... ORDER BY <列>`
```sql
-- GROUP BY 与 ORDER BY
SELECT department, AVG(salary) AS avg_salary
FROM employees
GROUP BY department
ORDER BY avg_salary DESC;
```

## HAVING 分组过滤

**HAVING：分组后过滤**
`HAVING <分组条件>`
```sql
-- HAVING: 对分组后的结果进行过滤
SELECT department, COUNT(*) AS emp_count, AVG(salary) AS avg_salary
FROM employees
GROUP BY department
HAVING COUNT(*) > 5 AND AVG(salary) > 50000;
```

**WHERE 与 HAVING 区别**
`WHERE <行条件> ... GROUP BY ... HAVING <组条件>`
```sql
-- WHERE: 分组前过滤（行级）
-- HAVING: 分组后过滤（组级）
SELECT department, COUNT(*) AS cnt
FROM employees
WHERE hire_date >= '2024-01-01'
GROUP BY department
HAVING COUNT(*) > 3;
```

## GROUP_CONCAT 字符串聚合

**GROUP_CONCAT：分组拼接字符串**
`GROUP_CONCAT([DISTINCT] <列> [ORDER BY <列>] SEPARATOR '<分隔符>')`
```sql
-- MySQL: GROUP_CONCAT
SELECT
  department,
  GROUP_CONCAT(name SEPARATOR ', ') AS employees
FROM employees
GROUP BY department;

-- 去重拼接
SELECT
  department,
  GROUP_CONCAT(DISTINCT job_title ORDER BY job_title SEPARATOR ', ') AS titles
FROM employees
GROUP BY department;
```

**STRING_AGG：PostgreSQL/SQL Server**
`STRING_AGG(<列>, '<分隔符>') [WITHIN GROUP (ORDER BY <列>)]`
```sql
-- PostgreSQL / SQL Server
SELECT
  department,
  STRING_AGG(name, ', ') AS employees
FROM employees
GROUP BY department;

-- 带排序的拼接
SELECT
  department,
  STRING_AGG(name, ', ' ORDER BY salary DESC) AS employees
FROM employees
GROUP BY department;
```

## 统计聚合函数

**STDDEV：标准差**
`STDDEV(<列>) | STDDEV_POP(<列>) | STDDEV_SAMP(<列>)`
```sql
-- STDDEV / STDDEV_SAMP: 样本标准差
SELECT STDDEV(salary) AS salary_std FROM employees;

-- STDDEV_POP: 总体标准差
SELECT STDDEV_POP(salary) AS salary_std_pop FROM employees;
```

**VARIANCE：方差**
`VARIANCE(<列>) | VAR_POP(<列>) | VAR_SAMP(<列>)`
```sql
-- VARIANCE / VAR_SAMP: 样本方差
SELECT VARIANCE(salary) AS salary_var FROM employees;

-- VAR_POP: 总体方差
SELECT VAR_POP(salary) AS salary_var_pop FROM employees;
```

## 布尔聚合

**BOOL_AND / BOOL_OR：布尔聚合**
`BOOL_AND(<表达式>) | BOOL_OR(<表达式>)`
```sql
-- PostgreSQL: BOOL_AND / BOOL_OR
SELECT
  department,
  BOOL_AND(salary > 50000) AS all_high_paid,
  BOOL_OR(salary > 100000) AS any_high_paid
FROM employees
GROUP BY department;
```

**MySQL 等价写法**
`MIN(<表达式>) | MAX(<表达式>)`
```sql
-- MySQL: 使用 MIN/MAX 模拟 BOOL_AND/BOOL_OR
SELECT
  department,
  MIN(salary > 50000) AS all_high_paid,  -- 等价于 BOOL_AND
  MAX(salary > 100000) AS any_high_paid  -- 等价于 BOOL_OR
FROM employees
GROUP BY department;
```

## JSON 聚合

**JSON_ARRAYAGG：聚合为 JSON 数组**
`JSON_ARRAYAGG(<列|表达式>)`
```sql
-- MySQL / PostgreSQL: JSON_ARRAYAGG
SELECT
  department,
  JSON_ARRAYAGG(name) AS employee_names
FROM employees
GROUP BY department;
```

**JSON_OBJECTAGG：聚合为 JSON 对象**
`JSON_OBJECTAGG(<键列>, <值列>)`
```sql
-- MySQL / PostgreSQL: JSON_OBJECTAGG
SELECT
  department,
  JSON_OBJECTAGG(name, salary) AS name_salary_map
FROM employees
GROUP BY department;
```
