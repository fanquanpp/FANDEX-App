# 数据操作

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## INSERT

**INSERT 单行：插入一条数据**
`INSERT INTO <table> [(<列>, ...)] VALUES (<值>, ...)`
```sql
-- 插入单行（指定列）
INSERT INTO employees (name, email, dept_id)
VALUES ('张三', 'zhangsan@example.com', 1);

-- 插入单行（所有列，按表定义顺序）
INSERT INTO employees
VALUES (1, '张三', 'zhangsan@example.com', 1, 50000.00, '2026-06-14', NOW());
```

**INSERT 多行：批量插入**
`INSERT INTO <table> [(<列>, ...)] VALUES (<值>, ...), (<值>, ...), ...`
```sql
-- 批量插入（推荐，减少网络往返）
INSERT INTO employees (name, email, dept_id) VALUES
  ('张三', 'zhangsan@example.com', 1),
  ('李四', 'lisi@example.com', 2),
  ('王五', 'wangwu@example.com', 1);
```

**INSERT ... SELECT：从查询插入**
`INSERT INTO <table> [(<列>, ...)] SELECT ...`
```sql
-- 从其他表插入数据
INSERT INTO employees (name, email, dept_id)
SELECT name, email, dept_id
FROM temp_employees
WHERE status = 'approved';
```

**INSERT ... ON DUPLICATE KEY UPDATE：插入或更新**
`INSERT INTO <table> ... VALUES (...) ON DUPLICATE KEY UPDATE <列>=<值>`
```sql
-- MySQL: 插入或更新（upsert）
INSERT INTO users (id, name, email)
VALUES (1, '张三', 'zhangsan@example.com')
ON DUPLICATE KEY UPDATE
  name = VALUES(name),
  email = VALUES(email);
```

**INSERT ... ON CONFLICT：PostgreSQL upsert**
`INSERT INTO <table> ... VALUES (...) ON CONFLICT (<列>) DO UPDATE SET ...`
```sql
-- PostgreSQL: ON CONFLICT
INSERT INTO users (id, name, email)
VALUES (1, '张三', 'zhangsan@example.com')
ON CONFLICT (id) DO UPDATE SET
  name = EXCLUDED.name,
  email = EXCLUDED.email;
```

## UPDATE

**UPDATE 基本语法：更新数据**
`UPDATE <table> SET <列>=<值> [, ...] WHERE <条件>`
```sql
-- 更新单列
UPDATE employees SET salary = 60000 WHERE id = 1;

-- 更新多列
UPDATE employees
SET salary = 60000, dept_id = 2
WHERE id = 1;

-- 无 WHERE 更新全表（危险！）
UPDATE employees SET status = 'active';
```

**UPDATE 使用表达式**
`UPDATE <table> SET <列>=<表达式> WHERE <条件>`
```sql
-- 加薪 10%
UPDATE employees SET salary = salary * 1.1 WHERE dept_id = 1;

-- 使用 CASE WHEN 条件更新
UPDATE employees
SET salary = CASE
  WHEN dept_id = 1 THEN salary * 1.10
  WHEN dept_id = 2 THEN salary * 1.05
  ELSE salary
END;
```

**UPDATE FROM：基于其他表更新**
`UPDATE <table> SET ... FROM <other_table> WHERE <连接条件>`
```sql
-- PostgreSQL: UPDATE FROM
UPDATE employees e
SET salary = d.avg_salary
FROM dept_avg_salary d
WHERE e.dept_id = d.dept_id;

-- MySQL: UPDATE JOIN
UPDATE employees e
JOIN dept_avg_salary d ON e.dept_id = d.dept_id
SET e.salary = d.avg_salary;
```

## DELETE

**DELETE 基本语法：删除数据**
`DELETE FROM <table> WHERE <条件>`
```sql
-- 删除指定行
DELETE FROM employees WHERE id = 1;

-- 删除多行
DELETE FROM employees WHERE dept_id = 5 AND status = 'inactive';

-- 无 WHERE 删除全表（危险！）
DELETE FROM employees;
```

**DELETE USING：基于其他表删除**
`DELETE FROM <table> USING <other_table> WHERE <连接条件>`
```sql
-- PostgreSQL: DELETE USING
DELETE FROM orders
USING customers
WHERE orders.customer_id = customers.id
  AND customers.status = 'deleted';

-- MySQL: DELETE JOIN
DELETE o FROM orders o
JOIN customers c ON o.customer_id = c.id
WHERE c.status = 'deleted';
```

## MERGE

**MERGE：合并插入/更新/删除**
`MERGE INTO <target> USING <source> ON <条件> WHEN MATCHED THEN ... WHEN NOT MATCHED THEN ...`
```sql
-- Oracle / SQL Server / PostgreSQL 15+
MERGE INTO employees target
USING temp_employees source
ON (target.id = source.id)
WHEN MATCHED THEN
  UPDATE SET target.salary = source.salary
WHEN NOT MATCHED THEN
  INSERT (id, name, salary) VALUES (source.id, source.name, source.salary);
```

## RETURNING

**RETURNING：返回被修改的行**
`INSERT|UPDATE|DELETE ... RETURNING <列>`
```sql
-- PostgreSQL / Oracle: RETURNING
INSERT INTO employees (name, salary)
VALUES ('张三', 50000)
RETURNING id, name;

UPDATE employees SET salary = 60000
WHERE id = 1
RETURNING id, name, salary;

DELETE FROM employees
WHERE id = 1
RETURNING id, name;
```

## 事务相关

**自动提交与手动事务**
`BEGIN | COMMIT | ROLLBACK`
```sql
-- 开始事务
BEGIN;

-- 执行多个操作
INSERT INTO orders (user_id, amount) VALUES (1, 100);
UPDATE users SET balance = balance - 100 WHERE id = 1;

-- 提交
COMMIT;

-- 回滚
-- ROLLBACK;
```

**保存点**
`SAVEPOINT <name> | ROLLBACK TO <name>`
```sql
-- 设置保存点
BEGIN;
INSERT INTO orders (user_id, amount) VALUES (1, 100);
SAVEPOINT after_insert;
UPDATE users SET balance = balance - 100 WHERE id = 1;

-- 回滚到保存点
ROLLBACK TO after_insert;

-- 提交
COMMIT;
```
