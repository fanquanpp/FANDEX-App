# 约束

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## PRIMARY KEY

**主键约束：唯一标识行**
`[CONSTRAINT <name>] PRIMARY KEY (<列> [, ...])`
```sql
-- 列级主键
CREATE TABLE users (
  id BIGINT PRIMARY KEY,
  name VARCHAR(50)
);

-- 表级主键
CREATE TABLE users (
  id BIGINT,
  name VARCHAR(50),
  CONSTRAINT pk_users PRIMARY KEY (id)
);

-- 复合主键
CREATE TABLE order_items (
  order_id BIGINT,
  product_id BIGINT,
  quantity INT,
  PRIMARY KEY (order_id, product_id)
);
```

**自增主键**
`AUTO_INCREMENT | SERIAL | IDENTITY`
```sql
-- MySQL: AUTO_INCREMENT
CREATE TABLE users (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(50)
);

-- PostgreSQL: SERIAL
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  name VARCHAR(50)
);

-- SQL Server: IDENTITY
CREATE TABLE users (
  id BIGINT PRIMARY KEY IDENTITY(1,1),
  name VARCHAR(50)
);
```

## NOT NULL

**非空约束：列不允许 NULL**
`<列> <类型> NOT NULL`
```sql
-- 列级 NOT NULL
CREATE TABLE users (
  id BIGINT PRIMARY KEY,
  name VARCHAR(50) NOT NULL,
  email VARCHAR(100) NOT NULL
);
```

---

## DEFAULT

**默认值约束：插入时未指定则使用默认值**
`<列> <类型> DEFAULT <值>`
```sql
-- 列级 DEFAULT
CREATE TABLE users (
  id BIGINT PRIMARY KEY,
  status VARCHAR(20) DEFAULT 'active',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  balance DECIMAL(10,2) DEFAULT 0.00
);
```

## UNIQUE

**唯一约束：列值唯一**
`[CONSTRAINT <name>] UNIQUE (<列> [, ...])`
```sql
-- 列级 UNIQUE
CREATE TABLE users (
  id BIGINT PRIMARY KEY,
  email VARCHAR(100) UNIQUE,
  phone VARCHAR(20) UNIQUE
);

-- 表级 UNIQUE（可命名）
CREATE TABLE users (
  id BIGINT PRIMARY KEY,
  email VARCHAR(100),
  phone VARCHAR(20),
  CONSTRAINT uk_email UNIQUE (email),
  CONSTRAINT uk_phone UNIQUE (phone)
);

-- 复合唯一约束
CREATE TABLE user_courses (
  user_id BIGINT,
  course_id BIGINT,
  CONSTRAINT uk_user_course UNIQUE (user_id, course_id)
);
```

## FOREIGN KEY

**外键约束：保证引用完整性**
`[CONSTRAINT <name>] FOREIGN KEY (<列>) REFERENCES <table>(<列>) [ON DELETE <action>] [ON UPDATE <action>]`
```sql
-- 基本外键
CREATE TABLE orders (
  id BIGINT PRIMARY KEY,
  user_id BIGINT,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 带删除/更新动作
CREATE TABLE orders (
  id BIGINT PRIMARY KEY,
  user_id BIGINT,
  CONSTRAINT fk_orders_user
    FOREIGN KEY (user_id) REFERENCES users(id)
    ON DELETE CASCADE
    ON UPDATE CASCADE
);
```

**ON DELETE / ON UPDATE 动作**
`ON DELETE|UPDATE RESTRICT | CASCADE | SET NULL | SET DEFAULT | NO ACTION`
```sql
-- RESTRICT: 禁止删除被引用的行（默认）
ON DELETE RESTRICT

-- CASCADE: 级联删除
ON DELETE CASCADE

-- SET NULL: 设为 NULL
ON DELETE SET NULL

-- SET DEFAULT: 设为默认值
ON DELETE SET DEFAULT

-- NO ACTION: 类似 RESTRICT（SQL 标准）
ON DELETE NO ACTION
```

## CHECK

**检查约束：自定义条件验证**
`[CONSTRAINT <name>] CHECK (<条件>)`
```sql
-- 列级 CHECK
CREATE TABLE products (
  id BIGINT PRIMARY KEY,
  name VARCHAR(100),
  price DECIMAL(10,2) CHECK (price > 0),
  stock INT CHECK (stock >= 0)
);

-- 表级 CHECK（可命名，可跨列）
CREATE TABLE employees (
  id BIGINT PRIMARY KEY,
  name VARCHAR(50),
  salary DECIMAL(10,2),
  bonus DECIMAL(10,2),
  CONSTRAINT chk_salary CHECK (salary > 0),
  CONSTRAINT chk_bonus CHECK (bonus >= 0 AND bonus <= salary * 0.5)
);
```

## 约束管理

**添加约束**
`ALTER TABLE <table> ADD [CONSTRAINT <name>] <约束类型> ...`
```sql
-- 添加主键
ALTER TABLE users ADD PRIMARY KEY (id);

-- 添加唯一约束
ALTER TABLE users ADD CONSTRAINT uk_email UNIQUE (email);

-- 添加外键
ALTER TABLE orders
ADD CONSTRAINT fk_orders_user
FOREIGN KEY (user_id) REFERENCES users(id);

-- 添加 CHECK 约束
ALTER TABLE products ADD CONSTRAINT chk_price CHECK (price > 0);

-- 添加 NOT NULL
ALTER TABLE users MODIFY COLUMN name VARCHAR(50) NOT NULL;
ALTER TABLE users ALTER COLUMN name SET NOT NULL;  -- PostgreSQL
```

**删除约束**
`ALTER TABLE <table> DROP CONSTRAINT <name> | DROP PRIMARY KEY | DROP FOREIGN KEY <name>`
```sql
-- 删除约束
ALTER TABLE users DROP CONSTRAINT uk_email;
ALTER TABLE users DROP CONSTRAINT chk_salary;

-- MySQL 简写
ALTER TABLE users DROP INDEX uk_email;
ALTER TABLE orders DROP FOREIGN KEY fk_orders_user;

-- 删除主键
ALTER TABLE users DROP PRIMARY KEY;
```

**禁用/启用约束**
`ALTER TABLE <table> DISABLE|ENABLE CONSTRAINT <name>`
```sql
-- Oracle / PostgreSQL
ALTER TABLE users DISABLE CONSTRAINT uk_email;
ALTER TABLE users ENABLE CONSTRAINT uk_email;

-- SQL Server
ALTER TABLE users NOCHECK CONSTRAINT chk_salary;
ALTER TABLE users CHECK CONSTRAINT chk_salary;
```

## 约束命名规范

**命名约定：约束命名规则**
`<前缀>_<表名>_<列名>`
```sql
-- 命名约定:
-- pk_<table>       主键
-- fk_<table>_<ref> 外键
-- uk_<table>_<col> 唯一约束
-- chk_<table>_<col> CHECK 约束
-- nn_<table>_<col> NOT NULL（非标准）

CREATE TABLE orders (
  id BIGINT,
  user_id BIGINT,
  email VARCHAR(100),
  CONSTRAINT pk_orders PRIMARY KEY (id),
  CONSTRAINT fk_orders_users FOREIGN KEY (user_id) REFERENCES users(id),
  CONSTRAINT uk_orders_email UNIQUE (email)
);
```
