# 子查询

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 标量子查询

**标量子查询：返回单个值**
`WHERE <列> <运算符> (SELECT ...)`
```sql
-- 返回单个值的子查询
SELECT name, salary
FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);

-- 在 SELECT 中使用
SELECT
  name,
  salary,
  (SELECT AVG(salary) FROM employees) AS avg_salary
FROM employees;
```

---

## 列子查询

**ANY/SOME：与子查询任一值比较**
`WHERE <列> <运算符> ANY (SELECT ...)`
```sql
-- ANY/SOME: 与子查询结果的任一值比较
SELECT name, salary FROM employees
WHERE salary > ANY (SELECT salary FROM employees WHERE dept_id = 5);
-- 等价于：salary > 子查询中的最小值

-- = ANY 等价于 IN
SELECT name, salary FROM employees
WHERE dept_id = ANY (SELECT id FROM departments WHERE region = 'East');
```

**ALL：与子查询所有值比较**
`WHERE <列> <运算符> ALL (SELECT ...)`
```sql
-- ALL: 与子查询结果的所有值比较
SELECT name, salary FROM employees
WHERE salary > ALL (SELECT salary FROM employees WHERE dept_id = 5);
-- 等价于：salary > 子查询中的最大值

-- <> ALL 等价于 NOT IN（无 NULL 时）
SELECT name, salary FROM employees
WHERE dept_id <> ALL (SELECT id FROM departments WHERE region = 'East');
```

## 行子查询

**行子查询：返回单行多列**
`WHERE (<列1>, <列2>) = (SELECT ...)`
```sql
-- 行子查询: 返回单行多列
SELECT * FROM employees
WHERE (dept_id, salary) = (
  SELECT dept_id, MAX(salary)
  FROM employees
  GROUP BY dept_id
  HAVING dept_id = 5
);

-- 多列 IN
SELECT * FROM orders
WHERE (customer_id, order_date) IN (
  SELECT customer_id, MAX(order_date)
  FROM orders
  GROUP BY customer_id
);
```

## 表子查询

**派生表：FROM 中的子查询**
`FROM (SELECT ...) AS <alias>`
```sql
-- 在 FROM 中使用（派生表）
SELECT dept_name, avg_salary
FROM (
  SELECT department AS dept_name, AVG(salary) AS avg_salary
  FROM employees
  GROUP BY department
) AS dept_stats
WHERE avg_salary > 50000;
```

---

## 相关子查询

**相关子查询：引用外层查询列**
`WHERE <列> = (SELECT ... FROM ... WHERE ... = <外层列>)`
```sql
-- 查询每个部门薪资最高的员工
SELECT name, department, salary
FROM employees e
WHERE salary = (
  SELECT MAX(salary)
  FROM employees e2
  WHERE e2.department = e.department
);
```

## EXISTS 与 NOT EXISTS

**EXISTS：检查子查询是否返回行**
`WHERE EXISTS (SELECT 1 FROM ... WHERE ...)`
```sql
-- EXISTS: 检查子查询是否返回行
SELECT d.department_name
FROM departments d
WHERE EXISTS (
  SELECT 1 FROM employees e
  WHERE e.dept_id = d.id AND e.salary > 100000
);
```

**NOT EXISTS：避免 NULL 陷阱**
`WHERE NOT EXISTS (SELECT 1 FROM ... WHERE ...)`
```sql
-- NOT EXISTS 不受 NULL 影响（推荐）
SELECT name FROM employees e
WHERE NOT EXISTS (
  SELECT 1 FROM employees e2
  WHERE e2.dept_id = e.dept_id AND e2.salary > 100000
);
```

## IN 与 NOT IN

**IN：值在子查询结果中**
`WHERE <列> IN (SELECT ...)`
```sql
-- IN: 检查值是否在子查询结果中
SELECT d.department_name
FROM departments d
WHERE d.id IN (
  SELECT dept_id FROM employees WHERE salary > 100000
);
```

**NOT IN 的 NULL 陷阱**
`WHERE <列> NOT IN (SELECT ...)`
```sql
-- NOT IN: 如果子查询包含 NULL，整个查询返回空
SELECT name FROM employees
WHERE dept_id NOT IN (SELECT id FROM departments WHERE region = 'East');
-- 如果 departments.id 中有 NULL，结果为空

-- 推荐使用 NOT EXISTS 替代 NOT IN
SELECT name FROM employees e
WHERE NOT EXISTS (
  SELECT 1 FROM departments d
  WHERE d.id = e.dept_id AND d.region = 'East'
);
```

## 子查询位置

**SELECT 中的子查询：标量子查询**
`SELECT <列>, (SELECT ...) AS <别名>`
```sql
-- SELECT 中使用标量子查询
SELECT
  name,
  salary,
  (SELECT AVG(salary) FROM employees) AS avg_salary,
  salary - (SELECT AVG(salary) FROM employees) AS diff
FROM employees;
```

**WHERE 中的子查询：条件过滤**
`WHERE <列> <运算符> (SELECT ...)`
```sql
-- WHERE 中使用子查询
SELECT name, salary FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);
```

**HAVING 中的子查询：分组后过滤**
`HAVING <聚合> <运算符> (SELECT ...)`
```sql
-- HAVING 中使用子查询
SELECT department, AVG(salary) AS avg_sal
FROM employees
GROUP BY department
HAVING AVG(salary) > (SELECT AVG(salary) FROM employees);
```

## 子查询与 JOIN 对比

**子查询 vs JOIN：等价改写**
`JOIN <table> ON ... | WHERE <列> IN (SELECT ...)`
```sql
-- 子查询写法
SELECT name
FROM employees
WHERE dept_id IN (SELECT id FROM departments WHERE region = 'East');

-- JOIN 写法（通常更高效）
SELECT e.name
FROM employees e
JOIN departments d ON e.dept_id = d.id
WHERE d.region = 'East';
```
