# 窗口函数

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本语法

**OVER 子句：定义窗口框架**
`<窗口函数>() OVER ([PARTITION BY <列>] [ORDER BY <列>] [frame_clause])`
```sql
-- 基本语法
SELECT
  column1,
  column2,
  window_function() OVER (
    [PARTITION BY partition_column]
    [ORDER BY sort_column]
    [frame_clause]
  ) AS alias
FROM table_name;
```

## 排名函数

**ROW_NUMBER：唯一行号**
`ROW_NUMBER() OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 按薪资降序排名（唯一，无并列）
SELECT
  name,
  salary,
  ROW_NUMBER() OVER (ORDER BY salary DESC) AS row_num
FROM employees;

-- 分部门排名
SELECT
  name,
  department,
  salary,
  ROW_NUMBER() OVER (PARTITION BY department ORDER BY salary DESC) AS dept_rank
FROM employees;
```

**RANK：允许并列，跳号**
`RANK() OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 允许并列，跳号
-- 示例: 1, 2, 2, 4, 5
SELECT
  name,
  salary,
  RANK() OVER (ORDER BY salary DESC) AS rank_num
FROM employees;
```

**DENSE_RANK：允许并列，不跳号**
`DENSE_RANK() OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 允许并列，不跳号
-- 示例: 1, 2, 2, 3, 4
SELECT
  name,
  salary,
  DENSE_RANK() OVER (ORDER BY salary DESC) AS dense_rank_num
FROM employees;
```

**NTILE：分桶**
`NTILE(<N>) OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 将数据分为 N 个桶
SELECT
  name,
  salary,
  NTILE(4) OVER (ORDER BY salary DESC) AS quartile
FROM employees;
```

## 聚合窗口函数

**SUM OVER：累计求和**
`SUM(<列>) OVER ([PARTITION BY <列>] [ORDER BY <列>])`
```sql
-- 累计求和
SELECT
  order_date,
  amount,
  SUM(amount) OVER (ORDER BY order_date) AS running_total
FROM orders;

-- 分组累计求和
SELECT
  department,
  name,
  salary,
  SUM(salary) OVER (PARTITION BY department ORDER BY hire_date) AS dept_running_total
FROM employees;
```

**AVG OVER：移动平均**
`AVG(<列>) OVER ([PARTITION BY <列>] [ORDER BY <列>] <frame>)`
```sql
-- 3天移动平均
SELECT
  date,
  price,
  AVG(price) OVER (
    ORDER BY date
    ROWS BETWEEN 2 PRECEDING AND CURRENT ROW
  ) AS moving_avg_3day
FROM stock_prices;
```

## 偏移函数

**LAG：访问前 N 行**
`LAG(<列>, [<offset>], [<default>]) OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 访问前 N 行的值
SELECT
  date,
  price,
  LAG(price, 1) OVER (ORDER BY date) AS prev_price,
  price - LAG(price, 1) OVER (ORDER BY date) AS price_change
FROM stock_prices;

-- 指定偏移量和默认值
SELECT
  date,
  price,
  LAG(price, 7, 0) OVER (ORDER BY date) AS price_7_days_ago
FROM stock_prices;
```

**LEAD：访问后 N 行**
`LEAD(<列>, [<offset>], [<default>]) OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 访问后 N 行的值
SELECT
  date,
  price,
  LEAD(price, 1) OVER (ORDER BY date) AS next_price
FROM stock_prices;
```

**FIRST_VALUE / LAST_VALUE：窗口首尾值**
`FIRST_VALUE(<列>) OVER (...) | LAST_VALUE(<列>) OVER (...)`
```sql
-- 每个部门薪资最高和最低的员工
SELECT
  name,
  department,
  salary,
  FIRST_VALUE(name) OVER (PARTITION BY department ORDER BY salary DESC) AS highest_paid,
  LAST_VALUE(name) OVER (
    PARTITION BY department
    ORDER BY salary DESC
    ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
  ) AS lowest_paid
FROM employees;
```

## 值函数

**CUME_DIST：累积分布**
`CUME_DIST() OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 累积分布（0~1）
SELECT
  name,
  salary,
  CUME_DIST() OVER (ORDER BY salary) AS cume_dist
FROM employees;
```

**PERCENT_RANK：百分位排名**
`PERCENT_RANK() OVER ([PARTITION BY <列>] ORDER BY <列>)`
```sql
-- 百分位排名（0~1）
SELECT
  name,
  salary,
  PERCENT_RANK() OVER (ORDER BY salary) AS percent_rank
FROM employees;
```

## 窗口框架

**ROWS BETWEEN：行范围**
`ROWS BETWEEN <start> AND <end>`
```sql
-- 窗口框架（frame_clause）
-- ROWS: 按物理行
-- RANGE: 按逻辑范围（值相同的一组行）

-- 累计求和（从第一行到当前行）
SUM(amount) OVER (
  ORDER BY date
  ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
)

-- 前2行到后2行
AVG(price) OVER (
  ORDER BY date
  ROWS BETWEEN 2 PRECEDING AND 2 FOLLOWING
)
```

**RANGE BETWEEN：值范围**
`RANGE BETWEEN <start> AND <end>`
```sql
-- RANGE: 按逻辑范围
-- 同一薪资值的行被视为一组
SELECT
  name,
  salary,
  SUM(salary) OVER (
    ORDER BY salary
    RANGE BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
  ) AS running_total
FROM employees;
```

**框架边界选项**
`UNBOUNDED PRECEDING | <N> PRECEDING | CURRENT ROW | <N> FOLLOWING | UNBOUNDED FOLLOWING`
```sql
-- 边界选项:
-- UNBOUNDED PRECEDING: 窗口起点
-- N PRECEDING: 前 N 行
-- CURRENT ROW: 当前行
-- N FOLLOWING: 后 N 行
-- UNBOUNDED FOLLOWING: 窗口终点

-- 默认框架:
-- 有 ORDER BY: RANGE BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
-- 无 ORDER BY: ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
```

## NTH_VALUE

**NTH_VALUE：窗口第 N 行的值**
`NTH_VALUE(<列>, <N>) OVER (...)`
```sql
-- 每个部门薪资第3高的员工
SELECT
  name,
  department,
  salary,
  NTH_VALUE(salary, 3) OVER (
    PARTITION BY department
    ORDER BY salary DESC
    ROWS BETWEEN UNBOUNDED PRECEDING AND UNBOUNDED FOLLOWING
  ) AS third_highest
FROM employees;
```

## Top-N 查询

**ROW_NUMBER 实现 Top-N**
`ROW_NUMBER() OVER (PARTITION BY <列> ORDER BY <列>)`
```sql
-- 每个部门薪资前3名
SELECT * FROM (
  SELECT
    name,
    department,
    salary,
    ROW_NUMBER() OVER (PARTITION BY department ORDER BY salary DESC) AS rn
  FROM employees
) ranked
WHERE rn <= 3;
```

## 去重

**DISTINCT 与窗口函数**
`SELECT DISTINCT ... FROM (SELECT ... ROW_NUMBER() OVER (...))`
```sql
-- 去重保留最新记录
SELECT DISTINCT user_id, latest_action
FROM (
  SELECT
    user_id,
    action AS latest_action,
    ROW_NUMBER() OVER (PARTITION BY user_id ORDER BY created_at DESC) AS rn
  FROM user_actions
) t
WHERE rn = 1;
```
