# 集合操作

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## UNION

**UNION：合并并去重**
`<查询1> UNION <查询2>`
```sql
-- 合并两个查询结果，自动去重
SELECT name FROM employees
UNION
SELECT name FROM contractors;

-- 等价于
SELECT DISTINCT name FROM (
  SELECT name FROM employees
  UNION ALL
  SELECT name FROM contractors
) combined;
```

**UNION ALL：合并不去重**
`<查询1> UNION ALL <查询2>`
```sql
-- 合并两个查询结果，保留重复行（性能更好）
SELECT name, 'employee' AS type FROM employees
UNION ALL
SELECT name, 'contractor' AS type FROM contractors;
```

---

## INTERSECT

**INTERSECT：交集**
`<查询1> INTERSECT <查询2>`
```sql
-- 返回两个查询结果中都存在的行（去重）
SELECT product_id FROM orders_2025
INTERSECT
SELECT product_id FROM orders_2026;

-- INTERSECT ALL（PostgreSQL）: 保留重复行
SELECT product_id FROM orders_2025
INTERSECT ALL
SELECT product_id FROM orders_2026;
```

---

## EXCEPT

**EXCEPT：差集**
`<查询1> EXCEPT <查询2>`
```sql
-- 返回在查询1中但不在查询2中的行（去重）
SELECT product_id FROM all_products
EXCEPT
SELECT product_id FROM discontinued_products;

-- EXCEPT ALL（PostgreSQL）: 保留重复行
SELECT product_id FROM all_products
EXCEPT ALL
SELECT product_id FROM discontinued_products;

-- MySQL 使用 MINUS（Oracle）或改写为 LEFT JOIN
SELECT a.product_id
FROM all_products a
LEFT JOIN discontinued_products d ON a.product_id = d.product_id
WHERE d.product_id IS NULL;
```

## 集合操作规则

**列数与类型匹配**
`<查询1> UNION|INTERSECT|EXCEPT <查询2>`
```sql
-- 规则:
-- 1. 两个查询的列数必须相同
-- 2. 对应列的数据类型必须兼容
-- 3. 结果的列名取自第一个查询
-- 4. ORDER BY 只能在最后一个查询后使用

-- 正确示例
SELECT name, salary FROM employees
UNION ALL
SELECT name, hourly_rate * 2080 AS salary FROM contractors
ORDER BY salary DESC;
```

## 多个集合操作

**组合多个集合操作**
`<查询1> UNION <查询2> INTERSECT <查询3>`
```sql
-- 多个集合操作（注意优先级: INTERSECT > UNION）
SELECT product_id FROM orders_2024
UNION
SELECT product_id FROM orders_2025
INTERSECT
SELECT product_id FROM orders_2026;

-- 使用括号控制优先级
(SELECT product_id FROM orders_2024
 UNION
 SELECT product_id FROM orders_2025)
INTERSECT
SELECT product_id FROM orders_2026;
```

## ORDER BY 与 LIMIT

**集合操作中的排序**
`<查询1> UNION <查询2> ORDER BY <列>`
```sql
-- ORDER BY 只能在最后，作用于整个结果集
SELECT name, salary FROM employees
UNION ALL
SELECT name, salary FROM contractors
ORDER BY salary DESC;

-- 错误: 不能在中间查询使用 ORDER BY
-- SELECT name FROM employees ORDER BY name
-- UNION
-- SELECT name FROM contractors;
```

**集合操作中的分页**
`<查询1> UNION <查询2> ORDER BY <列> LIMIT <N> OFFSET <M>`
```sql
-- LIMIT 作用于整个结果集
SELECT name, salary FROM employees
UNION ALL
SELECT name, salary FROM contractors
ORDER BY salary DESC
LIMIT 10 OFFSET 20;
```
