# PostgreSQL 索引类型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## B-tree 索引

**创建 B-tree 索引（默认）**
`CREATE INDEX <索引名> ON <表名>(<列名>)`
```sql
-- 默认 B-tree 索引
CREATE INDEX idx_employees_name ON employees(name);
```

---

## Hash 索引

**创建 Hash 索引**
`CREATE INDEX <索引名> ON <表名> USING HASH (<列名>)`
```sql
-- Hash 索引（仅支持等值查找）
CREATE INDEX idx_users_email_hash ON users USING HASH (email);
```

---

## GiST 索引

**创建空间索引**
`CREATE INDEX <索引名> ON <表名> USING GIST (<列名>)`
```sql
-- 空间索引（PostGIS）
CREATE INDEX idx_locations_geom ON locations USING GIST (geom);
```

**创建范围索引**
`CREATE INDEX <索引名> ON <表名> USING GIST (<范围列>)`
```sql
-- 范围类型索引
CREATE INDEX idx_reservations_period ON reservations USING GIST (period);
```

**创建全文索引（GiST）**
`CREATE INDEX <索引名> ON <表名> USING GIST (to_tsvector('<语言>', <列名>))`
```sql
-- 全文检索（较慢，GIN 更常用）
CREATE INDEX idx_articles_fts_gist ON articles USING GIST (to_tsvector('english', content));
```

---

## GIN 索引

**创建全文索引（推荐）**
`CREATE INDEX <索引名> ON <表名> USING GIN (to_tsvector('<语言>', <列名>))`
```sql
-- 全文检索（推荐 GIN）
CREATE INDEX idx_articles_fts ON articles USING GIN (to_tsvector('english', content));
```

**创建 JSONB 索引**
`CREATE INDEX <索引名> ON <表名> USING GIN (<JSONB 列>)`
```sql
-- JSONB 索引
CREATE INDEX idx_data_jsonb ON api_logs USING GIN (payload);

-- JSONB 路径索引
CREATE INDEX idx_data_jsonb_path ON api_logs USING GIN (payload jsonb_path_ops);
```

**创建数组索引**
`CREATE INDEX <索引名> ON <表名> USING GIN (<数组列>)`
```sql
-- 数组索引
CREATE INDEX idx_tags ON posts USING GIN (tags);
```

**启用 fastupdate**
`CREATE INDEX <索引名> ON <表名> USING GIN (<列名>) WITH (fastupdate = on)`
```sql
-- 延迟更新提升写入性能
CREATE INDEX idx_tags_fast ON posts USING GIN (tags) WITH (fastupdate = on);
```

---

## SP-GiST 索引

**创建 SP-GiST 索引**
`CREATE INDEX <索引名> ON <表名> USING SPGST (<列名> <操作符类>)`
```sql
-- 电话号码前缀索引
CREATE INDEX idx_phones ON contacts USING SPGST (phone prefix_range_ops);

-- 路由表索引
CREATE INDEX idx_routes ON routing USING SPGST (prefix);
```

---

## BRIN 索引

**创建 BRIN 索引**
`CREATE INDEX <索引名> ON <表名> USING BRIN (<列名>) [WITH (pages_per_range = <数量>)]`
```sql
-- 块范围索引（适合大表有序数据）
CREATE INDEX idx_logs_created ON logs USING BRIN (created_at)
    WITH (pages_per_range = 32);
```
