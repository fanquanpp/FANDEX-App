# PostgreSQL 序列与自增列

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 序列（SEQUENCE）

**创建序列**
`CREATE SEQUENCE <序列名> [START <起始值>] [INCREMENT <步长>]`
```sql
-- 创建序列
CREATE SEQUENCE order_seq START 1000 INCREMENT 1;
```

**获取下一个值**
`nextval('<序列名>')`
```sql
-- 获取下一个值
SELECT nextval('order_seq');
```

**获取当前值**
`currval('<序列名>')`
```sql
-- 获取当前会话的当前值
SELECT currval('order_seq');
```

**设置序列值**
`setval('<序列名>', <值>)`
```sql
-- 设置序列值
SELECT setval('order_seq', 5000);
```

**查看序列信息**
`SELECT * FROM information_schema.sequences WHERE sequence_name = '<序列名>'`
```sql
-- 查看序列信息
SELECT * FROM information_schema.sequences WHERE sequence_name = 'order_seq';
```

## SERIAL 类型

**使用 SERIAL**
`<列名> SERIAL [PRIMARY KEY]`
```sql
-- SERIAL 是序列+默认值的语法糖
CREATE TABLE orders (
    id SERIAL PRIMARY KEY,
    amount DECIMAL(10,2)
);
```

**SERVAL 等价写法**
`CREATE SEQUENCE <序列名>; CREATE TABLE ... DEFAULT nextval('<序列名>') ...`
```sql
-- 等价于 SERIAL
CREATE SEQUENCE orders_id_seq;
CREATE TABLE orders (
    id INTEGER NOT NULL DEFAULT nextval('orders_id_seq') PRIMARY KEY,
    amount DECIMAL(10,2)
);
ALTER SEQUENCE orders_id_seq OWNED BY orders.id;
```

## IDENTITY 列

**GENERATED ALWAYS**
`<列名> <类型> GENERATED ALWAYS AS IDENTITY [PRIMARY KEY]`
```sql
-- 不允许手动插入 ID（PostgreSQL 10+ 推荐）
CREATE TABLE users (
    id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
    name VARCHAR(100)
);
```

**GENERATED BY DEFAULT**
`<列名> <类型> GENERATED BY DEFAULT AS IDENTITY [PRIMARY KEY]`
```sql
-- 允许手动插入 ID
CREATE TABLE products (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name VARCHAR(100)
);
```

**重置序列**
`ALTER TABLE <表名> ALTER COLUMN <列名> RESTART WITH <值>`
```sql
-- 手动插入后重置序列
ALTER TABLE users ALTER COLUMN id RESTART WITH 1000;
```

## ID 生成策略

**自增 ID**
`<列名> BIGINT GENERATED ALWAYS AS IDENTITY`
```sql
-- 自增 ID（简单、有序）
CREATE TABLE users (
    id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
    name VARCHAR(100)
);
```

**UUID**
`<列名> UUID DEFAULT gen_random_uuid() PRIMARY KEY`
```sql
-- UUID（全局唯一、无序）
CREATE TABLE users (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name VARCHAR(100)
);
```
