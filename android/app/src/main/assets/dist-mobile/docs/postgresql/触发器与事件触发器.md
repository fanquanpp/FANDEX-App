# PostgreSQL 触发器与事件触发器

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 行级触发器

**创建触发器函数**
`CREATE OR REPLACE FUNCTION <函数名>() RETURNS TRIGGER AS $$ BEGIN <触发体> RETURN <NEW|OLD|NULL> END $$ LANGUAGE plpgsql`
```sql
-- 自动更新修改时间
CREATE OR REPLACE FUNCTION update_modified_at()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = NOW();
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;
```

**创建触发器**
`CREATE TRIGGER <触发器名> {BEFORE|AFTER} {INSERT|UPDATE|DELETE} ON <表名> FOR EACH ROW EXECUTE FUNCTION <函数名>()`
```sql
-- 更新前触发
CREATE TRIGGER trg_employees_modified
BEFORE UPDATE ON employees
FOR EACH ROW
EXECUTE FUNCTION update_modified_at();
```

## 审计触发器

**审计触发器函数**
`CREATE OR REPLACE FUNCTION <函数名>() RETURNS TRIGGER AS $$ BEGIN IF TG_OP = '<操作>' THEN <审计插入> END IF; RETURN NULL; END $$ LANGUAGE plpgsql`
```sql
-- 多操作审计
CREATE OR REPLACE FUNCTION audit_log()
RETURNS TRIGGER AS $$
BEGIN
    IF TG_OP = 'INSERT' THEN
        INSERT INTO audit_table (table_name, operation, new_data, changed_at)
        VALUES (TG_TABLE_NAME, 'INSERT', to_jsonb(NEW), NOW());
    ELSIF TG_OP = 'UPDATE' THEN
        INSERT INTO audit_table (table_name, operation, old_data, new_data, changed_at)
        VALUES (TG_TABLE_NAME, 'UPDATE', to_jsonb(OLD), to_jsonb(NEW), NOW());
    ELSIF TG_OP = 'DELETE' THEN
        INSERT INTO audit_table (table_name, operation, old_data, changed_at)
        VALUES (TG_TABLE_NAME, 'DELETE', to_jsonb(OLD), NOW());
    END IF;
    RETURN NULL;
END;
$$ LANGUAGE plpgsql;
```

**创建审计触发器**
`CREATE TRIGGER <触发器名> AFTER INSERT OR UPDATE OR DELETE ON <表名> FOR EACH ROW EXECUTE FUNCTION <函数名>()`
```sql
-- 创建审计触发器
CREATE TRIGGER trg_employees_audit
AFTER INSERT OR UPDATE OR DELETE ON employees
FOR EACH ROW EXECUTE FUNCTION audit_log();
```

## 事件触发器

**DDL 事件触发器函数**
`CREATE OR REPLACE FUNCTION <函数名>() RETURNS EVENT_TRIGGER AS $$ BEGIN <触发体> END $$ LANGUAGE plpgsql`
```sql
-- 阻止删除表
CREATE OR REPLACE FUNCTION prevent_drop_table()
RETURNS EVENT_TRIGGER AS $$
BEGIN
    RAISE EXCEPTION '不允许删除表！';
END;
$$ LANGUAGE plpgsql;
```

**创建 DDL 事件触发器**
`CREATE EVENT TRIGGER <触发器名> ON <事件> EXECUTE FUNCTION <函数名>()`
```sql
-- 在 sql_drop 事件上触发
CREATE EVENT TRIGGER trg_prevent_drop
ON sql_drop
EXECUTE FUNCTION prevent_drop_table();
```

**DDL 完成后触发**
`CREATE EVENT TRIGGER <触发器名> ON ddl_command_end EXECUTE FUNCTION <函数名>()`
```sql
-- DDL 完成后记录日志
CREATE OR REPLACE FUNCTION log_ddl()
RETURNS EVENT_TRIGGER AS $$
BEGIN
    INSERT INTO ddl_log (event, object_type, object_name, timestamp)
    VALUES (tg_tag, tg_event, tg_objectid, NOW());
END;
$$ LANGUAGE plpgsql;

CREATE EVENT TRIGGER trg_log_ddl
ON ddl_command_end
EXECUTE FUNCTION log_ddl();
```
