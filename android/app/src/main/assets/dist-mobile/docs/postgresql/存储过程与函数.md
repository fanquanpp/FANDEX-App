# PostgreSQL 存储过程与函数

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

## PL/pgSQL 函数

**创建函数**
`CREATE OR REPLACE FUNCTION <函数名>([<参数>]) RETURNS <返回类型> AS $$ DECLARE <变量> BEGIN <函数体> RETURN <值> END $$ LANGUAGE plpgsql`
```sql
-- 计算奖金
CREATE OR REPLACE FUNCTION calculate_bonus(
    p_salary NUMERIC,
    p_performance INTEGER
) RETURNS NUMERIC AS $$
DECLARE
    v_bonus NUMERIC;
BEGIN
    v_bonus := p_salary * p_performance / 100.0;
    RETURN v_bonus;
END;
$$ LANGUAGE plpgsql;
```

**调用函数**
`SELECT <函数名>([<参数>...])`
```sql
-- 调用函数
SELECT calculate_bonus(50000, 15);
```

## 存储过程

**创建存储过程**
`CREATE OR REPLACE PROCEDURE <过程名>([<参数>]) AS $$ BEGIN <过程体> END $$ LANGUAGE plpgsql`
```sql
-- 转账存储过程
CREATE OR REPLACE PROCEDURE transfer_funds(
    p_from INTEGER,
    p_to INTEGER,
    p_amount NUMERIC
) AS $$
BEGIN
    UPDATE accounts SET balance = balance - p_amount WHERE id = p_from;
    UPDATE accounts SET balance = balance + p_amount WHERE id = p_to;
    COMMIT;
END;
$$ LANGUAGE plpgsql;
```

**调用存储过程**
`CALL <过程名>([<参数>...])`
```sql
-- 调用存储过程
CALL transfer_funds(1, 2, 1000);
```

## 控制流

**IF 语句**
`IF <条件> THEN <语句> [ELSIF <条件> THEN <语句>] [ELSE <语句>] END IF`
```sql
-- 等级判断
CREATE OR REPLACE FUNCTION get_grade(p_score INTEGER)
RETURNS VARCHAR(10) AS $$
BEGIN
    IF p_score >= 90 THEN
        RETURN 'A';
    ELSIF p_score >= 80 THEN
        RETURN 'B';
    ELSIF p_score >= 70 THEN
        RETURN 'C';
    ELSE
        RETURN 'D';
    END IF;
END;
$$ LANGUAGE plpgsql;
```

## 游标与循环

**FOR 循环遍历查询**
`FOR <记录变量> IN <SELECT 语句> LOOP <循环体> END LOOP`
```sql
-- 遍历订单处理
CREATE OR REPLACE PROCEDURE process_orders() AS $$
DECLARE
    order_record RECORD;
BEGIN
    FOR order_record IN
        SELECT id, amount FROM orders WHERE status = 'pending'
    LOOP
        UPDATE orders SET status = 'processing' WHERE id = order_record.id;
    END LOOP;
END;
$$ LANGUAGE plpgsql;
```

## PL/Python

**创建 PL/Python 扩展**
`CREATE EXTENSION plpython3u`
```sql
-- 安装 PL/Python 扩展
CREATE EXTENSION plpython3u;
```

**PL/Python 函数**
`CREATE OR REPLACE FUNCTION <函数名>(<参数>) RETURNS <类型> AS $$ <Python 代码> $$ LANGUAGE plpython3u`
```sql
-- Python 哈希函数
CREATE OR REPLACE FUNCTION python_hash(p_text TEXT)
RETURNS TEXT AS $$
import hashlib
return hashlib.sha256(p_text.encode()).hexdigest()
$$ LANGUAGE plpython3u;
```
