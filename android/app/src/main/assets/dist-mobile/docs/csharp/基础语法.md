# C# 基础语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 变量声明

**显式类型声明**：声明指定类型的变量
`<类型> <变量名> = <初始值>;`
```csharp
// 声明整数、字符串、浮点、布尔变量
int age = 25;
string name = "张三";
double price = 99.99;
bool isActive = true;
```

---

**var 类型推断**：编译期自动推断变量类型
`var <变量名> = <初始值>;`
```csharp
// 编译期确定类型
var count = 42;                          // int
var message = "Hello";                   // string
var numbers = new[] { 1, 2, 3 };        // int[]
var dict = new Dictionary<string, int>(); // Dictionary<string, int>
```

---

**常量声明**：编译期确定的不可变值
`const <类型> <常量名> = <值>;`
```csharp
// 声明全局常量
const double Pi = 3.14159265358979;
const string AppName = "FANDEX";
```

---

**required 属性**：强制对象初始化时必须赋值
`public required <类型> <属性名> { get; init; }`
```csharp
// C# 11 required 修饰符
public class Person
{
    public required string Name { get; init; }
    public required int Age { get; init; }
}

// 必须初始化 required 属性
var person = new Person { Name = "李四", Age = 30 };
```

---

## 类型转换

**隐式转换**：安全无损的类型转换
`<目标类型> <变量> = <源变量>;`
```csharp
// int → long, int → double 自动转换
int num = 100;
long bigNum = num;        // int → long
double d = num;           // int → double
```

---

**显式转换**：可能丢失数据的强制转换
`(<目标类型>)<变量>`
```csharp
// 截断小数部分
double pi = 3.14159;
int intPi = (int)pi;      // 3
```

---

**Convert 类转换**：基础类型之间的转换
`Convert.To<类型>(<值>)`
```csharp
// 字符串转数值
string str = "123";
int parsed = Convert.ToInt32(str);
double dbl = Convert.ToDouble("3.14");
```

---

**Parse / TryParse**：字符串解析为数值
`<类型>.Parse(<字符串>) | <类型>.TryParse(<字符串>, out <结果>)`
```csharp
// 解析失败抛异常
int number = int.Parse("456");

// 安全解析，返回是否成功
if (int.TryParse("789", out int result))
{
    Console.WriteLine($"解析成功: {result}");
}
```

---

**is / as 类型检查**：运行时类型判断与转换
`<变量> is <类型> <变量名> | <变量> as <类型>`
```csharp
// is 模式匹配转换
object obj = "Hello";
if (obj is string s)
{
    Console.WriteLine(s.Length); // 5
}

// as 转换，失败返回 null
IAnimal? animal = dog as IAnimal;
```

## 字符串操作

**字符串插值**：内嵌表达式到字符串
`$"文本 {<表达式>}"`
```csharp
// 基本插值与表达式插值
var name = "世界";
var year = 2026;
Console.WriteLine($"你好, {name}! 当前年份: {year}");
Console.WriteLine($"2 + 3 = {2 + 3}");
Console.WriteLine($"大写: {name.ToUpper()}");
Console.WriteLine($"时间: {DateTime.Now:yyyy-MM-dd HH:mm:ss}");
```

---

**原始字符串字面量**：保留原始格式的多行字符串
`"""<内容>"""`
```csharp
// C# 11 三引号字符串
var json = """
    {
        "name": "张三",
        "age": 25,
        "email": "zhangsan@example.com"
    }
    """;
```

---

**插值原始字符串**：原始字符串中嵌入表达式
`$"""<内容 {<表达式>}>"""`
```csharp
// 在原始字符串中插值
var id = 1001;
var query = $"""
    SELECT * FROM Users
    WHERE Id = {id}
    AND IsActive = 1
    """;
```

---

**字符串查找**：检索子串位置或存在性
`<字符串>.Contains(<子串>) | <字符串>.IndexOf(<子串>) | <字符串>.StartsWith(<前缀>)`
```csharp
// 查找子串
string str = "Hello, C# World!";
bool contains = str.Contains("C#");         // true
int index = str.IndexOf("World");           // 9
bool startsWith = str.StartsWith("Hello");  // true
```

---

**字符串截取与分割**：提取子串或拆分为数组
`<字符串>.Substring(<起始>, <长度>) | <字符串>.Split(<分隔符>)`
```csharp
// 截取与分割
string str = "Hello, C# World!";
string sub = str.Substring(7, 2);           // "C#"
string[] parts = str.Split(' ');            // ["Hello,", "C#", "World!"]
```

---

**字符串修改**：转换大小写、去空白、替换
`<字符串>.ToUpper() | <字符串>.Trim() | <字符串>.Replace(<旧值>, <新值>)`
```csharp
// 修改字符串内容
string str = "Hello, C# World!";
string upper = str.ToUpper();               // "HELLO, C# WORLD!"
string trimmed = "  hello  ".Trim();        // "hello"
string replaced = str.Replace("C#", "F#");   // "Hello, F# World!"
```

---

**StringBuilder**：高性能字符串拼接
`new StringBuilder().AppendLine(<内容>).AppendFormat(<格式>, <参数>)`
```csharp
// 链式构建字符串
var sb = new StringBuilder();
sb.AppendLine("第一行");
sb.AppendLine("第二行");
sb.AppendFormat("数字: {0:N2}", 1234.5678);
string result = sb.ToString();
```

## Nullable 引用类型

**可空引用类型**：标记引用类型允许为 null
`<类型>? <变量名>`
```csharp
#nullable enable

// 引用类型默认不可为 null
string name = "张三";   // OK
// string name2 = null; // 警告

// 使用 ? 标记可为 null
string? nickname = null; // OK
```

---

**空条件运算符**：安全访问可能为 null 的成员
`<变量>?.<成员>`
```csharp
// null 时返回 null，不抛异常
string? nickname = null;
int? length = nickname?.Length;     // null
string upper = nickname?.ToUpper(); // null
```

---

**空合并运算符**：null 时提供默认值
`<变量> ?? <默认值> | <变量> ??= <默认值>`
```csharp
// 提供默认值或赋值
string? nickname = null;
string display = nickname ?? "匿名";       // "匿名"
string display2 = nickname ??= "匿名";     // 赋值并返回
```

---

**强制非空运算符**：告知编译器不为 null
`<变量>!`
```csharp
// 慎用，抑制 null 警告
string? nickname = null;
string forced = nickname!;
```

---

**值类型可空**：使值类型可以接受 null
`<值类型>? <变量名>`
```csharp
// Nullable<int> 包装
int? age = null;
bool hasValue = age.HasValue;
int value = age ?? 0;
int value2 = age.GetValueOrDefault(18);
```

## 控制流

**if-else 语句**：条件分支
`if (<条件>) <语句> else if (<条件>) <语句> else <语句>`
```csharp
// 多分支条件判断
int score = 85;
if (score >= 90)
    Console.WriteLine("优秀");
else if (score >= 80)
    Console.WriteLine("良好");
else if (score >= 60)
    Console.WriteLine("及格");
else
    Console.WriteLine("不及格");
```

---

**switch 语句**：多分支选择
`switch (<变量>) { case <值>: <语句>; break; default: <语句>; break; }`
```csharp
// 枚举分支
var day = DayOfWeek.Monday;
switch (day)
{
    case DayOfWeek.Saturday:
    case DayOfWeek.Sunday:
        Console.WriteLine("周末");
        break;
    default:
        Console.WriteLine("工作日");
        break;
}
```

---

**switch 表达式**：基于模式的表达式分支
`<变量> switch { <模式> => <结果>, _ => <默认> }`
```csharp
// switch 表达式与模式匹配
string label = day switch
{
    DayOfWeek.Saturday or DayOfWeek.Sunday => "周末",
    _ => "工作日"
};

// 类型模式
object obj = 42;
string typeName = obj switch
{
    int => "整数",
    string => "字符串",
    double => "浮点数",
    null   => "null",
    _      => "其他"
};
```

---

**for 循环**：计数循环
`for (<初始化>; <条件>; <更新>) <循环体>`
```csharp
// 计数迭代
for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}
```

---

**foreach 循环**：遍历可枚举集合
`foreach (<类型> <变量> in <集合>) <循环体>`
```csharp
// 遍历数组
var fruits = new[] { "苹果", "香蕉", "橙子" };
foreach (var fruit in fruits)
{
    Console.WriteLine(fruit);
}
```

---

**while 循环**：条件为真时循环
`while (<条件>) <循环体>`
```csharp
// 前置条件循环
int n = 10;
while (n > 0)
{
    Console.WriteLine(n--);
}
```

---

**do-while 循环**：至少执行一次的后置条件循环
`do <循环体> while (<条件>);`
```csharp
// 至少执行一次
string? input;
do
{
    Console.Write("请输入 (q 退出): ");
    input = Console.ReadLine();
} while (input != "q");
```

---

**范围与索引**：访问数组切片与末尾元素
`<数组>[^<索引>] | <数组>[<开始>..<结束>]`
```csharp
// 索引与切片
var numbers = new[] { 10, 20, 30, 40, 50 };
int last = numbers[^1];          // 50
int secondLast = numbers[^2];    // 40
var slice = numbers[1..4];       // [20, 30, 40]
var firstThree = numbers[..3];   // [10, 20, 30]
```

## 模式匹配

**is 模式**：类型检查并提取值
`<变量> is <类型> <变量名> and <条件>`
```csharp
// 组合条件匹配
object value = 42;
if (value is int num and > 0 and < 100)
{
    Console.WriteLine($"0-100 之间的整数: {num}");
}
```

---

**属性模式**：匹配对象属性值
`<变量> switch { { <属性>: <值> } => <结果> }`
```csharp
// 基于属性分支
public record Order(decimal Amount, string Status, Customer Customer);

string GetDiscount(Order order) => order switch
{
    { Status: "VIP", Amount: > 1000m } => "8折",
    { Status: "VIP" } => "9折",
    { Amount: > 500m } => "95折",
    _ => "无折扣"
};
```

---

**列表模式**：匹配数组或列表元素
`<数组> switch { [<元素模式>] => <结果> }`
```csharp
// C# 11 列表模式
int[] numbers = [1, 2, 3];
string label = numbers switch
{
    [] => "空列表",
    [single] => $"单个元素: {single}",
    [first, .., last] => $"首: {first}, 尾: {last}",
    [var a, var b, ..] => $"前两个: {a}, {b}"
};
```

---

**when 守卫**：为模式添加额外条件
`<模式> when <条件>`
```csharp
// 守卫子句
string Classify(int[] arr) => arr switch
{
    [var a, .., var b] when a == b => "首尾相同",
    [var a, .., var b] when a < b => "递增趋势",
    _ => "其他"
};
```

## 顶级语句与全局 Using

**全局 using**：全项目共享的命名空间引用
`global using <命名空间>;`
```csharp
// GlobalUsings.cs
global using System;
global using System.Collections.Generic;
global using System.Linq;
global using System.Threading.Tasks;
global using System.IO;
```

---

**文件范围命名空间**：单文件命名空间声明
`namespace <命名空间>;`
```csharp
// C# 10+ 文件范围命名空间
namespace MyApp.Services;

public class UserService
{
    // 整个文件都在该命名空间下
}
```

---

**顶级语句**：无需 Main 方法的入口点
`<语句>;`
```csharp
// 顶级语句支持异步
var data = await FetchDataAsync();
Console.WriteLine($"获取到 {data.Length} 条记录");

async Task<string[]> FetchDataAsync()
{
    await Task.Delay(100);
    return ["item1", "item2", "item3"];
}
```

## 运算符速查

**空合并赋值**：null 时赋值并返回
`<变量> ??= <值>`
```csharp
// null 时赋值
string? name = null;
name ??= "赋值";
```

---

**with 表达式**：非破坏性修改记录
`<记录> with { <属性> = <值> }`
```csharp
// 修改 record 创建副本
var original = new Point(1, 2);
var modified = original with { X = 10 };
```

---

**集合表达式**：统一的集合初始化语法
`[<元素>, ...] | [..<集合>, <元素>]`
```csharp
// C# 12 集合表达式
List<int> list = [1, 2, 3];
int[] arr = [..list, 4, 5]; // 展开 + 合并
```
