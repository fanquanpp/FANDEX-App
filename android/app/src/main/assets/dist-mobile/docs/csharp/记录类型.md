# C# 记录类型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 位置记录

**位置记录定义**：简洁的数据类型定义
`public record <名称>(<参数列表>);`
```csharp
// 位置记录：一行代码定义一个完整的数据类型
public record Person(string Name, int Age);

// 创建实例
var p1 = new Person("Alice", 25);
var p2 = new Person("Alice", 25);

// 值相等性：内容相同即相等
Console.WriteLine(p1 == p2);        // True（class 的话会是 False）
Console.WriteLine(p1.Equals(p2));   // True

// with 表达式：创建修改了部分属性的新实例
var p3 = p1 with { Age = 26 };
Console.WriteLine(p3);              // Person { Name = Alice, Age = 26 }

// 原实例不受影响
Console.WriteLine(p1.Age);          // 25（不可变）
```

---

**自动生成功能**：位置记录自动生成的成员
`public record <名称>(<参数列表>);`
```csharp
// 定义位置记录
public record Point(double X, double Y);

// 自动生成的功能：
// 1. 构造函数：new Point(1.0, 2.0)
// 2. 只读属性：X, Y
// 3. Deconstruct 方法：var (x, y) = point;
// 4. 值相等性：point1 == point2
// 5. ToString：Point { X = 1, Y = 2 }

var p = new Point(3.0, 4.0);

// 使用 Deconstruct 解构
var (x, y) = p;
Console.WriteLine($"X={x}, Y={y}");

// with 表达式
var p2 = p with { X = 5.0 };
Console.WriteLine(p2); // Point { X = 5, Y = 4 }
```

## 添加额外成员

**记录额外成员**：在记录中添加属性、方法
`public record <名称>(<参数>) { public <成员> }`
```csharp
// 添加额外属性和方法
public record Student(string Name, int Age, double Score)
{
    // 添加额外属性
    public string Grade { get; init; } = CalculateGrade(Score);

    // 添加方法
    public string GetInfo() => $"{Name}，{Age}岁，成绩{Score}，等级{Grade}";

    // 添加静态方法
    private static string CalculateGrade(double score) => score switch
    {
        >= 90 => "A",
        >= 80 => "B",
        >= 70 => "C",
        >= 60 => "D",
        _ => "F"
    };
}

var student = new Student("张三", 20, 85.5);
Console.WriteLine(student.GetInfo()); // 张三，20岁，成绩85.5，等级B
```

## record struct

**record struct**：值类型记录
`public record struct <名称>(<参数列表>);`
```csharp
// 值类型记录
public record struct Point(double X, double Y);

// 引用类型记录
public record PointRef(double X, double Y);

var s1 = new Point(1, 2);
var s2 = s1;         // 值复制
s2.X = 10;           // 修改 s2 不影响 s1
Console.WriteLine(s1.X); // 1

var r1 = new PointRef(1, 2);
var r2 = r1;         // 引用复制
```

---

**可变 record struct**：record struct 默认属性可变
`public record struct <名称>(<参数>);`
```csharp
// record struct 默认属性可变
public record struct MutablePoint(double X, double Y);

var mp = new MutablePoint(1, 2);
mp.X = 10;  // 允许修改

// 如果需要不可变，使用 init
public record struct ImmutablePoint(double X, double Y)
{
    public double X { get; init; } = X;
    public double Y { get; init; } = Y;
}
```

## 自定义构造函数和验证

**参数验证**：在记录中验证构造参数
`public record <名称>(<参数>) { public <属性> { get; init; } = <验证表达式>; }`
```csharp
// 验证构造函数参数
public record EmailAddress(string Address)
{
    // 验证构造函数参数
    public string Address { get; init; } =
        string.IsNullOrWhiteSpace(Address)
            ? throw new ArgumentException("邮箱地址不能为空")
            : Address;

    // 添加计算属性
    public string Domain => Address.Split('@').LastOrDefault() ?? "";

    // 自定义验证逻辑
    public bool IsValid => Address.Contains('@') && Address.Contains('.');
}

var email = new EmailAddress("user@example.com");
Console.WriteLine(email.Domain);    // example.com
Console.WriteLine(email.IsValid);   // True
```

## 继承

**记录继承**：记录类型支持继承
`public record <派生>(<参数>) : <基类>(<参数>);`
```csharp
// 基类记录
public record Animal(string Name, string Type);

// 派生记录
public record Dog(string Name, string Breed) : Animal(Name, "Dog");

public record Cat(string Name, bool IsIndoor) : Animal(Name, "Cat");

var dog = new Dog("旺财", "柴犬");
var cat = new Cat("小花", true);

Console.WriteLine(dog); // Dog { Name = 旺财, Type = Dog, Breed = 柴犬 }
Console.WriteLine(cat); // Cat { Name = 小花, Type = Cat, IsIndoor = True }

// 值相等性考虑继承层次
Animal a1 = new Dog("旺财", "柴犬");
Animal a2 = new Dog("旺财", "柴犬");
Console.WriteLine(a1 == a2); // True

Animal a3 = new Cat("旺财", true);
Console.WriteLine(a1 == a3); // False（类型不同）
```

## 模式匹配与记录

**属性模式匹配**：基于记录属性的分支
`<记录> switch { <类型> { <属性>: <值> } => <结果> }`
```csharp
public record Order(int Id, decimal Amount, string Status);

var order = new Order(1, 99.9m, "Shipped");

// 属性模式匹配
var message = order switch
{
    Order { Status: "Pending", Amount: < 100 } => "小额待处理订单",
    Order { Status: "Pending" } => "待处理订单",
    Order { Status: "Shipped" } => "已发货订单",
    Order { Status: "Delivered", Amount: > 1000 } => "大额已送达订单",
    _ => "其他订单"
};

// 解构模式
if (order is Order(_, var amount, "Shipped"))
{
    Console.WriteLine($"已发货，金额: {amount}");
}
```

## 集合中的记录

**记录集合操作**：值相等性在集合操作中的应用
`<集合>.Contains(<记录>) | <集合>.Distinct()`
```csharp
public record Product(string Name, decimal Price);

var products = new List<Product>
{
    new("笔记本电脑", 5999),
    new("手机", 3999),
    new("耳机", 299)
};

// 值相等性在集合操作中很有用
var target = new Product("手机", 3999);
var found = products.Contains(target); // True

// 使用 Distinct 去重
var duplicates = new List<Product>
{
    new("手机", 3999),
    new("手机", 3999),  // 与上一条值相等
    new("手机", 4999)   // 价格不同
};
var unique = duplicates.Distinct().ToList(); // 只剩2个

// 使用 with 批量修改
var discounted = products.Select(p => p with { Price = p.Price * 0.9m }).ToList();
```

## 领域事件

**领域事件记录**：用记录类型定义领域事件
`public record <事件>(<参数>) : <基类>(<参数>);`
```csharp
// 用记录类型定义领域事件
public record DomainEvent(DateTime OccurredAt)
{
    public Guid EventId { get; init; } = Guid.NewGuid();
}

public record UserCreated(Guid UserId, string Name) : DomainEvent(DateTime.UtcNow);
public record UserDeleted(Guid UserId, string Reason) : DomainEvent(DateTime.UtcNow);
public record OrderPlaced(Guid OrderId, decimal Amount) : DomainEvent(DateTime.UtcNow);

// 事件处理
void HandleEvent(DomainEvent e)
{
    switch (e)
    {
        case UserCreated created:
            Console.WriteLine($"用户创建: {created.Name}");
            break;
        case UserDeleted deleted:
            Console.WriteLine($"用户删除: {deleted.Reason}");
            break;
        case OrderPlaced placed:
            Console.WriteLine($"下单: {placed.Amount:C}");
            break;
    }
}
```

## 自定义相等性

**重写相等性**：自定义记录的相等比较逻辑
`public virtual bool Equals(<记录>? other) | public override int GetHashCode()`
```csharp
// 重写相等性逻辑
public record Money(decimal Amount, string Currency)
{
    // 自定义相等性比较：金额和货币都相同才相等
    public virtual bool Equals(Money? other)
    {
        if (other is null) return false;
        // 允许微小误差
        return Math.Abs(Amount - other.Amount) < 0.01m
            && Currency == other.Currency;
    }

    // 必须同时重写 GetHashCode
    public override int GetHashCode() => HashCode.Combine(
        Math.Round(Amount, 2), Currency);
}
```

## 建造者模式

**链式构建**：使用 record 实现建造者模式
`public record <名称>(<参数> = <默认值>) { public <名称> With<属性>(<值>) => this with { ... }; }`
```csharp
// 使用 record 实现建造者模式
public record QueryOptions(
    int Page = 1,
    int PageSize = 20,
    string SortBy = "Id",
    bool SortDescending = false,
    string? SearchTerm = null)
{
    // 链式构建
    public QueryOptions WithPage(int page) => this with { Page = page };
    public QueryOptions WithSearch(string term) => this with { SearchTerm = term };
    public QueryOptions WithSort(string field, bool desc = false) =>
        this with { SortBy = field, SortDescending = desc };
}

// 使用
var options = new QueryOptions()
    .WithPage(2)
    .WithSearch("关键词")
    .WithSort("Name", desc: true);
```
