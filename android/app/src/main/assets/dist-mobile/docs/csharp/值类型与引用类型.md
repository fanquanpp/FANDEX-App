# C# 值类型与引用类型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 赋值行为差异

**值类型赋值**：复制整个值，互不影响
`<值类型> <变量B> = <变量A>;`
```csharp
// struct 赋值复制值
var p1 = new Point(1, 2);
var p2 = p1;    // 复制值，p2 是独立的副本
p2.X = 10;      // 只修改 p2，p1 不受影响
Console.WriteLine(p1.X); // 1

struct Point { public int X; public int Y; public Point(int x, int y) => (X, Y) = (x, y); }
```

---

**引用类型赋值**：复制引用，指向同一对象
`<引用类型> <变量B> = <变量A>;`
```csharp
// class 赋值复制引用
var u1 = new User("张三");
var u2 = u1;    // 复制引用，指向同一个对象
u2.Name = "李四"; // 修改了同一个对象
Console.WriteLine(u1.Name); // "李四"

class User { public string Name; public User(string name) => Name = name; }
```

## 装箱与拆箱

**装箱**：值类型转换为引用类型
`object <变量> = <值类型变量>;`
```csharp
// 值类型 -> 堆上的引用类型
int x = 42;
object obj = x;        // 装箱
```

---

**拆箱**：引用类型转回值类型
`<值类型> <变量> = (<值类型>)<object变量>;`
```csharp
// 引用类型 -> 值类型
object obj = 42;
int y = (int)obj;      // 拆箱
```

---

**避免装箱**：使用泛型集合替代非泛型集合
`List<<值类型>> <变量> = new();`
```csharp
// 错误：ArrayList 每次添加 int 都装箱
var list = new ArrayList();
for (int i = 0; i < 1000000; i++) {
    list.Add(i); // 每次都装箱
}

// 正确：泛型列表无装箱
var list2 = new List<int>();
for (int i = 0; i < 1000000; i++) {
    list2.Add(i); // 无装箱，值类型直接存储
}
```

## struct 定义

**readonly struct**：不可变值类型
`public readonly struct <名称> { public <类型> <属性> { get; } }`
```csharp
// 适合小型不可变数据
public readonly struct Point {
    public double X { get; }
    public double Y { get; }
    public Point(double x, double y) => (X, Y) = (x, y);

    public double DistanceTo(Point other) =>
        Math.Sqrt(Math.Pow(X - other.X, 2) + Math.Pow(Y - other.Y, 2));
}
```

---

**ref struct**：只能存在于栈上的结构体
`public ref struct <名称> { ... }`
```csharp
// ref struct 不能逃逸到堆
public ref struct StackBuffer {
    private readonly Span<byte> _buffer;

    public StackBuffer(int size) {
        _buffer = stackalloc byte[size];
    }

    public void Write(int offset, byte value) => _buffer[offset] = value;
    public byte Read(int offset) => _buffer[offset];
}
```

## record struct

**record struct**：值类型记录，自动生成相等性比较
`public record struct <名称>(<参数列表>);`
```csharp
// 值类型记录
public record struct Point(double X, double Y);

var p1 = new Point(1, 2);
var p2 = new Point(1, 2);
Console.WriteLine(p1 == p2);      // True（值相等）
Console.WriteLine(p1.Equals(p2)); // True
```

---

**record class**：引用类型记录
`public record <名称>(<参数列表>);`
```csharp
// 引用类型记录
public record User(string Name, int Age);
var u1 = new User("张三", 25);
var u2 = u1 with { Age = 26 }; // 使用 with 创建副本并修改
```

## 接口与装箱

**接口导致装箱**：值类型通过接口调用会装箱
`<接口> <变量> = <值类型实例>;`
```csharp
// 值类型实现接口
interface IProcessor {
    void Process();
}

struct MyProcessor : IProcessor {
    public void Process() { /* ... */ }
}

MyProcessor processor = new();
processor.Process(); // 无装箱（直接调用）

IProcessor boxed = processor; // 装箱！
boxed.Process(); // 通过虚方法表调用，有额外开销
```

---

**泛型约束避免装箱**：使用泛型方法约束接口
`void Process<T>(T processor) where T : <接口>`
```csharp
// 使用泛型约束避免装箱
void Process<T>(T processor) where T : IProcessor {
    processor.Process(); // 无装箱
}
```

## 参数传递优化

**ref 传递**：通过引用传递，避免复制
`void <方法>(ref <类型> <参数>)`
```csharp
// 大 struct 使用 ref 传递
void ProcessLargeStruct(ref LargeData data) {
    data.Value = 42; // 通过引用传递，不复制
}
```

---

**in 只读引用传递**：通过引用传递且不允许修改
`void <方法>(in <类型> <参数>)`
```csharp
// 只读引用传递
void ReadLargeStruct(in LargeData data) {
    Console.WriteLine(data.Value); // 通过引用传递，且不允许修改
}
```

---

**ref 返回**：返回引用，调用者可直接修改
`ref <类型> <方法>(<参数>)`
```csharp
// 返回引用
ref int FindMax(int[] array) {
    int maxIndex = 0;
    for (int i = 1; i < array.Length; i++) {
        if (array[i] > array[maxIndex]) maxIndex = i;
    }
    return ref array[maxIndex]; // 返回引用，调用者可直接修改
}
```

## 结构体内存布局

**StructLayout 顺序布局**：控制内存布局用于互操作
`[StructLayout(LayoutKind.Sequential)] public struct <名称> { ... }`
```csharp
// 与 C/C++ 互操作的结构体布局
[StructLayout(LayoutKind.Sequential)]
public struct NativeHeader {
    public int Magic;
    public short Version;
    public short Flags;
    public int DataLength;
}
```

---

**StructLayout 显式布局**：精确控制字段偏移
`[StructLayout(LayoutKind.Explicit)] public struct <名称> { [FieldOffset(<偏移>)] public <类型> <字段>; }`
```csharp
// 联合体（union）效果
[StructLayout(LayoutKind.Explicit)]
public struct UnionValue {
    [FieldOffset(0)] public int IntValue;
    [FieldOffset(0)] public float FloatValue;
    [FieldOffset(0)] public double DoubleValue;
}

var v = new UnionValue { IntValue = 42 };
Console.WriteLine(v.FloatValue); // 42 的浮点表示
```

## MemoryMarshal 高级操作

**类型重解释**：零拷贝将字节数组转为其他类型数组
`MemoryMarshal.Cast<<源类型>, <目标类型>>(<Span>)`
```csharp
// 将 byte 数组重新解释为 int 数组（零拷贝）
byte[] bytes = new byte[16];
Span<int> ints = MemoryMarshal.Cast<byte, int>(bytes.AsSpan());
ints[0] = 42; // 直接修改 byte 数组的前4个字节
```

---

**结构体转字节**：将结构体转为只读字节跨度
`MemoryMarshal.AsBytes(MemoryMarshal.CreateReadOnlySpan(ref <结构体>, 1))`
```csharp
// 将结构体转为字节数组
var header = new NativeHeader { Magic = 0x4D42, Version = 1 };
ReadOnlySpan<byte> headerBytes = MemoryMarshal.AsBytes(
    MemoryMarshal.CreateReadOnlySpan(ref header, 1));
```
