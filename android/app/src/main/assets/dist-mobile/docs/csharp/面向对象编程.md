# C# 面向对象编程

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

## 类的定义

**类定义**：声明字段、属性、构造函数、方法
`public class <类名> { public <类型> <属性> { get; set; } public <类名>(<参数>) { ... } }`
```csharp
// 完整的类定义
public class Person
{
    // 字段
    private string _name;
    private int _age;

    // 属性（带验证）
    public string Name
    {
        get => _name;
        set => _name = value ?? throw new ArgumentNullException(nameof(value));
    }

    // 自动属性
    public string Email { get; set; } = string.Empty;

    // init 属性（C# 9+，仅初始化时赋值）
    public string Address { get; init; } = string.Empty;

    // 计算属性
    public bool IsAdult => Age >= 18;

    // 构造函数
    public Person(string name, int age)
    {
        _name = name;
        _age = age;
    }

    // 方法重载
    public string Greet() => $"你好，我是{Name}";
    public string Greet(string greeting) => $"{greeting}，我是{Name}";
}
```

## 构造函数

**构造函数链**：调用其他构造函数
`public <类名>(<参数>) : this(<参数>) { ... }`
```csharp
// 构造函数链调用
public class Product
{
    public string Name { get; set; }
    public decimal Price { get; set; }
    public string Category { get; set; }

    public Product(string name, decimal price)
    {
        Name = name;
        Price = price;
        Category = "未分类";
    }

    // 链式调用
    public Product(string name, decimal price, string category) : this(name, price)
    {
        Category = category;
    }
}
```

## 继承

**基本继承**：派生类继承基类
`public class <派生类> : <基类> { public <派生类>(<参数>) : base(<参数>) { ... } }`
```csharp
// 基类
public class Animal
{
    public string Name { get; set; }
    public int Age { get; set; }

    public Animal(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // virtual 允许子类重写
    public virtual string Speak() => $"{Name}发出了声音";

    // sealed 阻止进一步重写
    public sealed string GetInfo() => $"{Name}, {Age}岁";
}

// 派生类
public class Dog : Animal
{
    public string Breed { get; set; }

    // 调用基类构造函数
    public Dog(string name, int age, string breed) : base(name, age)
    {
        Breed = breed;
    }

    // override 重写虚方法
    public override string Speak() => $"{Name}汪汪叫！";

    // 方法隐藏（不推荐）
    public new string GetInfo() => $"{Name} ({Breed}), {Age}岁";
}
```

---

**多态**：运行时根据实际类型调用方法
`<基类> <变量> = new <派生类>(<参数>);`
```csharp
// 运行时多态
Animal animal = new Dog("旺财", 3, "金毛");
Console.WriteLine(animal.Speak()); // "旺财汪汪叫！"

// 多态集合
List<Animal> animals =
[
    new Dog("旺财", 3, "金毛"),
    new Cat("咪咪", 2, "橘猫"),
    new Bird("小鸟", 1, "鹦鹉")
];

foreach (var a in animals)
{
    Console.WriteLine(a.Speak()); // 每个对象调用各自实现
}
```

## 抽象类与接口

**抽象类**：包含抽象成员的基类
`public abstract class <类名> { public abstract <返回类型> <方法>(); }`
```csharp
// 抽象类定义
public abstract class Shape
{
    public string Color { get; set; }

    protected Shape(string color) => Color = color;

    // 抽象方法：子类必须实现
    public abstract double Area();
    public abstract double Perimeter();

    // 虚方法：子类可选重写
    public virtual string Describe() =>
        $"{GetType().Name} (颜色: {Color}, 面积: {Area():F2})";
}

public class Circle : Shape
{
    public double Radius { get; }

    public Circle(double radius, string color) : base(color) => Radius = radius;

    public override double Area() => Math.PI * Radius * Radius;
    public override double Perimeter() => 2 * Math.PI * Radius;
}
```

---

**接口定义**：定义契约
`public interface <接口名> { <返回类型> <方法>(); }`
```csharp
// 接口定义契约
public interface IReadable
{
    string Read();
}

public interface IWritable
{
    void Write(string content);
}

// 接口默认实现（C# 8+）
public interface ILogger
{
    void Log(string message);

    // 默认实现
    void LogWarning(string message) => Log($"[WARN] {message}");
    void LogError(string message) => Log($"[ERROR] {message}");
}
```

---

**多接口实现**：一个类实现多个接口
`public class <类名> : <接口1>, <接口2> { ... }`
```csharp
// 多接口实现
public class FileStorage : IReadable, IWritable, ILogger
{
    private readonly string _path;

    public FileStorage(string path) => _path = path;

    public string Read() => File.ReadAllText(_path);
    public void Write(string content) => File.WriteAllText(_path, content);
    public void Log(string message) => Console.WriteLine(message);
}
```

## 属性与索引器

**派生属性**：基于其他属性计算
`public <类型> <属性名> { get => <表达式>; set => <赋值>; }`
```csharp
// 派生属性
public class Temperature
{
    private double _celsius;

    public double Celsius
    {
        get => _celsius;
        set => _celsius = value;
    }

    // 派生属性
    public double Fahrenheit
    {
        get => _celsius * 9 / 5 + 32;
        set => _celsius = (value - 32) * 5 / 9;
    }

    // required 属性（C# 11）
    public required string Unit { get; init; }
}
```

---

**索引器**：通过索引访问对象成员
`public <类型> this[<参数>] { get; set; }`
```csharp
// 索引器
public class Matrix
{
    private readonly double[,] _data;

    public int Rows => _data.GetLength(0);
    public int Cols => _data.GetLength(1);

    public Matrix(int rows, int cols)
    {
        _data = new double[rows, cols];
    }

    // 索引器
    public double this[int row, int col]
    {
        get => _data[row, col];
        set => _data[row, col] = value;
    }
}

var matrix = new Matrix(3, 3);
matrix[0, 0] = 1.0;
matrix[1, 1] = 2.0;
var value = matrix[2, 2];
```

## 运算符重载

**二元运算符重载**：自定义运算符行为
`public static <类型> operator <符号>(<类型> <参数1>, <类型> <参数2>)`
```csharp
// 二元运算符
public class Vector
{
    public double X { get; }
    public double Y { get; }

    public Vector(double x, double y) => (X, Y) = (x, y);

    public static Vector operator +(Vector a, Vector b) =>
        new(a.X + b.X, a.Y + b.Y);

    public static Vector operator -(Vector a, Vector b) =>
        new(a.X - b.X, a.Y - b.Y);

    // 标量乘法
    public static Vector operator *(Vector v, double scalar) =>
        new(v.X * scalar, v.Y * scalar);
}
```

---

**比较运算符重载**：成对重载 == 和 !=
`public static bool operator ==(<类型> a, <类型> b) | public static bool operator !=(<类型> a, <类型> b)`
```csharp
// 比较运算符（需成对重载）
public static bool operator ==(Vector a, Vector b) =>
    a.X == b.X && a.Y == b.Y;

public static bool operator !=(Vector a, Vector b) => !(a == b);

// 必须重写 Equals 和 GetHashCode
public override bool Equals(object? obj) =>
    obj is Vector v && this == v;

public override int GetHashCode() => HashCode.Combine(X, Y);
```

**隐式/显式转换**：自定义类型转换
`public static implicit operator <目标类型>(<源类型> <参数>) | public static explicit operator <目标类型>(<源类型> <参数>)`
```csharp
// 隐式转换
public static implicit operator Vector((double x, double y) tuple) =>
    new(tuple.x, tuple.y);

// 使用
Vector v5 = (5.0, 6.0); // 元组隐式转换
```

## 记录类型 (Record)

**位置记录**：简洁的不可变数据类型
`public record <名称>(<参数列表>);`
```csharp
// 位置记录（最简洁）
public record Person(string Name, int Age);

// 可添加额外成员
public record Student(string Name, int Age, string School) : Person(Name, Age)
{
    public double GPA { get; init; }
    public string Grade => GPA switch
    {
        >= 3.7 => "A",
        >= 3.0 => "B",
        >= 2.0 => "C",
        _ => "D"
    };
}
```

---

**record struct**：值类型记录
`public record struct <名称>(<参数列表>) { ... }`
```csharp
// record struct（C# 10，值类型记录）
public record struct Point(double X, double Y)
{
    public double Distance => Math.Sqrt(X * X + Y * Y);
}

// readonly record struct
public readonly record struct Money(decimal Amount, string Currency);
```

---

**记录类型特性**：值相等性、with 表达式、解构
`<记录1> == <记录2> | <记录> with { <属性> = <值> } | var (<变量>) = <记录>`
```csharp
// 值相等性（与 class 不同）
var p1 = new Person("张三", 25);
var p2 = new Person("张三", 25);
Console.WriteLine(p1 == p2);       // True（class 为 False）

// with 表达式（非破坏性修改）
var p3 = p1 with { Age = 26 };

// 解构
var (name, age) = p1;
Console.WriteLine($"{name}, {age}"); // 张三, 25

// ToString 自动生成
Console.WriteLine(p1); // Person { Name = 张三, Age = 25 }
```
