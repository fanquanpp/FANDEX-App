# 模板

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 函数模板

**基本函数模板**
`template <typename T> <return_type> <func>(T <param>) { ... }`
```cpp
// 基本函数模板
template <typename T>
T max(T a, T b) {
    return a > b ? a : b;
}
// 使用
int i = max(10, 20);                    // T = int
double d = max(3.14, 2.71);              // T = double
std::string s = max(std::string("hello"), std::string("world"));    // T = std::string
```

---

**显式指定模板参数**
`<func><<type>>(<args>)`
```cpp
template <typename T>
T add(T a, T b) {
    return a + b;
}
// 显式指定模板参数
int result = add<int>(10, 20);
double result2 = add<double>(10.5, 20.5);
```

---

**模板重载与特化**
`template <> <return_type> <func><<type>>(<type> <a>, <type> <b>) { ... }`
```cpp
// 通用版本
template <typename T>
T max(T a, T b) {
    return a > b ? a : b;
}
// 针对 const char* 的重载
const char* max(const char* a, const char* b) {
    return strcmp(a, b) > 0 ? a : b;
}
// 特化版本
template <>
int max<int>(int a, int b) {
    return a > b ? a : b;
}
```

---

**多个模板参数**
`template <typename T1, typename T2, ...> <return_type> <func>(T1 <a>, T2 <b>) { ... }`
```cpp
template <typename T1, typename T2, typename T3>
typename std::common_type<T1, T2, T3>::type max(T1 a, T2 b, T3 c) {
    return max(max(a, b), c);
}
// 使用
auto result = max(10, 20.5, 15);    // 返回 double 类型
```

## 类模板

**基本类模板**
`template <typename T> class <Name> { ... };`
```cpp
template <typename T>
class Stack {
private:
    std::vector<T> elements;
public:
    void push(const T& item) {
        elements.push_back(item);
    }
    T pop() {
        if (elements.empty()) {
            throw std::runtime_error("Stack is empty");
        }
        T top = std::move(elements.back());
        elements.pop_back();
        return top;
    }
    bool empty() const {
        return elements.empty();
    }
};
// 使用
Stack<int> intStack;
intStack.push(1);
intStack.push(2);
std::cout << intStack.pop() << std::endl;    // 输出 2
```

---

**模板参数默认值**
`template <typename T, typename <Default> = <default_type>> class <Name> { ... };`
```cpp
template <typename T, typename Allocator = std::allocator<T>>
class MyVector {
private:
    std::vector<T, Allocator> data;
public:
    void push_back(const T& value) {
        data.push_back(value);
    }
    T& operator[](size_t index) {
        return data[index];
    }
};
// 使用默认分配器
MyVector<int> v1;
```

---

**类模板特化**
`template <> class <Name><<type>> { ... };`
```cpp
// 主模板
template <typename T>
class MyType {
public:
    static void print() { std::cout << "General template" << std::endl; }
};
// 特化版本
template <>
class MyType<int> {
public:
    static void print() { std::cout << "Specialized for int" << std::endl; }
};
// 部分特化
template <typename T>
class MyType<T*> {
public:
    static void print() { std::cout << "Specialized for pointer" << std::endl; }
};
```

## 可变参数模板

**可变参数模板（递归展开）**
`template <typename T, typename... Args> void <func>(T <first>, Args... <rest>) { ... }`
```cpp
// 递归终止条件
void print() {
    std::cout << std::endl;
}
// 可变参数模板
template <typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first << " ";
    print(rest...);    // 递归调用
}
// 使用
print(1, 2.5, "hello", true);    // 输出 1 2.5 hello 1
```

---

**折叠表达式（C++17）**
`(<expr> <op> ...) | (... <op> <expr>)`
```cpp
// 使用折叠表达式求和
template <typename... Args>
auto sum(Args... args) {
    return (args + ...);    // 二元左折叠
}
// 使用折叠表达式打印
template <typename... Args>
void print_fold(Args... args) {
    (std::cout << ... << args) << std::endl;    // 二元左折叠
}
// 使用
std::cout << "Sum: " << sum(1, 2, 3, 4, 5) << std::endl;    // 15
print_fold(1, " ", 2.5, " ", "hello");
```

---

**完美转发**
`template <typename... Args> void <func>(Args&&... args) { <target>(std::forward<Args>(args)...); }`
```cpp
// 完美转发函数
template <typename... Args>
void forward_args(Args&&... args) {
    print(std::forward<Args>(args)...);
}
// 使用
int x = 10;
forward_args(1, "hello", std::move(x));
```

## 模板元编程

**编译期计算**
`template <int N> struct <Name> { static constexpr int value = ...; };`
```cpp
// 编译期计算阶乘
template <int N>
struct Factorial {
    static constexpr int value = N * Factorial<N-1>::value;
};
// 特化版本作为递归终止条件
template <>
struct Factorial<0> {
    static constexpr int value = 1;
};
// 使用
constexpr int fact5 = Factorial<5>::value;    // 编译期计算 120
```

---

**类型 traits**
`template <typename T> struct <TraitName> { static constexpr bool value = ...; };`
```cpp
// 自定义类型 trait
template <typename T>
struct IsIntegral {
    static constexpr bool value = false;
};
// 特化
template <>
struct IsIntegral<int> {
    static constexpr bool value = true;
};
// 使用 if constexpr（C++17）
template <typename T>
void process(T value) {
    if constexpr (IsIntegral<T>::value) {
        std::cout << "Processing integral type: " << value << std::endl;
    } else {
        std::cout << "Processing non-integral type" << std::endl;
    }
}
```

## 概念约束（C++20）

**使用概念约束模板参数**
`template <typename T> requires <concept> <return_type> <func>(T <param>) { ... }`
```cpp
#include <concepts>
// 使用概念约束
template <typename T>
requires std::integral<T>
T add(T a, T b) {
    return a + b;
}
// 简写形式
template <std::integral T>
T multiply(T a, T b) {
    return a * b;
}
// 使用
int result = add(10, 20);    // 正确
// add(1.5, 2.5);    // 错误：double 不满足 std::integral
```
