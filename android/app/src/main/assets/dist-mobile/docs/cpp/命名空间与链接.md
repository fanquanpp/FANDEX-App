# 命名空间与链接

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 命名空间定义

**基本命名空间**
`namespace <Name> { <members> }`
```cpp
namespace MyMath {
    const double PI = 3.14159265358979323846;
    int add(int a, int b) { return a + b; }
    int subtract(int a, int b) { return a - b; }
    namespace Advanced {
        double power(double base, int exp) {
            return std::pow(base, exp);
        }
    }
}
```

---

**嵌套命名空间（C++17 简写）**
`namespace <A>::<B>::<C> { ... }`
```cpp
// C++17 简写形式
namespace Company::Project::Module {
    void function() { std::cout << "Nested namespace" << std::endl; }
}
// 等价的传统写法
namespace Company {
    namespace Project {
        namespace Module {
            void function() { std::cout << "Nested namespace" << std::endl; }
        }
    }
}
```

---

## 使用命名空间

**using 声明**
`using <namespace>::<member>;`
```cpp
using std::cout;
using std::endl;
using std::string;
int main() {
    cout << "Hello" << endl;
    string s = "World";
    return 0;
}
```

---

**using 指令**
`using namespace <name>;`
```cpp
using namespace std;
int main() {
    cout << "Hello" << endl;
    vector<int> v;
    return 0;
}
```

---

**命名空间别名**
`namespace <alias> = <long_namespace>;`
```cpp
namespace very_long_namespace_name {
    void function() {}
}
namespace vln = very_long_namespace_name;
int main() {
    vln::function();
    return 0;
}
```

## 链接性

**外部链接**
`extern <type> <var>;`
```cpp
// file1.cpp
int global_var = 100;           // 定义，外部链接
void external_func() {}         // 定义，外部链接
// file2.cpp
extern int global_var;          // 声明，引用 file1.cpp 中的定义
extern void external_func();    // 声明，引用 file1.cpp 中的定义
void use_external() {
    std::cout << global_var;
    external_func();
}
```

---

**内部链接**
`static <type> <var>;` 或匿名命名空间
```cpp
// 使用 static
static int internal_var = 50;    // 仅当前文件可见
static void internal_func() {}   // 仅当前文件可见
// 使用匿名命名空间（推荐）
namespace {
    int anonymous_var = 200;
    void anonymous_func() {}
}
```

---

**无链接**
`<type> <var>;` （局部变量）
```cpp
void function() {
    int local_var = 10;    // 无链接，仅函数内可见
    static int static_local = 20;    // 无链接，但生命周期为整个程序
}
```

---

## inline 变量（C++17）

**头文件中定义变量**
`inline <type> <var> = <value>;`
```cpp
// header.h
inline int global_counter = 0;    // C++17：可在头文件中定义
inline const double PI = 3.14159;
// 多个源文件包含此头文件不会导致重复定义
```

---

## extern 模板

**显式实例化声明**
`extern template class <ClassName><<type>>;`
```cpp
// 模板定义在 header.h 中
template <typename T>
class MyContainer { /* ... */ };
// 在 source.cpp 中显式实例化
// template class MyContainer<int>;
// 在其他文件中声明，避免重复实例化
// extern template class MyContainer<int>;
```

## 匿名命名空间

**匿名命名空间**
`namespace { <members> }`
```cpp
// 匿名命名空间：内容仅当前文件可见
namespace {
    const int INTERNAL_CONSTANT = 42;
    void internal_helper() {
        std::cout << "Internal helper" << std::endl;
    }
}
void public_function() {
    internal_helper();    // 可以调用
    std::cout << INTERNAL_CONSTANT;
}
```

## 命名空间与 ADL

**参数依赖查找（ADL）**
`<func>(<args>); // 编译器根据参数类型查找命名空间`
```cpp
namespace MyLib {
    class MyClass {
    public:
        int value;
    };
    // 在命名空间中定义运算符
    std::ostream& operator<<(std::ostream& os, const MyClass& obj) {
        return os << "MyClass(" << obj.value << ")";
    }
    void print(const MyClass& obj) {
        std::cout << obj << std::endl;    // ADL 查找 operator<<
    }
}
int main() {
    MyLib::MyClass obj{42};
    MyLib::print(obj);    // 正确调用
    // operator<< 通过 ADL 自动找到
    std::cout << obj << std::endl;    // 无需 using 声明
    return 0;
}
```
