# 字符串处理

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## std::string 基础

**字符串声明与初始化**
`std::string <str> = "<value>";` 或 `std::string <str>(<count>, <char>);`
```cpp
#include <string>
std::string s1;                          // 空字符串
std::string s2 = "Hello";                // 从 C 字符串初始化
std::string s3("World");                // 构造函数初始化
std::string s4(5, 'a');                  // "aaaaa"
std::string s5(s2);                      // 拷贝构造
std::string s6(s2, 1, 3);                // "ell"（从位置 1 开始取 3 个字符）
std::string s7 = s2 + " " + s3;          // "Hello World"
```

---

**字符串赋值与拼接**
`<str> = <value>; | <str> += <value>; | <str>.append(<value>);`
```cpp
std::string s = "Hello";
s = "World";                    // 赋值
s += "!";                       // 追加
s.append(" C++");                // 追加
s.push_back('!');                // 追加字符
// 使用 + 运算符
std::string result = s + " " + "World";
```

---

## 字符串访问

**下标访问**
`<str>[<index>]` 或 `<str>.at(<index>)`
```cpp
std::string s = "Hello";
char c1 = s[0];              // 'H'，不检查边界
char c2 = s.at(1);           // 'e'，检查边界，越界抛出异常
char c3 = s.front();         // 'H'，首字符
char c4 = s.back();          // 'o'，尾字符
```

---

**迭代器访问**
`for (auto it = <str>.begin(); it != <str>.end(); ++it) { ... }`
```cpp
std::string s = "Hello";
// 使用迭代器
for (auto it = s.begin(); it != s.end(); ++it) {
    std::cout << *it << " ";
}
// 范围 for 循环
for (char& c : s) {
    c = std::toupper(c);
}
```

---

## 字符串信息

**获取字符串信息**
`<str>.size() | <str>.length() | <str>.empty() | <str>.capacity()`
```cpp
std::string s = "Hello";
size_t len = s.size();           // 5
size_t len2 = s.length();        // 5
bool empty = s.empty();          // false
size_t cap = s.capacity();       // 容量
s.reserve(100);                  // 预留容量
s.shrink_to_fit();               // 释放多余容量
```

## 字符串查找

**查找子串**
`<str>.find(<substr>[, <pos>]) | <str>.rfind(<substr>[, <pos>])`
```cpp
std::string s = "Hello, World!";
size_t pos = s.find("World");           // 7
size_t pos2 = s.find("o", 5);           // 8（从位置 5 开始查找）
size_t pos3 = s.rfind("o");             // 8（从后向前查找）
if (pos != std::string::npos) {
    std::cout << "Found at: " << pos;
}
```

---

**查找字符**
`<str>.find_first_of(<chars>) | <str>.find_last_of(<chars>)`
```cpp
std::string s = "Hello, World!";
size_t pos = s.find_first_of("aeiou");    // 1（第一个元音）
size_t pos2 = s.find_last_of("aeiou");    // 10（最后一个元音）
size_t pos3 = s.find_first_not_of("Hello, ");    // 7
size_t pos4 = s.find_last_not_of("!");    // 12
```

---

## 字符串子串与修改

**获取子串**
`<str>.substr(<pos>[, <count>])`
```cpp
std::string s = "Hello, World!";
std::string sub1 = s.substr(7);        // "World!"
std::string sub2 = s.substr(7, 5);     // "World"
std::string sub3 = s.substr(0, 5);     // "Hello"
```

---

**插入与删除**
`<str>.insert(<pos>, <str>) | <str>.erase(<pos>[, <count>])`
```cpp
std::string s = "Hello World";
s.insert(5, ",");          // "Hello, World"
s.insert(0, "Say: ");      // "Say: Hello, World"
s.erase(0, 4);             // "Hello, World"（删除前 4 个字符）
s.erase(5);                 // "Hello"（删除位置 5 之后的所有字符）
s.replace(0, 5, "Hi");      // "Hi, World"
```

---

**替换**
`<str>.replace(<pos>, <count>, <new_str>)`
```cpp
std::string s = "Hello, World!";
s.replace(7, 5, "C++");     // "Hello, C++!"
```

## 字符串比较

**比较运算符**
`<str1> <==|!=|<|>|<=|>=> <str2>`
```cpp
std::string s1 = "Hello";
std::string s2 = "World";
bool eq = (s1 == s2);       // false
bool less = (s1 < s2);      // true
int cmp = s1.compare(s2);   // <0（s1 < s2）
```

---

**compare 函数**
`<str>.compare(<other>) | <str>.compare(<pos>, <len>, <other>)`
```cpp
std::string s = "Hello, World!";
int result = s.compare("Hello");            // >0
int result2 = s.compare(0, 5, "Hello");     // 0
int result3 = s.compare(7, 5, "World");    // 0
```

## 字符串转换

**数值转字符串**
`std::to_string(<number>)`
```cpp
std::string s1 = std::to_string(42);          // "42"
std::string s2 = std::to_string(3.14);         // "3.140000"
std::string s3 = std::to_string(-100);         // "-100"
```

---

**字符串转数值**
`std::stoi(<str>) | std::stod(<str>) | std::stol(<str>)`
```cpp
std::string s = "42";
int i = std::stoi(s);           // 42
double d = std::stod("3.14");    // 3.14
long l = std::stol("123456");   // 123456
// 带错误处理
try {
    size_t pos;
    int val = std::stoi("42abc", &pos);    // val=42, pos=2
} catch (const std::invalid_argument& e) {
    std::cerr << "Invalid argument: " << e.what();
} catch (const std::out_of_range& e) {
    std::cerr << "Out of range: " << e.what();
}
```

## 字符串流

**字符串流拼接**
`std::ostringstream <oss>; <oss> << <data>; std::string <result> = <oss>.str();`
```cpp
#include <sstream>
std::ostringstream oss;
oss << "Name: " << "Alice" << ", Age: " << 25 << ", Score: " << 95.5;
std::string result = oss.str();
// "Name: Alice, Age: 25, Score: 95.5"
```

---

**字符串流解析**
`std::istringstream <iss>(<str>); <iss> >> <var>;`
```cpp
std::string data = "10 3.14 hello";
std::istringstream iss(data);
int i;
double d;
std::string s;
iss >> i >> d >> s;    // i=10, d=3.14, s="hello"
```

## C 风格字符串

**转换为 C 字符串**
`<str>.c_str()` 或 `<str>.data()`
```cpp
std::string s = "Hello";
const char* cstr = s.c_str();    // 返回 const char*
char* data = s.data();           // C++17 返回 char*
// 使用 C 函数
printf("%s\n", s.c_str());
FILE* f = fopen(s.c_str(), "r");
```

---

**从 C 字符串构造**
`std::string <str>(<cstr>[, <count>])`
```cpp
const char* cstr = "Hello, World!";
std::string s1(cstr);            // "Hello, World!"
std::string s2(cstr, 5);         // "Hello"
```

## 字符串视图（C++17）

**std::string_view**
`std::string_view <sv> = "<string>";`
```cpp
#include <string_view>
// 不拥有字符串，仅引用
std::string_view sv = "Hello, World!";
std::string_view sv2(sv.substr(7, 5));    // "World"
// 函数参数使用 string_view 避免拷贝
void print(std::string_view sv) {
    std::cout << sv << std::endl;
}
print("Hello");
print(std::string("World"));
```

## 字符串格式化（C++20）

**std::format**
`std::format("<format_string>", <args>...)`
```cpp
#include <format>
std::string result = std::format("Name: {}, Age: {}", "Alice", 25);
std::string result2 = std::format("Pi: {:.2f}", 3.14159);
std::string result3 = std::format("{1} before {0}", "World", "Hello");
```
