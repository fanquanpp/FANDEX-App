# 右值引用与移动语义

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 左值与右值

**左值引用**
`<type>& <ref> = <lvalue>;`
```cpp
int x = 10;
int& lref = x;    // 左值引用：绑定到左值 x
lref = 20;
std::cout << x << std::endl;    // 20
```

---

**右值引用**
`<type>&& <ref> = <rvalue>;`
```cpp
// 右值引用：绑定到右值
int&& rref = 42;
std::cout << rref << std::endl;    // 42
// 右值引用可以修改临时值
rref = 100;
std::cout << rref << std::endl;    // 100
```

---

## std::move

**将左值转换为右值**
`std::move(<lvalue>)`
```cpp
std::string s1 = "Hello";
// std::move 将 s1 转换为右值引用
std::string s2 = std::move(s1);
// s1 现在处于有效但未定义状态
std::cout << s2 << std::endl;    // Hello
// std::move 本质上是 static_cast<T&&>
```

## 移动构造函数

**移动构造函数定义**
`<ClassName>(<ClassName>&& <other>) noexcept { ... }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    // 构造函数
    Buffer(size_t size) : data_(new int[size]), size_(size) {}
    // 移动构造函数
    Buffer(Buffer&& other) noexcept
        : data_(other.data_), size_(other.size_) {
        other.data_ = nullptr;    // 置空源对象
        other.size_ = 0;
    }
    ~Buffer() { delete[] data_; }
};
// 使用
Buffer b1(100);
Buffer b2 = std::move(b1);    // 调用移动构造函数
```

## 移动赋值运算符

**移动赋值运算符定义**
`<ClassName>& operator=(<ClassName>&& <other>) noexcept { ... return *this; }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    // 移动赋值运算符
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data_;            // 释放当前资源
            data_ = other.data_;       // 接管资源
            size_ = other.size_;
            other.data_ = nullptr;    // 置空源对象
            other.size_ = 0;
        }
        return *this;
    }
};
// 使用
Buffer b1(100);
Buffer b2(50);
b2 = std::move(b1);    // 调用移动赋值运算符
```

## 完美转发

**std::forward 完美转发**
`template <typename T> void <func>(T&& <arg>) { <target>(std::forward<T>(<arg>)); }`
```cpp
#include <utility>
// 完美转发函数
template <typename T>
void wrapper(T&& arg) {
    target(std::forward<T>(arg));    // 保持 arg 的左右值属性
}
void target(int& x) { std::cout << "lvalue" << std::endl; }
void target(int&& x) { std::cout << "rvalue" << std::endl; }
// 使用
int x = 10;
wrapper(x);     // 输出 lvalue
wrapper(42);    // 输出 rvalue
```

---

**可变参数完美转发**
`template <typename... Args> void <func>(Args&&... args) { <target>(std::forward<Args>(args)...); }`
```cpp
template <typename... Args>
void emplace_back_wrapper(std::vector<int>& vec, Args&&... args) {
    vec.emplace_back(std::forward<Args>(args)...);
}
// 使用
std::vector<int> v;
emplace_back_wrapper(v, 42);
```

## 引用折叠规则

**引用折叠规则**
`T& & -> T& | T& && -> T& | T&& & -> T& | T&& && -> T&&`
```cpp
template <typename T>
void check_type(T&& x) {
    // T 为 int& 时，T&& 折叠为 int&
    // T 为 int 时，T&& 保持为 int&&
}
int x = 10;
check_type(x);     // T = int&，x 为左值
check_type(42);    // T = int，42 为右值
```

## std::move 与 std::forward 对比

**move 与 forward 的区别**
`std::move(<arg>) // 无条件转换为右值 | std::forward<T>(<arg>) // 有条件转换`
```cpp
// std::move：无条件转换为右值
template <typename T>
void move_example(T&& x) {
    auto y = std::move(x);    // 总是右值
}
// std::forward：有条件转换
template <typename T>
void forward_example(T&& x) {
    auto y = std::forward<T>(x);    // 保持原始左右值属性
}
```

## 移动语义与容器

**vector 的移动语义**
`std::vector<<Type>> <vec>; <vec>.push_back(std::move(<obj>));`
```cpp
class HeavyObject {
    int* data_;
    size_t size_;
public:
    HeavyObject(size_t size) : data_(new int[size]), size_(size) {}
    ~HeavyObject() { delete[] data_; }
    // 移动构造函数
    HeavyObject(HeavyObject&& other) noexcept
        : data_(other.data_), size_(other.size_) {
        other.data_ = nullptr;
        other.size_ = 0;
    }
};
std::vector<HeavyObject> objects;
HeavyObject obj(1000);
objects.push_back(std::move(obj));    // 移动而非拷贝
```

---

**emplace_back 与 push_back**
`<vec>.emplace_back(<args>...) // 直接构造 | <vec>.push_back(<obj>) // 先构造再移动/拷贝`
```cpp
std::vector<std::string> vec;
// push_back：先构造临时对象，再移动
vec.push_back(std::string("Hello"));
// emplace_back：直接在容器中构造
vec.emplace_back("Hello");    // 更高效
```

## 返回值优化（RVO/NRVO）

**返回局部对象**
`<Type> <func>() { <Type> <local>; ... return <local>; }`
```cpp
std::string create_string() {
    std::string s = "Hello";
    s += ", World!";
    return s;    // RVO：编译器优化，避免拷贝
}
std::string result = create_string();    // 无拷贝
```

---

**返回 std::move 的陷阱**
`return std::move(<local>); // 错误：阻止 RVO`
```cpp
std::string create_string_bad() {
    std::string s = "Hello";
    return std::move(s);    // 错误：阻止 RVO
}
std::string create_string_good() {
    std::string s = "Hello";
    return s;    // 正确：允许 RVO
}
```

## 移动语义最佳实践

**noexcept 移动操作**
`<ClassName>(<ClassName>&& <other>) noexcept { ... }`
```cpp
class MyClass {
public:
    // 移动操作必须 noexcept，否则容器不会使用移动
    MyClass(MyClass&& other) noexcept { /* ... */ }
    MyClass& operator=(MyClass&& other) noexcept { /* ... */ return *this; }
};
```

---

**五法则**
`class <Name> { <Name>() = default; ~<Name>() = default; <Name>(const <Name>&) = default; <Name>(<Name>&&) = default; <Name>& operator=(const <Name>&) = default; <Name>& operator=(<Name>&&) = default; };`
```cpp
class MyClass {
public:
    MyClass() = default;
    ~MyClass() = default;
    MyClass(const MyClass&) = default;             // 拷贝构造
    MyClass(MyClass&&) noexcept = default;         // 移动构造
    MyClass& operator=(const MyClass&) = default; // 拷贝赋值
    MyClass& operator=(MyClass&&) noexcept = default; // 移动赋值
};
```
