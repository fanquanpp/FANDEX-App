# 指针

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 指针基础

**指针声明与初始化**
`<type>* <ptr> = &<var>;`
```cpp
int x = 10;          // 声明一个整数变量
int* p = &x;         // 声明指向整数的指针，指向 x 的地址
*p = 20;             // 通过指针修改 x 的值
std::cout << x;     // 输出 20
```

---

## 指针类型

**各种指针类型**
`<type>* <ptr> = &<var>; | const <type>* <ptr> = &<var>; | <type>* const <ptr> = &<var>;`
```cpp
int x = 10;
int* p = &x;                  // 指向整数的指针
const int* cp = &x;          // 指向常量整数的指针（不能通过指针修改值）
int* const pc = &x;          // 常量指针（指针本身不可修改）
const int* const cpc = &x;   // 指向常量的常量指针
void* v = &x;                // 通用指针（无类型）
```

## 指针运算

**指针算术**
`<ptr> + <n> | <ptr> - <n> | <ptr>++ | <ptr>--`
```cpp
int arr[] = {1, 2, 3, 4, 5};
int* p = arr;              // 指向数组的第一个元素
std::cout << *p << std::endl;       // 输出 1
std::cout << *(p + 1) << std::endl; // 输出 2
p++;                       // 指针向后移动一个元素
std::cout << *p << std::endl;       // 输出 2
p--;                       // 指针向前移动一个元素
std::cout << *p << std::endl;       // 输出 1
```

## 空指针

**空指针声明**
`<type>* <ptr> = nullptr;`
```cpp
// 空指针（C++11 推荐使用 nullptr）
int* p1 = nullptr;
// 传统方式
int* p2 = NULL;
int* p3 = 0;
// 检查指针是否为空
if (p1 != nullptr) {
    *p1 = 10;
}
```

## 指针与函数

**指针作为函数参数（传地址）**
`void <func>(<type>* <param>) { (*<param>)++; }`
```cpp
void increment(int* p) {
    (*p)++;
}
int main() {
    int x = 10;
    increment(&x);
    std::cout << x;    // 输出 11
    return 0;
}
```

---

**指针作为函数返回值**
`<type>* <func>(<params>) { return new <type>[<size>]; }`
```cpp
int* create_array(int size) {
    return new int[size];
}
```

---

## 指针数组与数组指针

**指针数组：数组元素是指针**
`<type>* <array_name>[<size>];`
```cpp
int* ptrs[5];    // 5 个指向整数的指针
int a = 10, b = 20;
ptrs[0] = &a;
ptrs[1] = &b;
```

---

**数组指针：指向数组的指针**
`<type> (*<ptr>)[<size>] = &<array>;`
```cpp
int arr[5] = {1, 2, 3, 4, 5};
int (*p)[5] = &arr;    // 指向包含 5 个整数的数组的指针
std::cout << (*p)[0] << std::endl;    // 输出 1
std::cout << (*p)[1] << std::endl;    // 输出 2
```

## 动态内存分配

**new 分配内存**
`<type>* <ptr> = new <type>;` 或 `<type>* <ptr> = new <type>[<size>];`
```cpp
// 分配单个变量
int* p = new int;
*p = 42;
// 分配数组
int* arr = new int[10];
for (int i = 0; i < 10; i++) {
    arr[i] = i;
}
```

---

**delete 释放内存**
`delete <ptr>;` 或 `delete[] <ptr>;`
```cpp
// 释放单个变量
delete p;
// 释放数组
delete[] arr;
```

## 智能指针

**unique_ptr：独占所有权**
`std::unique_ptr<<Type>> <ptr> = std::make_unique<<Type>>(<args>);`
```cpp
#include <memory>
// 创建 unique_ptr
auto p = std::make_unique<int>(42);
std::cout << *p << std::endl;    // 输出 42
// 所有权转移
auto p2 = std::move(p);
// p 现在为空，p2 拥有对象
```

---

**shared_ptr：共享所有权**
`std::shared_ptr<<Type>> <ptr> = std::make_shared<<Type>>(<args>);`
```cpp
#include <memory>
// 创建 shared_ptr
auto p1 = std::make_shared<int>(42);
auto p2 = p1;    // 引用计数 +1
std::cout << *p1 << " " << *p2 << std::endl;    // 输出 42 42
std::cout << p1.use_count() << std::endl;        // 输出 2
```

---

**weak_ptr：打破循环引用**
`std::weak_ptr<<Type>> <wp> = <shared_ptr>;`
```cpp
#include <memory>
auto sp = std::make_shared<int>(42);
std::weak_ptr<int> wp = sp;    // 不增加引用计数
// 从 weak_ptr 获取 shared_ptr
if (auto locked = wp.lock()) {
    std::cout << *locked << std::endl;    // 输出 42
}
```
