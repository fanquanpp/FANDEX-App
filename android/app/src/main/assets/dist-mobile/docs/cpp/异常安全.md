# 异常安全

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 异常抛出

**throw 抛出异常**
`throw <exception_object>;`
```cpp
#include <stdexcept>
// 抛出基本类型
throw 42;
throw "Error occurred";
// 抛出标准异常
throw std::runtime_error("Runtime error");
throw std::logic_error("Logic error");
throw std::out_of_range("Index out of range");
// 抛出自定义异常
class MyException : public std::exception {
public:
    const char* what() const noexcept override {
        return "My custom exception";
    }
};
throw MyException();
```

## 异常捕获

**try-catch 基本结构**
`try { ... } catch (<exception_type> <name>) { ... }`
```cpp
try {
    throw std::runtime_error("Something went wrong");
} catch (const std::runtime_error& e) {
    std::cerr << "Runtime error: " << e.what() << std::endl;
} catch (const std::exception& e) {
    std::cerr << "Standard exception: " << e.what() << std::endl;
} catch (...) {
    std::cerr << "Unknown exception" << std::endl;
}
```

---

**捕获多种异常类型**
`try { ... } catch (<type1>& e) { ... } catch (<type2>& e) { ... }`
```cpp
try {
    if (error_code == 1) {
        throw std::runtime_error("Runtime error");
    } else if (error_code == 2) {
        throw std::logic_error("Logic error");
    } else {
        throw 42;    // 基本类型
    }
} catch (const std::runtime_error& e) {
    std::cerr << "Runtime: " << e.what() << std::endl;
} catch (const std::logic_error& e) {
    std::cerr << "Logic: " << e.what() << std::endl;
} catch (int code) {
    std::cerr << "Error code: " << code << std::endl;
} catch (...) {
    std::cerr << "Unknown error" << std::endl;
}
```

## 异常重抛

**异常重抛**
`throw;`
```cpp
try {
    try {
        throw std::runtime_error("Original error");
    } catch (const std::exception& e) {
        std::cerr << "Inner catch: " << e.what() << std::endl;
        throw;    // 重抛当前异常
    }
} catch (const std::exception& e) {
    std::cerr << "Outer catch: " << e.what() << std::endl;
}
```

## noexcept 说明符

**noexcept 声明**
`<return_type> <func>(<params>) noexcept { ... }`
```cpp
// noexcept：承诺不抛出异常
void safe_function() noexcept {
    // 如果抛出异常，程序将调用 std::terminate
}
// noexcept(false)：可能抛出异常
void risky_function() noexcept(false) {
    throw std::runtime_error("Error");
}
// 条件 noexcept
template <typename T>
void swap_values(T& a, T& b) noexcept(noexcept(T(std::move(a)))) {
    T temp = std::move(a);
    a = std::move(b);
    b = std::move(temp);
}
```

---

**noexcept 运算符**
`noexcept(<expression>)`
```cpp
// 检查表达式是否不会抛出异常
static_assert(noexcept(1 + 1), "Addition should not throw");
template <typename T>
void process(T value) {
    if constexpr (noexcept(value.process())) {
        // 编译期检查
    }
}
```

## RAII 模式

**RAII 资源管理**
`class <Name> { <resource>* <ptr>; public: <Name>() { <ptr> = new ...; } ~<Name>() { delete <ptr>; } };`
```cpp
class FileGuard {
    FILE* file_;
public:
    explicit FileGuard(const char* filename) {
        file_ = fopen(filename, "w");
        if (!file_) {
            throw std::runtime_error("Cannot open file");
        }
    }
    ~FileGuard() {
        if (file_) {
            fclose(file_);
        }
    }
    // 禁用拷贝
    FileGuard(const FileGuard&) = delete;
    FileGuard& operator=(const FileGuard&) = delete;
    FILE* get() const { return file_; }
};
// 使用：无论是否抛出异常，析构函数都会被调用
void process_file() {
    FileGuard fg("output.txt");
    fprintf(fg.get(), "Hello, RAII!");
    // 如果这里抛出异常，fg 的析构函数仍会调用
}
```

## 自定义异常类

**继承 std::exception**
`class <Name> : public std::exception { ... };`
```cpp
#include <exception>
#include <string>
class DatabaseException : public std::exception {
private:
    std::string message_;
    int error_code_;
public:
    DatabaseException(const std::string& msg, int code)
        : message_(msg), error_code_(code) {}
    const char* what() const noexcept override {
        return message_.c_str();
    }
    int code() const noexcept { return error_code_; }
};
// 使用
try {
    throw DatabaseException("Connection failed", 1001);
} catch (const DatabaseException& e) {
    std::cerr << "Error " << e.code() << ": " << e.what() << std::endl;
}
```

---

**继承具体异常类型**
`class <Name> : public std::<exception_type> { ... };`
```cpp
class InvalidInputException : public std::invalid_argument {
public:
    InvalidInputException(const std::string& msg)
        : std::invalid_argument(msg) {}
};
class NetworkException : public std::runtime_error {
private:
    int status_code_;
public:
    NetworkException(const std::string& msg, int status)
        : std::runtime_error(msg), status_code_(status) {}
    int statusCode() const noexcept { return status_code_; }
};
```

## 异常安全等级

**基本保证**
`void <func>() { ... } // 不泄漏资源，不破坏不变式`
```cpp
// 基本保证：异常发生时对象处于有效状态
void process_data(std::vector<int>& data) {
    std::vector<int> temp;
    for (int val : data) {
        temp.push_back(val * 2);    // 可能抛出异常
    }
    data = std::move(temp);    // 仅在成功时修改原数据
}
```

---

**强保证（事务语义）**
`void <func>() { <Type> copy = original; ... original = std::move(copy); }`
```cpp
// 强保证：要么完全成功，要么回滚
void add_items(std::vector<int>& vec, const std::vector<int>& items) {
    auto copy = vec;    // 创建副本
    for (int item : items) {
        copy.push_back(item);    // 在副本上操作
    }
    vec = std::move(copy);    // 仅在成功时提交
}
```

## 异常与构造函数

**构造函数中的异常**
`<ClassName>(<params>) : <member>(<init>) { if (<error>) throw ...; }`
```cpp
class Config {
    std::string filename_;
    int* data_;
    size_t size_;
public:
    Config(const std::string& filename, size_t size)
        : filename_(filename), data_(new int[size]), size_(size) {
        FILE* f = fopen(filename.c_str(), "r");
        if (!f) {
            delete[] data_;    // 手动清理已分配的资源
            throw std::runtime_error("Cannot open config file");
        }
        fclose(f);
    }
    ~Config() { delete[] data_; }
};
```

## 异常处理最佳实践

**使用智能指针避免泄漏**
`std::unique_ptr<<Type>> <ptr> = std::make_unique<<Type>>(<args>);`
```cpp
void safe_function() {
    // 使用智能指针，异常时自动释放
    auto ptr = std::make_unique<int[]>(1000);
    if (some_error) {
        throw std::runtime_error("Error");
    }
    // 即使抛出异常，ptr 也会自动释放
}
```

---

**catch 引用避免对象切片**
`catch (const <Base>& <e>) { ... }`
```cpp
try {
    throw DerivedException("Derived error");
} catch (const BaseException& e) {    // 使用引用避免切片
    e.what();    // 正确调用派生类的 what()
}
```
