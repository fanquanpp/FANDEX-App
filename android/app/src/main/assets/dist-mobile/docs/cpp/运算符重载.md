# 运算符重载

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 二元算术运算符

**复合赋值运算符（成员函数）**
`<Type>& operator<+=|-=|*=|/=>(const <Type>& <rhs>) { ... return *this; }`
```cpp
class Vector2D {
    double x_, y_;
public:
    Vector2D(double x = 0, double y = 0) : x_(x), y_(y) {}
    // 复合赋值运算符
    Vector2D& operator+=(const Vector2D& rhs) {
        x_ += rhs.x_;
        y_ += rhs.y_;
        return *this;
    }
};
```

---

**二元算术运算符（非成员函数）**
`<Type> operator<+|-|*|/>(<Type> <lhs>, const <Type>& <rhs>) { <lhs> <op>= <rhs>; return <lhs>; }`
```cpp
class Vector2D {
    double x_, y_;
public:
    Vector2D(double x = 0, double y = 0) : x_(x), y_(y) {}
    Vector2D& operator+=(const Vector2D& rhs);
    friend Vector2D operator+(Vector2D lhs, const Vector2D& rhs) {
        lhs += rhs;    // 通过 += 实现 +
        return lhs;
    }
};
```

## 比较运算符

**C++20 三路比较运算符**
`auto operator<=>(const <Type>&) const = default;`
```cpp
class Point {
    int x_, y_;
public:
    Point(int x, int y) : x_(x), y_(y) {}
    // C++20：自动生成 < <= > >= == !=
    auto operator<=>(const Point&) const = default;
};
```

---

**C++20 之前手动定义比较运算符**
`bool operator<==|!=|<|>|<=|>=>(const <Type>& <other>) const { ... }`
```cpp
class PointOld {
    int x_, y_;
public:
    bool operator==(const PointOld& other) const {
        return x_ == other.x_ && y_ == other.y_;
    }
    bool operator<(const PointOld& other) const {
        return x_ < other.x_ || (x_ == other.x_ && y_ < other.y_);
    }
};
```

## 下标运算符

**下标运算符重载**
`<type>& operator[](size_t <index>) { ... return <data>[<index>]; }`
```cpp
class IntArray {
    int* data_;
    size_t size_;
public:
    explicit IntArray(size_t n) : data_(new int[n]()), size_(n) {}
    ~IntArray() { delete[] data_; }
    // 非常量版本：可修改元素
    int& operator[](size_t index) {
        return data_[index];
    }
    // 常量版本：只读访问
    const int& operator[](size_t index) const {
        return data_[index];
    }
};
```

## 函数调用运算符

**函数对象（Functor）**
`<type> operator()(<params>) const { ... }`
```cpp
class LinearInterpolator {
    double x1_, y1_, x2_, y2_;
public:
    LinearInterpolator(double x1, double y1, double x2, double y2)
        : x1_(x1), y1_(y1), x2_(x2), y2_(y2) {}
    double operator()(double x) const {
        double t = (x - x1_) / (x2_ - x1_);
        return y1_ + t * (y2_ - y1_);
    }
};
// 使用
LinearInterpolator interp(0, 0, 10, 100);
double y = interp(5);    // 50.0
```

## 流运算符

**输出运算符（非成员函数）**
`friend std::ostream& operator<<(std::ostream& <os>, const <Type>& <obj>) { <os> << <data>; return <os>; }`
```cpp
class Fraction {
    int numerator_, denominator_;
public:
    Fraction(int n, int d) : numerator_(n), denominator_(d) {}
    friend std::ostream& operator<<(std::ostream& os, const Fraction& f) {
        os << f.numerator_ << "/" << f.denominator_;
        return os;
    }
};
Fraction f(3, 4);
std::cout << f << std::endl;    // 输出: 3/4
```

---

**输入运算符（非成员函数）**
`friend std::istream& operator>>(std::istream& <is>, <Type>& <obj>) { <is> >> <data>; return <is>; }`
```cpp
class Fraction {
    int numerator_, denominator_;
public:
    friend std::istream& operator>>(std::istream& is, Fraction& f) {
        char slash;
        is >> f.numerator_ >> slash >> f.denominator_;
        return is;
    }
};
```

## 自增自减运算符

**前置自增**
`<Type>& operator++() { ++<member>; return *this; }`
```cpp
class Counter {
    int value_;
public:
    explicit Counter(int v = 0) : value_(v) {}
    // 前置 ++：返回引用
    Counter& operator++() {
        ++value_;
        return *this;
    }
};
```

---

**后置自增**
`<Type> operator++(int) { <Type> temp = *this; ++<member>; return temp; }`
```cpp
class Counter {
    int value_;
public:
    // 后置 ++：返回旧值的副本，int 参数是区分标志
    Counter operator++(int) {
        Counter temp = *this;
        ++value_;
        return temp;
    }
};
```

---

**前置与后置自减**
`<Type>& operator--() { ... } | <Type> operator--(int) { ... }`
```cpp
class Counter {
    int value_;
public:
    // 前置 --
    Counter& operator--() {
        --value_;
        return *this;
    }
    // 后置 --
    Counter operator--(int) {
        Counter temp = *this;
        --value_;
        return temp;
    }
};
```

## 类型转换运算符

**显式类型转换运算符**
`explicit operator <target_type>() const { ... }`
```cpp
class SafeBool {
    bool value_;
public:
    explicit SafeBool(bool v) : value_(v) {}
    // C++11：explicit 防止隐式转换
    explicit operator bool() const { return value_; }

SafeBool sb(true);
if (sb) { /* 正确：显式转换上下文 */ }
```

## 智能指针风格运算符

**解引用与箭头运算符**
`<Type>& operator*() const { return *<ptr>; } | <Type>* operator->() const { return <ptr>; }`
```cpp
template<typename T>
class ScopedPtr {
    T* ptr_;
public:
    explicit ScopedPtr(T* p = nullptr) : ptr_(p) {}
    ~ScopedPtr() { delete ptr_; }
    // 解引用
    T& operator*() const { return *ptr_; }
    // 箭头运算符
    T* operator->() const { return ptr_; }
    // 布尔转换
    explicit operator bool() const { return ptr_ != nullptr; }
};
```

## 迭代器运算符

**迭代器风格运算符**
`<Iter>& operator++() { ... } | <Type>& operator*() const { ... } | bool operator<==|!=>(const <Iter>&) const { ... }`
```cpp
template<typename T>
class ArrayIterator {
    T* ptr_;
public:
    explicit ArrayIterator(T* p) : ptr_(p) {}
    T& operator*() const { return *ptr_; }
    T* operator->() const { return ptr_; }
    ArrayIterator& operator++() { ++ptr_; return *this; }
    ArrayIterator operator++(int) { auto tmp = *this; ++ptr_; return tmp; }
    bool operator==(const ArrayIterator& other) const { return ptr_ == other.ptr_; }
    bool operator!=(const ArrayIterator& other) const { return ptr_ != other.ptr_; }
};
```

## C++20 三路比较进阶

**自定义三路比较逻辑**
`std::strong_ordering operator<=>(const <Type>& <other>) const { ... }`
```cpp
#include <compare>
class Version {
    int major_, minor_, patch_;
public:
    Version(int maj, int min, int pat) : major_(maj), minor_(min), patch_(pat) {}
    std::strong_ordering operator<=>(const Version& other) const {
        if (auto cmp = major_ <=> other.major_; cmp != 0) return cmp;
        if (auto cmp = minor_ <=> other.minor_; cmp != 0) return cmp;
        return patch_ <=> other.patch_;
    }
};
Version v1(1, 2, 3), v2(1, 2, 4);
bool less = v1 < v2;     // true
bool equal = v1 == v2;   // false
```

## 自定义字面量

**字面量运算符**
`constexpr <type> operator""_<suffix>(unsigned long long <value>) { ... }`
```cpp
#include <chrono>
// 自定义字面量运算符
constexpr std::chrono::milliseconds operator""_ms(unsigned long long value) {
    return std::chrono::milliseconds(value);
}
constexpr std::chrono::seconds operator""_s(unsigned long long value) {
    return std::chrono::seconds(value);
}
// 使用
auto timeout = 100_ms;
auto delay = 5_s;
```
