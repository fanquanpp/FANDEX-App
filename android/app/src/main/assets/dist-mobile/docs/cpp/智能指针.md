# 智能指针

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## unique_ptr

**创建 unique_ptr**
`std::unique_ptr<<Type>> <ptr> = std::make_unique<<Type>>(<args>);`
```cpp
#include <memory>
// 创建 unique_ptr
auto p1 = std::make_unique<int>(42);
auto p2 = std::make_unique<std::string>("Hello");
// 直接构造
std::unique_ptr<int> p3(new int(100));
// 数组版本
auto arr = std::make_unique<int[]>(10);
arr[0] = 1;
```

---

**访问与解引用**
`*<ptr>` 或 `<ptr>-><member>`
```cpp
auto p = std::make_unique<int>(42);
std::cout << *p << std::endl;    // 解引用
auto obj = std::make_unique<std::string>("Hello");
std::cout << obj->size() << std::endl;    // 成员访问
```

---

**所有权转移**
`std::move(<ptr>)`
```cpp
auto p1 = std::make_unique<int>(42);
// auto p2 = p1;    // 错误：不能拷贝
auto p2 = std::move(p1);    // 转移所有权
// p1 现在为空
if (!p1) {
    std::cout << "p1 is null" << std::endl;
}
std::cout << *p2 << std::endl;    // 42
```

---

**release 与 reset**
`<ptr>.release() | <ptr>.reset(<new_ptr>)`
```cpp
auto p = std::make_unique<int>(42);
// release：释放所有权，返回原始指针
int* raw = p.release();
// p 现在为空，需要手动管理 raw
delete raw;
// reset：替换管理的对象
auto p2 = std::make_unique<int>(10);
p2.reset(new int(20));    // 释放旧对象，管理新对象
p2.reset();               // 释放对象，p2 变为空
```

---

**自定义删除器**
`std::unique_ptr<<Type>, <Deleter>> <ptr>(<raw>, <deleter>);`
```cpp
// 自定义删除器
struct FileDeleter {
    void operator()(FILE* f) const {
        if (f) fclose(f);
    }
};
std::unique_ptr<FILE, FileDeleter> file_ptr(fopen("test.txt", "r"));
// 使用 lambda 删除器
auto deleter = [](int* p) { delete[] p; };
std::unique_ptr<int, decltype(deleter)> array_ptr(new int[10], deleter);
```

## shared_ptr

**创建 shared_ptr**
`std::shared_ptr<<Type>> <ptr> = std::make_shared<<Type>>(<args>);`
```cpp
#include <memory>
// 创建 shared_ptr
auto p1 = std::make_shared<int>(42);
auto p2 = std::make_shared<std::string>("Hello");
// 直接构造（不推荐，两次内存分配）
std::shared_ptr<int> p3(new int(100));
```

---

**共享所有权**
`auto <p2> = <p1>; // 引用计数 +1`
```cpp
auto p1 = std::make_shared<int>(42);
auto p2 = p1;    // 引用计数 +1
auto p3 = p1;    // 引用计数 +1
std::cout << *p1 << " " << *p2 << " " << *p3 << std::endl;    // 42 42 42
std::cout << "Count: " << p1.use_count() << std::endl;        // 3
p2.reset();      // 引用计数 -1
std::cout << "Count: " << p1.use_count() << std::endl;        // 2
```

---

**引用计数查询**
`<ptr>.use_count() | <ptr>.unique()`
```cpp
auto p1 = std::make_shared<int>(42);
std::cout << p1.use_count() << std::endl;    // 1
auto p2 = p1;
std::cout << p1.use_count() << std::endl;    // 2
// C++17 前
// bool is_unique = p1.unique();    // C++17 移除
bool is_unique = (p1.use_count() == 1);
```

---

**自定义删除器**
`std::shared_ptr<<Type>> <ptr>(<raw>, <deleter>);`
```cpp
// 数组删除器
std::shared_ptr<int> arr(new int[10], std::default_delete<int[]>());
// 自定义删除器
std::shared_ptr<FILE> file(fopen("test.txt", "r"), [](FILE* f) {
    if (f) fclose(f);
});
```

---

**make_shared 与直接构造对比**
`std::make_shared<<Type>>(<args>) // 推荐`
```cpp
// make_shared：单次内存分配，高效
auto p1 = std::make_shared<std::string>("Hello");
// 直接构造：两次内存分配
std::shared_ptr<std::string> p2(new std::string("Hello"));
// make_shared 的缺点：内存延迟释放（weak_ptr 延长生命周期）
```

## weak_ptr

**创建 weak_ptr**
`std::weak_ptr<<Type>> <wp> = <shared_ptr>;`
```cpp
auto sp = std::make_shared<int>(42);
std::weak_ptr<int> wp = sp;    // 不增加引用计数
std::cout << sp.use_count() << std::endl;    // 1
```

---

**检查与锁定**
`<wp>.expired() | <wp>.lock()`
```cpp
std::weak_ptr<int> wp;
{
    auto sp = std::make_shared<int>(42);
    wp = sp;
}
// sp 离开作用域，对象被销毁
if (wp.expired()) {
    std::cout << "Object has been destroyed" << std::endl;
}
// 安全获取 shared_ptr
auto sp2 = std::make_shared<int>(100);
std::weak_ptr<int> wp2 = sp2;
if (auto locked = wp2.lock()) {
    std::cout << *locked << std::endl;    // 100
}
```

---

**use_count 与 expired**
`<wp>.use_count() | <wp>.expired()`
```cpp
auto sp = std::make_shared<int>(42);
std::weak_ptr<int> wp = sp;
std::cout << wp.use_count() << std::endl;    // 1
std::cout << wp.expired() << std::endl;      // 0 (false)
sp.reset();
std::cout << wp.expired() << std::endl;      // 1 (true)
```

## enable_shared_from_this

**从对象获取 shared_ptr**
`class <Name> : public std::enable_shared_from_this<<Name>> { ... };`
```cpp
class Node : public std::enable_shared_from_this<Node> {
public:
    std::shared_ptr<Node> getPtr() {
        return shared_from_this();    // 安全获取指向自己的 shared_ptr
    }
};
// 使用
auto node = std::make_shared<Node>();
auto same_node = node->getPtr();
std::cout << (node.get() == same_node.get()) << std::endl;    // 1
```

## 智能指针与数组

**管理动态数组**
`std::unique_ptr<<Type>[]> <ptr> = std::make_unique<<Type>[]>(<size>);`
```cpp
// C++11/14：unique_ptr 管理数组
auto arr = std::make_unique<int[]>(10);
for (int i = 0; i < 10; ++i) {
    arr[i] = i * 2;
}
// shared_ptr 管理数组（需要自定义删除器）
std::shared_ptr<int> sarr(new int[10], std::default_delete<int[]>());
```

## 智能指针转换

**static_pointer_cast**
`std::static_pointer_cast<<TargetType>>(<ptr>)`
```cpp
class Base { public: virtual ~Base() = default; };
class Derived : public Base {};
auto derived = std::make_shared<Derived>();
// 向上转型
std::shared_ptr<Base> base = std::static_pointer_cast<Base>(derived);
// 向下转型
auto back = std::static_pointer_cast<Derived>(base);
```

---

**dynamic_pointer_cast**
`std::dynamic_pointer_cast<<TargetType>>(<ptr>)`
```cpp
class Base { public: virtual ~Base() = default; };
class Derived : public Base { public: void derivedFunc() {} };
std::shared_ptr<Base> base = std::make_shared<Derived>();
// 安全向下转型
auto derived = std::dynamic_pointer_cast<Derived>(base);
if (derived) {
    derived->derivedFunc();
}
```

## 使用场景

**工厂函数返回智能指针**
`std::unique_ptr<<Type>> <create>(<args>) { return std::make_unique<<Type>>(<args>); }`
```cpp
class Product {
public:
    virtual void use() = 0;
    virtual ~Product() = default;
};
class ConcreteProduct : public Product {
public:
    void use() override { std::cout << "Using product" << std::endl; }
};
std::unique_ptr<Product> createProduct() {
    return std::make_unique<ConcreteProduct>();
}
// 使用
auto product = createProduct();
product->use();
```

---

**容器存储智能指针**
`std::vector<std::unique_ptr<<Type>>> <vec>;`
```cpp
class Animal {
public:
    virtual void makeSound() = 0;
    virtual ~Animal() = default;
};
class Dog : public Animal {
public:
    void makeSound() override { std::cout << "Woof!" << std::endl; }
};
class Cat : public Animal {
public:
    void makeSound() override { std::cout << "Meow!" << std::endl; }
};
std::vector<std::unique_ptr<Animal>> animals;
animals.push_back(std::make_unique<Dog>());
animals.push_back(std::make_unique<Cat>());
for (const auto& animal : animals) {
    animal->makeSound();
}
```
