# 基础语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 数据类型

**基本数据类型声明**
`<signed|unsigned> <char|short|int|long|long long> <var>;`
```cpp
char c = 'A';              // 1 字节字符
short s = 32767;           // 2 字节短整数
int x = 10;                // 4 字节整数
long l = 1000000L;         // 4 或 8 字节长整数
long long ll = 10000000000LL;  // 8 字节长长整数
float f = 3.14f;           // 4 字节单精度浮点
double d = 3.1415926535;   // 8 字节双精度浮点
bool is_valid = true;      // 1 字节布尔值
```

---

**类型修饰符**
`<const|volatile|constexpr|auto|decltype> <type> <var>;`
```cpp
const int MAX = 100;       // 常量
volatile int flag = 0;     // 易变类型
constexpr int SIZE = 10;   // 编译期常量（C++11）
auto i = 10;               // 自动类型推断（C++11）
decltype(i) j = 20;        // 类型推导（C++11）
```

## 类型转换

**隐式类型转换**
`<type> <var> = <other_type_var>;`
```cpp
int i = 10;
double d = i;    // int 隐式转换为 double
char c = 'A';
i = c;           // char 隐式转换为 int
```

---

**显式类型转换（C++ 风格）**
`static_cast<<target_type>>(<expr>) | dynamic_cast<<target_type>>(<expr>) | const_cast<<target_type>>(<expr>) | reinterpret_cast<<target_type>>(<expr>)`
```cpp
// C 风格转换
double d = 3.14;
int i = (int)d;    // 截断小数部分
// static_cast：静态类型转换
i = static_cast<int>(d);
// dynamic_cast：动态类型转换（用于多态）
Base* base = new Derived();
Derived* derived = dynamic_cast<Derived*>(base);
// const_cast：移除 const 修饰
const int& const_ref = i;
int& ref = const_cast<int&>(const_ref);
// reinterpret_cast：重新解释类型
int* p = &i;
long addr = reinterpret_cast<long>(p);
```

## 条件判断

**if-else 语句**
`if (<condition>) { ... } else if (<condition>) { ... } else { ... }`
```cpp
int score = 85;
if (score >= 90) {
    std::cout << "优秀" << std::endl;
} else if (score >= 80) {
    std::cout << "良好" << std::endl;
} else if (score >= 60) {
    std::cout << "及格" << std::endl;
} else {
    std::cout << "不及格" << std::endl;
}
```

---

**switch 语句**
`switch (<expr>) { case <val>: ... break; [default: ...] }`
```cpp
int day = 3;
switch (day) {
    case 1: std::cout << "星期一" << std::endl; break;
    case 2: std::cout << "星期二" << std::endl; break;
    case 3: std::cout << "星期三" << std::endl; break;
    case 6:
    case 7: std::cout << "周末" << std::endl; break;
    default: std::cout << "无效的日期" << std::endl;
}
```

## 循环结构

**for 循环**
`for (<init>; <condition>; <update>) { ... }`
```cpp
// 传统 for 循环
for (int i = 0; i < 10; ++i) {
    std::cout << i << " ";
}
// 多变量 for 循环
for (int i = 0, j = 10; i < 5 && j > 5; ++i, --j) {
    std::cout << "i: " << i << ", j: " << j << std::endl;
}
```

---

**范围 for 循环（C++11）**
`for (<type> <var> : <container>) { ... }`
```cpp
std::vector<int> numbers = {1, 2, 3, 4, 5};
// 值遍历
for (int num : numbers) {
    std::cout << num << " ";
}
// auto 推导
for (auto num : numbers) {
    std::cout << num << " ";
}
// const 引用遍历（避免复制）
for (const auto& num : numbers) {
    std::cout << num << " ";
}
// 引用遍历（可修改元素）
for (auto& num : numbers) {
    num *= 2;    // 每个元素乘以 2
}
```

---

**while 循环**
`while (<condition>) { ... }`
```cpp
int i = 0;
while (i < 10) {
    std::cout << i << " ";
    ++i;
}
```

---

**do-while 循环**
`do { ... } while (<condition>);`
```cpp
int i = 0;
do {
    std::cout << i << " ";
    ++i;
} while (i < 10);
```

## 跳转语句

**break 语句**
`break;`
```cpp
for (int i = 0; i < 10; ++i) {
    if (i == 5) {
        break;    // 跳出循环
    }
    std::cout << i << " ";
}
```

---

**continue 语句**
`continue;`
```cpp
for (int i = 0; i < 10; ++i) {
    if (i % 2 == 0) {
        continue;    // 跳过偶数
    }
    std::cout << i << " ";
}
```

---

**return 语句**
`return <expr>;` 或 `return;`
```cpp
int add(int a, int b) {
    return a + b;    // 返回值并结束函数
}
void print_hello() {
    std::cout << "Hello!" << std::endl;
    return;    // 可选
}
```

## 标准输入输出

**标准输出**
`std::cout << <expr> [<< <expr2>] [<< std::endl];`
```cpp
#include <iostream>
int main() {
    std::cout << "Hello, C++!" << std::endl;
    int x = 10;
    double y = 3.14;
    std::cout << "x = " << x << ", y = " << y << std::endl;
    // 使用 \n 仅换行（不刷新缓冲区）
    std::cout << "Line 1\nLine 2" << std::endl;
    // 输出布尔值
    bool flag = true;
    std::cout << std::boolalpha << "Flag: " << flag << std::endl;
    return 0;
}
```

---

**标准输入**
`std::cin >> <var>;`
```cpp
#include <iostream>
#include <string>
int main() {
    int x;
    std::cin >> x;    // 输入整数
    std::string name;
    std::cin >> name;    // 输入字符串（遇空格停止）
    // 输入一行字符串
    std::string line;
    std::getline(std::cin, line);    // 读取一行（包括空格）
    return 0;
}
```

---

**输入验证**
`if (std::cin >> <var>) { ... } else { std::cin.clear(); std::cin.ignore(...); }`
```cpp
int age;
while (true) {
    std::cout << "Enter your age: ";
    if (std::cin >> age) {
        break;    // 输入成功
    } else {
        std::cin.clear();    // 清除错误状态
        std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
        std::cout << "Invalid input." << std::endl;
    }
}
```

## 格式化输出

**使用 iomanip 格式化**
`std::setw(<n>) | std::setprecision(<n>) | std::fixed | std::hex | std::oct`
```cpp
#include <iostream>
#include <iomanip>
int main() {
    // 设置输出宽度
    std::cout << std::setw(10) << "Name" << std::setw(10) << "Age" << std::endl;
    // 设置填充字符
    std::cout << std::setw(10) << std::setfill('*') << "Hello" << std::endl;
    // 设置精度
    double pi = 3.1415926535;
    std::cout << std::fixed << std::setprecision(2) << pi << std::endl;
    // 十六进制输出
    int x = 255;
    std::cout << "Hex: " << std::hex << x << std::endl;
    // 八进制输出
    std::cout << "Octal: " << std::oct << x << std::endl;
    // 重置为十进制
    std::cout << "Decimal: " << std::dec << x << std::endl;
    return 0;
}
```

## 文件输入输出

**文件写入**
`std::ofstream <out>("<filename>"); <out> << <data>;`
```cpp
#include <fstream>
std::ofstream outfile("example.txt");
if (outfile.is_open()) {
    outfile << "Hello, File!" << std::endl;
    outfile << "Number: " << 42 << std::endl;
    outfile.close();
}
```

---

**文件读取**
`std::ifstream <in>("<filename>"); while (std::getline(<in>, <line>)) { ... }`
```cpp
std::ifstream infile("example.txt");
if (infile.is_open()) {
    std::string line;
    while (std::getline(infile, line)) {
        std::cout << line << std::endl;
    }
    infile.close();
}
```

## 命名空间

**命名空间定义**
`namespace <Name> { <members> }`
```cpp
namespace MyNamespace {
    int add(int a, int b) { return a + b; }
    namespace Nested {
        int multiply(int a, int b) { return a * b; }
    }
}
// 使用
int result1 = MyNamespace::add(5, 3);
int result2 = MyNamespace::Nested::multiply(5, 3);
```

```

---

**using 指令**
`using namespace <name>;`

---

**命名空间别名**

```

## 作用域

**块作用域**
`{ <type> <var>; ... }`
```cpp
int main() {
    int global_var = 10;    // 函数作用域
    if (true) {
        int local_var = 20;    // 块作用域
        std::cout << local_var << std::endl;
    }
    // local_var 在此处不可访问
    return 0;
}
```

---

**类作用域**
`class <Name> { public: <type> <member>; private: <type> <member>; };`
```cpp
class MyClass {
public:
    int public_var;    // 类作用域，外部可访问
private:
    int private_var;    // 类作用域，仅类内访问
};
int main() {
    MyClass obj;
    obj.public_var = 10;    // 可以访问
    return 0;
}
```
