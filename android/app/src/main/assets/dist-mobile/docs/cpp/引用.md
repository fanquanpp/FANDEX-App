# 引用

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 引用基础

**引用声明与初始化**
`<type>& <ref> = <var>;`
```cpp
int x = 10;
int& r = x;    // 声明对 x 的引用
r = 20;        // 修改 r 就是修改 x
std::cout << x;    // 输出 20
```

---

## 引用与函数

**引用作为函数参数（传引用）**
`void <func>(<type>& <param>) { ... }`
```cpp
// 交换两个变量的值
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}
int main() {
    int x = 10, y = 20;
    swap(x, y);
    std::cout << x << " " << y;    // 输出 20 10
    return 0;
}
```

---

**引用作为函数返回值**
`<type>& <func>(<params>) { ... return <var>; }`
```cpp
// 返回数组元素的引用，可用于修改
int& get_element(int arr[], int index) {
    return arr[index];
}
int main() {
    int arr[] = {1, 2, 3, 4, 5};
    get_element(arr, 2) = 10;    // 修改数组元素
    std::cout << arr[2];         // 输出 10
    return 0;
}
```

---

## 常量引用

**常量引用绑定到常量或临时值**
`const <type>& <ref> = <value>;`
```cpp
// 常量引用可以绑定到常量或临时值
const int& cr = 10;    // 合法，临时值会被延长生命周期
// 常量引用可以绑定到不同类型（隐式转换）
double d = 3.14;
const int& cr2 = d;    // 合法，d 会被转换为 int
// 非常量引用不能绑定到临时值
// int& r = 10;    // 错误
```

---

## 右值引用

**右值引用声明**
`<type>&& <ref> = <rvalue>;`
```cpp
// 右值引用：只能绑定右值
int&& rref = 42;
// 常量左值引用：可以绑定右值
const int& cref = 42;
// 使用 std::move 将左值转换为右值
int x = 10;
int&& r = std::move(x);    // x 变为有效但未定义状态
```

---

## 引用与指针

**引用与指针对比**
`<type>& <ref> = <var>; | <type>* <ptr> = &<var>;`
```cpp
int x = 10;
// 引用：必须初始化，不可重绑定，不能为空
int& ref = x;
ref = 20;    // 修改 x
// 指针：可以为空，可以改变指向
int* ptr = &x;
*ptr = 30;       // 修改 x
int y = 40;
ptr = &y;        // 指针重新指向 y
```
