# 面向对象基础

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 类定义

**基本类定义**
`class <ClassName> { public: ... protected: ... private: ... };`
```cpp
class Person {
private:
    std::string name_;    // 私有成员变量
    int age_;
public:
    // 构造函数
    Person(const std::string& name, int age) : name_(name), age_(age) {}
    // 成员函数
    void display() const {
        std::cout << "Name: " << name_ << ", Age: " << age_ << std::endl;
    }
    // Getter
    const std::string& getName() const { return name_; }
    int getAge() const { return age_; }
    // Setter
    void setAge(int age) { age_ = age; }
};
```

---

**struct 与 class 对比**
`struct <Name> { <members> };`
```cpp
// struct 默认访问权限为 public
struct Point {
    double x;    // 默认 public
    double y;
    Point(double x = 0, double y = 0) : x(x), y(y) {}
};
// class 默认访问权限为 private
class Rectangle {
    double width_, height_;    // 默认 private
public:
    Rectangle(double w, double h) : width_(w), height_(h) {}
};
```

## 构造函数

**默认构造函数**
`<ClassName>() { ... }` 或 `<ClassName>() = default;`
```cpp
class MyClass {
public:
    MyClass() { std::cout << "Default constructor" << std::endl; }
};
// 使用 = default 生成默认构造函数
class DefaultClass {
public:
    DefaultClass() = default;
};
```

---

**参数化构造函数**
`<ClassName>(<params>) : <member>(<value>), ... { ... }`
```cpp
class Student {
    std::string name_;
    int id_;
    double gpa_;
public:
    // 成员初始化列表
    Student(const std::string& name, int id, double gpa)
        : name_(name), id_(id), gpa_(gpa) {}
};
```

---

**委托构造函数（C++11）**
`<ClassName>(<params>) : <ClassName>(<other_params>) { ... }`
```cpp
class Student {
    std::string name_;
    int id_;
    double gpa_;
public:
    // 主构造函数
    Student(const std::string& name, int id, double gpa)
        : name_(name), id_(id), gpa_(gpa) {}
    // 委托构造函数
    Student(const std::string& name, int id)
        : Student(name, id, 0.0) {}
    Student() : Student("Unknown", 0, 0.0) {}
};
```

---

**explicit 关键字**
`explicit <ClassName>(<params>) { ... }`
```cpp
class Distance {
    double meters_;
public:
    // 防止隐式转换
    explicit Distance(double m) : meters_(m) {}
};
Distance d1(10.5);           // 正确：直接初始化
// Distance d2 = 10.5;      // 错误：explicit 防止隐式转换
Distance d3 = Distance(10.5); // 正确：显式构造
```

## 析构函数

**析构函数定义**
`~<ClassName>() { ... }` 或 `~<ClassName>() = default;`
```cpp
class FileHandler {
    FILE* file_;
public:
    FileHandler(const char* filename) {
        file_ = fopen(filename, "r");
    }
    // 析构函数：释放资源
    ~FileHandler() {
        if (file_) {
            fclose(file_);
        }
    }
};
```

## 拷贝构造与拷贝赋值

**拷贝构造函数**
`<ClassName>(const <ClassName>& <other>) { ... }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    Buffer(size_t size) : data_(new int[size]), size_(size) {}
    // 拷贝构造函数：深拷贝
    Buffer(const Buffer& other) : data_(new int[other.size_]), size_(other.size_) {
        std::copy(other.data_, other.data_ + size_, data_);
    }
    ~Buffer() { delete[] data_; }
};
```

---

**拷贝赋值运算符**
`<ClassName>& operator=(const <ClassName>& <other>) { ... return *this; }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    // 拷贝赋值运算符
    Buffer& operator=(const Buffer& other) {
        if (this != &other) {    // 防止自赋值
            delete[] data_;
            size_ = other.size_;
            data_ = new int[size_];
            std::copy(other.data_, other.data_ + size_, data_);
        }
        return *this;
    }
};
```

---

**禁用拷贝**
`<ClassName>(const <ClassName>&) = delete; <ClassName>& operator=(const <ClassName>&) = delete;`
```cpp
class NonCopyable {
public:
    NonCopyable() = default;
    // C++11：禁用拷贝构造和拷贝赋值
    NonCopyable(const NonCopyable&) = delete;
    NonCopyable& operator=(const NonCopyable&) = delete;
};
```

## 移动构造与移动赋值

**移动构造函数**
`<ClassName>(<ClassName>&& <other>) noexcept { ... }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    // 移动构造函数
    Buffer(Buffer&& other) noexcept : data_(other.data_), size_(other.size_) {
        other.data_ = nullptr;
        other.size_ = 0;
    }
};
```

---

**移动赋值运算符**
`<ClassName>& operator=(<ClassName>&& <other>) noexcept { ... return *this; }`
```cpp
class Buffer {
    int* data_;
    size_t size_;
public:
    // 移动赋值运算符
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data_;
            data_ = other.data_;
            size_ = other.size_;
            other.data_ = nullptr;
            other.size_ = 0;
        }
        return *this;
    }
};
```

## 三/五/零法则

**零法则：使用智能指针管理资源**
`class <Name> { std::unique_ptr<<Type>> <ptr>; ... };`
```cpp
#include <memory>
class ResourceManager {
    std::unique_ptr<int[]> data_;    // 智能指针自动管理
public:
    ResourceManager(size_t size) : data_(std::make_unique<int[]>(size)) {}
    // 无需自定义析构、拷贝、移动
};
```

## 继承

**基本继承**
`class <Derived> : <access> <Base> { ... };`
```cpp
class Animal {
protected:
    std::string name_;
public:
    Animal(const std::string& name) : name_(name) {}
    virtual void makeSound() const {
        std::cout << "Some sound" << std::endl;
    }
};
class Dog : public Animal {
public:
    Dog(const std::string& name) : Animal(name) {}
    void makeSound() const override {
        std::cout << "Woof!" << std::endl;
    }
};
```

---

**多重继承**
`class <Derived> : <access> <Base1>, <access> <Base2> { ... };`
```cpp
class Drawable {
public:
    virtual void draw() const = 0;
};
class Serializable {
public:
    virtual std::string serialize() const = 0;
};
class Shape : public Drawable, public Serializable {
public:
    void draw() const override { std::cout << "Drawing shape" << std::endl; }
    std::string serialize() const override { return "Shape"; }
};
```

---

**虚继承解决菱形继承**
`class <Derived> : virtual <access> <Base> { ... };`
```cpp
class Base { public: int data; };
class Derived1 : virtual public Base {};
class Derived2 : virtual public Base {};
class Final : public Derived1, public Derived2 {
public:
    Final() { Base::data = 10; }    // 无歧义
};
```

## 多态

**虚函数**
`virtual <return_type> <func>(<params>) [const] { ... }`
```cpp
class Shape {
public:
    virtual double area() const {
        return 0.0;
    }
    virtual void draw() const {
        std::cout << "Drawing shape" << std::endl;
    }
};
```

---

**纯虚函数与抽象类**
`virtual <return_type> <func>(<params>) [const] = 0;`
```cpp
class Shape {
public:
    // 纯虚函数：使类成为抽象类
    virtual double area() const = 0;
    virtual void draw() const = 0;
    virtual ~Shape() = default;
};
// 派生类必须实现纯虚函数才能实例化
class Circle : public Shape {
    double radius_;
public:
    Circle(double r) : radius_(r) {}
    double area() const override {
        return 3.14159 * radius_ * radius_;
    }
    void draw() const override {
        std::cout << "Drawing circle" << std::endl;
    }
};
```

---

**override 与 final**
`virtual <return_type> <func>(<params>) [const] override [final] { ... }`
```cpp
class Base {
public:
    virtual void func() {}
    virtual void final_func() {}
};
class Derived : public Base {
public:
    void func() override {}    // 确保正确重写
    void final_func() override final {}    // 禁止进一步重写
};
// class Further : public Derived {
//     void final_func() override {}    // 错误：final 禁止重写
// };
```

## 静态成员

**静态成员变量**
`static <type> <var>;`
```cpp
class Counter {
public:
    static int count_;    // 静态成员变量声明
    Counter() { ++count_; }
    ~Counter() { --count_; }
    static int getCount() { return count_; }
};
// 静态成员变量定义（在类外）
int Counter::count_ = 0;
// 使用
Counter c1, c2;
std::cout << Counter::getCount();    // 输出 2
```

---

**静态成员函数**
`static <return_type> <func>(<params>) { ... }`
```cpp
class MathUtils {
public:
    static double square(double x) { return x * x; }
    static double cube(double x) { return x * x * x; }
};
// 使用
double s = MathUtils::square(5.0);    // 25.0
```

## 友元

**友元函数**
`friend <return_type> <func>(<params>);`
```cpp
class Vector {
    double x_, y_;
public:
    Vector(double x, double y) : x_(x), y_(y) {}
    // 友元函数可以访问私有成员
    friend double dotProduct(const Vector& a, const Vector& b);
};
double dotProduct(const Vector& a, const Vector& b) {
    return a.x_ * b.x_ + a.y_ * b.y_;
}
```

---

**友元类**
`friend class <ClassName>;`
```cpp
class Engine {
    friend class Car;    // Car 可以访问 Engine 的私有成员
private:
    int horsepower_ = 200;
};
class Car {
public:
    void displayEngine(const Engine& e) {
        std::cout << "HP: " << e.horsepower_ << std::endl;
    }
};
```
