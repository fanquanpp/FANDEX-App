# JavaScript 模块化

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## ES 模块

**命名导出**：导出多个变量
`export <声明>;`
```javascript
// math.js
export const PI = 3.14159;
export function add(a, b) {
  return a + b;
}
export function subtract(a, b) {
  return a - b;
}
```

---

**命名导入**：导入指定变量
`import { <名称1>, <名称2> } from '<模块>';`
```javascript
// main.js
import { PI, add } from './math.js';
console.log(PI);      // 3.14159
console.log(add(2, 3));  // 5
```

---

**默认导出**：每个模块一个默认导出
`export default <值>;`
```javascript
// calculator.js
export default class Calculator {
  add(a, b) { return a + b; }
}

// 或导出函数
// export default function add(a, b) { return a + b; }
```

---

**默认导入**：导入默认导出
`import <名称> from '<模块>';`
```javascript
// main.js
import Calculator from './calculator.js';
const calc = new Calculator();
console.log(calc.add(2, 3));  // 5
```

---

**混合导入**：同时导入默认和命名
`import <默认名>, { <命名1>, <命名2> } from '<模块>';`
```javascript
// module.js
export const PI = 3.14159;
export function helper() { return 'helper'; }
export default function main() { return 'main'; }

// main.js
import main, { PI, helper } from './module.js';
console.log(main());      // 'main'
console.log(PI);          // 3.14159
console.log(helper());    // 'helper'
```

---

**重命名导入**：使用 as 关键字
`import { <原名> as <别名> } from '<模块>';`
```javascript
import { add as plus, PI as pi } from './math.js';
console.log(plus(2, 3));  // 5
console.log(pi);          // 3.14159
```

---

**重命名导出**：使用 as 关键字
`export { <原名> as <别名> };`
```javascript
// math.js
const add = (a, b) => a + b;
const subtract = (a, b) => a - b;
export { add as plus, subtract as minus };
```

---

**导入全部**：使用 * 导入所有
`import * as <命名空间> from '<模块>';`
```javascript
import * as math from './math.js';
console.log(math.PI);        // 3.14159
console.log(math.add(2, 3));  // 5
```

---

**动态导入**：按需加载模块
`const <模块> = await import('<模块>');`
```javascript
async function loadModule() {
  const module = await import('./math.js');
  console.log(module.add(2, 3));  // 5
}
loadModule();

// 条件加载
if (condition) {
  const { default: Calculator } = await import('./calculator.js');
  const calc = new Calculator();
}
```

## 聚合导出

**聚合模块**：将多个模块合并导出
`export { <名称> } from '<模块>';`
```javascript
// index.js - 聚合多个模块
export { add, subtract } from './math.js';
export { default as Calculator } from './calculator.js';
export { PI } from './constants.js';

// 使用者只需导入一个文件
// import { add, Calculator, PI } from './index.js';
```

---

**聚合默认导出**：重命名默认导出
`export { default as <名称> } from '<模块>';`
```javascript
export { default as Calculator } from './calculator.js';
```

## 模块路径

**相对路径**：使用 ./ 或 ../
`import <模块> from './<文件>';`
```javascript
import { add } from './math.js';        // 同级目录
import { helper } from '../utils.js';    // 上级目录
import { config } from './config/index.js';  // 子目录
```

---

**绝对路径**：使用包名
`import <模块> from '<包名>';`
```javascript
import React from 'react';
import { useState } from 'react';
import axios from 'axios';
```

---

**省略扩展名**：某些打包工具支持
`import <模块> from './<文件>';`
```javascript
// 在 Node.js 或打包工具中
import { add } from './math';  // 省略 .js
import { add } from './math.js';  // 完整路径
```

## CommonJS 模块

**module.exports**：导出模块
`module.exports = <值>;`
```javascript
// math.js
const add = (a, b) => a + b;
const subtract = (a, b) => a - b;
module.exports = { add, subtract };

// 或导出单个值
module.exports = function add(a, b) {
  return a + b;
};
```

---

**require**：导入模块
`const <变量> = require('<模块>');`
```javascript
const { add, subtract } = require('./math');
console.log(add(2, 3));  // 5

// 导入单个值
const add = require('./math');
console.log(add(2, 3));  // 5
```

---

**exports 简写**：添加属性到 exports
`exports.<名称> = <值>;`
```javascript
// math.js
exports.add = (a, b) => a + b;
exports.subtract = (a, b) => a - b;
exports.PI = 3.14159;

// 使用
const math = require('./math');
console.log(math.add(2, 3));  // 5
```

## 模块模式

**立即执行函数表达式（IIFE）**：创建私有作用域
`const <模块名> = (function() { ... return { ... }; })();`
```javascript
const counter = (function () {
  let count = 0;  // 私有变量
  return {
    increment() { return ++count; },
    decrement() { return --count; },
    getCount() { return count; },
  };
})();
console.log(counter.increment());  // 1
console.log(counter.getCount());    // 1
```

---

**命名空间模式**：使用对象组织模块
`const <命名空间> = { <属性>: <值>, <方法>() { ... } };`
```javascript
const App = {
  config: {
    apiUrl: 'https://api.example.com',
    timeout: 5000,
  },
  utils: {
    formatDate(date) { return date.toISOString(); },
    parseJSON(str) { return JSON.parse(str); },
  },
  api: {
    async get(endpoint) {
      const response = await fetch(`${this.config.apiUrl}${endpoint}`);
      return response.json();
    },
  },
};
```

## 动态导入

**按需加载**：使用 import() 函数
`import('<模块>').then(<模块> => { ... });`
```javascript
// 按钮点击时加载模块
document.getElementById('loadModule').addEventListener('click', async () => {
  const module = await import('./heavyModule.js');
  module.doSomething();
});
```

---

**条件加载**：根据条件加载不同模块
`const <模块> = await import(<条件> ? '<模块1>' : '<模块2>');`
```javascript
async function loadAdapter() {
  const adapter = await import(
    process.env.NODE_ENV === 'production'
      ? './productionAdapter.js'
      : './developmentAdapter.js'
  );
  return adapter;
}
```

## 循环依赖处理

**循环依赖**：模块间相互引用
`// moduleA.js: import { b } from './moduleB.js';`
```javascript
// moduleA.js
import { b } from './moduleB.js';
export const a = () => b() + 1;

// moduleB.js
import { a } from './moduleA.js';
export const b = () => a() + 1;

// 解决方案：重构代码，提取共享逻辑到第三个模块
```

---

**重构循环依赖**：提取共享模块
`// shared.js: export const shared = ...;`
```javascript
// shared.js - 提取共享逻辑
export const sharedValue = 42;

// moduleA.js
import { sharedValue } from './shared.js';
export const a = () => sharedValue + 1;

// moduleB.js
import { sharedValue } from './shared.js';
export const b = () => sharedValue + 1;
```

## 模块导出最佳实践

**单一职责**：每个模块只导出相关功能
`export <相关功能>;`
```javascript
// userUtils.js - 只导出用户相关功能
export function createUser(name) { ... }
export function updateUser(user) { ... }
export function deleteUser(id) { ... }

// postUtils.js - 只导出文章相关功能
export function createPost(title) { ... }
export function updatePost(post) { ... }
```

---

**使用 index.js 聚合**：统一入口
`export * from '<模块>';`
```javascript
// utils/index.js
export * from './math.js';
export * from './string.js';
export * from './array.js';

// 使用者只需导入一个文件
import { add, formatString, uniqueArray } from './utils';
```
