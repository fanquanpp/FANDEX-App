# JavaScript 异步编程

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 同步与异步

**同步代码**：按顺序执行
`console.log(<消息>);`
```javascript
console.log('开始');
console.log('执行同步任务');
console.log('结束');
// 输出顺序: 开始 -> 执行同步任务 -> 结束
```

---

**异步代码**：非阻塞执行
`setTimeout(() => { <操作> }, <延迟>);`
```javascript
console.log('开始');
setTimeout(() => {
  console.log('执行异步任务');
}, 1000);
console.log('结束');
// 输出顺序: 开始 -> 结束 -> 执行异步任务（1秒后）
```

---

## 事件循环

**事件循环示例**：微任务优先于宏任务
`Promise.resolve().then(() => { <操作> });`
```javascript
console.log('1. 同步任务开始');
setTimeout(() => {
  console.log('4. 宏任务执行');
}, 0);
Promise.resolve().then(() => {
  console.log('3. 微任务执行');
});
console.log('2. 同步任务结束');
// 输出顺序: 1 -> 2 -> 3 -> 4
```

## 回调函数

**基本回调**：异步操作完成后调用
`function <函数名>(<参数>, <回调>) { ... <回调>(<结果>); }`
```javascript
function fetchData(callback) {
  setTimeout(() => {
    const data = { name: '张三', age: 30 };
    callback(data);
  }, 1000);
}
fetchData((data) => {
  console.log('获取到数据:', data);
});
```

---

**回调地狱**：嵌套回调
`<函数1>((<结果1>) => { <函数2>((<结果2>) => { <函数3>(...); }); });`
```javascript
fetchUser((user) => {
  fetchPosts(user.id, (posts) => {
    fetchComments(posts[0].id, (comments) => {
      console.log('评论:', comments);
    });
  });
});
```

## Promise

**创建 Promise**：使用 new Promise
`const <变量名> = new Promise((<resolve>, <reject>) => { ... });`
```javascript
const promise = new Promise((resolve, reject) => {
  setTimeout(() => {
    const random = Math.random();
    if (random > 0.5) {
      resolve('操作成功');
    } else {
      reject('操作失败');
    }
  }, 1000);
});
```

---

**then()**：处理成功状态
`<promise>.then((<结果>) => { <处理> })`
```javascript
promise.then((result) => {
  console.log('成功:', result);
});
```

---

**catch()**：处理失败状态
`<promise>.catch((<错误>) => { <处理> })`
```javascript
promise.catch((error) => {
  console.log('失败:', error);
});
```

---

**finally()**：无论成功失败都执行
`<promise>.finally(() => { <清理> })`
```javascript
promise
  .then((result) => { console.log('成功:', result); })
  .catch((error) => { console.log('失败:', error); })
  .finally(() => { console.log('操作完成'); });
```

---

**链式调用**：Promise 链
`<promise>.then(<回调1>).then(<回调2>).catch(<回调>)`
```javascript
const promise = Promise.resolve('成功');
promise
  .then((result) => {
    console.log('处理成功:', result);
    return '处理后的结果';
  })
  .then((result) => {
    console.log('再次处理:', result);
  });
```

## Promise 静态方法

**Promise.all()**：等待所有 Promise 完成
`Promise.all([<promise1>, <promise2>, ...])`
```javascript
const promise1 = Promise.resolve(1);
const promise2 = Promise.resolve(2);
const promise3 = Promise.resolve(3);
Promise.all([promise1, promise2, promise3]).then((results) => {
  console.log('所有 Promise 完成:', results);  // [1, 2, 3]
});
```

---

**Promise.race()**：返回第一个完成的 Promise
`Promise.race([<promise1>, <promise2>, ...])`
```javascript
const promise1 = new Promise((resolve) => setTimeout(() => resolve('第一个'), 1000));
const promise2 = new Promise((resolve) => setTimeout(() => resolve('第二个'), 500));
Promise.race([promise1, promise2]).then((result) => {
  console.log('第一个完成的 Promise:', result);  // 第二个
});
```

---

**Promise.resolve()**：返回已解决的 Promise
`Promise.resolve(<值>)`
```javascript
const promise = Promise.resolve('成功');
promise.then((result) => {
  console.log(result);  // 成功
});
```

---

**Promise.reject()**：返回已拒绝的 Promise
`Promise.reject(<原因>)`
```javascript
const promise = Promise.reject('失败');
promise.catch((error) => {
  console.log(error);  // 失败
});
```

## async/await

**async 函数**：声明异步函数
`async function <函数名>() { ... }`
```javascript
async function fetchData() {
  try {
    const response = await new Promise((resolve) => {
      setTimeout(() => {
        resolve({ data: { name: '张三', age: 30 } });
      }, 1000);
    });
    console.log('获取到数据:', response.data);
    return response.data;
  } catch (error) {
    console.error('错误:', error);
    throw error;
  }
}
```

---

**await 表达式**：等待 Promise 完成
`const <结果> = await <promise>;`
```javascript
async function getData() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  console.log(data);
  return data;
}
```

---

**错误处理**：使用 try-catch
`try { <await 操作> } catch (<错误>) { <处理> }`
```javascript
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    if (!response.ok) {
      throw new Error('网络请求失败');
    }
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('错误:', error);
    throw error;
  }
}
```

```javascript

clearInterval(intervalId);
```

## 网络请求

**Fetch API GET 请求**：获取数据
`fetch(<URL>).then(<回调>).catch(<回调>)`
```javascript
fetch('https://api.example.com/data')
  .then((response) => {
    if (!response.ok) {
      throw new Error('网络请求失败');
    }
    return response.json();
  })
  .then((data) => {
    console.log('获取到数据:', data);
  })
  .catch((error) => {
    console.error('错误:', error);
  });
```

---

**Fetch API POST 请求**：提交数据
`fetch(<URL>, { method: 'POST', body: <数据> })`
```javascript
fetch('https://api.example.com/data', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ name: '张三', age: 30 }),
})
  .then((response) => response.json())
  .then((data) => {
    console.log('提交成功:', data);
  });
```

---

**async/await 请求**：使用 async/await
`const <响应> = await fetch(<URL>);`
```javascript
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
```

## 并行执行

**Promise.all 并行**：并行执行多个异步操作
`const [结果1, 结果2] = await Promise.all([<promise1>, <promise2>]);`
```javascript
async function fetchData() {
  const [user, posts, comments] = await Promise.all([
    fetchUser(),
    fetchPosts(),
    fetchComments(),
  ]);
  return { user, posts, comments };
}
```

---

**避免不必要的 await**：并行获取
`const [结果1, 结果2] = await Promise.all([<promise1>, <promise2>]);`
```javascript
// 不好的做法（串行执行）
async function processData() {
  const data1 = await fetchData1();
  const data2 = await fetchData2();
  return { data1, data2 };
}

// 好的做法（并行执行）
async function processData() {
  const promise1 = fetchData1();
  const promise2 = fetchData2();
  const [data1, data2] = await Promise.all([promise1, promise2]);
  return { data1, data2 };
}
```

## 超时控制

**带超时的请求**：使用 Promise.race
`Promise.race([<promise>, <超时promise>])`
```javascript
function fetchWithTimeout(url, timeout = 5000) {
  return Promise.race([
    fetch(url),
    new Promise((_, reject) =>
      setTimeout(() => reject(new Error('Timeout')), timeout)
    ),
  ]);
}
```

## 重试机制

**批量重试**：失败后重试
`async function fetchWithRetry(<URL>, <重试次数>) { ... }`
```javascript
async function fetchWithRetry(url, retries = 3) {
  for (let i = 0; i < retries; i++) {
    try {
      return await fetch(url);
    } catch (error) {
      if (i === retries - 1) throw error;
      await new Promise((r) => setTimeout(r, 1000 * Math.pow(2, i)));
    }
  }
}
```
