# JavaScript 正则表达式

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 正则表达式创建

**字面量创建**：使用斜杠包裹
`const <变量名> = /<模式>/[<标志>];`
```javascript
const pattern = /hello/;
const patternWithFlags = /hello/gi;  // g 全局，i 忽略大小写
```

---

**构造函数创建**：使用 RegExp
`const <变量名> = new RegExp('<模式>'[, '<标志>']);`
```javascript
const pattern = new RegExp('hello');
const patternWithFlags = new RegExp('hello', 'gi');
```

---

## 正则标志

**标志组合**：控制匹配行为
`/<模式>/<标志>`
```javascript
const str = 'Hello World, hello JavaScript';
console.log(str.match(/hello/g));    // ['hello'] - 全局匹配
console.log(str.match(/hello/gi));   // ['Hello', 'hello'] - 全局且忽略大小写
console.log(str.match(/hello/m));    // 多行模式
console.log(str.match(/hello/s));    // dotAll 模式，. 匹配换行符
console.log(str.match(/hello/u));    // Unicode 模式
console.log(str.match(/hello/y));    // 粘连模式
```

---

## 字符类

**字符类**：匹配字符集合
`[<字符集合>]`
```javascript
console.log('cat'.match(/[abc]at/));   // ['cat']
console.log('123'.match(/[0-9]+/));    // ['123']
console.log('abc'.match(/[a-z]+/));    // ['abc']
console.log('ABC'.match(/[A-Z]+/));    // ['ABC']
```

---

**否定字符类**：匹配不在集合中的字符
`[^<字符集合>]`
```javascript
console.log('cat'.match(/[^abc]at/));  // null（c 在集合中）
console.log('bat'.match(/[^abc]at/));  // null（b 在集合中）
console.log('dat'.match(/[^abc]at/));  // ['dat']（d 不在集合中）
```

---

**预定义字符类**：常用字符简写
`<字符类>`
```javascript
console.log('hello123'.match(/\d+/));    // ['123'] - 数字
console.log('hello'.match(/\D+/));       // ['hello'] - 非数字
console.log('hello world'.match(/\w+/g)); // ['hello', 'world'] - 单词字符
console.log('hello world'.match(/\W+/));  // [' '] - 非单词字符
console.log('hello world'.match(/\s+/)); // [' '] - 空白字符
console.log('hello'.match(/\S+/));        // ['hello'] - 非空白字符
```

---

## 量词

**量词**：指定匹配次数
`<字符><量词>`
```javascript
console.log('aaaa'.match(/a*/));     // ['aaaa'] - 0 次或多次
console.log('aaaa'.match(/a+/));     // ['aaaa'] - 1 次或多次
console.log('a'.match(/a?/));         // ['a'] - 0 次或 1 次
console.log('aaa'.match(/a{3}/));     // ['aaa'] - 恰好 3 次
console.log('aaaa'.match(/a{2,3}/));  // ['aaa'] - 2 到 3 次
console.log('aaaaa'.match(/a{2,}/));  // ['aaaaa'] - 至少 2 次
```

---

**贪婪与非贪婪**：量词后加 ?
`<量词>?`
```javascript
// 贪婪匹配（默认）
console.log('<a><b>'.match(/<.+>/));   // ['<a><b>'] - 匹配到最后一个 >

// 非贪婪匹配
console.log('<a><b>'.match(/<.+?>/));  // ['<a>'] - 匹配到第一个 >
```

## 边界匹配

**边界匹配**：匹配字符串边界
`^<模式>` | `<模式>$`
```javascript
console.log('hello'.match(/^hello/));  // ['hello'] - 开头匹配
console.log('hello'.match(/hello$/));  // ['hello'] - 结尾匹配
console.log('hello world'.match(/^hello$/));  // null - 不完全匹配
console.log('hello'.match(/^hello$/));  // ['hello'] - 完全匹配
```

---

**单词边界**：使用 \b
`\b<单词>\b`
```javascript
const str = 'cat category catch';
console.log(str.match(/\bcat\b/g));     // ['cat'] - 只匹配独立的 cat
console.log(str.match(/\bcat/g));        // ['cat', 'cat'] - 匹配以 cat 开头的单词
```

---

## 分组与引用

**捕获组**：使用括号分组
`(<子模式>)`
```javascript
const date = '2024-01-15';
const match = date.match(/(\d{4})-(\d{2})-(\d{2})/);
console.log(match[0]);  // '2024-01-15' - 完整匹配
console.log(match[1]);  // '2024' - 第一个捕获组
console.log(match[2]);  // '01' - 第二个捕获组
console.log(match[3]);  // '15' - 第三个捕获组
```

---

**非捕获组**：使用 ?: 不捕获
`(?:<子模式>)`
```javascript
const str = 'foo123bar';
const match = str.match(/(?:foo)(\d+)/);
console.log(match[0]);  // 'foo123'
console.log(match[1]);  // '123' - 只有捕获组
```

---

**反向引用**：使用 \1, \2 等
`(<子模式>)\1`
```javascript
// 匹配重复的单词
const str = 'hello hello world';
console.log(str.match(/\b(\w+)\s+\1\b/));  // ['hello hello', 'hello']

// 匹配成对的标签
const html = '<div>content</div>';
console.log(html.match(/<(\w+)>(.*?)<\/\1>/));  // ['<div>content</div>', 'div', 'content']
```

---

**命名捕获组**：使用 ?<name>
`(?<<名称><子模式>)`
```javascript
const date = '2024-01-15';
const match = date.match(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/);
console.log(match.groups.year);   // '2024'
console.log(match.groups.month);  // '01'
console.log(match.groups.day);    // '15'
```

## 字符串方法

**match()**：返回匹配结果
`<字符串>.match(<正则>)`
```javascript
const str = 'Hello World, hello JavaScript';
console.log(str.match(/hello/gi));  // ['Hello', 'hello']
console.log(str.match(/xyz/));       // null
```

---

**matchAll()**：返回所有匹配的迭代器
`<字符串>.matchAll(<正则>)`
```javascript
const str = '2024-01-15, 2024-02-20';
const matches = str.matchAll(/(\d{4})-(\d{2})-(\d{2})/g);
for (const match of matches) {
  console.log(match[0], match[1], match[2], match[3]);
}
```

---

**replace()**：替换匹配内容
`<字符串>.replace(<正则>, <替换字符串>)`
```javascript
const str = 'Hello World';
console.log(str.replace(/world/i, 'JavaScript'));  // 'Hello JavaScript'
console.log(str.replace(/o/g, '0'));  // 'Hell0 W0rld'

// 使用回调函数
console.log(str.replace(/\w+/g, (word) => word.toUpperCase()));  // 'HELLO WORLD'
```

---

**replaceAll()**：替换所有匹配
`<字符串>.replaceAll(<正则>, <替换字符串>)`
```javascript
const str = 'aaa';
console.log(str.replaceAll('a', 'b'));  // 'bbb'
console.log(str.replaceAll(/a/g, 'c'));  // 'ccc'
```

---

**search()**：返回匹配索引
`<字符串>.search(<正则>)`
```javascript
const str = 'Hello World';
console.log(str.search(/world/i));  // 6 - 返回索引
console.log(str.search(/xyz/));     // -1 - 未找到
```

---

**split()**：按正则分割字符串
`<字符串>.split(<正则>)`
```javascript
const str = 'apple, banana; orange|grape';
console.log(str.split(/[,;|]/));  // ['apple', ' banana', ' orange', 'grape']
console.log(str.split(/[,;|]\s*/));  // ['apple', 'banana', 'orange', 'grape']
```

---

## 正则对象方法

**test()**：测试是否匹配
`<正则>.test(<字符串>)`
```javascript
const pattern = /\d+/;
console.log(pattern.test('hello123'));  // true
console.log(pattern.test('hello'));     // false
```

---

**exec()**：执行匹配，返回详细信息
`<正则>.exec(<字符串>)`
```javascript
const pattern = /(\d+)-(\d+)/;
const str = '12-34, 56-78';
let match;
while ((match = pattern.exec(str)) !== null) {
  console.log(match[0], match[1], match[2]);
}
// 输出: 12-34 12 34
// 输出: 56-78 56 78
```

## 常用正则模式

**邮箱验证**：匹配邮箱格式
`/^[\w.-]+@[\w.-]+\.\w+$/`
```javascript
const emailPattern = /^[\w.-]+@[\w.-]+\.\w+$/;
console.log(emailPattern.test('user@example.com'));  // true
console.log(emailPattern.test('invalid-email'));     // false
```

---

**手机号验证**：匹配手机号
`/^1[3-9]\d{9}$/`
```javascript
const phonePattern = /^1[3-9]\d{9}$/;
console.log(phonePattern.test('13812345678'));  // true
console.log(phonePattern.test('12345678901'));  // false
```

---

**URL 验证**：匹配 URL
`/^https?:\/\/[\w.-]+/`
```javascript
const urlPattern = /^https?:\/\/[\w.-]+/;
console.log(urlPattern.test('https://example.com'));  // true
console.log(urlPattern.test('ftp://example.com'));    // false
```

---

**IP 地址验证**：匹配 IPv4
`/^(\d{1,3}\.){3}\d{1,3}$/`
```javascript
const ipPattern = /^(\d{1,3}\.){3}\d{1,3}$/;
console.log(ipPattern.test('192.168.1.1'));  // true
console.log(ipPattern.test('999.999.999.999'));  // true（格式正确但值无效）
```

---

**日期格式验证**：匹配 YYYY-MM-DD
`/^\d{4}-\d{2}-\d{2}$/`
```javascript
const datePattern = /^\d{4}-\d{2}-\d{2}$/;
console.log(datePattern.test('2024-01-15'));  // true
console.log(datePattern.test('2024/01/15'));  // false
```

## 前瞻与后顾

**正向前瞻**：匹配后面跟着的模式
`<模式>(?=<后顾模式>)`
```javascript
const str = 'price: $100, quantity: 5';
console.log(str.match(/\d+(?=\d)/g));  // 匹配数字后面跟着数字
console.log(str.match(/\w+(?=:)/g));   // ['price', 'quantity'] - 匹配后面跟着冒号的单词
```

---

**负向前瞻**：匹配后面不跟着的模式
`<模式>(?!<后顾模式>)`
```javascript
const str = 'foo123 bar456';
console.log(str.match(/\d+(?!\d)/g));  // ['3', '6'] - 匹配后面不跟着数字的数字
```

---

**正向后顾**：匹配前面是某模式的
`(?<=<前瞻模式>)<模式>`
```javascript
const str = 'price: $100, tax: $20';
console.log(str.match(/(?<=\$)\d+/g));  // ['100', '20'] - 匹配 $ 后面的数字
```

---

**负向后顾**：匹配前面不是某模式的
`(?<!<前瞻模式>)<模式>`
```javascript
const str = 'abc123 def456';
console.log(str.match(/(?<!\d)\d+/g));  // ['123', '456'] - 匹配前面不是数字的数字
```
