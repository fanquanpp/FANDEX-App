# JavaScript 数组高阶方法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## reduce 归约方法

**基本语法**：对数组元素进行归约
`<数组>.reduce((<累加器>, <当前值>, <索引>, <数组>) => <新累加器>, <初始值>)`
```javascript
const sum = [1, 2, 3, 4].reduce((acc, cur) => acc + cur, 0);  // 10
const product = [1, 2, 3, 4].reduce((acc, cur) => acc * cur, 1);  // 24
```

---

**对象分组**：按属性分组
`<数组>.reduce((<累加器>, <元素>) => { ... return <累加器>; }, {})`
```javascript
const people = [
  { name: 'Alice', age: 25 },
  { name: 'Bob', age: 30 },
  { name: 'Charlie', age: 25 },
];
const byAge = people.reduce((acc, person) => {
  const key = person.age;
  (acc[key] ??= []).push(person);
  return acc;
}, {});
// { 25: [{ name: 'Alice', age: 25 }, { name: 'Charlie', age: 25 }],
//   30: [{ name: 'Bob', age: 30 }] }
```

---

**数组扁平化**：使用 reduce
`<数组>.reduce((acc, cur) => acc.concat(cur), [])`
```javascript
const nested = [[1, 2], [3, [4, 5]], [6]];
const flat = nested.reduce((acc, cur) => acc.concat(cur), []);
// [1, 2, 3, [4, 5], 6] - 只扁平一层
```

---

**管道函数**：组合多个函数
`const pipe = (...fns) => (input) => fns.reduce((acc, fn) => fn(acc), input);`
```javascript
const pipe = (...fns) => (input) => fns.reduce((acc, fn) => fn(acc), input);

const transform = pipe(
  (str) => str.trim(),
  (str) => str.toLowerCase(),
  (str) => str.replace(/\s+/g, '-')
);
transform('  Hello World  ');  // 'hello-world'
```

---

**统计词频**：统计元素出现次数
`<数组>.reduce((acc, <元素>) => { acc[<元素>] = (acc[<元素>] || 0) + 1; return acc; }, {})`
```javascript
const words = ['apple', 'banana', 'apple', 'cherry', 'banana', 'apple'];
const frequency = words.reduce((acc, word) => {
  acc[word] = (acc[word] || 0) + 1;
  return acc;
}, {});
// { apple: 3, banana: 2, cherry: 1 }
```

---

**reduce 初始值陷阱**：空数组必须提供初始值
`<数组>.reduce(<回调>, <初始值>)`
```javascript
// 错误：空数组没有初始值会报错
// [].reduce((acc, cur) => acc + cur);  // TypeError

// 正确：始终提供初始值
[].reduce((acc, cur) => acc + cur, 0);  // 0
```

## flatMap 映射并扁平化

**基本语法**：映射后扁平化一层
`<数组>.flatMap((<元素>, <索引>, <数组>) => <新数组或元素>)`
```javascript
const sentences = ['Hello World', 'Good Morning'];
const words = sentences.flatMap((s) => s.split(' '));
// ['Hello', 'World', 'Good', 'Morning']
```

---

**过滤与映射**：同时过滤和映射
`<数组>.flatMap((<元素>) => <条件> ? [<结果>] : [])`
```javascript
const numbers = [-2, -1, 0, 1, 2, 3];
const result = numbers.flatMap((n) => (n > 0 ? [n * 2] : []));
// [2, 4, 6]
```

---

**提取嵌套属性**：从嵌套结构提取数据
`<数组>.flatMap((<元素>) => <元素>.<属性>.map(<回调>))`
```javascript
const users = [
  { name: 'Alice', orders: [{ id: 1 }, { id: 2 }] },
  { name: 'Bob', orders: [{ id: 3 }] },
  { name: 'Charlie', orders: [] },
];
const orderIds = users.flatMap((user) => user.orders.map((o) => o.id));
// [1, 2, 3]
```

---

**展开树结构**：将树结构展开为数组
`<数组>.flatMap((<元素>) => [<元素>.<属性>, ...<元素>.<子属性>])`
```javascript
const categories = [
  { name: 'Electronics', subcategories: ['Phones', 'Laptops'] },
  { name: 'Clothing', subcategories: ['Shirts', 'Pants'] },
];
const allCategories = categories.flatMap((cat) => [cat.name, ...cat.subcategories]);
// ['Electronics', 'Phones', 'Laptops', 'Clothing', 'Shirts', 'Pants']
```

## find 与 findIndex

**find()**：查找第一个满足条件的元素
`<数组>.find((<元素>) => <条件>)`
```javascript
const users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
];
const user = users.find((u) => u.id === 2);
// { id: 2, name: 'Bob' }
```

---

**findIndex()**：查找第一个满足条件的索引
`<数组>.findIndex((<元素>) => <条件>)`
```javascript
const users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
];
const index = users.findIndex((u) => u.id === 2);
// 1
```

---

**findLast() 和 findLastIndex()**：从末尾查找
`<数组>.findLast(<回调>)` | `<数组>.findLastIndex(<回调>)`
```javascript
const users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
  { id: 3, name: 'Charlie' },
];
users.findLast((u) => u.name.startsWith('B'));       // { id: 2, name: 'Bob' }
users.findLastIndex((u) => u.name.startsWith('B'));   // 1
```

## every 与 some

**every()**：检查所有元素是否满足条件
`<数组>.every((<元素>) => <条件>)`
```javascript
const ages = [18, 25, 30, 42];
const allAdults = ages.every((age) => age >= 18);  // true
```

---

**some()**：检查是否有元素满足条件
`<数组>.some((<元素>) => <条件>)`
```javascript
const ages = [18, 25, 30, 42];
const hasSenior = ages.some((age) => age > 40);  // true
```

## Array.from 映射

**创建范围数组**：使用 Array.from
`Array.from({ length: <长度> }, (_, i) => <表达式>)`
```javascript
const range = Array.from({ length: 5 }, (_, i) => i * 2);
// [0, 2, 4, 6, 8]
```

---

**去重**：使用 Set 和 Array.from
`Array.from(new Set(<数组>))`
```javascript
const unique = Array.from(new Set([1, 2, 2, 3, 3]));
// [1, 2, 3]
```

## at() 方法

**at()**：支持负索引访问
`<数组>.at(<索引>)`
```javascript
const arr = [10, 20, 30, 40, 50];
arr.at(0);   // 10
arr.at(-1);  // 50 - 负索引从末尾计数
arr.at(-2);  // 40
```

## 链式调用

**方法链**：组合多个数组方法
`<数组>.filter(<回调>).map(<回调>).reduce(<回调>, <初始值>)`
```javascript
const result = orders
  .filter((order) => order.status === 'completed')
  .map((order) => order.total)
  .reduce((sum, total) => sum + total, 0);
```

---

**函数组合**：自定义 compose
`const compose = (...fns) => (arr) => fns.reduce((acc, fn) => fn(acc), arr);`
```javascript
const compose = (...fns) => (arr) => fns.reduce((acc, fn) => fn(acc), arr);

const processNumbers = compose(
  (arr) => arr.filter((n) => n > 0),
  (arr) => arr.map((n) => n * 2),
  (arr) => arr.reduce((sum, n) => sum + n, 0)
);

processNumbers([-1, 2, -3, 4, 5]);  // 22
```
