# JavaScript 程序结构与基本语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 脚本引入

**内部脚本**：在 HTML 中嵌入 JavaScript
`<script> <代码> </script>`
```html
<script>
  // 内部脚本
  console.log('Hello from inline script!');
</script>
```

---

**外部脚本**：引入外部 JavaScript 文件
`<script src="<文件路径>"></script>`
```html
<!-- 引入外部脚本 -->
<script src="app.js"></script>
```

---

**ES 模块**：引入 ES 模块
`<script type="module" src="<文件路径>"></script>`
```html
<!-- 引入 ES 模块 -->
<script type="module" src="main.js"></script>
```

---

**脚本加载控制**：defer 与 async
`<script src="<路径>" [defer|async]>`
```html
<!-- 延迟脚本：HTML 解析完成后按顺序执行 -->
<script src="deferred.js" defer></script>

<!-- 异步脚本：下载完成后立即执行 -->
<script src="async.js" async></script>
```

---

## 语句与分号

**基本语句**：以分号结尾
`<语句>;`
```javascript
// 变量声明语句
let x = 10;

// 赋值语句
x = 20;

// 函数调用语句
console.log(x);
```

---

**代码块**：使用大括号包裹
`{ <语句集合> }`
```javascript
{
  let blockVar = '只在块内可见';
  console.log(blockVar);
}
```

---

## 注释

**单行注释**：使用 //
`// <注释内容>`
```javascript
// 这是一个单行注释
let x = 10; // 这也是一个单行注释
```

---

**多行注释**：使用 /* */
`/* <注释内容> */`
```javascript
/*
 * 这是一个
 * 多行注释
 */
let y = 20;
```

---

**文档注释**：JSDoc 格式
`/** <注释内容> */`
```javascript
/**
 * 计算两个数的和
 * @param {number} a - 第一个数
 * @param {number} b - 第二个数
 * @returns {number} 两个数的和
 */
function add(a, b) {
  return a + b;
}
```

## 变量声明

**var 声明**：函数作用域，存在变量提升
`var <变量名> = <值>;`
```javascript
// 变量提升 - 可以在声明前使用
console.log(x); // 输出: undefined
var x = 10;

// 函数作用域
function test() {
  var y = 20;
  console.log(y); // 输出: 20
}
```

---

**let 声明**：块级作用域，不存在变量提升
`let <变量名> = <值>;`
```javascript
// 块级作用域
if (true) {
  let z = 20;
  console.log(z); // 输出: 20
}
// console.log(z); // 报错: z is not defined

// 可以修改值
let a = 10;
a = 30;
```

---

**const 声明**：块级作用域，常量
`const <常量名> = <值>;`
```javascript
// 必须初始化
const PI = 3.14159;

// 不能修改值
// PI = 3.14; // 报错

// 对象和数组的内容可以修改
const person = { name: 'Alice' };
person.name = 'Bob'; // 允许
```

## 标识符命名

**变量和函数**：小驼峰命名法
`<lowerCamelCase>`
```javascript
let userName = 'Alice';
let userAge = 30;
function calculateTotalPrice(items) {
  // 函数体
}
```

---

**常量**：全大写下划线分隔
`<UPPER_SNAKE_CASE>`
```javascript
const MAX_SIZE = 100;
const API_URL = 'https://api.example.com';
```

---

**类**：大驼峰命名法
`<PascalCase>`
```javascript
class User {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}
```

---

## 严格模式

**全局严格模式**：在脚本顶部声明
`'use strict';`
```javascript
'use strict';
// 严格模式下的代码
let x = 10;
```

---

**函数严格模式**：在函数内部声明
`function <函数名>() { 'use strict'; ... }`
```javascript
function strictFunction() {
  'use strict';
  // 严格模式下的代码
  let y = 20;
}
```

---

**严格模式限制**：禁止未声明变量
`<变量> = <值>;`
```javascript
'use strict';
// x = 10; // 报错: x is not defined
let x = 10; // 正确
```
