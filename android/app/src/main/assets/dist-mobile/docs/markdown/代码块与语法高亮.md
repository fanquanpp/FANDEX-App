# 代码块与语法高亮

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 行内代码

**行内代码：使用反引号包围**
`` `<代码>` ``
```markdown
使用 `console.log('Hello, World!');` 输出内容
```

---

## 基本代码块

**基本代码块：使用三个反引号包围**
` ``` \n<代码>\n``` `
```markdown
```
function hello() {
  console.log('Hello, World!');
}
```
```

---

## 语法高亮

**指定语言：在三个反引号后指定语言名称**
` ```<语言> \n<代码>\n``` `
```markdown
```javascript
function hello() {
  console.log('Hello, World!');
}
```

```python
def hello():
    print('Hello, World!')
```

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```sql
SELECT * FROM users WHERE age > 18;
```

```json
{
  "name": "John",
  "age": 30
}
```

```yaml
server:
  port: 8080
```
```

## 代码块高级功能

**显示行号：添加 linenos 选项**
` ```<语言> {linenos} `
```markdown
```javascript {linenos}
function hello() {
  console.log('Hello, World!');
}
```
```

**高亮特定行：添加 hl_lines 选项**
` ```<语言> {hl_lines=[<行号>]} `
```markdown
```javascript {hl_lines=[2,4]}
function hello() {
  console.log('Hello, World!');
}
hello();
```
```
