# 段落与换行

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 段落

**段落分隔：使用空行分隔段落**
`<文本>\n\n<文本>`
```markdown
这是第一个段落。

这是第二个段落，与第一个段落之间有一个空行。
```

---

## 换行

**行尾空格换行：行尾添加两个或更多空格**
`<文本>  \n<文本>`
```markdown
这是第一行  
这是第二行，与第一行属于同一段落
```

**HTML 换行标签：使用 br 标签**
`<文本><br>\n<文本>`
```markdown
这是第一行<br>
这是第二行，与第一行属于同一段落
```

**反斜杠换行：行尾添加反斜杠**
`<文本>\\n<文本>`
```markdown
这是第一行\
这是第二行，与第一行属于同一段落
```

---

## 空行

**元素间空行：分隔不同元素**
`<元素>\n\n<元素>`
```markdown
# 标题

这是一个段落。

- 列表项 1
- 列表项 2

```python
print("Hello!")
```

这是另一个段落。
```
