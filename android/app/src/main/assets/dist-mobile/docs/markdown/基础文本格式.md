# 基础文本格式

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 斜体

**斜体：使用 * 或 _ 包裹文本**
`*<文本>* | _<文本>_`
```markdown
*斜体文本*
_斜体文本_
```

---

## 粗体

**粗体：使用 ** 或 __ 包裹文本**
`**<文本>** | __<文本>__`
```markdown
**粗体文本**
__粗体文本__
```

---

## 粗斜体

**粗斜体：使用 *** 或 ___ 包裹文本**
`***<文本>*** | ___<文本>___`
```markdown
***粗斜体文本***
___粗斜体文本___
```

---

## 删除线

**删除线：使用 ~~ 包裹文本（GFM 扩展）**
`~~<文本>~~`
```markdown
~~删除线文本~~
```

---

## 下划线

**下划线：使用 HTML u 标签**
`<u><文本></u>`
```markdown
<u>下划线文本</u>
```

---

## 上标与下标

**上标：使用 HTML sup 标签**
`<sup><文本></sup>`
```markdown
2<sup>2</sup> = 4
```

**下标：使用 HTML sub 标签**
`<sub><文本></sub>`
```markdown
H<sub>2</sub>O
```

---

## 高亮文本

**高亮文本：使用 == 包裹文本（GFM 扩展）**
`==<文本>==`
```markdown
==高亮文本==
```

---

## 行内代码

**行内代码：使用反引号包裹文本**
`` `<文本>` ``
```markdown
`行内代码`
```
