# 表格

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本表格

**基本表格：使用管道符和分隔线**
`| <表头1> | <表头2> |\n| --- | --- |\n| <单元格1> | <单元格2> |`
```markdown
| 姓名 | 年龄 | 城市 |
| ---- | ---- | ---- |
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
| 王五 | 28   | 广州 |
```

---

## 表格对齐

**左对齐：冒号在左**
`| :--- |`
```markdown
| 左对齐   | 居中对齐 | 右对齐 |
| :------- | :------: | -----: |
| 内容 1   |  内容 2  |   内容 3 |
| 长内容 1 | 长内容 2 | 长内容 3 |
```

**居中对齐：冒号在两侧**
`| :---: |`
```markdown
| 姓名 | 年龄 |
| :--: | :--: |
| 张三 |  25  |
```

**右对齐：冒号在右**
`| ---: |`
```markdown
| 商品 | 价格 |
| ---- | ---: |
| 手机 |  2999 |
| 电脑 |  5999 |
```

---

## 表格中的特殊内容

**表格中换行：使用 br 标签**
`| <文本><br><文本> |`
```markdown
| 姓名 | 地址                             |
| ---- | -------------------------------- |
| 张三 | 北京市朝阳区<br>建国路 88 号     |
| 李四 | 上海市浦东新区<br>陆家嘴金融中心 |
```

**表格中的链接**
`| [<文本>](<URL>) |`
```markdown
| 名称   | 链接                         |
| ------ | ---------------------------- |
| GitHub | [GitHub](https://github.com) |
| Google | [Google](https://google.com) |
```

**表格中的代码**
`| `<代码>` |`
```markdown
| 语言       | 代码                    |
| ---------- | ----------------------- |
| JavaScript | `console.log('Hello');` |
| Python     | `print('Hello')`        |
```

**表格中的管道符转义**
`| <文本>\|<文本> | | <文本>&#124;<文本> |`
```markdown
| 命令           | 说明     |
| -------------- | -------- |
| `grep \| file` | 管道操作 |
```

## 合并单元格

**HTML 表格合并：使用 colspan 属性**
`<th colspan="<N>">`
```markdown
<table>
  <tr>
    <th colspan="2">个人信息</th>
  </tr>
  <tr>
    <td>姓名</td>
    <td>张三</td>
  </tr>
  <tr>
    <td>年龄</td>
    <td>25</td>
  </tr>
</table>
```
