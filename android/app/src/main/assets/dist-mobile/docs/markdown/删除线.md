# 删除线

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本语法

**删除线：使用两个波浪号包裹文本（GFM 扩展）**
`~~<文本>~~`
```markdown
~~这段文字会被加上删除线~~
```

---

## 语法规则

**两个波浪号：必须成对出现**
`~~<文本>~~`
```markdown
~~正确写法~~
~错误写法~（单波浪号无效）
~~ 未闭合（缺少结束标记）
```

**紧贴文本：波浪号与文本之间不能有空格**
`~~<文本>~~`
```markdown
~~正确~~
~~ 错误 ~~（空格会被保留）
```

---

## 与其他格式组合

**删除线与斜体组合**
`~~_<文本>_~~`
```markdown
~~_删除且斜体_~~
```

**删除线与粗体组合**
`~~**<文本>**~~`
```markdown
~~**删除且粗体**~~
```

**删除线与粗斜体组合**
`~~**_<文本>_**~~`
```markdown
~~**_删除且粗斜体_**~~
```

**删除线与行内代码组合**
`~~`<代码>`~~`
```markdown
~~`删除的代码`~~
```

---

## HTML 替代

**del 标签：表示已删除内容（推荐）**
`<del><文本></del>`
```markdown
<del>这段文字会被加上删除线</del>
```

**s 标签：表示不再准确的内容**
`<s><文本></s>`
```markdown
<s>这段文字会被加上删除线</s>
```

**strike 标签：已废弃的标签**
`<strike><文本></strike>`
```markdown
<strike>这段文字会被加上删除线</strike>
```

**del 标签带属性**
`<del datetime="<日期>" cite="<引用URL>"><文本></del>`
```markdown
<del datetime="2026-06-14">已删除的内容</del>
<del cite="https://example.com/reason">删除原因见链接</del>
```

## 使用场景

**价格标注**
`~~<原价>~~ <现价>`
```markdown
限时优惠：~~¥299~~ ¥199
```

**文档修订**
`~~<旧内容>~~ <新内容>`
```markdown
项目使用 ~~Webpack~~ Vite 作为构建工具。
```

**变更记录**
`~~<旧方法>~~ → <新方法>`
```markdown
- ~~`oldMethod()`~~ → `newMethod()`
- ~~`Config.default`~~ → `Config.defaults`
```
