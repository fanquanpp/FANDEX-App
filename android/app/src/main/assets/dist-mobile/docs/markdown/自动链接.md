# 自动链接

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 尖括号自动链接

**URL 自动链接：使用尖括号包裹 URL**
`<<URL>>`
```markdown
<https://github.com>
<http://example.com/path?q=1>
```

**邮箱自动链接：使用尖括号包裹邮箱**
`<<邮箱>>`
```markdown
<user@example.com>
<contact@company.org>
```

---

## GFM 裸 URL 自动链接

**裸 URL 自动识别：GFM 扩展**
`<URL>`
```markdown
访问 https://github.com 了解更多
浏览 www.example.com 查看
```

**裸邮箱自动识别**
`<邮箱>`
```markdown
联系 user@example.com 获取更多信息
```

---

## 自动链接规则

**URL 必须包含协议**
`<http://> | <https://>`
```markdown
<!-- 有效 -->
<https://example.com>
<http://example.com/path?q=1>

<!-- 无效（缺少协议） -->
<example.com>
```

**尖括号内不能有空格**
`<<URL>>`
```markdown
<!-- 有效 -->
<https://example.com>

<!-- 无效（有空格） -->
< https://example.com >
```

---

## 自定义链接文本

**使用标准链接语法自定义文本**
`[<文本>](<URL>)`
```markdown
<!-- 自动链接：显示 URL -->
<https://github.com>

<!-- 标准链接：自定义文本 -->
[GitHub](https://github.com)
```

---

## 新窗口打开

**HTML a 标签：在新窗口打开**
`<a href="<URL>" target="_blank"><文本></a>`
```markdown
<a href="https://example.com" target="_blank">在新窗口打开</a>
```

**添加 nofollow 属性**
`<a href="<URL>" rel="nofollow"><文本></a>`
```markdown
<a href="https://example.com" rel="nofollow">不追踪的链接</a>
```
