# 转义字符

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 反斜杠转义

**反斜杠转义：原样显示特殊字符**
`\\<字符>`
```markdown
\*不是强调\*
\#不是标题
\`不是代码
```

---

## 可转义字符

**CommonMark 可转义字符列表**
`\\ | \` | \* | \_ | \{ | \} | \[ | \] | \( | \) | \# | \+ | \- | \. | \! | \|`
```markdown
\\   反斜杠
\`   反引号
\*   星号
\_   下划线
\{   花括号
\}   花括号
\[   方括号
\]   方括号
\(   圆括号
\)   圆括号
\#   井号
\+   加号
\-   减号
\.   句点
\!   感叹号
\|   管道符
```

---

## 转义规则

**规则一：反斜杠后跟可转义字符时，字符原样显示**
`\\<可转义字符>`
```markdown
\*不是强调\*
```

**规则二：反斜杠后跟不可转义字符时，反斜杠保留**
`\\<不可转义字符>`
```markdown
\A → \A
\3 → \3
```

**规则三：反斜杠在行尾表示硬换行**
`<文本>\\n<文本>`
```markdown
第一行\
第二行
```

## HTML 实体转义

**常用 HTML 实体**
`&<实体名>; | &#<实体编号>;`
```markdown
&lt;       小于号 <
&gt;       大于号 >
&amp;      & 符号
&quot;     双引号 "
&apos;     单引号 '
&copy;     版权符号
&reg;      注册商标
&trade;    商标符号
&nbsp;     不换行空格
```

---

## 表格中的管道符

**表格中转义管道符**
`\\| | &#124;`
```markdown
| 命令           | 说明     |
| -------------- | -------- |
| `grep \| file` | 管道操作 |
```

---

## 强调中的星号

**转义强调中的星号**
`\\*<文本>\\*`
```markdown
\*文件名: readme\*
```
