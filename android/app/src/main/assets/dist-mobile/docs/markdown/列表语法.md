# 列表语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 无序列表

**无序列表：使用 -、* 或 + 加空格**
`- <列表项> | * <列表项> | + <列表项>`
```markdown
- 无序列表项 1
- 无序列表项 2
- 无序列表项 3
```

**列表项换行：第二行与第一行文本对齐**
`- <第一行>\n  <第二行>`
```markdown
- 这是一个很长的列表项，
  换行后需要与第一行文本对齐
```

---

## 有序列表

**有序列表：使用数字加句点加空格**
`<数字>. <列表项>`
```markdown
1. 有序列表项 1
2. 有序列表项 2
3. 有序列表项 3
```

**序号自动调整：数字顺序不影响渲染**
`1. <项1>\n1. <项2>\n1. <项3>`
```markdown
1. 第一项
1. 第二项
1. 第三项
```

---

## 任务列表

**任务列表：GFM 扩展语法**
`- [ ] <未完成项> | - [x] <已完成项>`
```markdown
- [ ] 未完成的待办任务
- [x] 已完成的待办任务
```

---

## 嵌套列表

**嵌套列表：使用 4 个空格或 1 个制表符缩进**
`- <一级项>\n    - <二级项>`
```markdown
- 一级无序列表项
    - 二级无序列表项
        - 三级无序列表项
    - 另一个二级项
- 另一个一级项
```

**混合嵌套：无序与有序混合**
`- <无序项>\n    1. <有序项>`
```markdown
- 无序列表项
    1. 有序列表项
    2. 有序列表项
- 无序列表项
```

---

## 列表中使用代码块

**列表中的代码块：缩进对齐**
`- <项>\n\n  ```<语言>\n  <代码>\n  ````
```markdown
- 列表项 1

  ```python
  print("Hello, World!")
  ```

- 列表项 2
```

---

## 列表中使用引用

**列表中的引用：缩进对齐**
`- <项>\n  > <引用>`
```markdown
- 列表项 1
  > 这是一个引用
  > 可以跨越多行
- 列表项 2
```
