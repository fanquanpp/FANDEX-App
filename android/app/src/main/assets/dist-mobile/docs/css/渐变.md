# CSS 渐变

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 线性渐变

**linear-gradient 基本语法**：创建线性渐变
`linear-gradient(<方向>, <颜色1>, <颜色2>, ...);`
```css
/* 基本线性渐变 */
.element { background: linear-gradient(to right, #3498db, #2ecc71); }

/* 使用角度 */
.element { background: linear-gradient(45deg, #3498db, #2ecc71); }
.element { background: linear-gradient(135deg, #3498db, #2ecc71); }

/* 多色渐变 */
.element { background: linear-gradient(to right, #3498db, #9b59b6, #e74c3c); }
```

---

**方向参数**：控制渐变方向
`linear-gradient(<方向>, <颜色>);`
```css
/* 关键字方向 */
.element { background: linear-gradient(to right, #3498db, #2ecc71); }
.element { background: linear-gradient(to left, #3498db, #2ecc71); }
.element { background: linear-gradient(to top, #3498db, #2ecc71); }
.element { background: linear-gradient(to bottom, #3498db, #2ecc71); }

/* 对角方向 */
.element { background: linear-gradient(to top right, #3498db, #2ecc71); }
.element { background: linear-gradient(to bottom left, #3498db, #2ecc71); }

/* 角度（0deg 向上，90deg 向右，180deg 向下，270deg 向左） */
.element { background: linear-gradient(0deg, #3498db, #2ecc71); }
.element { background: linear-gradient(90deg, #3498db, #2ecc71); }
.element { background: linear-gradient(180deg, #3498db, #2ecc71); }
```

---

**颜色停止点**：控制颜色位置
`linear-gradient(<方向>, <颜色> <位置>, <颜色> <位置>);`
```css
/* 百分比位置 */
.element { background: linear-gradient(to right, #3498db 0%, #2ecc71 100%); }

/* 像素位置 */
.element { background: linear-gradient(to right, #3498db 50px, #2ecc71 200px); }

/* 硬切换（无渐变过渡） */
.element { background: linear-gradient(to right, #3498db 50%, #2ecc71 50%); }

/* 多停止点 */
.element {
  background: linear-gradient(to right,
    #3498db 0%,
    #9b59b6 25%,
    #e74c3c 50%,
    #f39c12 75%,
    #2ecc71 100%);
}
```

---

**透明度渐变**：使用 rgba 创建透明过渡
`linear-gradient(<方向>, rgba(<颜色>, <透明度>), rgba(<颜色>, <透明度>));`
```css
/* 从透明到不透明 */
.element {
  background: linear-gradient(to right, rgba(52, 152, 219, 0), rgba(52, 152, 219, 1));
}

/* 从不透明到透明 */
.fade-overlay {
  background: linear-gradient(to bottom, rgba(0, 0, 0, 0.8), transparent);
}
```

## 径向渐变

**radial-gradient 基本语法**：创建径向渐变
`radial-gradient(<形状> <大小> at <位置>, <颜色1>, <颜色2>);`
```css
/* 基本径向渐变 */
.element { background: radial-gradient(circle, #3498db, #2ecc71); }

/* 椭圆渐变 */
.element { background: radial-gradient(ellipse, #3498db, #2ecc71); }

/* 指定位置 */
.element { background: radial-gradient(circle at top left, #3498db, #2ecc71); }
.element { background: radial-gradient(circle at 50% 50%, #3498db, #2ecc71); }
.element { background: radial-gradient(circle at 25% 75%, #3498db, #2ecc71); }
```

---

**径向渐变大小**：控制渐变尺寸
`radial-gradient(<大小> <形状>, <颜色>);`
```css
/* 关键字大小 */
.element { background: radial-gradient(closest-side, #3498db, #2ecc71); }
.element { background: radial-gradient(farthest-side, #3498db, #2ecc71); }
.element { background: radial-gradient(closest-corner, #3498db, #2ecc71); }
.element { background: radial-gradient(farthest-corner, #3498db, #2ecc71); }

/* 具体尺寸 */
.element { background: radial-gradient(circle 100px, #3498db, #2ecc71); }
.element { background: radial-gradient(ellipse 100px 50px, #3498db, #2ecc71); }
```

## 圆锥渐变

**conic-gradient 基本语法**：创建圆锥渐变
`conic-gradient(from <角度> at <位置>, <颜色1>, <颜色2>);`
```css
/* 基本圆锥渐变 */
.element { background: conic-gradient(#3498db, #2ecc71, #e74c3c, #3498db); }

/* 指定起始角度 */
.element { background: conic-gradient(from 0deg, #3498db, #2ecc71, #e74c3c); }
.element { background: conic-gradient(from 90deg, #3498db, #2ecc71, #e74c3c); }

/* 指定中心位置 */
.element { background: conic-gradient(from 0deg at 50% 50%, #3498db, #2ecc71, #e74c3c); }

/* 饼图效果 */
.pie {
  background: conic-gradient(#3498db 0% 25%, #2ecc71 25% 50%, #e74c3c 50% 75%, #f39c12 75% 100%);
}
```

## 重复渐变

**repeating-linear-gradient**：重复线性渐变
`repeating-linear-gradient(<方向>, <颜色1> <位置>, <颜色2> <位置>);`
```css
/* 斜纹背景 */
.stripes {
  background: repeating-linear-gradient(
    45deg,
    #3498db,
    #3498db 10px,
    #2980b9 10px,
    #2980b9 20px
  );
}

/* 水平条纹 */
.horizontal-stripes {
  background: repeating-linear-gradient(
    to bottom,
    #3498db 0px,
    #3498db 20px,
    #2ecc71 20px,
    #2ecc71 40px
  );
}
```

---

**repeating-radial-gradient**：重复径向渐变
`repeating-radial-gradient(<形状>, <颜色1> <位置>, <颜色2> <位置>);`
```css
/* 同心圆 */
.rings {
  background: repeating-radial-gradient(
    circle,
    #3498db 0px,
    #3498db 10px,
    #2ecc71 10px,
    #2ecc71 20px
  );
}
```

---

**repeating-conic-gradient**：重复圆锥渐变
`repeating-conic-gradient(from <角度>, <颜色1> <位置>, <颜色2> <位置>);`
```css
/* 放射状重复 */
.sunburst {
  background: repeating-conic-gradient(
    from 0deg,
    #3498db 0deg 30deg,
    #2ecc71 30deg 60deg
  );
}
```

## 渐变实战

**按钮渐变**：按钮背景渐变
`background: linear-gradient(<方向>, <颜色1>, <颜色2>);`
```css
.gradient-button {
  background: linear-gradient(135deg, #3498db, #2980b9);
  color: white;
  border: none;
  padding: 12px 24px;
  border-radius: 4px;
  cursor: pointer;
  transition: filter 0.3s ease;
}

.gradient-button:hover {
  filter: brightness(1.1);
}
```

---

**渐变文字**：渐变背景裁剪到文字
`background: <渐变>; -webkit-background-clip: text; color: transparent;`
```css
.gradient-text {
  background: linear-gradient(45deg, #3498db, #9b59b6, #e74c3c);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
  -webkit-text-fill-color: transparent;
}
```

---

**渐变边框**：使用 border-image 实现渐变边框
`border: <宽度> solid; border-image: <渐变> 1;`
```css
.gradient-border {
  border: 4px solid;
  border-image: linear-gradient(45deg, #3498db, #e74c3c) 1;
}
```

---

**渐变叠加**：使用多重背景叠加渐变
`background: <渐变1>, <渐变2>, <颜色>;`
```css
/* 多重渐变叠加 */
.complex-gradient {
  background:
    linear-gradient(45deg, rgba(52, 152, 219, 0.5), transparent),
    radial-gradient(circle at top right, rgba(231, 76, 60, 0.5), transparent),
    #2c3e50;
}

/* 渐变 + 图片叠加 */
.image-overlay {
  background:
    linear-gradient(to bottom, transparent, rgba(0, 0, 0, 0.8)),
    url('image.jpg');
  background-size: cover;
  background-position: center;
}
```

---

**渐变动画**：动画化渐变背景
`background-size: <大小>; animation: <名称> <时间>;`
```css
.animated-gradient {
  background: linear-gradient(45deg, #3498db, #9b59b6, #e74c3c, #f39c12, #2ecc71);
  background-size: 400% 400%;
  animation: gradientShift 8s ease infinite;
}

@keyframes gradientShift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

---

**网格图案**：使用渐变创建网格
`background: linear-gradient(<方向>, <颜色>, <颜色>);`
```css
.grid-pattern {
  background-image:
    linear-gradient(rgba(0, 0, 0, 0.1) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0, 0, 0, 0.1) 1px, transparent 1px);
  background-size: 20px 20px;
}

/* 棋盘格 */
.checkerboard {
  background-image:
    linear-gradient(45deg, #ccc 25%, transparent 25%),
    linear-gradient(-45deg, #ccc 25%, transparent 25%),
    linear-gradient(45deg, transparent 75%, #ccc 75%),
    linear-gradient(-45deg, transparent 75%, #ccc 75%);
  background-size: 20px 20px;
  background-position: 0 0, 0 10px, 10px -10px, -10px 0px;
}
```

## CSS 变量管理渐变

**渐变变量**：使用 CSS 变量管理渐变
`:root { --gradient-<名称>: <渐变值>; }`
```css
:root {
  --gradient-primary: linear-gradient(135deg, #3498db, #2980b9);
  --gradient-success: linear-gradient(135deg, #2ecc71, #27ae60);
  --gradient-danger: linear-gradient(135deg, #e74c3c, #c0392b);
  --gradient-rainbow: linear-gradient(45deg, #3498db, #9b59b6, #e74c3c, #f39c12, #2ecc71);
}

.button-primary { background: var(--gradient-primary); }
.button-success { background: var(--gradient-success); }
.button-danger { background: var(--gradient-danger); }
```
