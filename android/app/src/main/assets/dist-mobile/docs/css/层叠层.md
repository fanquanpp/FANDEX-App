# CSS 层叠层

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## @layer 声明

**声明层**：定义命名层
`@layer <层名1>, <层名2>, ...;`
```css
/* 声明多个层（顺序决定优先级，后声明的优先级高） */
@layer reset, base, components, utilities;
```

---

**层内样式**：在指定层内编写样式
`@layer <层名> { <样式> }`
```css
@layer reset {
  * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }
}

@layer base {
  body {
    font-size: 16px;
    line-height: 1.6;
  }
}

@layer components {
  .button {
    padding: 10px 20px;
    border-radius: 4px;
  }
}

@layer utilities {
  .text-center {
    text-align: center;
  }
}
```

---

## 层优先级

**层顺序声明**：优先级从低到高
`@layer <低优先级>, <高优先级>;`
```css
/* 优先级从低到高 */
@layer reset, base, components, utilities;

/* 后声明的层优先级更高 */
@layer reset {
  h1 { font-size: 2em; }
}

@layer base {
  /* base 层的 h1 会覆盖 reset 层 */
  h1 { font-size: 1.5em; color: #333; }
}
```

---

**未分层样式优先级**：未分层样式始终最高
`<选择器> { <样式> }`
```css
@layer base {
  .button {
    color: blue;
  }
}

/* 未分层的样式优先级高于所有层 */
.button {
  color: red; /* 此样式生效 */
}
```

---

## 层嵌套

**嵌套层**：在层内声明子层
`@layer <父层> { @layer <子层> { <样式> } }`
```css
@layer components {
  @layer buttons {
    .button { padding: 10px 20px; }
  }

  @layer cards {
    .card { padding: 20px; }
  }
}

/* 引用嵌套层 */
@layer components.buttons {
  .button-primary { background: #3498db; }
}
```

---

## import 层

**@import 层**：将导入的样式表放入指定层
`@import url('<文件>') layer(<层名>);`
```css
/* 将第三方库样式放入指定层 */
@import url('normalize.css') layer(reset);
@import url('bootstrap.css') layer(framework);

/* 自定义层顺序 */
@layer reset, framework, base, components, utilities;
```

## 实战应用

**完整层架构**：项目层架构示例
`@layer reset, base, layout, components, utilities;`
```css
/* 声明层顺序 */
@layer reset, base, layout, components, utilities;

/* 重置层 */
@layer reset {
  *, *::before, *::after {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }
}

/* 基础层 */
@layer base {
  :root {
    --color-primary: #3498db;
    --font-size-base: 16px;
  }

  body {
    font-family: -apple-system, BlinkMacSystemFont, sans-serif;
    font-size: var(--font-size-base);
    line-height: 1.6;
  }
}

/* 布局层 */
@layer layout {
  .container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
  }

  .grid {
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: 20px;
  }
}

/* 组件层 */
@layer components {
  .button {
    display: inline-block;
    padding: 10px 20px;
    border: none;
    border-radius: 4px;
    cursor: pointer;
    transition: all 0.3s ease;
  }

  .button--primary {
    background: var(--color-primary);
    color: white;
  }
}

/* 工具层 */
@layer utilities {
  .text-center { text-align: center; }
  .mt-4 { margin-top: 1rem; }
  .hidden { display: none; }
}
```

---

**覆盖第三方样式**：使用层控制优先级
`@layer <第三方层>, <自定义层>;`
```css
/* 第三方样式在低优先级层 */
@import url('bootstrap.css') layer(framework);

/* 自定义样式在高优先级层 */
@layer framework, custom;

@layer custom {
  .btn-primary {
    background-color: #e74c3c; /* 覆盖 Bootstrap */
  }
}
```
