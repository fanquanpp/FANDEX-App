# CSS 盒模型详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 盒模型组成

**盒模型属性**：定义元素的内容、内边距、边框、外边距
`width: <宽度>; height: <高度>; padding: <内边距>; border: <边框>; margin: <外边距>;`
```css
/* 完整盒模型示例 */
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #333;
  margin: 15px;
  background-color: #f0f0f0;
}
```

---

## box-sizing 盒模型类型

**content-box 标准盒模型**：width 仅包含内容区
`box-sizing: content-box;`
```css
/* 标准盒模型：实际宽度 = width + padding + border */
.standard-box {
  box-sizing: content-box;
  width: 200px;
  padding: 20px;
  border: 5px solid #333;
  /* 实际宽度: 200 + 20*2 + 5*2 = 250px */
}
```

---

**border-box 怪异盒模型**：width 包含内容、padding、border
`box-sizing: border-box;`
```css
/* 怪异盒模型：实际宽度 = width */
.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid #333;
  /* 实际宽度: 200px（内容宽度被压缩为 150px） */
}
```

---

**全局盒模型设置**：统一所有元素的盒模型
`* { box-sizing: border-box; }`
```css
/* 方法 1：全局设置 */
* {
  box-sizing: border-box;
}

/* 方法 2：包含伪元素 */
*,
*::before,
*::after {
  box-sizing: border-box;
}

/* 方法 3：继承方式 */
html {
  box-sizing: border-box;
}
*,
*::before,
*::after {
  box-sizing: inherit;
}
```

---

## margin 外边距

**margin 简写**：设置四个方向的外边距
`margin: <上> <右> <下> <左>;`
```css
/* 四个方向都是 10px */
margin: 10px;

/* 上下 10px，左右 20px */
margin: 10px 20px;

/* 上 10px，左右 20px，下 30px */
margin: 10px 20px 30px;

/* 上 10px，右 20px，下 30px，左 40px */
margin: 10px 20px 30px 40px;

/* 单个方向 */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;
```

---

**水平居中**：块级元素水平居中
`margin: 0 auto;`
```css
/* 块级元素水平居中（必须指定宽度） */
.centered {
  width: 50%;
  margin: 0 auto;
}

/* 上下 20px，左右自动 */
.centered-box {
  width: 50%;
  margin: 20px auto;
}
```

---

## padding 内边距

**padding 简写**：设置四个方向的内边距
`padding: <上> <右> <下> <左>;`
```css
/* 四个方向都是 10px */
padding: 10px;

/* 上下 10px，左右 20px */
padding: 10px 20px;

/* 上 10px，左右 20px，下 30px */
padding: 10px 20px 30px;

/* 上 10px，右 20px，下 30px，左 40px */
padding: 10px 20px 30px 40px;

/* 单个方向 */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;
```

---

## border 边框

**border 简写**：设置边框宽度、样式、颜色
`border: <宽度> <样式> <颜色>;`
```css
/* 完整边框 */
border: 1px solid #333;

/* 单边边框 */
border-top: 2px dashed #e74c3c;
border-right: 1px dotted #3498db;
border-bottom: 3px double #2ecc71;
border-left: 1px solid #f39c12;

/* 仅设置宽度/样式/颜色 */
border-width: 2px;
border-style: solid;
border-color: #333;
```

## 外边距塌陷

**相邻元素塌陷**：垂直外边距取最大值
`<元素1> { margin-bottom: <值>; } <元素2> { margin-top: <值>; }`
```css
/* 相邻元素外边距塌陷 */
.box1 {
  margin-bottom: 30px;
}
.box2 {
  margin-top: 20px;
}
/* 实际间距: 30px（取最大值），而非 50px */

/* 解决方案：使用 BFC */
.container {
  overflow: hidden; /* 触发 BFC */
}
```

---

**父子元素塌陷**：子元素 margin 穿透父元素
`<父> { overflow: hidden; } | <父> { padding-top: 1px; } | <父> { border-top: 1px solid transparent; }`
```css
/* 父子元素外边距塌陷 */
.parent {
  /* 方法 1：触发 BFC */
  overflow: hidden;

  /* 方法 2：添加边框 */
  /* border: 1px solid transparent; */

  /* 方法 3：添加内边距 */
  /* padding: 1px; */
}
.child {
  margin-top: 30px;
}
```

## BFC 块级格式化上下文

**触发 BFC**：创建独立的渲染区域
`float: left; | position: absolute; | display: flex; | overflow: hidden; | display: inline-block;`
```css
/* 触发 BFC 的条件 */
.bfc-trigger {
  /* 方式 1：浮动 */
  float: left;

  /* 方式 2：绝对定位 */
  /* position: absolute; */

  /* 方式 3：行内块 */
  /* display: inline-block; */

  /* 方式 4：弹性容器 */
  /* display: flex; */

  /* 方式 5：overflow 不为 visible */
  /* overflow: hidden; */
}
```

---

**BFC 清除浮动**：解决父元素高度塌陷
`<父> { overflow: hidden; }`
```css
/* 父元素包含浮动子元素时高度塌陷 */
.parent {
  background-color: #f0f0f0;
  overflow: hidden; /* 触发 BFC 清除浮动 */
}
.child {
  float: left;
  width: 100px;
  height: 100px;
  margin: 10px;
}
```

---

**BFC 两栏布局**：防止文字环绕浮动元素
`<主内容> { overflow: hidden; }`
```css
/* 经典两栏布局 */
.sidebar {
  float: left;
  width: 200px;
  height: 300px;
  background-color: #f0f0f0;
}
.content {
  overflow: hidden; /* 触发 BFC，不与浮动元素重叠 */
  height: 300px;
  background-color: #e0e0e0;
}
```

## CSS 变量控制盒模型

**变量化盒模型**：使用 CSS 变量统一控制
`:root { --<变量>: <值>; }`
```css
/* 定义盒模型变量 */
:root {
  --box-padding: 20px;
  --box-border: 5px;
  --box-margin: 15px;
}

/* 使用变量 */
.box {
  padding: var(--box-padding);
  border: var(--box-border) solid #333;
  margin: var(--box-margin);
}

/* 响应式调整 */
@media (max-width: 768px) {
  :root {
    --box-padding: 10px;
    --box-border: 3px;
    --box-margin: 10px;
  }
}
```
