# CSS 背景增强

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 背景基础

**background 简写**：背景复合属性
`background: <颜色> <图片> <重复> <定位> / <大小> <固定方式>;`
```css
/* 简写形式 */
.element {
  background: #f0f0f0 url('image.jpg') no-repeat center / cover;
}

/* 等价于分拆写法 */
.element {
  background-color: #f0f0f0;
  background-image: url('image.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}
```

---

**background-color**：设置背景颜色
`background-color: <颜色>;`
```css
.element {
  background-color: #3498db;
  background-color: rgba(52, 152, 219, 0.5);
  background-color: hsla(210, 80%, 50%, 0.5);
}
```

---

**background-image**：设置背景图片
`background-image: url('<路径>');`
```css
.element {
  background-image: url('image.jpg');
  background-image: url('image1.jpg'), url('image2.jpg');
}
```

---

## 背景重复与定位

**background-repeat**：背景图片重复方式
`background-repeat: repeat | no-repeat | repeat-x | repeat-y | round | space;`
```css
.element { background-repeat: repeat; }      /* 默认：水平和垂直重复 */
.element { background-repeat: no-repeat; }   /* 不重复 */
.element { background-repeat: repeat-x; }    /* 水平重复 */
.element { background-repeat: repeat-y; }    /* 垂直重复 */
.element { background-repeat: round; }       /* 缩放以完整平铺 */
.element { background-repeat: space; }       /* 不缩放，以间距平铺 */
```

---

**background-position**：背景图片位置
`background-position: <x> <y>;`
```css
.element { background-position: center; }              /* 居中 */
.element { background-position: top left; }            /* 左上角 */
.element { background-position: 50% 50%; }             /* 百分比 */
.element { background-position: 20px 30px; }           /* 像素值 */
.element { background-position: right 20px bottom 10px; } /* 偏移量 */
```

---

**background-size**：背景图片大小
`background-size: cover | contain | <宽度> <高度>;`
```css
.element { background-size: cover; }       /* 覆盖整个容器，可能裁剪 */
.element { background-size: contain; }     /* 完整显示，可能留白 */
.element { background-size: 100% 100%; }   /* 拉伸填满 */
.element { background-size: 200px 100px; } /* 指定尺寸 */
.element { background-size: 50% auto; }    /* 宽度 50%，高度自适应 */
```

---

## 背景附着与裁剪

**background-attachment**：背景滚动方式
`background-attachment: scroll | fixed | local;`
```css
.element { background-attachment: scroll; }  /* 默认：随页面滚动 */
.element { background-attachment: fixed; }   /* 固定不动（视差效果） */
.element { background-attachment: local; }   /* 随元素内容滚动 */
```

---

**background-clip**：背景绘制区域
`background-clip: border-box | padding-box | content-box | text;`
```css
.element { background-clip: border-box; }   /* 默认：包含边框 */
.element { background-clip: padding-box; }   /* 包含内边距 */
.element { background-clip: content-box; }   /* 仅内容区域 */

/* 文字背景裁剪 */
.text-clip {
  background: linear-gradient(45deg, #3498db, #e74c3c);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
}
```

---

**background-origin**：背景定位区域
`background-origin: border-box | padding-box | content-box;`
```css
.element { background-origin: border-box; }   /* 从边框开始 */
.element { background-origin: padding-box; }  /* 从内边距开始（默认） */
.element { background-origin: content-box; }  /* 从内容区域开始 */
```

---

## 多重背景

**多重背景**：叠加多个背景
`background: <背景1>, <背景2>, ...;`
```css
.element {
  background:
    url('top-image.png') no-repeat top center,
    url('bottom-image.png') no-repeat bottom center,
    linear-gradient(to bottom, #3498db, #2ecc71);
}
```

---

**多重背景定位**：为每个背景设置不同位置
`background-position: <位置1>, <位置2>;`
```css
.element {
  background-image:
    url('image1.png'),
    url('image2.png'),
    url('image3.png');
  background-position:
    top left,
    center,
    bottom right;
  background-repeat: no-repeat, no-repeat, no-repeat;
}
```

---

## 渐变背景

**线性渐变**：线性渐变背景
`background: linear-gradient(<方向>, <颜色1>, <颜色2>, ...);`
```css
.element { background: linear-gradient(to right, #3498db, #2ecc71); }
.element { background: linear-gradient(45deg, #3498db, #2ecc71); }
.element { background: linear-gradient(135deg, #3498db 0%, #9b59b6 50%, #e74c3c 100%); }
```

---

**径向渐变**：径向渐变背景
`background: radial-gradient(<形状> <大小> at <位置>, <颜色1>, <颜色2>);`
```css
.element { background: radial-gradient(circle, #3498db, #2ecc71); }
.element { background: radial-gradient(circle at top left, #3498db, #2ecc71); }
.element { background: radial-gradient(ellipse 50% 50% at center, #3498db, #2ecc71); }
```

---

**圆锥渐变**：圆锥渐变背景
`background: conic-gradient(from <角度> at <位置>, <颜色1>, <颜色2>);`
```css
.element { background: conic-gradient(from 0deg, #3498db, #2ecc71, #e74c3c, #3498db); }
.element { background: conic-gradient(from 90deg at 50% 50%, #3498db 0%, #2ecc71 25%, #e74c3c 50%, #3498db 100%); }
```

## 背景实战

**视差滚动**：固定背景实现视差
`background-attachment: fixed;`
```css
.parallax {
  height: 400px;
  background-image: url('background.jpg');
  background-attachment: fixed;
  background-position: center;
  background-size: cover;
}
```

---

**渐变文字**：渐变背景裁剪到文字
`background-clip: text; color: transparent;`
```css
.gradient-text {
  background: linear-gradient(45deg, #3498db, #9b59b6, #e74c3c);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
  -webkit-text-fill-color: transparent;
}
```

---

**毛玻璃效果**：backdrop-filter 实现毛玻璃
`backdrop-filter: blur(<半径>);`
```css
.glass {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.3);
}
```

---

**图案背景**：使用渐变创建图案
`background: <渐变> <重复>;`
```css
/* 网格图案 */
.grid-pattern {
  background-image:
    linear-gradient(rgba(0, 0, 0, 0.1) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0, 0, 0, 0.1) 1px, transparent 1px);
  background-size: 20px 20px;
}

/* 斜纹图案 */
.stripes {
  background: repeating-linear-gradient(
    45deg,
    #3498db,
    #3498db 10px,
    #2980b9 10px,
    #2980b9 20px
  );
}

/* 圆点图案 */
.dots {
  background-image: radial-gradient(circle, #3498db 1px, transparent 1px);
  background-size: 20px 20px;
}
```
