# CSS 边框圆角

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## border-radius 基础

**border-radius 简写**：设置四个角的圆角
`border-radius: <左上> <右上> <右下> <左下>;`
```css
/* 四个角相同 */
.element { border-radius: 10px; }

/* 左上右下 10px，右上左下 20px */
.element { border-radius: 10px 20px; }

/* 左上 10px，右上左下 20px，右下 30px */
.element { border-radius: 10px 20px 30px; }

/* 分别设置四个角 */
.element { border-radius: 10px 20px 30px 40px; }

/* 单独设置 */
.element {
  border-top-left-radius: 10px;
  border-top-right-radius: 20px;
  border-bottom-right-radius: 30px;
  border-bottom-left-radius: 40px;
}
```

---

**百分比圆角**：使用百分比设置圆角
`border-radius: <百分比>;`
```css
/* 50% 实现圆形 */
.circle {
  width: 100px;
  height: 100px;
  border-radius: 50%;
}

/* 椭圆 */
.ellipse {
  width: 200px;
  height: 100px;
  border-radius: 50%;
}

/* 部分圆角 */
.partial {
  border-radius: 50% 50% 0 0;  /* 上半圆 */
  border-radius: 0 0 50% 50%;  /* 下半圆 */
}
```

---

**椭圆圆角**：分别设置水平和垂直半径
`border-radius: <水平半径> / <垂直半径>;`
```css
/* 椭圆圆角 */
.element {
  border-radius: 50px / 25px;
}

/* 分别设置每个角的椭圆 */
.element {
  border-radius: 50px 30px 20px 10px / 25px 15px 10px 5px;
}
```

## 边框样式

**border-style**：边框样式
`border-style: solid | dashed | dotted | double | groove | ridge | inset | outset | none;`
```css
.element { border-style: solid; }    /* 实线 */
.element { border-style: dashed; } /* 虚线 */
.element { border-style: dotted; } /* 点线 */
.element { border-style: double; }  /* 双线 */
.element { border-style: groove; } /* 凹槽 */
.element { border-style: ridge; }  /* 凸脊 */
.element { border-style: inset; }   /* 内陷 */
.element { border-style: outset; }  /* 外凸 */
.element { border-style: none; }    /* 无边框 */
```

---

**单边边框**：设置单个边的样式
`border-<方向>: <宽度> <样式> <颜色>;`
```css
.element {
  border-top: 2px solid #333;
  border-right: 1px dashed #999;
  border-bottom: 3px double #666;
  border-left: 1px dotted #ccc;
}
```

---

## 边框颜色与宽度

**border-color**：边框颜色
`border-color: <颜色>;`
```css
.element { border-color: #3498db; }
.element { border-color: rgba(52, 152, 219, 0.5); }

/* 四边不同颜色 */
.element {
  border-color: #3498db #2ecc71 #e74c3c #f39c12;
}
```

---

**border-width**：边框宽度
`border-width: <宽度>;`
```css
.element { border-width: 1px; }
.element { border-width: 2px 4px; }
.element { border-width: thin; }    /* 细 */
.element { border-width: medium; }  /* 中 */
.element { border-width: thick; }   /* 粗 */
```

---

## 高级边框

**border-image**：使用图片作为边框
`border-image: <图片源> <切片> <重复方式>;`
```css
.element {
  border: 20px solid transparent;
  border-image: url('border.png') 30 round;
}

/* 拉伸方式 */
.element {
  border-image: url('border.png') 30 stretch;
}

/* 完整语法 */
.element {
  border-image-source: url('border.png');
  border-image-slice: 30;
  border-image-width: 20px;
  border-image-outset: 0;
  border-image-repeat: round;
}
```

---

**outline 轮廓**：不占空间的边框
`outline: <宽度> <样式> <颜色>;`
```css
.element {
  outline: 2px solid #3498db;
  outline-offset: 4px; /* 轮廓与边框的距离 */
}

/* 移除默认轮廓 */
.element:focus {
  outline: none;
}
```

## 实战效果

**渐变边框**：使用 border-image 实现渐变边框
`border: <宽度> solid; border-image: <渐变> <切片>;`
```css
.gradient-border {
  border: 4px solid;
  border-image: linear-gradient(45deg, #3498db, #e74c3c) 1;
}
```

---

**渐变边框（mask 方式）**：使用 mask 实现渐变边框
`border: <宽度> solid transparent; background: <渐变>; mask: <遮罩>;`
```css
.gradient-border-mask {
  position: relative;
  background: white;
  border: 4px solid transparent;
  border-radius: 8px;
  background-clip: padding-box;
}

.gradient-border-mask::before {
  content: '';
  position: absolute;
  inset: -4px;
  border-radius: inherit;
  background: linear-gradient(45deg, #3498db, #e74c3c);
  z-index: -1;
}
```

---

**虚线动画**：使用 stroke-dasharray 实现路径动画
`stroke-dasharray: <长度>; stroke-dashoffset: <偏移>;`
```css
.animated-path {
  stroke-dasharray: 1000;
  stroke-dashoffset: 1000;
  animation: drawPath 2s forwards;
}

@keyframes drawPath {
  to {
    stroke-dashoffset: 0;
  }
}
```

---

**三角形**：使用 border 创建三角形
`border-<方向>: <宽度> solid <颜色>;`
```css
/* 向上的三角形 */
.triangle-up {
  width: 0;
  height: 0;
  border-left: 50px solid transparent;
  border-right: 50px solid transparent;
  border-bottom: 100px solid #3498db;
}

/* 向下的三角形 */
.triangle-down {
  width: 0;
  height: 0;
  border-left: 50px solid transparent;
  border-right: 50px solid transparent;
  border-top: 100px solid #3498db;
}

/* 向右的三角形 */
.triangle-right {
  width: 0;
  height: 0;
  border-top: 50px solid transparent;
  border-bottom: 50px solid transparent;
  border-left: 100px solid #3498db;
}
```

---

**不规则圆角**：创建有机形状
`border-radius: <百分比> <百分比> <百分比> <百分比> / <百分比> <百分比> <百分比> <百分比>;`
```css
/* 有机形状 */
.blob {
  width: 200px;
  height: 200px;
  background: #3498db;
  border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;
}

/* 叶子形状 */
.leaf {
  width: 200px;
  height: 200px;
  background: #2ecc71;
  border-radius: 0 100% 0 100%;
}
```
