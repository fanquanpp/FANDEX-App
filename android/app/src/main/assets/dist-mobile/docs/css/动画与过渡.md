# CSS 动画与过渡

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 过渡 Transition

**transition 简写**：属性过渡效果
`transition: <属性> <持续时间> <时序函数> <延迟>;`
```css
/* 单个属性过渡 */
.element {
  transition: background-color 0.3s ease-in-out 0.1s;
}

/* 多个属性过渡 */
.element {
  transition:
    background-color 0.3s ease-in-out 0.1s,
    transform 0.3s ease-in-out 0.1s;
}

/* 所有属性过渡 */
.element {
  transition: all 0.3s ease;
}

.element:hover {
  background-color: #3498db;
  transform: scale(1.05);
}
```

---

**transition 分拆属性**：分别设置过渡参数
`transition-property: <属性>; transition-duration: <时间>; transition-timing-function: <函数>; transition-delay: <时间>;`
```css
.element {
  transition-property: background-color, transform;
  transition-duration: 0.3s;
  transition-timing-function: ease-in-out;
  transition-delay: 0.1s;
}
```

---

**时序函数**：控制过渡速度曲线
`transition-timing-function: <函数>;`
```css
/* 预定义时序函数 */
.box1 { transition-timing-function: ease; }        /* 默认：慢-快-慢 */
.box2 { transition-timing-function: linear; }      /* 匀速 */
.box3 { transition-timing-function: ease-in; }     /* 慢-快 */
.box4 { transition-timing-function: ease-out; }     /* 快-慢 */
.box5 { transition-timing-function: ease-in-out; }  /* 慢-快-慢 */

/* 贝塞尔曲线 */
.box6 { transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); }

/* 步进函数 */
.box7 { transition-timing-function: steps(4, end); }
.box8 { transition-timing-function: steps(10, start); }
```

---

**可过渡属性**：支持过渡的 CSS 属性
`transition: <可过渡属性> <时间>;`
```css
/* 支持过渡的常见属性 */
.supported {
  /* 颜色 */
  transition: color 0.3s, background-color 0.3s, border-color 0.3s;
  /* 尺寸 */
  transition: width 0.3s, height 0.3s, margin 0.3s, padding 0.3s;
  /* 变换 */
  transition: transform 0.3s, opacity 0.3s;
  /* 阴影 */
  transition: box-shadow 0.3s, text-shadow 0.3s;
}

/* display:none 无法过渡，使用 opacity + visibility 替代 */
.fade-element {
  opacity: 0;
  visibility: hidden;
  transition: opacity 0.3s ease, visibility 0.3s ease;
}
.fade-element.visible {
  opacity: 1;
  visibility: visible;
}
```

## 动画 Animation

**@keyframes 关键帧**：定义动画关键帧
`@keyframes <名称> { from { <样式> } to { <样式> } }`
```css
/* 定义关键帧 */
@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* 多关键帧动画 */
@keyframes bounce {
  0% { transform: translateY(0); }
  25% { transform: translateY(-20px); }
  50% { transform: translateY(0); }
  75% { transform: translateY(-10px); }
  100% { transform: translateY(0); }
}
```

---

**animation 简写**：动画复合属性
`animation: <名称> <持续时间> <时序函数> <延迟> <次数> <方向> <填充模式> <播放状态>;`
```css
/* 应用动画 */
.slide-in {
  animation: slideIn 0.5s ease-out forwards;
}

.bounce {
  animation: bounce 1s ease infinite;
}

/* 完整简写 */
.animation-demo {
  animation: slideIn 0.5s ease-out 0.2s 1 normal forwards;
}
```

---

**animation 分拆属性**：分别设置动画参数
`animation-name: <名称>; animation-duration: <时间>; animation-iteration-count: <次数>;`
```css
.animation-demo {
  animation-name: slideIn;
  animation-duration: 0.5s;
  animation-timing-function: ease-out;
  animation-delay: 0.2s;
  animation-iteration-count: 1;          /* 数字 | infinite */
  animation-direction: normal;           /* normal | reverse | alternate | alternate-reverse */
  animation-fill-mode: forwards;         /* none | forwards | backwards | both */
  animation-play-state: running;         /* running | paused */
}
```

---

**animation-fill-mode**：动画填充模式
`animation-fill-mode: none | forwards | backwards | both;`
```css
/* none：动画前后都应用原始样式 */
.fill-none { animation-fill-mode: none; }

/* forwards：动画结束后保持最后一帧 */
.fill-forwards { animation-fill-mode: forwards; }

/* backwards：动画延迟期间应用第一帧 */
.fill-backwards { animation-fill-mode: backwards; }

/* both：同时应用 forwards 和 backwards */
.fill-both { animation-fill-mode: both; }
```

## 实用动画效果

**加载旋转动画**：无限旋转
`@keyframes spin { to { transform: rotate(360deg); } }`
```css
@keyframes spin {
  to { transform: rotate(360deg); }
}

.spinner {
  width: 40px;
  height: 40px;
  border: 4px solid #e0e0e0;
  border-top-color: #3498db;
  border-radius: 50%;
  animation: spin 0.8s linear infinite;
}
```

---

**脉冲动画**：缩放透明度变化
`@keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.05); } }`
```css
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.05);
    opacity: 0.8;
  }
}

.pulse {
  animation: pulse 2s ease-in-out infinite;
}
```

---

**打字机效果**：宽度变化 + 闪烁光标
`@keyframes typing { from { width: 0; } to { width: 100%; } }`
```css
@keyframes typing {
  from { width: 0; }
  to { width: 100%; }
}

@keyframes blink {
  50% { border-color: transparent; }
}

.typewriter {
  overflow: hidden;
  white-space: nowrap;
  border-right: 2px solid #333;
  width: 0;
  animation:
    typing 3s steps(20) forwards,
    blink 0.7s step-end infinite;
}
```

---

**摇晃动画**：左右晃动
`@keyframes shake { 0%, 100% { transform: translateX(0); } }`
```css
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  10%, 30%, 50%, 70%, 90% { transform: translateX(-5px); }
  20%, 40%, 60%, 80% { transform: translateX(5px); }
}

.shake {
  animation: shake 0.5s ease-in-out;
}
```

## 变换 Transform

**2D 变换**：平移、旋转、缩放、倾斜
`transform: <函数>(<参数>);`
```css
.transform-demo {
  /* 平移 */
  transform: translate(50px, 100px);
  transform: translateX(50px);
  transform: translateY(100px);
  transform: translate(-50%, -50%);

  /* 旋转 */
  transform: rotate(45deg);
  transform: rotate(-0.25turn);

  /* 缩放 */
  transform: scale(1.5);
  transform: scale(1.5, 2);
  transform: scaleX(2);

  /* 倾斜 */
  transform: skew(10deg, 20deg);
  transform: skewX(10deg);

  /* 组合变换（从右到左应用） */
  transform: translate(50px, 0) rotate(45deg) scale(1.2);
}
```

---

**3D 变换**：三维空间变换
`transform: rotateX(<角度>); | transform: rotateY(<角度>); | transform: translateZ(<长度>);`
```css
.transform-3d {
  transform-style: preserve-3d;
  perspective: 1000px;

  /* 3D 旋转 */
  transform: rotateX(45deg);
  transform: rotateY(45deg);
  transform: rotate3d(1, 1, 0, 45deg);

  /* 3D 平移 */
  transform: translateZ(100px);
}
```

---

**翻转卡片**：3D 翻转效果
`transform: rotateY(180deg); backface-visibility: hidden;`
```css
.flip-card {
  width: 300px;
  height: 200px;
  perspective: 1000px;
}
.flip-card-inner {
  position: relative;
  width: 100%;
  height: 100%;
  transition: transform 0.6s;
  transform-style: preserve-3d;
}
.flip-card:hover .flip-card-inner {
  transform: rotateY(180deg);
}
.flip-card-front,
.flip-card-back {
  position: absolute;
  width: 100%;
  height: 100%;
  backface-visibility: hidden;
}
.flip-card-back {
  transform: rotateY(180deg);
}
```

---

**transform-origin**：变换原点
`transform-origin: <x> <y>;`
```css
/* 变换原点 */
.origin-center { transform-origin: center center; }   /* 默认 */
.origin-top-left { transform-origin: top left; }
.origin-custom { transform-origin: 30% 70%; }
.origin-pixel { transform-origin: 50px 100px; }

/* 不同原点下的旋转效果 */
.rotate-center {
  transform-origin: center;
  transform: rotate(45deg);
}
.rotate-corner {
  transform-origin: bottom right;
  transform: rotate(45deg);
}
```

## 性能优化

**高性能动画属性**：使用 GPU 加速属性
`transition: transform <时间>, opacity <时间>;`
```css
/* 推荐：仅触发 Composite 的属性（GPU 加速） */
.good-animation {
  transition: transform 0.3s ease, opacity 0.3s ease;
}

/* 避免：触发 Layout 的属性（性能差） */
.bad-animation {
  transition: width 0.3s, height 0.3s, top 0.3s, left 0.3s;
}
```

---

**will-change 提示**：提前告知浏览器优化
`will-change: <属性>;`
```css
/* 提示浏览器提前优化 */
.will-animate {
  will-change: transform, opacity;
}
```

---

**prefers-reduced-motion**：尊重用户减少动画偏好
`@media (prefers-reduced-motion: reduce) { <选择器> { animation-duration: 0.01ms; } }`
```css
/* 尊重用户的减少动画偏好 */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```
