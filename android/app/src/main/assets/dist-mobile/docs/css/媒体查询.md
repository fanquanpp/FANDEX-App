# CSS 媒体查询

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## @media 语法

**基本语法**：媒体查询基本结构
`@media <类型> and (<特性>) { <样式> }`
```css
@media screen and (min-width: 768px) {
  .container {
    max-width: 720px;
  }
}
```

---

**媒体类型**：指定目标设备类型
`@media <类型> { <样式> }`
```css
/* all：所有设备（默认） */
@media all { }

/* screen：屏幕设备 */
@media screen { }

/* print：打印设备 */
@media print {
  body { font-size: 12pt; }
}

/* speech：语音合成器 */
@media speech { }
```

---

**逻辑操作符**：组合媒体查询条件
`@media (<条件1>) and (<条件2>) { } | @media (<条件1>), (<条件2>) { } | @media not (<条件>) { } | @media only (<条件>) { }`
```css
/* and：同时满足 */
@media (min-width: 768px) and (max-width: 1024px) { }

/* 逗号（or）：满足任一 */
@media (max-width: 480px), (min-width: 1200px) { }

/* not：取反 */
@media not screen { }

/* only：仅支持媒体查询的浏览器 */
@media only screen and (min-width: 768px) { }
```

## 媒体特性

**视口尺寸**：基于视口宽高查询
`@media (min-width: <值>) | @media (max-width: <值>) | @media (min-height: <值>)`
```css
/* 视口宽度 */
@media (min-width: 768px) { }
@media (max-width: 1024px) { }

/* 视口高度 */
@media (min-height: 600px) { }
@media (max-height: 800px) { }

/* 同时限制宽高 */
@media (min-width: 768px) and (max-width: 1024px) and (orientation: landscape) { }
```

---

**方向**：设备方向查询
`@media (orientation: portrait | landscape) { }`
```css
/* 竖屏 */
@media (orientation: portrait) {
  .layout { flex-direction: column; }
}

/* 横屏 */
@media (orientation: landscape) {
  .layout { flex-direction: row; }
}
```

---

**深色模式**：检测系统颜色方案
`@media (prefers-color-scheme: dark | light) { }`
```css
/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #1a1a1a;
    --text: #e0e0e0;
  }
}

/* 浅色模式 */
@media (prefers-color-scheme: light) {
  :root {
    --bg: #ffffff;
    --text: #333333;
  }
}
```

---

**减少动画**：检测用户动画偏好
`@media (prefers-reduced-motion: reduce) { }`
```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

---

**悬停能力**：检测设备是否支持悬停
`@media (hover: hover | none) { }`
```css
/* 支持悬停的设备 */
@media (hover: hover) {
  .button:hover {
    background-color: #2980b9;
  }
}

/* 不支持悬停的设备（触摸屏） */
@media (hover: none) {
  .button:active {
    background-color: #2980b9;
  }
}
```

---

**指针精度**：检测指针设备精度
`@media (pointer: fine | coarse | none) { }`
```css
/* 精确指针（鼠标） */
@media (pointer: fine) {
  .tooltip { display: block; }
}

/* 粗略指针（触摸屏） */
@media (pointer: coarse) {
  .button { padding: 12px 24px; }
}
```

## 响应式断点

**常用断点**：标准断点设置
`@media (min-width: <断点>) { <样式> }`
```css
/* 移动设备优先的断点 */
.container {
  padding: 1rem;
}

/* 小屏幕 */
@media (min-width: 576px) {
  .container { padding: 1.5rem; }
}

/* 中等屏幕 */
@media (min-width: 768px) {
  .container { padding: 2rem; }
}

/* 大屏幕 */
@media (min-width: 992px) {
  .container {
    max-width: 960px;
    margin: 0 auto;
  }
}

/* 超大屏幕 */
@media (min-width: 1200px) {
  .container { max-width: 1140px; }
}
```

## 深色模式实现

**CSS 变量切换**：使用变量实现主题切换
`:root { --<变量>: <值>; } @media (prefers-color-scheme: dark) { :root { --<变量>: <值>; } }`
```css
/* 默认浅色主题 */
:root {
  --bg: #ffffff;
  --text: #333333;
  --border: #e0e0e0;
  --accent: #3498db;
}

/* 深色主题 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #1a1a2e;
    --text: #e0e0e0;
    --border: #333355;
    --accent: #5dade2;
  }
}

body {
  background-color: var(--bg);
  color: var(--text);
}
```

---

**手动主题切换**：使用 data 属性切换
`[data-theme="dark"] { --<变量>: <值>; }`
```css
/* 浅色主题（默认） */
:root {
  --bg: #ffffff;
  --text: #333333;
}

/* 深色主题 */
[data-theme="dark"] {
  --bg: #1a1a2e;
  --text: #e0e0e0;
}

/* 系统偏好检测（无手动设置时） */
@media (prefers-color-scheme: dark) {
  :root:not([data-theme]) {
    --bg: #1a1a2e;
    --text: #e0e0e0;
  }
}
```

## JavaScript 检测

**matchMedia**：JS 中检测媒体查询
`window.matchMedia('(<条件>)').matches`
```javascript
// 检测深色模式
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;

// 监听变化
window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', (e) => {
  if (e.matches) {
    document.documentElement.setAttribute('data-theme', 'dark');
  } else {
    document.documentElement.setAttribute('data-theme', 'light');
  }
});
```

## @import 媒体查询

**条件导入**：根据条件加载样式表
`@import url('<文件>') (<条件>);`
```css
/* 仅在屏幕宽度大于 768px 时加载 */
@import url('desktop.css') screen and (min-width: 768px);

/* 仅在打印时加载 */
@import url('print.css') print;
```
