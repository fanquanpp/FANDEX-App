# CSS 选择器系统

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基础选择器

**元素选择器**：匹配所有指定标签
`<标签名> { <样式> }`
```css
/* 匹配所有段落 */
p {
  margin: 0;
  line-height: 1.6;
}

/* 匹配所有 div */
div {
  box-sizing: border-box;
}
```

---

**类选择器**：匹配指定 class 的元素
`.<类名> { <样式> }`
```css
/* 匹配 class="card" 的元素 */
.card {
  background: white;
  border-radius: 8px;
}

/* 多类组合 */
.card.highlighted {
  border-color: #3498db;
}
```

---

**ID 选择器**：匹配指定 id 的元素
`#<ID> { <样式> }`
```css
/* 匹配 id="header" 的元素 */
#header {
  position: sticky;
  top: 0;
  z-index: 100;
}
```

---

**通配选择器**：匹配所有元素
`* { <样式> }`
```css
/* 重置所有元素 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* 匹配某元素的所有后代 */
.card * {
  font-size: 14px;
}
```

---

## 组合选择器

**后代选择器**：匹配所有层级后代
`<父> <后代> { <样式> }`
```css
/* 匹配 .nav 内所有 a 元素（任意层级） */
.nav a {
  color: #333;
  text-decoration: none;
}
```

---

**子代选择器**：仅匹配直接子元素
`<父> > <子> { <样式> }`
```css
/* 仅匹配 .list 的直接 li 子元素 */
.list > li {
  list-style: none;
}
```

---

**相邻兄弟选择器**：匹配紧邻的下一个兄弟
`<前> + <后> { <样式> }`
```css
/* 匹配 h1 后紧邻的 p */
h1 + p {
  margin-top: 0;
}
```

---

**通用兄弟选择器**：匹配后续所有兄弟
`<前> ~ <后> { <样式> }`
```css
/* 匹配 h1 后所有的 p */
h1 ~ p {
  color: #666;
}
```

---

**分组选择器**：同时匹配多个选择器
`<选择器1>, <选择器2> { <样式> }`
```css
/* 同时匹配 h1、h2、h3 */
h1, h2, h3 {
  font-weight: bold;
  color: #222;
}
```

---

## 属性选择器

**存在属性匹配**：匹配具有指定属性的元素
`<元素>[<属性>] { <样式> }`
```css
/* 匹配具有 disabled 属性的 input */
input[disabled] {
  opacity: 0.5;
  cursor: not-allowed;
}

/* 匹配具有 target 属性的 a */
a[target] {
  background: yellow;
}
```

---

**精确值匹配**：属性值完全等于指定值
`<元素>[<属性>="<值>"] { <样式> }`
```css
/* 匹配 type="text" 的 input */
input[type="text"] {
  border: 1px solid #ddd;
  padding: 8px;
}

/* 匹配 href 等于指定 URL */
a[href="https://example.com"] {
  color: red;
}
```

---

**前缀匹配**：属性值以指定值开头
`<元素>[<属性>^="<前缀>"] { <样式> }`
```css
/* 匹配 https 开头的链接 */
a[href^="https"] {
  color: green;
}

/* 匹配 tel: 开头的链接 */
a[href^="tel:"] {
  text-decoration: none;
}
```

---

**后缀匹配**：属性值以指定值结尾
`<元素>[<属性>$="<后缀>"] { <样式> }`
```css
/* 匹配 .pdf 结尾的链接 */
a[href$=".pdf"] {
  background: url('pdf-icon.png') no-repeat left center;
  padding-left: 20px;
}
```

---

**包含匹配**：属性值包含指定子串
`<元素>[<属性>*="<子串>"] { <样式> }`
```css
/* 匹配 class 包含 "button" 的元素 */
div[class*="button"] {
  cursor: pointer;
}
```

---

**空格分隔匹配**：属性值是以空格分隔的列表且包含指定值
`<元素>[<属性>~="<值>"] { <样式> }`
```css
/* 匹配 class 列表中包含 "active" 的元素 */
div[class~="active"] {
  font-weight: bold;
}
```

---

**连字符分隔匹配**：属性值以连字符分隔且以指定值开头
`<元素>[<属性>|="<值>"] { <样式> }`
```css
/* 匹配 lang 属性以 "zh" 开头（如 zh-CN、zh-TW） */
html[lang|="zh"] {
  font-family: "Microsoft YaHei", sans-serif;
}
```

## 伪类选择器

**交互状态伪类**：匹配用户交互状态
`<元素>:hover | <元素>:focus | <元素>:active | <元素>:visited`
```css
/* 鼠标悬停 */
a:hover {
  color: #3498db;
  text-decoration: underline;
}

/* 获得焦点 */
input:focus {
  border-color: #3498db;
  outline: none;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
}

/* 键盘聚焦可见 */
input:focus-visible {
  outline: 2px solid #3498db;
}

/* 激活（按下） */
button:active {
  transform: scale(0.98);
}

/* 已访问链接 */
a:visited {
  color: #9b59b6;
}
```

---

**表单状态伪类**：匹配表单元素状态
`<元素>:checked | <元素>:disabled | <元素>:enabled | <元素>:valid | <元素>:invalid`
```css
/* 选中的复选框 */
input[type="checkbox"]:checked + label {
  color: #2ecc71;
  font-weight: bold;
}

/* 禁用的输入框 */
input:disabled {
  background: #f5f5f5;
  cursor: not-allowed;
}

/* 启用的输入框 */
input:enabled {
  background: white;
}

/* 校验通过 */
input:valid {
  border-color: #2ecc71;
}

/* 校验失败 */
input:invalid {
  border-color: #e74c3c;
}
```

---

**位置伪类**：基于 DOM 位置匹配
`<元素>:first-child | <元素>:last-child | <元素>:only-child | <元素>:nth-child(<公式>)`
```css
/* 第一个子元素 */
li:first-child {
  border-top: none;
}

/* 最后一个子元素 */
li:last-child {
  border-bottom: none;
}

/* 唯一子元素 */
div:only-child {
  margin: 0 auto;
}

/* 奇数行 */
tr:nth-child(odd) {
  background: #f9f9f9;
}

/* 偶数行 */
tr:nth-child(even) {
  background: white;
}

/* 第 3 个 */
li:nth-child(3) {
  color: red;
}

/* 每 3 个选第 1 个 */
li:nth-child(3n + 1) {
  color: blue;
}

/* 前 3 个 */
li:nth-child(-n + 3) {
  font-weight: bold;
}
```

---

**类型位置伪类**：基于同类型元素的位置匹配
`<元素>:first-of-type | <元素>:last-of-type | <元素>:nth-of-type(<公式>)`
```css
/* 第一个段落 */
p:first-of-type {
  margin-top: 0;
}

/* 最后一个段落 */
p:last-of-type {
  margin-bottom: 0;
}

/* 每 2 个段落的第 1 个 */
p:nth-of-type(2n + 1) {
  background: #f0f0f0;
}
```

---

**空元素伪类**：匹配没有子元素的元素
`<元素>:empty`
```css
/* 匹配没有内容的 div */
div:empty {
  display: none;
}

/* 匹配空段落 */
p:empty::before {
  content: "（空内容）";
  color: #999;
}
```

---

**否定伪类**：排除匹配指定条件的元素
`<元素>:not(<选择器>)`
```css
/* 匹配非最后一个 li */
li:not(:last-child) {
  border-bottom: 1px solid #ddd;
}

/* 匹配非特殊 class 的 div */
div:not(.special) {
  padding: 10px;
}

/* 匹配非禁用的 input */
input:not(:disabled) {
  border-color: #ccc;
}
```

---

**匹配伪类**：复杂条件匹配
`:is(<选择器列表>) | :where(<选择器列表>) | :has(<选择器>)`
```css
/* is：匹配列表中任一选择器，优先级取最高 */
:is(h1, h2, h3):hover {
  color: #3498db;
}

/* 匹配深层结构 */
:is(.card, .panel) :is(.title, .heading) {
  font-weight: bold;
}

/* where：与 is 相同，但优先级始终为 0 */
:where(h1, h2, h3) {
  margin: 0;
}

/* has：匹配包含指定后代的元素 */
a:has(> img) {
  border: none;
}

/* 匹配包含选中复选框的 div */
div:has(input:checked) {
  background: #e8f5e9;
}
```

---

**焦点伪类**：匹配包含焦点元素的容器
`<元素>:focus-within`
```css
/* 当表单内任意元素获得焦点时 */
form:focus-within {
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
}

/* 当搜索框内 input 获得焦点时 */
.search:focus-within {
  border-color: #3498db;
}
```

---

**根伪类**：匹配文档根元素
`:root`
```css
/* 定义全局 CSS 变量 */
:root {
  --primary-color: #3498db;
  --font-size-base: 16px;
  --spacing-unit: 8px;
}
```

## 伪元素选择器

**内容伪元素**：在元素前后插入内容
`<元素>::before | <元素>::after`
```css
/* 在引用前插入左引号 */
.quote::before {
  content: "\201C";
  font-size: 2em;
  color: #999;
}

/* 在必填字段后插入星号 */
.required::after {
  content: " *";
  color: #e74c3c;
}

/* 清除浮动 */
.clearfix::after {
  content: "";
  display: table;
  clear: both;
}
```

---

**首行首字母伪元素**：匹配首行或首字母
`<元素>::first-line | <元素>::first-letter`
```css
/* 段落首行加粗 */
p::first-line {
  font-weight: bold;
  color: #333;
}

/* 段落首字母放大 */
p::first-letter {
  font-size: 3em;
  float: left;
  line-height: 1;
  margin-right: 5px;
}
```

---

**选中文本伪元素**：匹配用户选中的文本
`::selection`
```css
/* 自定义选中文本样式 */
::selection {
  background: #3498db;
  color: white;
}

/* 特定元素的选中样式 */
.highlight::selection {
  background: #e74c3c;
}
```

---

**占位符伪元素**：匹配输入框占位符
`<元素>::placeholder`
```css
/* 自定义 placeholder 样式 */
input::placeholder {
  color: #999;
  font-style: italic;
}

/* 不同浏览器前缀 */
input::-webkit-input-placeholder {
  color: #999;
}
input::-moz-placeholder {
  color: #999;
}
```

---

## 选择器优先级

**优先级计算**：内联 > ID > 类/属性/伪类 > 元素/伪元素
`<内联> > #<ID> > .<类> > <元素>`
```css
/* 优先级从低到高 */
* { }              /* 0,0,0 */
p { }              /* 0,0,1 */
.card { }          /* 0,1,0 */
#header { }        /* 1,0,0 */
style="..."       /* 1,0,0,0 */

/* 组合优先级 */
.card p { }        /* 0,1,1 */
#nav .item { }     /* 1,1,0 */
#nav .item:hover { } /* 1,2,0 */

/* !important 覆盖优先级 */
.button {
  color: white !important;
}
```
