# CSS 响应式设计

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 媒体查询

**基本语法**：根据条件应用样式
`@media (<条件>) { <选择器> { <样式> } }`
```css
@media (max-width: 767px) {
  /* 移动设备样式 */
  .container {
    padding: 1rem;
  }
}
```

---

**断点设置**：常用响应式断点
`@media (min-width: <宽度>) and (max-width: <宽度>) { <样式> }`
```css
/* 移动设备 */
@media (max-width: 767px) {
  .container { padding: 1rem; }
}

/* 平板设备 */
@media (min-width: 768px) and (max-width: 1023px) {
  .container { padding: 1.5rem; }
}

/* 桌面设备 */
@media (min-width: 1024px) {
  .container {
    max-width: 960px;
    margin: 0 auto;
    padding: 2rem;
  }
}
```

---

**常用媒体特性**：检测设备特性
`@media (<特性>: <值>) { <样式> }`
```css
/* 视口宽度 */
@media (min-width: 768px) { }

/* 设备方向 */
@media (orientation: portrait) { }
@media (orientation: landscape) { }

/* 深色模式 */
@media (prefers-color-scheme: dark) { }

/* 减少动画 */
@media (prefers-reduced-motion: reduce) { }

/* 支持悬停 */
@media (hover: hover) { }

/* 精确指针 */
@media (pointer: fine) { }
```

## 弹性布局

**弹性网格系统**：使用 flex 实现响应式
`display: flex; flex-wrap: wrap; gap: <间距>;`
```css
.container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}
.item {
  flex: 1 1 300px;
}
```

---

**Grid 响应式**：使用 auto-fit 自动适应
`grid-template-columns: repeat(auto-fit, minmax(<最小宽度>, 1fr));`
```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}
```

## 响应式图像

**自适应图像**：图像随容器缩放
`img { max-width: 100%; height: auto; }`
```css
img {
  max-width: 100%;
  height: auto;
}
```

---

**picture 元素**：根据屏幕加载不同图片
`<picture><source media="(<条件>)" srcset="<图片>"><img src="<默认图片>"></picture>`
```html
<picture>
  <source media="(max-width: 768px)" srcset="small-image.jpg" />
  <source media="(min-width: 769px)" srcset="large-image.jpg" />
  <img src="fallback-image.jpg" alt="描述" />
</picture>
```

## 响应式排版

**相对字体单位**：使用 rem 和媒体查询
`:root { font-size: <值>; }`
```css
:root {
  font-size: 16px;
}

@media (max-width: 768px) {
  :root {
    font-size: 14px;
  }
}

body {
  font-size: 1rem;
}

h1 {
  font-size: 2.5rem;
}
```

---

**clamp 流式排版**：使用 clamp 实现流式字体
`font-size: clamp(<最小值>, <理想值>, <最大值>);`
```css
:root {
  --font-size-base: clamp(1rem, 0.875rem + 0.5vw, 1.25rem);
  --font-size-xl: clamp(1.75rem, 1.25rem + 2vw, 3rem);
}

body {
  font-size: var(--font-size-base);
}

h1 {
  font-size: var(--font-size-xl);
}
```

## 响应式导航

**汉堡菜单**：移动端折叠导航
`@media (max-width: 768px) { .nav-links { display: none; } }`
```css
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
  background: #333;
  color: white;
}
.nav-links {
  display: flex;
  list-style: none;
  gap: 1rem;
}
.menu-toggle {
  display: none;
  cursor: pointer;
}

@media (max-width: 768px) {
  .nav-links {
    position: absolute;
    top: 70px;
    left: 0;
    right: 0;
    background: #333;
    flex-direction: column;
    align-items: center;
    padding: 1rem;
    gap: 1rem;
    display: none;
  }
  .nav-links.active {
    display: flex;
  }
  .menu-toggle {
    display: block;
  }
}
```

## 响应式表格

**可滚动表格**：小屏幕水平滚动
`@media (max-width: 768px) { .table-container { overflow-x: auto; } }`
```css
.table-container {
  width: 100%;
}

@media (max-width: 768px) {
  .table-container {
    overflow-x: auto;
    -webkit-overflow-scrolling: touch;
  }
}
```
