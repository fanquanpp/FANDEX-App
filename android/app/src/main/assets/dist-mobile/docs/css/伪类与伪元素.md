# CSS 伪类与伪元素

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 伪类

**交互状态伪类**：匹配用户交互状态
`<元素>:hover | <元素>:focus | <元素>:active | <元素>:visited`
```css
/* 鼠标悬停 */
a:hover {
  color: #3498db;
  text-decoration: underline;
}

/* 获得焦点 */
input:focus {
  border-color: #3498db;
  outline: none;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
}

/* 键盘聚焦可见 */
input:focus-visible {
  outline: 2px solid #3498db;
}

/* 激活（按下） */
button:active {
  transform: scale(0.98);
}

/* 已访问链接 */
a:visited {
  color: #9b59b6;
}
```

---

**表单状态伪类**：匹配表单元素状态
`<元素>:checked | <元素>:disabled | <元素>:enabled | <元素>:valid | <元素>:invalid`
```css
/* 选中的复选框 */
input[type='checkbox']:checked + label {
  color: #2ecc71;
  font-weight: bold;
}

/* 禁用的输入框 */
input:disabled {
  background: #f5f5f5;
  cursor: not-allowed;
}

/* 启用的输入框 */
input:enabled {
  background: white;
}

/* 校验通过 */
input:valid {
  border-color: #2ecc71;
}

/* 校验失败 */
input:invalid {
  border-color: #e74c3c;
}
```

---

**位置伪类**：基于 DOM 位置匹配
`<元素>:first-child | <元素>:last-child | <元素>:only-child | <元素>:nth-child(<公式>)`
```css
/* 第一个子元素 */
li:first-child {
  border-top: none;
}

/* 最后一个子元素 */
li:last-child {
  border-bottom: none;
}

/* 唯一子元素 */
div:only-child {
  margin: 0 auto;
}

/* 奇数行 */
tr:nth-child(odd) {
  background: #f9f9f9;
}

/* 偶数行 */
tr:nth-child(even) {
  background: white;
}

/* 第 3 个 */
li:nth-child(3) {
  color: red;
}

/* 每 3 个选第 1 个 */
li:nth-child(3n + 1) {
  color: blue;
}

/* 前 3 个 */
li:nth-child(-n + 3) {
  font-weight: bold;
}
```

---

**类型位置伪类**：基于同类型元素的位置匹配
`<元素>:first-of-type | <元素>:last-of-type | <元素>:nth-of-type(<公式>)`
```css
/* 第一个段落 */
p:first-of-type {
  margin-top: 0;
}

/* 最后一个段落 */
p:last-of-type {
  margin-bottom: 0;
}

/* 每 2 个段落的第 1 个 */
p:nth-of-type(2n + 1) {
  background: #f0f0f0;
}
```

---

**空元素伪类**：匹配没有子元素的元素
`<元素>:empty`
```css
/* 匹配没有内容的 div */
div:empty {
  display: none;
}

/* 匹配空段落 */
p:empty::before {
  content: '（空内容）';
  color: #999;
}
```

---

**否定伪类**：排除匹配指定条件的元素
`<元素>:not(<选择器>)`
```css
/* 匹配非最后一个 li */
li:not(:last-child) {
  border-bottom: 1px solid #ddd;
}

/* 匹配非特殊 class 的 div */
div:not(.special) {
  padding: 10px;
}

/* 匹配非禁用的 input */
input:not(:disabled) {
  border-color: #ccc;
}
```

---

**匹配伪类**：复杂条件匹配
`:is(<选择器列表>) | :where(<选择器列表>) | :has(<选择器>)`
```css
/* is：匹配列表中任一选择器，优先级取最高 */
:is(h1, h2, h3):hover {
  color: #3498db;
}

/* 匹配深层结构 */
:is(.card, .panel) :is(.title, .heading) {
  font-weight: bold;
}

/* where：与 is 相同，但优先级始终为 0 */
:where(h1, h2, h3) {
  margin: 0;
}

/* has：匹配包含指定后代的元素 */
a:has(> img) {
  border: none;
}

/* 匹配包含选中复选框的 div */
div:has(input:checked) {
  background: #e8f5e9;
}
```

---

**焦点伪类**：匹配包含焦点元素的容器
`<元素>:focus-within`
```css
/* 当表单内任意元素获得焦点时 */
form:focus-within {
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
}

/* 当搜索框内 input 获得焦点时 */
.search:focus-within {
  border-color: #3498db;
}
```

---

**根伪类**：匹配文档根元素
`:root`
```css
/* 定义全局 CSS 变量 */
:root {
  --primary-color: #3498db;
  --font-size-base: 16px;
  --spacing-unit: 8px;
}
```

## 伪元素

**内容伪元素**：在元素前后插入内容
`<元素>::before | <元素>::after`
```css
/* 在引用前插入左引号 */
.quote::before {
  content: '\201C';
  font-size: 2em;
  color: #999;
}

/* 在必填字段后插入星号 */
.required::after {
  content: ' *';
  color: #e74c3c;
}

/* 清除浮动 */
.clearfix::after {
  content: '';
  display: table;
  clear: both;
}
```

---

**首行首字母伪元素**：匹配首行或首字母
`<元素>::first-line | <元素>::first-letter`
```css
/* 段落首行加粗 */
p::first-line {
  font-weight: bold;
  color: #333;
}

/* 段落首字母放大 */
p::first-letter {
  font-size: 3em;
  float: left;
  line-height: 1;
  margin-right: 5px;
}
```

---

**选中文本伪元素**：匹配用户选中的文本
`::selection`
```css
/* 自定义选中文本样式 */
::selection {
  background: #3498db;
  color: white;
}

/* 特定元素的选中样式 */
.highlight::selection {
  background: #e74c3c;
}
```

---

**占位符伪元素**：匹配输入框占位符
`<元素>::placeholder`
```css
/* 自定义 placeholder 样式 */
input::placeholder {
  color: #999;
  font-style: italic;
}

/* 不同浏览器前缀 */
input::-webkit-input-placeholder {
  color: #999;
}
input::-moz-placeholder {
  color: #999;
}
```

## 实战应用

**自定义复选框**：隐藏原生复选框，自定义样式
`input[type="checkbox"]:checked + <label>`
```css
/* 隐藏原生复选框 */
input[type='checkbox'] {
  appearance: none;
  width: 20px;
  height: 20px;
  border: 2px solid #ccc;
  border-radius: 4px;
  position: relative;
  cursor: pointer;
}

/* 选中状态 */
input[type='checkbox']:checked {
  background-color: #3498db;
  border-color: #3498db;
}

/* 选中标记 */
input[type='checkbox']:checked::after {
  content: '';
  position: absolute;
  left: 6px;
  top: 2px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}
```

---

**tooltip 提示框**：使用 ::before 实现提示
`<元素>::before { content: attr(<属性>); }`
```css
.tooltip {
  position: relative;
  display: inline-block;
}

.tooltip::before {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  padding: 5px 10px;
  background: #333;
  color: white;
  border-radius: 4px;
  white-space: nowrap;
  opacity: 0;
  visibility: hidden;
  transition: opacity 0.3s;
}

.tooltip:hover::before {
  opacity: 1;
  visibility: visible;
}
```

---

**清除浮动**：使用 ::after 清除浮动
`<元素>::after { content: ''; display: table; clear: both; }`
```css
/* 现代清除浮动方案 */
.clearfix::after {
  content: '';
  display: table;
  clear: both;
}

/* 或使用 flow-root */
.container {
  display: flow-root;
}
```

---

**面包屑分隔符**：使用 ::before 添加分隔符
`<元素>::before { content: '<分隔符>'; }`
```css
.breadcrumb {
  list-style: none;
  display: flex;
  padding: 0;
}

.breadcrumb li + li::before {
  content: '/';
  margin: 0 10px;
  color: #999;
}
```
