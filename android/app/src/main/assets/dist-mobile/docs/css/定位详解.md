# CSS 定位详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## position 属性

**static 静态定位**：默认定位方式
`position: static;`
```css
/* 默认值，不脱离文档流 */
.element {
  position: static;
  /* top/left/right/bottom 不生效 */
}
```

---

**relative 相对定位**：相对自身原位置偏移
`position: relative; top: <值>; left: <值>;`
```css
/* 不脱离文档流，原位置保留 */
.element {
  position: relative;
  top: 10px;
  left: 20px;
}

/* 常作 absolute 的参照容器 */
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 0;
  right: 0;
}
```

---

**absolute 绝对定位**：相对最近定位祖先
`position: absolute; top: <值>; left: <值>;`
```css
/* 脱离文档流，参照最近 position 非 static 的祖先 */
.parent {
  position: relative;
  height: 200px;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%); /* 居中 */
}
```

---

**fixed 固定定位**：相对视口固定
`position: fixed; top: <值>; left: <值>;`
```css
/* 固定在视口顶部 */
.header {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  z-index: 100;
}

/* 固定在右下角 */
.back-to-top {
  position: fixed;
  bottom: 20px;
  right: 20px;
}
```

---

**sticky 粘性定位**：阈值前 relative，达到后 fixed
`position: sticky; top: <值>;`
```css
/* 滚动到 top:20px 时固定 */
.sidebar {
  position: sticky;
  top: 20px;
}

/* 表头固定 */
th {
  position: sticky;
  top: 0;
  background: white;
  z-index: 1;
}

/* 底部粘性 */
.footer {
  position: sticky;
  bottom: 0;
}
```

---

## 定位偏移属性

**top/right/bottom/left**：设置定位偏移
`top: <值>; right: <值>; bottom: <值>; left: <值>;`
```css
/* 四个方向偏移 */
.element {
  position: absolute;
  top: 10px;
  right: 20px;
  bottom: 10px;
  left: 20px;
}

/* 撑满父容器 */
.full {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}
```

---

## z-index 层叠顺序

**z-index**：控制定位元素的层叠顺序
`z-index: <数值>;`
```css
/* 数值越大越在上层 */
.modal {
  position: fixed;
  z-index: 1000;
}

.dropdown {
  position: absolute;
  z-index: 100;
}

/* 使用 CSS 变量管理层级 */
:root {
  --z-dropdown: 100;
  --z-sticky: 200;
  --z-modal: 1000;
  --z-toast: 1100;
}

.dropdown {
  z-index: var(--z-dropdown);
}
.modal {
  z-index: var(--z-modal);
}
```

---

**层叠上下文**：创建新的层叠上下文
`position: <定位>; z-index: <数值>; | opacity: <值>; | transform: <值>;`
```css
/* 创建层叠上下文的条件 */
.stacking-context {
  position: relative;
  z-index: 0; /* 创建层叠上下文 */

  /* 其他创建条件 */
  /* opacity: 0.99; */
  /* transform: translateZ(0); */
  /* filter: blur(0); */
  /* isolation: isolate; */
}

/* 子元素的 z-index 仅在父层叠上下文内比较 */
```

## 定位实战

**绝对定位居中**：使用 transform 实现居中
`position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);`
```css
/* 水平垂直居中 */
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 仅水平居中 */
.center-x {
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
```

---

**margin auto 居中**：绝对定位的 margin auto 居中
`position: absolute; top: 0; right: 0; bottom: 0; left: 0; margin: auto;`
```css
/* 需要指定宽高 */
.center-auto {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  width: 200px;
  height: 100px;
  margin: auto;
}
```

---

**tooltip 气泡**：使用伪元素实现提示框
`position: absolute; ::before { position: absolute; }`
```css
.tooltip {
  position: relative;
  display: inline-block;
}

.tooltip::before {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  padding: 5px 10px;
  background: #333;
  color: white;
  border-radius: 4px;
  white-space: nowrap;
  opacity: 0;
  visibility: hidden;
  transition: opacity 0.3s;
}

.tooltip:hover::before {
  opacity: 1;
  visibility: visible;
}
```
