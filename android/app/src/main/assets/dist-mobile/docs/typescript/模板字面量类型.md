# 模板字面量类型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本模板字面量类型

**模板字面量类型：基于字符串模板生成类型**
`type <类型> = \`<前缀>\${<类型>}\``

```typescript
// 基本模板字面量类型
type HelloWorld = `hello ${string}`;
let greeting: HelloWorld = 'hello world';
let greeting2: HelloWorld = 'hello typescript';
// greeting3: HelloWorld = 'hi world'; // 错误
// 组合字面量类型
type Direction = 'left' | 'right';
type Margin = `margin-${Direction}`;
let margin: Margin = 'margin-left';
let margin2: Margin = 'margin-right';
```

## 字符串操作类型

**Uppercase：转换为大写**
`type <类型> = Uppercase<<字符串类型>>`

```typescript
type Upper = Uppercase<'hello'>; // 'HELLO'
type UpperGreeting = Uppercase<'hello world'>; // 'HELLO WORLD'
```

---

**Lowercase：转换为小写**
`type <类型> = Lowercase<<字符串类型>>`

```typescript
type Lower = Lowercase<'HELLO'>; // 'hello'
type LowerGreeting = Lowercase<'HELLO WORLD'>; // 'hello world'
```

---

**Capitalize：首字母大写**
`type <类型> = Capitalize<<字符串类型>>`

```typescript
type Cap = Capitalize<'hello'>; // 'Hello'
type CapGreeting = Capitalize<'hello world'>; // 'Hello world'
```

---

**Uncapitalize：首字母小写**
`type <类型> = Uncapitalize<<字符串类型>>`

```typescript
type Uncap = Uncapitalize<'Hello'>; // 'hello'
type UncapGreeting = Uncapitalize<'Hello World'>; // 'hello World'
```

---

## 组合模板字面量类型

**组合多个字面量类型**
`type <类型> = \`${<类型1>}-${<类型2>}\``

```typescript
// 组合多个字面量类型
type Horizontal = 'left' | 'right';
type Vertical = 'top' | 'bottom';
type Position = `${Vertical}-${Horizontal}`;
// 等价于: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
let position: Position = 'top-left';
let position2: Position = 'bottom-right';
```

## 动态属性名

**动态属性名：使用模板字面量类型作为键**
`type <类型> = { [K in \`prefix-\${string}\`]: <值类型> }`

```typescript
// 动态属性名
type CSSProperties = {
  [K in `--${string}`]?: string;
};
const styles: CSSProperties = {
  '--primary-color': '#2196F3',
  '--font-size': '16px',
  '--margin': '20px',
};
```

## 事件处理器类型

**事件处理器：基于事件名生成类型**
`type <类型> = \`on${Capitalize<string>}\``

```typescript
// 事件处理器类型
type EventName = 'click' | 'change' | 'submit';
type EventHandler = `on${Capitalize<EventName>}`;
// 等价于: 'onClick' | 'onChange' | 'onSubmit'
type EventMap = {
  [K in EventHandler]?: (...args: any[]) => void;
};
const handlers: EventMap = {
  onClick: () => console.log('clicked'),
  onChange: () => console.log('changed'),
  onSubmit: () => console.log('submitted'),
};
```

## Getter/Setter 类型

**Getter 类型：自动生成 getter 方法名**
`type <类型><T> = { [K in keyof T as \`get${Capitalize<string & K>}\`]: () => T[K] }`

```typescript
interface User {
  id: number;
  name: string;
  age: number;
}
// Getter 类型
type Getters<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K];
};
type UserGetters = Getters<User>;
// 等价于: { getId: () => number; getName: () => string; getAge: () => number }
// Setter 类型
type Setters<T> = {
  [K in keyof T as `set${Capitalize<string & K>}`]: (value: T[K]) => void;
};
type UserSetters = Setters<User>;
// 等价于: { setId: (value: number) => void; setName: (value: string) => void; setAge: (value: number) => void }
```

## 路由参数类型

**路由参数：基于路径生成参数类型**
`type <类型> = \`/api/\${string}/\${string}\``

```typescript
// 路由参数类型
type Route = `/api/${string}/${string}`;
let route: Route = '/api/users/123';
let route2: Route = '/api/products/456';
// API 端点类型
type ApiVersion = 'v1' | 'v2';
type ApiEndpoint = `/api/${ApiVersion}/${string}`;
let endpoint: ApiEndpoint = '/api/v1/users';
let endpoint2: ApiEndpoint = '/api/v2/products';
```

## CSS 属性类型

**CSS 属性：基于属性名生成类型**
`type <类型> = \`${string}-\${string}\``

```typescript
// CSS 属性类型
type CSSProperty = `${string}-${string}`;
let prop: CSSProperty = 'background-color';
let prop2: CSSProperty = 'font-size';
let prop3: CSSProperty = 'margin-top';
// CSS 值类型
type CSSValue = string | number;
type CSSRule = {
  [K in CSSProperty]?: CSSValue;
};
const style: CSSRule = {
  'background-color': 'red',
  'font-size': 16,
  'margin-top': '10px',
};
```

## 复杂模板字面量类型

**递归模板字面量类型**
`type <类型> = ...`

```typescript
// 递归替换字符串中的字符
type ReplaceAll<S extends string, From extends string, To extends string> =
  From extends '' ? S :
  S extends `${infer Start}${From}${infer End}` ? `${Start}${To}${ReplaceAll<End, From, To>}` : S;
type Result = ReplaceAll<'hello world', 'o', '0'>; // 'hell0 w0rld'
```

## 提取字符串部分

**提取字符串前缀**
`type <类型><S> = S extends \`${infer Prefix}_${string}\` ? Prefix : never`

```typescript
// 提取字符串前缀
type GetPrefix<S extends string> = S extends `${infer Prefix}_${string}` ? Prefix : never;
type P1 = GetPrefix<'user_name'>; // 'user'
type P2 = GetPrefix<'product_price'>; // 'product'
type P3 = GetPrefix<'hello'>; // never
```

---

**提取字符串后缀**
`type <类型><S> = S extends \`${string}_${infer Suffix}\` ? Suffix : never`

```typescript
// 提取字符串后缀
type GetSuffix<S extends string> = S extends `${string}_${infer Suffix}` ? Suffix : never;
type S1 = GetSuffix<'user_name'>; // 'name'
type S2 = GetSuffix<'product_price'>; // 'price'
type S3 = GetSuffix<'hello'>; // never
```

## 实际应用

**表单字段类型**
`type <类型> = \`form_\${string}\``

```typescript
// 表单字段类型
type FormField = `form_${string}`;
type FormValues = {
  [K in FormField]?: string;
};
const form: FormValues = {
  'form_username': 'Alice',
  'form_password': 'secret',
  'form_email': 'alice@example.com',
};
```

---

**配置键类型**
`type <类型> = \`${string}_\${string}\``

```typescript
// 配置键类型
type ConfigKey = `${string}_${string}`;
type Config = {
  [K in ConfigKey]?: string | number | boolean;
};
const config: Config = {
  'api_url': 'https://api.example.com',
  'api_timeout': 5000,
  'ui_theme': 'dark',
};
```
