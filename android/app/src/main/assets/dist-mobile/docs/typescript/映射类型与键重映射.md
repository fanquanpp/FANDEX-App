# 映射类型与键重映射

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 映射类型基础

**映射类型：基于旧类型创建新类型**
`type <类型><T> = { [P in keyof T]: <新类型> }`

```typescript
// 基本映射类型
type Stringify<T> = {
  [P in keyof T]: string;
};
// 使用
interface User {
  id: number;
  name: string;
  age: number;
}
type StringUser = Stringify<User>;
// 等价于: { id: string; name: string; age: string }
```

## 内置映射类型

**Partial：所有属性变为可选**
`type <类型> = Partial<T>`

```typescript
interface Todo {
  title: string;
  description: string;
  completed: boolean;
}
// Partial 实现
type MyPartial<T> = {
  [P in keyof T]?: T[P];
};
type PartialTodo = MyPartial<Todo>;
// 等价于: { title?: string; description?: string; completed?: boolean }
```

---

**Required：所有属性变为必需**
`type <类型> = Required<T>`

```typescript
interface Todo {
  title?: string;
  description?: string;
  completed?: boolean;
}
// Required 实现
type MyRequired<T> = {
  [P in keyof T]-?: T[P];
};
type RequiredTodo = MyRequired<Todo>;
// 等价于: { title: string; description: string; completed: boolean }
```

---

**Readonly：所有属性变为只读**
`type <类型> = Readonly<T>`

```typescript
interface Todo {
  title: string;
  description: string;
  completed: boolean;
}
// Readonly 实现
type MyReadonly<T> = {
  readonly [P in keyof T]: T[P];
};
type ReadonlyTodo = MyReadonly<Todo>;
// 等价于: { readonly title: string; readonly description: string; readonly completed: boolean }
```

---

**Pick：选择部分属性**
`type <类型> = Pick<T, K>`

```typescript
interface Todo {
  title: string;
  description: string;
  completed: boolean;
}
// Pick 实现
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P];
};
type TodoPreview = MyPick<Todo, 'title' | 'completed'>;
// 等价于: { title: string; completed: boolean }
```

---

**Record：键值对类型**
`type <类型> = Record<K, T>`

```typescript
// Record 实现
type MyRecord<K extends keyof any, T> = {
  [P in K]: T;
};
type StringMap = MyRecord<string, number>;
// 等价于: { [key: string]: number }
type PageMap = MyRecord<'home' | 'about' | 'contact', string>;
// 等价于: { home: string; about: string; contact: string }
```

## 修饰符控制

**添加 readonly 修饰符**
`type <类型><T> = { readonly [P in keyof T]: T[P] }`

```typescript
interface User {
  id: number;
  name: string;
}
type ReadonlyUser = {
  readonly [P in keyof User]: User[P];
};
```

---

**移除 readonly 修饰符**
`type <类型><T> = { -readonly [P in keyof T]: T[P] }`

```typescript
interface ReadonlyUser {
  readonly id: number;
  readonly name: string;
}
type MutableUser = {
  -readonly [P in keyof ReadonlyUser]: ReadonlyUser[P];
};
// 等价于: { id: number; name: string }
```

---

**添加可选修饰符**
`type <类型><T> = { [P in keyof T]?: T[P] }`

```typescript
interface User {
  id: number;
  name: string;
}
type OptionalUser = {
  [P in keyof User]?: User[P];
};
// 等价于: { id?: number; name?: string }
```

---

**移除可选修饰符**
`type <类型><T> = { [P in keyof T]-?: T[P] }`

```typescript
interface OptionalUser {
  id?: number;
  name?: string;
}
type RequiredUser = {
  [P in keyof OptionalUser]-?: OptionalUser[P];
};
// 等价于: { id: number; name: string }
```

## 键重映射

**键重映射：使用 as 重命名键**
`type <类型><T> = { [P in keyof T as <新键>]: T[P] }`

```typescript
interface User {
  id: number;
  name: string;
  age: number;
}
// 将所有键转为小写
type LowercaseKeys<T> = {
  [P in keyof T as Lowercase<string & P>]: T[P];
};
type LowerUser = LowercaseKeys<User>;
// 等价于: { id: number; name: string; age: number }
// 将所有键转为大写
type UppercaseKeys<T> = {
  [P in keyof T as Uppercase<string & P>]: T[P];
};
type UpperUser = UppercaseKeys<User>;
// 等价于: { ID: number; NAME: string; AGE: number }
```

---

**键重映射：添加前缀**
`type <类型><T> = { [P in keyof T as \`prefix_\${string & P}\`]: T[P] }`

```typescript
interface User {
  id: number;
  name: string;
  age: number;
}
// 添加前缀
type Getters<T> = {
  [P in keyof T as `get${Capitalize<string & P>}`]: () => T[P];
};
type UserGetters = Getters<User>;
// 等价于: { getId: () => number; getName: () => string; getAge: () => number }
```

---

**键重映射：过滤键**
`type <类型><T> = { [P in keyof T as <条件> ? P : never]: T[P] }`

```typescript
interface User {
  id: number;
  name: string;
  age: number;
  email: string;
}
// 过滤键：只保留 string 类型的属性
type StringValues<T> = {
  [P in keyof T as T[P] extends string ? P : never]: T[P];
};
type StringProps = StringValues<User>;
// 等价于: { name: string; email: string }
// 过滤键：排除特定键
type RemoveId<T> = {
  [P in keyof T as P extends 'id' ? never : P]: T[P];
};
type UserWithoutId = RemoveId<User>;
// 等价于: { name: string; age: number; email: string }
```

## 复杂映射类型

**深度 Partial**
`type <类型><T> = { [P in keyof T]?: T[P] extends object ? <递归调用> : T[P] }`

```typescript
// 深度 Partial
type DeepPartial<T> = {
  [P in keyof T]?: T[P] extends object ? DeepPartial<T[P]> : T[P];
};
// 使用
interface User {
  id: number;
  name: string;
  profile: {
    age: number;
    address: {
      city: string;
      country: string;
    };
  };
}
type PartialUser = DeepPartial<User>;
// 所有属性都变为可选，包括嵌套属性
const user: PartialUser = {
  name: 'Alice',
  profile: {
    address: {
      city: 'New York',
    },
  },
};
```

---

**深度 Readonly**
`type <类型><T> = { readonly [P in keyof T]: T[P] extends object ? <递归调用> : T[P] }`

```typescript
// 深度 Readonly
type DeepReadonly<T> = {
  readonly [P in keyof T]: T[P] extends object ? DeepReadonly<T[P]> : T[P];
};
// 使用
interface User {
  id: number;
  name: string;
  profile: {
    age: number;
    address: {
      city: string;
    };
  };
}
type ReadonlyUser = DeepReadonly<User>;
const user: ReadonlyUser = {
  id: 1,
  name: 'Alice',
  profile: {
    age: 30,
    address: {
      city: 'New York',
    },
  },
};
// user.id = 2; // 错误：只读属性
// user.profile.address.city = 'Boston'; // 错误：只读属性
```

## 映射类型与条件类型组合

**条件映射类型**
`type <类型><T> = { [P in keyof T]: T[P] extends <约束> ? <类型1> : <类型2> }`

```typescript
// 条件映射类型
type Nullable<T> = {
  [P in keyof T]: T[P] | null;
};
// 使用
interface User {
  id: number;
  name: string;
  age: number;
}
type NullableUser = Nullable<User>;
// 等价于: { id: number | null; name: string | null; age: number | null }
// 根据值类型选择不同类型
type StringifyNumbers<T> = {
  [P in keyof T]: T[P] extends number ? string : T[P];
};
type StringifiedUser = StringifyNumbers<User>;
// 等价于: { id: string; name: string; age: string }
```

## 自定义映射类型

**自定义工具类型**
`type <类型><T> = ...`

```typescript
// 将所有属性变为可空
type Nullable<T> = { [P in keyof T]: T[P] | null };
// 反转属性的可选性
type Mutable<T> = { -readonly [P in keyof T]: T[P] };
// 将所有属性变为布尔值（标记类型）
type BooleanFlags<T> = {
  [P in keyof T]: boolean;
};
// 使用
interface User {
  id: number;
  name: string;
  age: number;
}
type UserFlags = BooleanFlags<User>;
// 等价于: { id: boolean; name: boolean; age: boolean }
// 提取特定类型的属性
type PickByValueType<T, ValueType> = {
  [P in keyof T as T[P] extends ValueType ? P : never]: T[P];
};
type StringProps = PickByValueType<User, string>;
// 等价于: { name: string }
```
