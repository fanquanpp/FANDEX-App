# 类与装饰器

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 类定义

**基本类：使用 class 关键字**
`class <类名> { <属性>: <类型>; constructor(<参数>) { ... } <方法>() { ... } }`

```typescript
// 基本类定义
class Person {
  // 属性
  name: string;
  age: number;
  // 构造函数
  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }
  // 方法
  greet(): string {
    return `Hello, I'm ${this.name}!`;
  }
}
// 创建实例
const person = new Person('Alice', 30);
console.log(person.greet()); // Hello, I'm Alice!
```

## 访问修饰符

**public、private、protected：控制成员访问**
`class <类名> { public <属性>: <类型>; private <属性>: <类型>; protected <属性>: <类型> }`

```typescript
class Person {
  public name: string;       // 公有，外部可访问
  private age: number;      // 私有，仅类内部可访问
  protected email: string;  // 受保护，类和子类可访问
  constructor(name: string, age: number, email: string) {
    this.name = name;
    this.age = age;
    this.email = email;
  }
  public getAge(): number {
    return this.age;
  }
}
const person = new Person('Alice', 30, 'alice@example.com');
console.log(person.name); // Alice
// console.log(person.age); // 错误：私有属性
// console.log(person.email); // 错误：受保护属性
console.log(person.getAge()); // 30
```

---

**参数属性：构造函数参数简写**
`constructor(public <属性>: <类型>, private <属性>: <类型>)`

```typescript
class Person {
  constructor(
    public name: string,
    public age: number,
    private email: string
  ) {}
  greet(): string {
    return `Hello, I'm ${this.name}!`;
  }
}
const person = new Person('Alice', 30, 'alice@example.com');
console.log(person.name); // Alice
console.log(person.age); // 30
```

## 只读属性

**readonly：只读属性**
`class <类名> { readonly <属性>: <类型> }`

```typescript
class Person {
  readonly id: number;
  constructor(id: number, public name: string) {
    this.id = id;
```

## 静态成员

**static：静态属性和方法**
`class <类名> { static <属性>: <类型>; static <方法>(): ... }`

```typescript
class MathUtils {
  static pi: number = 3.14159265359;
  static add(a: number, b: number): number {
    return a + b;
  }
  static multiply(a: number, b: number): number {
    return a * b;
  }
}
// 访问静态成员
console.log(MathUtils.pi); // 3.14159265359
console.log(MathUtils.add(5, 3)); // 8
console.log(MathUtils.multiply(5, 3)); // 15
```

## 抽象类

**abstract：抽象类和抽象方法**
`abstract class <类名> { abstract <方法>(): ...; }`

```typescript
abstract class Animal {
  constructor(public name: string) {}
  // 抽象方法：子类必须实现
  abstract makeSound(): string;
  // 具体方法
  describe(): string {
    return `This is ${this.name}`;
  }
}
class Dog extends Animal {
  makeSound(): string {
    return 'Woof!';
  }
}
class Cat extends Animal {
  makeSound(): string {
    return 'Meow!';
  }
}
const dog = new Dog('Buddy');
console.log(dog.makeSound()); // Woof!
console.log(dog.describe()); // This is Buddy
// const animal = new Animal('Test'); // 错误：不能实例化抽象类
```

## 继承

**extends：类继承**
`class <子类> extends <父类> { ... }`

```typescript
class Animal {
  constructor(public name: string) {}
  move(distance: number = 0): void {
    console.log(`${this.name} moved ${distance}m.`);
  }
}
class Dog extends Animal {
  bark(): void {
    console.log('Woof! Woof!');
  }
}
class Cat extends Animal {
  constructor(name: string, public color: string) {
    super(name); // 必须调用 super()
  }
  meow(): void {
    console.log('Meow!');
  }
  move(distance: number = 5): void {
    super.move(distance); // 调用父类方法
    console.log(`${this.name} is a ${this.color} cat.`);
  }
}
const dog = new Dog('Buddy');
dog.bark(); // Woof! Woof!
dog.move(10); // Buddy moved 10m.
const cat = new Cat('Kitty', 'black');
cat.meow(); // Meow!
cat.move(); // Kitty moved 5m. Kitty is a black cat.
```

## 存取器

**getter 和 setter：属性访问控制**
`class <类名> { get <属性>(): ...; set <属性>(value): void }`

```typescript
class Employee {
  private _fullName: string = '';
  get fullName(): string {
    return this._fullName;
  }
  set fullName(newName: string) {
    if (newName.length < 3) {
      throw new Error('Name is too short');
    }
    this._fullName = newName;
  }
}
const employee = new Employee();
employee.fullName = 'Alice Smith';
console.log(employee.fullName); // Alice Smith
// employee.fullName = 'Al'; // 错误：Name is too short
```

## 接口实现

**implements：类实现接口**
`class <类> implements <接口> { ... }`

```typescript
interface ClockInterface {
  currentTime: Date;
  setTime(d: Date): void;
}
class Clock implements ClockInterface {
  currentTime: Date = new Date();
  setTime(d: Date): void {
    this.currentTime = d;
  }
}
const clock = new Clock();
console.log(clock.currentTime);
clock.setTime(new Date());
```

## 泛型类

**泛型类：使用类型参数**
`class <类名><T> { ... }`

```typescript
class GenericNumber<T> {
  zeroValue: T;
  add: (x: T, y: T) => T;
  constructor(zeroValue: T, addFn: (x: T, y: T) => T) {
    this.zeroValue = zeroValue;
    this.add = addFn;
  }
}
// 使用泛型类
let myGenericNumber = new GenericNumber<number>(0, (x, y) => x + y);
console.log(myGenericNumber.add(5, 3)); // 8
let stringGeneric = new GenericNumber<string>('', (x, y) => x + y);
console.log(stringGeneric.add('Hello, ', 'World!')); // Hello, World!
```

## 装饰器

**类装饰器：修改或替换类定义**
`function <装饰器>(<目标>: Function): void { ... }`
`@<装饰器> class <类> { ... }`

```typescript
// 类装饰器
function sealed(constructor: Function) {
  Object.seal(constructor);
  Object.seal(constructor.prototype);
}
@sealed
class Greeter {
  greeting: string;
  constructor(message: string) {
    this.greeting = message;
  }
  greet() {
    return `Hello, ${this.greeting}`;
  }
}
```

---

**类装饰器工厂：带参数的装饰器**
`function <装饰器>(<参数>): ClassDecorator`

```typescript
// 类装饰器工厂
function classDecorator<T extends { new (...args: any[]): {} }>(constructor: T) {
  return class extends constructor {
    newProperty = 'new property';
    hello = 'override';
  };
}
@classDecorator
class Greeter {
  property = 'property';
  hello: string;
  constructor(m: string) {
    this.hello = m;
  }
}
const greeter = new Greeter('world');
console.log(greeter);
```

---

**方法装饰器：修改方法行为**
`function <装饰器>(target: any, propertyKey: string, descriptor: PropertyDescriptor): void`

```typescript
// 方法装饰器
function log(target: any, propertyKey: string, descriptor: PropertyDescriptor) {
  const originalMethod = descriptor.value;
  descriptor.value = function (...args: any[]) {
    console.log(`Calling ${propertyKey} with args: ${JSON.stringify(args)}`);
    const result = originalMethod.apply(this, args);
    console.log(`Method ${propertyKey} returned: ${result}`);
    return result;
  };
}
class Calculator {
  @log
  add(a: number, b: number): number {
    return a + b;
  }
}
const calc = new Calculator();
calc.add(5, 3);
// Calling add with args: [5,3]
// Method add returned: 8
```

---

**属性装饰器：监听属性变化**
`function <装饰器>(target: any, propertyKey: string): void`

```typescript
// 属性装饰器
function readonly(target: any, propertyKey: string) {
  let value = target[propertyKey];
  const getter = () => value;
  const setter = (newValue: any) => {
    throw new Error(`Cannot set readonly property ${propertyKey}`);
  };
  Object.defineProperty(target, propertyKey, {
    get: getter,
    set: setter,
    enumerable: true,
    configurable: true,
  });
}
class Person {
  @readonly
  name: string = 'Alice';
}
const person = new Person();
console.log(person.name); // Alice
// person.name = 'Bob'; // 错误：Cannot set readonly property name
```

---

**参数装饰器：验证函数参数**
`function <装饰器>(target: any, propertyKey: string, parameterIndex: number): void`

```typescript
// 参数装饰器
function required(target: any, propertyKey: string, parameterIndex: number) {
  console.log(`Parameter at index ${parameterIndex} in ${propertyKey} is required`);
}
class UserService {
  getUser(@required id: number) {
    console.log(`Getting user with id: ${id}`);
  }
}
const service = new UserService();
service.getUser(1);
// Parameter at index 0 in getUser is required
// Getting user with id: 1
```

## 装饰器工厂

**装饰器工厂：带参数的装饰器**
`function <装饰器>(<参数>): <装饰器类型>`

```typescript
// 装饰器工厂
function logClass(prefix: string) {
  return function <T extends { new (...args: any[]): {} }>(constructor: T) {
    return class extends constructor {
      constructor(...args: any[]) {
        super(...args);
        console.log(`${prefix}: Creating instance with args: ${JSON.stringify(args)}`);
      }
    };
  };
}
@logClass('DEBUG')
class Person {
  constructor(public name: string, public age: number) {}
}
const person = new Person('Alice', 30);
// DEBUG: Creating instance with args: ["Alice",30]
```
