# MySQL 事务与锁机制

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 事务控制

**开启事务**
`START TRANSACTION` / `BEGIN`
```sql
-- 开启事务
START TRANSACTION;

-- 或
BEGIN;
```

**提交事务**
`COMMIT`
```sql
-- 提交事务
START TRANSACTION;
INSERT INTO users (username, email) VALUES ('张三', 'zhangsan@example.com');
UPDATE accounts SET balance = balance - 100 WHERE user_id = 1;
UPDATE accounts SET balance = balance + 100 WHERE user_id = 2;
COMMIT;
```

**回滚事务**
`ROLLBACK`
```sql
-- 回滚事务
ROLLBACK;
```

**保存点**
`SAVEPOINT <保存点名>` / `ROLLBACK TO <保存点名>` / `RELEASE SAVEPOINT <保存点名>`
```sql
-- 使用保存点
START TRANSACTION;
INSERT INTO users (username) VALUES ('张三');
SAVEPOINT sp1;
INSERT INTO users (username) VALUES ('李四');
ROLLBACK TO sp1;
COMMIT;
```

## 隔离级别

**查看隔离级别**
`SELECT @@transaction_isolation`
```sql
-- 查看当前隔离级别
SELECT @@transaction_isolation;

-- 查看旧变量名
SELECT @@tx_isolation;
```

**设置会话隔离级别**
`SET SESSION TRANSACTION ISOLATION LEVEL <级别>`
```sql
-- 设置会话隔离级别
SET SESSION TRANSACTION ISOLATION LEVEL READ COMMITTED;
```

**设置全局隔离级别**
`SET GLOBAL TRANSACTION ISOLATION LEVEL <级别>`
```sql
-- 设置全局隔离级别
SET GLOBAL TRANSACTION ISOLATION LEVEL SERIALIZABLE;
```

**通过变量设置**
`SET [SESSION|GLOBAL] transaction_isolation = '<级别>'`
```sql
-- 设置全局隔离级别
SET GLOBAL transaction_isolation = 'READ-COMMITTED';

-- 设置会话隔离级别
SET SESSION transaction_isolation = 'REPEATABLE-READ';
```

## 锁机制

**共享锁（读锁）**
`SELECT ... LOCK IN SHARE MODE`
```sql
-- 加共享锁
SELECT * FROM users WHERE id = 1 LOCK IN SHARE MODE;
```

**排他锁（写锁）**
`SELECT ... FOR UPDATE`
```sql
-- 加排他锁
SELECT * FROM users WHERE id = 1 FOR UPDATE;
```

**DML 自动加排他锁**
`INSERT|UPDATE|DELETE ...`
```sql
-- 写操作自动加排他锁
INSERT INTO users (name) VALUES ('John');
UPDATE users SET name = 'John' WHERE id = 1;
DELETE FROM users WHERE id = 1;
```

## 锁等待与超时

**查看锁等待超时**
`SELECT @@innodb_lock_wait_timeout`
```sql
-- 查看锁等待超时时间
SELECT @@innodb_lock_wait_timeout;
```

**设置锁等待超时**
`SET SESSION innodb_lock_wait_timeout = <秒数>`
```sql
-- 设置锁等待超时为 30 秒
SET SESSION innodb_lock_wait_timeout = 30;

-- 设置锁等待超时为 60 秒
SET SESSION innodb_lock_wait_timeout = 60;
```

## 事务实战

**转账事务**
`START TRANSACTION; <DML>; COMMIT;`
```sql
-- 转账事务
START TRANSACTION;
UPDATE accounts SET balance = balance - 1000 WHERE user_id = 1;
UPDATE accounts SET balance = balance + 1000 WHERE user_id = 2;
COMMIT;
```

**条件提交**
`IF <条件> THEN COMMIT; ELSE ROLLBACK; END IF`
```sql
-- 检查余额后决定提交或回滚
START TRANSACTION;
UPDATE accounts SET balance = balance - 1000 WHERE user_id = 1;
UPDATE accounts SET balance = balance + 1000 WHERE user_id = 2;

IF (SELECT balance FROM accounts WHERE user_id = 1) < 0 THEN
  ROLLBACK;
ELSE
  COMMIT;
END IF;
```

**订单创建事务**
`START TRANSACTION; <DML>; SET @变量; <DML>; COMMIT;`
```sql
-- 订单创建事务
START TRANSACTION;
INSERT INTO orders (user_id, total_amount) VALUES (1, 500);
SET @order_id = LAST_INSERT_ID();
INSERT INTO order_items (order_id, product_id, quantity, price) VALUES
  (@order_id, 101, 2, 200),
  (@order_id, 102, 1, 100);
UPDATE products SET stock = stock - 3 WHERE id IN (101, 102);
COMMIT;
```

**悲观锁查询**
`SELECT ... FOR UPDATE`
```sql
-- 先锁定再更新
SELECT * FROM users WHERE id = 1 FOR UPDATE;
UPDATE users SET status = 0 WHERE last_login_time < '2023-01-01';
```

**批量删除事务**
`START TRANSACTION; DELETE ... LIMIT <N>; COMMIT;`
```sql
-- 批量删除避免长事务
START TRANSACTION;
UPDATE users SET status = 0 WHERE last_login_time < '2023-01-01';
UPDATE stats SET inactive_users = inactive_users + 1;
COMMIT;

-- 分批删除
DELETE FROM logs WHERE created_at < '2023-01-01' LIMIT 1000;
```
