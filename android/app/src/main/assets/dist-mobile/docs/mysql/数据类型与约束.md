# MySQL 数据类型与约束

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 数值类型

**整数类型：用于自增主键、状态枚举**
`<TINYINT|INT|BIGINT> [UNSIGNED] [ZEROFILL]`
```sql
-- 自增主键使用 BIGINT
id BIGINT UNSIGNED NOT NULL PRIMARY KEY AUTO_INCREMENT;

-- 状态枚举使用 TINYINT
status TINYINT NOT NULL DEFAULT 1;
```

**定点数：用于金额存储**
`DECIMAL(<精度>, <小数位数>)`
```sql
-- 金额字段使用 DECIMAL 避免浮点误差
balance DECIMAL(10, 2) DEFAULT 0.00;
```

**浮点数：用于近似值**
`<FLOAT|DOUBLE> [(<精度>, <小数位数>)]`
```sql
-- 测量数据使用 DOUBLE
temperature DOUBLE;
```

---

## 字符串类型

**定长字符串：用于固定长度编码**
`CHAR(<长度>)`
```sql
-- 国家码、短编码使用 CHAR
country_code CHAR(2) NOT NULL;
```

**变长字符串：用于长度变化字段**
`VARCHAR(<最大长度>)`
```sql
-- 昵称、标题使用 VARCHAR
username VARCHAR(50) NOT NULL;
email VARCHAR(100) NOT NULL;
```

**长文本：用于文章内容**
`<TINYTEXT|TEXT|MEDIUMTEXT|LONGTEXT>`
```sql
-- 文章内容使用 TEXT
content TEXT;
```

---

## 日期与时间类型

**日期类型：用于仅日期**
`DATE`
```sql
-- 仅保存日期
birthday DATE;
```

**日期时间：用于业务发生时间**
`DATETIME`
```sql
-- 业务发生时间使用 DATETIME
created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP;
```

**时间戳：用于时区相关**
`TIMESTAMP [DEFAULT CURRENT_TIMESTAMP] [ON UPDATE CURRENT_TIMESTAMP]`
```sql
-- 更新时间自动维护
updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP;
```

## JSON 类型

**JSON 类型：用于结构化扩展字段**
`JSON`
```sql
-- 扩展字段使用 JSON
CREATE TABLE users (
  id BIGINT UNSIGNED NOT NULL PRIMARY KEY,
  profile JSON
);
```

## 字符集与排序规则

**指定字符集与排序规则**
`CHARACTER SET <字符集> COLLATE <排序规则>`
```sql
-- 统一使用 utf8mb4
CREATE DATABASE mydb
  CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;
```

## 约束

**非空约束**
`<列名> <类型> NOT NULL`
```sql
-- 必填字段
email VARCHAR(255) NOT NULL;
```

**默认值约束**
`<列名> <类型> DEFAULT <默认值>`
```sql
-- 默认状态为 1
status TINYINT NOT NULL DEFAULT 1;

-- 默认创建时间
created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP;
```

**唯一约束**
`UNIQUE [KEY <索引名>] (<列名>[, <列名>...])`
```sql
-- 单列唯一
UNIQUE KEY uk_email (email);

-- 组合唯一
UNIQUE KEY uk_tenant_email (tenant_id, email);
```

**主键约束**
`PRIMARY KEY (<列名>[, <列名>...])`
```sql
-- 单列主键
id BIGINT UNSIGNED NOT NULL PRIMARY KEY;

-- 复合主键
PRIMARY KEY (tenant_id, user_id);
```

**外键约束**
`FOREIGN KEY (<列名>) REFERENCES <父表>(<父列>) [ON DELETE <行为>] [ON UPDATE <行为>]`
```sql
-- 外键关联
FOREIGN KEY (user_id) REFERENCES users(id)
  ON DELETE RESTRICT
  ON UPDATE CASCADE;
```

**检查约束**
`CHECK (<条件表达式>)`
```sql
-- 金额必须非负
CHECK (total_amount >= 0);

-- 状态值限定
CHECK (status IN (1, 2, 3, 4, 5));
```

**自增约束**
`<列名> <整数类型> AUTO_INCREMENT`
```sql
-- 自增主键
id INT PRIMARY KEY AUTO_INCREMENT;
```

## 建表示例

**完整建表语句**
`CREATE TABLE <表名> (<列定义> [, <约束定义>...])`
```sql
-- 用户账户表示例
CREATE TABLE user_account (
  id BIGINT UNSIGNED NOT NULL PRIMARY KEY,
  email VARCHAR(255) NOT NULL,
  status TINYINT NOT NULL DEFAULT 1,
  created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_email (email)
);
```
