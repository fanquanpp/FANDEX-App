# Kotlin 协程基础速查

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 启动协程

**launch 启动协程**
`<scope>.launch { <body> }`
```kotlin
import kotlinx.coroutines.*;
// launch — 启动协程（不返回结果）
runBlocking {
    val job = launch {
        delay(1000);
        println("协程执行完成");
    };
    job.join(); // 等待协程完成
};
```

**async 启动协程**
`<scope>.async { <body> }`
```kotlin
// async — 启动协程（返回结果）
runBlocking {
    val deferred = async { fetchUser() };
    val user = deferred.await(); // 获取结果
    println(user);
};
```

## 协程作用域

**viewModelScope**
`viewModelScope.launch { <body> }`
```kotlin
// viewModelScope — ViewModel 销毁时自动取消
class MyViewModel : ViewModel() {
    fun loadData() {
        viewModelScope.launch {
            val data = repository.fetch();
            _state.value = data;
        };
    };
};
```

**lifecycleScope**
`lifecycleScope.launch { <body> }`
```kotlin
// lifecycleScope — Activity/Fragment 生命周期绑定
class MyActivity : AppCompatActivity() {
    fun loadData() {
        lifecycleScope.launch {
            val data = repository.fetch();
            updateUI(data);
        };
    };
};
```

**coroutineScope**
`suspend fun <name>() = coroutineScope { <body> }`
```kotlin
// coroutineScope — 创建临时作用域，所有子协程完成后才结束
suspend fun fetchAll() = coroutineScope {
    val user = async { fetchUser() };
    val orders = async { fetchOrders() };
    Pair(user.await(), orders.await());
};
```

## 挂起函数

**suspend 函数**
`suspend fun <name>(<params>): <ReturnType>`
```kotlin
// suspend 修饰的函数可以在协程中调用，也可以调用其他挂起函数
suspend fun fetchUser(): User {
    delay(1000); // 模拟网络请求（挂起点）
    return User("张三", 25);
};
// 挂起函数是安全的，不会阻塞线程
suspend fun loadData() {
    val user = fetchUser();       // 挂起，不阻塞
    val orders = fetchOrders();   // 挂起，不阻塞
    updateUI(user, orders);
};
```

**withContext 切换调度器**
`withContext(<dispatcher>) { <body> }`
```kotlin
// withContext 切换调度器
suspend fun saveData(data: Data) {
    // 在 IO 调度器上执行，完成后切回原调度器
    val result = withContext(Dispatchers.IO) {
        database.save(data);
    };
    showResult(result);
};
```

## 协程取消与超时

**取消协程**
`<job>.cancel() | <job>.cancelAndJoin()`
```kotlin
// 取消协程
val job = launch {
    repeat(1000) {
        // 检查取消状态
        ensureActive();
        // 或者使用可取消的挂起函数
        delay(100);
        println("处理: $it");
    };
};
delay(500);
job.cancel(); // 取消协程
job.join();   // 等待取消完成
// 简写：job.cancelAndJoin();
```

**超时控制**
`withTimeout(<ms>) { <body> }`
```kotlin
// 超时控制
try {
    withTimeout(3000) {
        fetchData(); // 超过3秒抛出 TimeoutCancellationException
    };
} catch (e: TimeoutCancellationException) {
    println("请求超时");
};
// 超时返回 null 而非抛异常
val result = withTimeoutOrNull(3000) {
    fetchData();
};
if (result == null) {
    println("请求超时");
};
```

## 异常处理

**SupervisorJob**
`CoroutineScope(SupervisorJob() + Dispatchers.Main)`
```kotlin
// SupervisorJob — 子协程失败不影响其他子协程
val scope = CoroutineScope(SupervisorJob() + Dispatchers.Main);
```

**supervisorScope**
`supervisorScope { <body> }`
```kotlin
// supervisorScope — 临时作用域
supervisorScope {
    launch {
        throw RuntimeException("子协程1失败"); // 不影响子协程2
    };
    launch {
        delay(100);
        println("子协程2正常执行");
    };
};
```

**CoroutineExceptionHandler**
`val <handler> = CoroutineExceptionHandler { _, <exception> -> <body> }`
```kotlin
// CoroutineExceptionHandler — 全局异常处理
val handler = CoroutineExceptionHandler { _, exception ->
    println("捕获异常: ${exception.message}");
};
scope.launch(handler) {
    throw RuntimeException("出错了");
};
```

## 并发请求

**async 并发执行**
`suspend fun <name>(): <Type> = coroutineScope { val <a> = async { <body> }; val <b> = async { <body> }; <Type>(<a>.await(), <b>.await()) }`
```kotlin
// 使用 async 并发执行多个请求
suspend fun loadPage(): PageData = coroutineScope {
    val user = async { apiService.getUser() };
    val posts = async { apiService.getPosts() };
    val notifications = async { apiService.getNotifications() };
    PageData(user.await(), posts.await(), notifications.await());
};
```

## 重试机制

**简单重试**
`suspend fun <T> retry(<times>, <delay>, <block>): T`
```kotlin
// 简单重试
suspend fun <T> retry(
    times: Int = 3,
    delay: Long = 1000,
    block: suspend () -> T
): T {
    repeat(times) { attempt ->
        try {
            return block();
        } catch (e: Exception) {
            if (attempt == times - 1) throw e;
            delay(delay * (attempt + 1)); // 递增延迟
        };
    };
    throw IllegalStateException("不应到达此处");
};
// 使用
val data = retry(times = 3) { apiService.fetchData() };
```

## Channel 通信

**Channel 在协程间传递数据**
`val <channel> = Channel<<Type>>()`
```kotlin
// 使用 Channel 在协程间传递数据
val channel = Channel<Int>();
// 生产者
launch {
    for (i in 1..10) {
        channel.send(i);
    };
    channel.close();
};
// 消费者
launch {
    for (item in channel) {
        println("收到: $item");
    };
};
```

**produce 构建器**
`fun CoroutineScope.<name>() = produce { <body> }`
```kotlin
// produce 构建器
fun CoroutineScope.numberProducer() = produce {
    for (i in 1..10) {
        send(i);
    };
};
```

## Select 等待多个操作

**select 同时等待多个挂起操作**
`select { <async>.onAwait { <body> } }`
```kotlin
// select 同时等待多个挂起操作，哪个先完成就处理哪个
suspend fun fetchFastest(): String = coroutineScope {
    val local = async { cache.get() };
    val remote = async { apiService.fetch() };
    select {
        local.onAwait { "缓存: $it" };
        remote.onAwait { "网络: $it" };
    };
};
```
