# Kotlin 空安全速查

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 可空类型

**可空类型与不可空类型**
`var <name>: <Type> | var <name>: <Type>?`
```kotlin
var name: String = "Alice";  // 不可空类型，永远不能赋值为 null
var nick: String? = null;    // 可空类型，允许 null 值
// 不可空类型不能赋值为 null
// name = null;  // 编译错误
// 可空类型的成员不能直接访问
// nick.length;  // 编译错误：可能为 null
```

---

## 安全调用操作符

**?. 安全调用**
`<obj>?.<prop>`
```kotlin
var name: String? = "Alice";
// 安全调用：如果 name 为 null，整个表达式返回 null
val len: Int? = name?.length;
// 链式安全调用
val city: String? = user?.address?.city;
```

**链式安全调用**
`<obj>?.<prop1>?.<prop2>?.<prop3>`
```kotlin
// 深层嵌套对象的空安全访问
val street = company?.department?.manager?.address?.street ?: "未知地址";
// 等价于 Java 的：
// if (company != null && company.getDepartment() != null && ...)
```

---

## Elvis 运算符

**?: Elvis 运算符**
`<expr> ?: <default>`
```kotlin
// Elvis 运算符：为 null 时提供默认值
val length: Int = name?.length ?: 0;
val name2 = user?.name ?: "未知用户";
// Elvis 配合 return/throw
val name3 = user?.name ?: return;  // 为 null 时提前返回
val config = loadConfig() ?: throw IllegalStateException("配置加载失败");
```

## 非空断言

**!! 非空断言**
`<obj>!!.<prop>`
```kotlin
var value: String? = "Hello";
// 非空断言：断言值不为 null，否则抛出 NPE
val length: Int = value!!.length;  // 如果 value 为 null 则崩溃
// 应尽量避免使用 !!，只在确定不为 null 时使用
// 错误示例：过度使用 !!
val result = getResult()!!;  // 危险：可能为 null
```

## 安全转换

**as? 安全转换**
`<obj> as? <Type>`
```kotlin
// as? 安全转换：失败时返回 null
val num: Int? = "123" as? Int;     // null
val str: String? = 42 as? String;  // null
val obj: Any = "hello";
val length = (obj as? String)?.length ?: 0;  // 5
```

## 智能转换

**条件判断后智能转换**
`if (<obj> != null) { <body with obj as non-null> }`
```kotlin
// 条件判断后自动智能转换
fun process(s: String?) {
    if (s != null) {
        // s 自动从 String? 转换为 String
        println(s.length);  // 无需 ?. 或 !!
    };
};
```

**when 表达式中的智能转换**
`when (<obj>) { is <Type> -> <body with obj as Type> }`
```kotlin
fun describe(x: Any) = when (x) {
    is String -> x.length;      // x 自动转换为 String
    is Int -> x.toDouble();     // x 自动转换为 Int
    is List<*> -> x.size;       // x 自动转换为 List
    else -> "未知类型";
};
```

**is 检查后的智能转换**
`if (<obj> is <Type>) { <body with obj as Type> }`
```kotlin
fun handle(value: Any) {
    if (value is String) {
        println(value.uppercase()); // value 自动转换为 String
    };
};
```

**while 循环中的智能转换**
`while (<obj> != null) { <body with obj as non-null> }`
```kotlin
var input: String? = readLine();
while (input != null) {
    process(input); // input 自动转换为 String
    input = readLine();
};
```

## 可空类型的集合操作

**filterNotNull**
`<list>.filterNotNull()`
```kotlin
val names: List<String?> = listOf("Alice", null, "Bob", null, "Charlie");
// 过滤 null 值
val nonNull: List<String> = names.filterNotNull();
// ["Alice", "Bob", "Charlie"]
```

**mapNotNull**
`<list>.mapNotNull { <transform> }`
```kotlin
// 映射并过滤 null
val lengths: List<Int> = names.mapNotNull { it?.length };
// [5, 3, 7]
```

**forEach 中处理 null**
`<list>.forEach { <item> -> <item>?.let { <body> } }`
```kotlin
names.forEach { name ->
    name?.let {
        println("名字: $it, 长度: ${it.length}");
    };
};
```

## let、run、with 处理空值

**let 处理非空值**
`<obj>?.let { <body with it> }`
```kotlin
// let：对非空值执行操作
val name: String? = getName();
name?.let {
    // it 为 String 类型（非空）
    println("名字: $it, 长度: ${it.length}");
};
```

**run 处理非空值**
`<obj>?.run { <body with this> }`
```kotlin
// run：对非空值执行代码块
name?.run {
    // this 为 String 类型
    println("名字: $this, 长度: ${length}");
};
```

**takeIf / takeUnless**
`<obj>?.takeIf { <cond> } | <obj>?.takeUnless { <cond> }`
```kotlin
// takeIf / takeUnless：条件判断
val validName = name?.takeIf { it.length > 2 };
val invalidName = name?.takeUnless { it.isBlank() };
```

## API 响应处理

**处理可能为 null 的 API 响应**
`fun <name>(<param>: <Type>?) { val <local> = <param>?.<prop> ?: run { <body>; return; } }`
```kotlin
// 处理可能为 null 的 API 响应
fun handleResponse(response: ApiResponse?) {
    val data = response?.data ?: run {
        println("响应为空");
        return;
    };
    val userName = data.user?.name ?: "未知用户";
    val userEmail = data.user?.email ?: "无邮箱";
    println("用户: $userName, 邮箱: $userEmail");
};
```

## 可空类型与 Java 互操作

**平台类型处理**
`val <name>: <Type>? = <javaMethod>()`
```kotlin
// Java 返回值可能为 null，使用平台类型
val javaResult = javaService.getData();  // 类型为 String!（平台类型）
// 明确标记为可空
val safeResult: String? = javaService.getData();
// 使用 @Nullable 和 @NonNull 注解帮助 Kotlin 推断
// Java: @Nullable String getName()
// Kotlin 自动识别为 String?
```

## 自定义空安全逻辑

**requireNotNull 检查参数非空**
`requireNotNull(<value>) { <message> }`
```kotlin
// 使用 require 检查参数非空
fun processName(name: String?) {
    requireNotNull(name) { "姓名不能为空" };
    // 此处 name 自动转换为 String（非空）
    println(name.length);
};
```

**check 检查状态**
`val <local> = <prop> ?: throw <Exception>`
```kotlin
// 使用 check 检查状态
fun getState(): String {
    val state = currentState ?: throw IllegalStateException("状态未初始化");
    return state;
};
```

**委托属性处理空值**
`val <name>: <Type> get() = <prop> ?: throw <Exception>`
```kotlin
// 委托属性处理空值
class Config {
    private var _value: String? = null;
    val value: String
        get() = _value ?: throw IllegalStateException("配置未加载");
};
```

## 可空类型与协程

**协程中处理空值**
`suspend fun <name>(): <Type>? { try { <body> } catch (e: Exception) { null } }`
```kotlin
// 协程中处理空值
suspend fun fetchUser(id: String): User? {
    return try {
        apiService.getUser(id);
    } catch (e: Exception) {
        null;
    };
};
// 使用
viewModelScope.launch {
    val user = fetchUser(userId);
    user?.let {
        updateUI(it);
    } ?: run {
        showError("用户不存在");
    };
};
```
