# 流

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 添加消息

**XADD 自动生成 ID：添加消息到 Stream**
`XADD <key> * <field> <value> [field value ...]`
```bash
# 添加消息，* 表示自动生成ID
XADD mystream * field1 value1 field2 value2
# 返回消息ID：1718334600000-0
```

**XADD 指定 ID：手动指定消息 ID**
`XADD <key> <id> <field> <value> [field value ...]`
```bash
# 指定ID格式
XADD mystream 1718334600000-0 field1 value1
```

**XADD 限制长度：添加消息并限制 Stream 长度**
`XADD <key> MAXLEN <count> * <field> <value> [field value ...]`
```bash
# 限制Stream长度为1000
XADD mystream MAXLEN 1000 * field1 value1
```

## 读取消息

**XREAD：从头读取消息**
`XREAD COUNT <count> STREAMS <key> <id>`
```bash
# 从头读取10条消息
XREAD COUNT 10 STREAMS mystream 0
```

**XREAD 阻塞读取：阻塞等待新消息**
`XREAD BLOCK <ms> COUNT <count> STREAMS <key> $`
```bash
# 阻塞读取新消息，最多等待5000ms
XREAD BLOCK 5000 COUNT 10 STREAMS mystream $
```

**XRANGE：范围读取消息**
`XRANGE <key> <start> <end> [COUNT <count>]`
```bash
# 读取所有消息
XRANGE mystream - + COUNT 10
# 按ID范围读取
XRANGE mystream 1718334600000-0 1718334700000-0
```

## 消费者组

**XGROUP CREATE：创建消费者组**
`XGROUP CREATE <key> <group> <id>`
```bash
# 从最新消息开始
XGROUP CREATE mystream mygroup $
# 从头开始
XGROUP CREATE mystream mygroup 0
```

**XREADGROUP：消费者读取消息**
`XREADGROUP GROUP <group> <consumer> COUNT <count> STREAMS <key> >`
```bash
# 消费者读取1条消息
XREADGROUP GROUP mygroup consumer1 COUNT 1 STREAMS mystream >
```

**XACK：确认消息已处理**
`XACK <key> <group> <id> [id ...]`
```bash
# 确认消息
XACK mystream mygroup 1718334600000-0
```

**XPENDING：查看待处理消息**
`XPENDING <key> <group>`
```bash
# 查看待处理消息
XPENDING mystream mygroup
```

**XCLAIM：认领超时消息**
`XCLAIM <key> <group> <consumer> <min-idle-time> <id> [id ...]`
```bash
# 认领空闲超过3600000ms的消息
XCLAIM mystream mygroup consumer1 3600000 1718334600000-0
```

## 消息积压处理

**XINFO STREAM：查看 Stream 信息**
`XINFO STREAM <key>`
```bash
# 查看Stream信息
XINFO STREAM mystream
```

**XTRIM：修剪 Stream 长度**
`XTRIM <key> MAXLEN <count> | MINID <id>`
```bash
# 按最大长度修剪
XTRIM mystream MAXLEN 10000
# 按最小ID修剪
XTRIM mystream MINID 1718334600000-0
```

**XINFO GROUPS：查看消费者组信息**
`XINFO GROUPS <key>`
```bash
# 查看所有消费者组
XINFO GROUPS mystream
```

**XINFO CONSUMERS：查看消费者信息**
`XINFO CONSUMERS <key> <group>`
```bash
# 查看消费者组内的消费者
XINFO CONSUMERS mystream mygroup
```
