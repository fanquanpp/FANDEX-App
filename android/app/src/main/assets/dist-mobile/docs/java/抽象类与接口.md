# Java 抽象类与接口

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 抽象类

**抽象类定义**：使用 abstract 修饰的类
`<修饰符> abstract class <类名> { <抽象方法> <普通方法> }`
```java
public abstract class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }

    public abstract void makeSound();  // 抽象方法（无方法体）

    public void eat() {  // 普通方法（有方法体）
        System.out.println(name + " is eating.");
    }
}
```

---

**抽象类继承**：子类实现抽象方法
`<修饰符> class <子类> extends <抽象类> { @Override public void <方法>() { ... } }`
```java
public class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    @Override
    public void makeSound() {
        System.out.println(name + " barks.");
    }
}

public class Cat extends Animal {
    public Cat(String name) {
        super(name);
    }

    @Override
    public void makeSound() {
        System.out.println(name + " meows.");
    }
}
```

## 接口定义

**接口定义**：使用 interface 关键字
`<修饰符> interface <接口名> { <常量> <抽象方法> }`
```java
public interface Shape {
    double PI = 3.14159;  // 常量（默认 public static final）

    double calculateArea();      // 抽象方法（默认 public abstract）
    double calculatePerimeter();
}
```

---

## 接口默认方法

**默认方法**：Java 8+，提供方法的默认实现
`default <返回类型> <方法名>(<参数>) { ... }`
```java
public interface Vehicle {
    void start();
    void stop();

    default void honk() {  // 默认方法
        System.out.println("Beep beep!");
    }
}
```

---

## 接口静态方法

**静态方法**：Java 8+，提供工具方法
`static <返回类型> <方法名>(<参数>) { ... }`
```java
public interface MathUtils {
    static int add(int a, int b) {
        return a + b;
    }

    static int subtract(int a, int b) {
        return a - b;
    }
}
```

---

## 接口私有方法

**私有方法**：Java 9+，供接口内部使用
`private <返回类型> <方法名>(<参数>) { ... }`
```java
public interface StringUtils {
    default String reverse(String str) {
        return reverseImpl(str);
    }

    default boolean isPalindrome(String str) {
        String reversed = reverseImpl(str);
        return str.equals(reversed);
    }

    private String reverseImpl(String str) {  // 私有方法
        return new StringBuilder(str).reverse().toString();
    }
}
```

## 接口实现

**接口实现**：使用 implements 关键字
`<修饰符> class <类名> implements <接口> { @Override ... }`
```java
public class Circle implements Shape {
    private double radius;

    public Circle(double radius) {
        this.radius = radius;
    }

    @Override
    public double calculateArea() {
        return PI * radius * radius;
    }

    @Override
    public double calculatePerimeter() {
        return 2 * PI * radius;
    }
}
```

## 接口多继承

**接口多继承**：接口可以继承多个接口
`<修饰符> interface <接口> extends <接口1>, <接口2> { ... }`
```java
public interface Movable {
    void move();
}

public interface Flyable {
    void fly();
}

// 多继承接口
public interface Bird extends Movable, Flyable {
    void sing();
}
```

---

## 继承与实现

**继承一个类，实现多个接口**
`<修饰符> class <类> extends <父类> implements <接口1>, <接口2> { ... }`
```java
public class Eagle extends Animal implements Bird, Predator {
    // 实现所有抽象方法
}
```

## 默认方法冲突解决

**默认方法冲突**：实现多个接口时解决冲突
`<接口>.super.<方法>();`
```java
public interface A {
    default void method() {
        System.out.println("A.method()");
    }
}

public interface B {
    default void method() {
        System.out.println("B.method()");
    }
}

// 解决冲突：重写默认方法
public class C implements A, B {
    @Override
    public void method() {
        A.super.method();  // 调用接口 A 的默认方法
        System.out.println("C.method()");
    }
}
```

## 模板方法模式

**模板方法模式**：抽象类定义算法骨架
`public final void <模板方法>() { <步骤1>(); <步骤2>(); <步骤3>(); }`
```java
public abstract class AbstractProcessor {
    public final void process() {  // 模板方法
        initialize();
        doProcess();
        cleanup();
    }

    protected abstract void doProcess();  // 抽象方法，由子类实现

    protected void initialize() {
        System.out.println("Initializing...");
    }

    protected void cleanup() {
        System.out.println("Cleaning up...");
    }
}

public class FileProcessor extends AbstractProcessor {
    @Override
    protected void doProcess() {
        System.out.println("Processing file...");
    }
}
```

## 策略模式

**策略模式**：接口定义算法族
`<接口> <变量> = new <实现类>(); <变量>.<方法>();`
```java
public interface PaymentStrategy {
    void pay(double amount);
}

public class CreditCardPayment implements PaymentStrategy {
    private String cardNumber;

    public CreditCardPayment(String cardNumber) {
        this.cardNumber = cardNumber;
    }

    @Override
    public void pay(double amount) {
        System.out.println("Paying " + amount + " with credit card: " + cardNumber);
    }
}

public class PayPalPayment implements PaymentStrategy {
    private String email;

    public PayPalPayment(String email) {
        this.email = email;
    }

    @Override
    public void pay(double amount) {
        System.out.println("Paying " + amount + " with PayPal: " + email);
    }
}

public class ShoppingCart {
    private PaymentStrategy paymentStrategy;

    public void setPaymentStrategy(PaymentStrategy paymentStrategy) {
        this.paymentStrategy = paymentStrategy;
    }

    public void checkout(double amount) {
        paymentStrategy.pay(amount);
    }
}
```
