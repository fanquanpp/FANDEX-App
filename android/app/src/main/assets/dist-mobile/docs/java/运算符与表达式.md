# Java 运算符与表达式

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 算术运算符

**基本算术运算**：执行数学运算
`<操作数1> <运算符> <操作数2>`
```java
int a = 10;
int b = 3;
System.out.println(a + b);  // 13 加法
System.out.println(a - b);  // 7 减法
System.out.println(a * b);  // 30 乘法
System.out.println(a / b);  // 3 整数除法
System.out.println(a % b);  // 1 取模
```

---

**自增自减**：变量值加 1 或减 1
`<变量>++` | `++<变量>` | `<变量>--` | `--<变量>`
```java
int c = 5;
System.out.println(c++);  // 5 先用后加
System.out.println(c);     // 6
System.out.println(++c);  // 7 先加后用

int d = 5;
System.out.println(d--);  // 5 先用后减
System.out.println(--d);  // 3 先减后用
```

---

**字符串拼接**：使用 + 运算符
`<字符串> + <其他类型>`
```java
String str1 = "Hello";
String str2 = "World";
System.out.println(str1 + " " + str2);  // "Hello World"
System.out.println("The answer is: " + 42);  // "The answer is: 42"
```

---

## 关系运算符

**关系运算**：比较两个值的大小
`<操作数1> <关系运算符> <操作数2>`
```java
int a = 10;
int b = 3;
System.out.println(a == b);  // false 等于
System.out.println(a != b);  // true 不等于
System.out.println(a > b);   // true 大于
System.out.println(a < b);   // false 小于
System.out.println(a >= b);  // true 大于等于
System.out.println(a <= b);  // false 小于等于
```

---

**引用类型比较**：使用 equals() 比较内容
`<对象1>.equals(<对象2>)`
```java
String s1 = "Hello";
String s2 = "Hello";
String s3 = new String("Hello");
System.out.println(s1 == s2);        // true（字符串常量池）
System.out.println(s1 == s3);        // false（不同对象）
System.out.println(s1.equals(s3));  // true（内容相同）
```

---

## 逻辑运算符

**短路与**：第一个为 false 则不计算第二个
`<布尔表达式1> && <布尔表达式2>`
```java
int x = 5;
boolean result1 = (x > 10) && (x++ > 0);
System.out.println(result1);  // false
System.out.println(x);         // 5（x++ 未执行）
```

---

**短路或**：第一个为 true 则不计算第二个
`<布尔表达式1> || <布尔表达式2>`
```java
int y = 5;
boolean result2 = (y < 10) || (y++ > 0);
System.out.println(result2);  // true
System.out.println(y);         // 5（y++ 未执行）
```

---

**逻辑非**：取反
`!<布尔表达式>`
```java
boolean flag = true;
System.out.println(!flag);  // false
```

---

**逻辑异或**：相同为 false，不同为 true
`<布尔表达式1> ^ <布尔表达式2>`
```java
boolean a = true;
boolean b = false;
System.out.println(a ^ b);  // true
```

## 位运算符

**按位与**：二进制位与运算
`<操作数1> & <操作数2>`
```java
int a = 6;   // 二进制: 110
int b = 3;   // 二进制: 011
System.out.println(a & b);  // 2 (010)
```

---

**按位或**：二进制位或运算
`<操作数1> | <操作数2>`
```java
int a = 6;
int b = 3;
System.out.println(a | b);  // 7 (111)
```

---

**按位异或**：二进制位异或运算
`<操作数1> ^ <操作数2>`
```java
int a = 6;
int b = 3;
System.out.println(a ^ b);  // 5 (101)
```

---

**按位取反**：二进制位取反
`~<操作数>`
```java
int a = 6;
System.out.println(~a);  // -7
```

---

**左移**：二进制位左移
`<操作数> << <位数>`
```java
int a = 6;
System.out.println(a << 1);  // 12 (1100)，相当于乘以 2
```

---

**右移**：二进制位右移（带符号）
`<操作数> >> <位数>`
```java
int a = 6;
System.out.println(a >> 1);  // 3 (011)，相当于除以 2

int c = -6;
System.out.println(c >> 1);  // -3（带符号右移）
```

---

**无符号右移**：二进制位右移（高位补 0）
`<操作数> >>> <位数>`
```java
int c = -6;
System.out.println(c >>> 1);  // 2147483645（无符号右移）
```

## 赋值运算符

**简单赋值**：给变量赋值
`<变量> = <值>`
```java
int a = 10;
a = 20;
```

---

**复合赋值**：运算并赋值
`<变量> <运算符>= <值>`
```java
int a = 10;
a += 5;  // 等价于 a = a + 5，结果 15
a -= 3;  // 等价于 a = a - 3，结果 12
a *= 2;  // 等价于 a = a * 2，结果 24
a /= 4;  // 等价于 a = a / 4，结果 6
a %= 3;  // 等价于 a = a % 3，结果 0

// 位运算复合赋值
int b = 10;
b <<= 2;  // 等价于 b = b << 2，结果 40
b >>= 1;  // 等价于 b = b >> 1，结果 20
b &= 5;   // 等价于 b = b & 5
b |= 3;   // 等价于 b = b | 3
b ^= 2;   // 等价于 b = b ^ 2
```

---

## 三元运算符

**三元运算符**：根据条件选择值
`<条件表达式> ? <表达式1> : <表达式2>`
```java
int a = 10;
int b = 20;
int max = (a > b) ? a : b;
System.out.println(max);  // 20

// 嵌套使用
int x = 5, y = 10, z = 15;
int largest = (x > y) ? ((x > z) ? x : z) : ((y > z) ? y : z);
System.out.println(largest);  // 15

// 用于赋值
String result = (a > b) ? "a is larger" : "b is larger";
System.out.println(result);  // "b is larger"
```

## 运算符优先级

**优先级示例**：使用括号明确运算顺序
`(<表达式>)`
```java
int a = 10, b = 5, c = 3, d = 2;

// 先乘除后加减
int result1 = a + b * c - d;  // 等价于 a + (b * c) - d = 23

// 使用括号
int result2 = (a + b) * (c - d);  // 15 * 1 = 15

// 先关系运算，后逻辑运算
boolean result3 = a > b && c < d;  // (a > b) && (c < d) = false
```

## 整数溢出处理

**Math.addExact()**：溢出时抛出异常
`Math.<addExact|multiplyExact>(<a>, <b>)`
```java
try {
    int result = Math.addExact(Integer.MAX_VALUE, 1);
} catch (ArithmeticException e) {
    System.out.println("发生溢出: " + e.getMessage());
}

// 使用更大范围类型
long max = Integer.MAX_VALUE;
long result = max + 1;  // 输出 2147483648，正确
```

## 字符串拼接性能

**StringBuilder**：高效拼接字符串
`StringBuilder <变量名> = new StringBuilder();`
```java
// 性能较差的方式
String result = "";
for (int i = 0; i < 1000; i++) {
    result += " " + i;
}

// 性能较好的方式
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(" ").append(i);
}
String result2 = sb.toString();
```

## 位运算应用

**位掩码**：表示一组布尔标志
`int <标志> = 1 << <位数>`
```java
int FLAG_READ = 1 << 0;      // 0b0001
int FLAG_WRITE = 1 << 1;     // 0b0010
int FLAG_EXECUTE = 1 << 2;   // 0b0100

int permissions = FLAG_READ | FLAG_WRITE;  // 0b0011

// 检查权限
if ((permissions & FLAG_READ) != 0) {
    System.out.println("Read permission granted");
}

// 高效的乘除运算
int a = 10;
int multiplyBy2 = a << 1;  // 等价于 a * 2，结果 20
int divideBy2 = a >> 1;    // 等价于 a / 2，结果 5
```
