# Java 泛型详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 泛型类

**泛型类定义**：使用类型参数
`<修饰符> class <类名><<类型参数>> { <类型参数> <字段>; }`
```java
public class Box<T> {
    private T content;

    public void set(T content) {
        this.content = content;
    }

    public T get() {
        return content;
    }
}

// 使用
Box<String> stringBox = new Box<>();
stringBox.set("Hello");
String value = stringBox.get();
```

---

**多类型参数泛型类**：使用多个类型参数
`<修饰符> class <类名><<T1>, <T2>> { ... }`
```java
public class Pair<K, V> {
    private K key;
    private V value;

    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }

    public K getKey() { return key; }
    public V getValue() { return value; }
}

// 使用
Pair<String, Integer> pair = new Pair<>("age", 25);
```

## 泛型接口

**泛型接口定义**：使用类型参数
`<修饰符> interface <接口名><<T>> { <T> <方法>(); }`
```java
public interface Container<T> {
    void add(T item);
    T get(int index);
    int size();
}

// 实现方式 1：指定具体类型
public class StringContainer implements Container<String> {
    private List<String> items = new ArrayList<>();

    @Override
    public void add(String item) { items.add(item); }

    @Override
    public String get(int index) { return items.get(index); }

    @Override
    public int size() { return items.size(); }
}

// 实现方式 2：保留泛型
public class GenericContainer<T> implements Container<T> {
    private List<T> items = new ArrayList<>();

    @Override
    public void add(T item) { items.add(item); }

    @Override
    public T get(int index) { return items.get(index); }

    @Override
    public int size() { return items.size(); }
}
```

## 泛型方法

**泛型方法定义**：在返回值前声明类型参数
`public <T> <返回类型> <方法名>(T <参数>) { ... }`
```java
public class Util {
    public <T> void printArray(T[] array) {
        for (T item : array) {
            System.out.println(item);
        }
    }

    public <T> T getFirst(List<T> list) {
        if (list.isEmpty()) {
            return null;
        }
        return list.get(0);
    }
}

// 使用
Util util = new Util();
util.printArray(new String[]{"a", "b", "c"});
String first = util.getFirst(Arrays.asList("x", "y", "z"));
```

---

**多类型参数泛型方法**：使用多个类型参数
`public <T, U> <返回类型> <方法名>(T <参数1>, U <参数2>) { ... }`
```java
public <T, U> String combine(T first, U second) {
    return first.toString() + second.toString();
}
```

## 类型通配符

**上界通配符**：接受指定类型及其子类型
`<? extends <类型>>`
```java
// 接受 Number 及其子类（Integer, Double 等）
public double sumOfList(List<? extends Number> list) {
    double sum = 0;
    for (Number n : list) {
        sum += n.doubleValue();
    }
    return sum;
}

List<Integer> ints = Arrays.asList(1, 2, 3);
List<Double> doubles = Arrays.asList(1.1, 2.2, 3.3);
System.out.println(sumOfList(ints));    // 6.0
System.out.println(sumOfList(doubles)); // 6.6
```

---

**下界通配符**：接受指定类型及其父类型
`<? super <类型>>`
```java
// 接受 Integer 及其父类（Number, Object）
public void addNumbers(List<? super Integer> list) {
    for (int i = 1; i <= 5; i++) {
        list.add(i);
    }
}

List<Number> numbers = new ArrayList<>();
addNumbers(numbers);  // 可以添加 Integer 到 List<Number>

List<Object> objects = new ArrayList<>();
addNumbers(objects);  // 可以添加 Integer 到 List<Object>
```

---

**无界通配符**：接受任何类型
`<?>`
```java
public void printList(List<?> list) {
    for (Object item : list) {
        System.out.println(item);
    }
}

List<String> strings = Arrays.asList("a", "b", "c");
List<Integer> ints = Arrays.asList(1, 2, 3);
printList(strings);
printList(ints);
```

## 类型约束

**泛型上界约束**：限制类型参数的范围
`<<T> extends <类型>>`
```java
// T 必须是 Number 或其子类
public <T extends Number> double sum(T[] numbers) {
    double total = 0;
    for (T num : numbers) {
        total += num.doubleValue();
    }
    return total;
}

// T 必须实现 Comparable 接口
public <T extends Comparable<T>> T findMax(List<T> list) {
    if (list.isEmpty()) return null;
    T max = list.get(0);
    for (T item : list) {
        if (item.compareTo(max) > 0) {
            max = item;
        }
    }
    return max;
}
```

---

**多边界约束**：类型参数必须同时满足多个约束
`<<T> extends <类型1> & <接口2>>`
```java
// T 必须同时是 Number 的子类且实现 Comparable 接口
public <T extends Number & Comparable<T>> T findMax(T a, T b) {
    return (a.compareTo(b) >= 0) ? a : b;
}
```

## 泛型类型推断

**菱形语法**：Java 7+，自动推断类型
`<类名><<类型>> <变量> = new <类名><>();`
```java
// Java 7 之前
Map<String, List<Integer>> map1 = new HashMap<String, List<Integer>>();

// Java 7+ 菱形语法
Map<String, List<Integer>> map2 = new HashMap<>();
```

## 泛型数组

**泛型数组创建限制**：不能直接创建泛型数组
`List<String>[] <数组名> = new ArrayList[10];`
```java
// 错误：不能创建泛型数组
// List<String>[] array = new ArrayList<String>[10];

// 正确：使用原始类型数组
List<String>[] array = new ArrayList[10];
array[0] = new ArrayList<>();
array[0].add("Hello");
```

## 泛型擦除

**类型擦除**：运行时泛型类型信息被擦除
`Class<<类型>> <变量> = <类>.class;`
```java
List<String> strings = new ArrayList<>();
List<Integer> integers = new ArrayList<>();

// 运行时类型相同
System.out.println(strings.getClass() == integers.getClass());  // true
System.out.println(strings.getClass().getName());  // java.util.ArrayList
```

## 泛型工厂模式

**泛型工厂模式**：使用泛型创建对象
`public static <T> T <方法>(Class<T> <参数>) { return <参数>.newInstance(); }`
```java
public class GenericFactory {
    public static <T> T createInstance(Class<T> clazz) throws Exception {
        return clazz.getDeclaredConstructor().newInstance();
    }
}

// 使用
String str = GenericFactory.createInstance(String.class);
StringBuilder sb = GenericFactory.createInstance(StringBuilder.class);
```

## 泛型缓存

**泛型缓存**：使用 Class 作为缓存键
`private static final Map<Class<?>, Object> CACHE = new HashMap<>();`
```java
public class GenericCache {
    private static final Map<Class<?>, Object> CACHE = new HashMap<>();

    @SuppressWarnings("unchecked")
    public static <T> T get(Class<T> clazz, Supplier<T> supplier) {
        return (T) CACHE.computeIfAbsent(clazz, k -> supplier.get());
    }
}
```
