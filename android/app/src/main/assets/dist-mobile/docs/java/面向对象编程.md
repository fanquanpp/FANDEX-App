# Java 面向对象编程

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 类的定义

**类定义**：定义对象的模板
`<修饰符> class <类名> { <成员变量> <方法> }`
```java
public class Student {
    private String name;    // 属性（成员变量）
    private int age;
    private String major;

    public void study() {   // 行为（方法）
        System.out.println(name + " is studying.");
    }

    public void introduce() {
        System.out.println("I'm " + name + ", " + age + " years old, major in " + major);
    }
}
```

---

## 对象创建

**对象创建**：使用 new 关键字
`<类名> <对象名> = new <类名>();`
```java
Student student = new Student();
student.setName("Alice");
student.setAge(20);
student.setMajor("Computer Science");
student.study();
student.introduce();
```

## 构造器

**默认构造器**：无参构造器
`<修饰符> <类名>() { }`
```java
public class Student {
    public Student() {  // 无参构造器
    }
}
```

---

**自定义构造器**：带参数的构造器
`<修饰符> <类名>(<参数列表>) { <初始化> }`
```java
public class Student {
    private String name;
    private int age;
    private String major;

    public Student() {  // 无参构造器
    }

    public Student(String name, int age) {  // 带参数的构造器
        this.name = name;
        this.age = age;
    }

    public Student(String name, int age, String major) {  // 全参构造器
        this.name = name;
        this.age = age;
        this.major = major;
    }
}
```

---

**构造器调用**：使用 new 调用
`<类名> <对象名> = new <类名>(<参数>);`
```java
Student student1 = new Student();                              // 无参构造器
Student student2 = new Student("Bob", 21);                     // 带参数构造器
Student student3 = new Student("Charlie", 22, "Software Engineering");  // 全参构造器
```

## 封装

**封装实现**：private 属性 + getter/setter
`private <类型> <属性>;` | `public <类型> get<X>() { ... }` | `public void set<X>(<类型> <参数>) { ... }`
```java
public class Student {
    private String name;
    private int age;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        if (age >= 0 && age <= 150) {  // 添加验证逻辑
            this.age = age;
        } else {
            System.out.println("Invalid age");
        }
    }
}
```

## 继承

**继承语法**：使用 extends 关键字
`<修饰符> class <子类> extends <父类> { ... }`
```java
// 父类
public class Animal {
    protected String name;

    public void eat() {
        System.out.println(name + " is eating.");
    }
}

// 子类
public class Dog extends Animal {
    public void bark() {
        System.out.println(name + " is barking.");
    }
}
```

---

**super 访问父类成员**：访问父类的成员变量
`super.<成员变量> = <值>;`
```java
public class Dog extends Animal {
    private String breed;

    public Dog(String name, String breed) {
        super.name = name;  // 访问父类的成员变量
        this.breed = breed;
    }
}
```

---

**super 调用父类方法**：调用父类的方法
`super.<方法名>();`
```java
public class Dog extends Animal {
    @Override
    public void eat() {
        super.eat();  // 调用父类的 eat 方法
        System.out.println("Dog eats bones.");
    }
}
```

---

**super 调用父类构造器**：调用父类的构造器
`super(<参数>);`
```java
public class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }
}

public class Dog extends Animal {
    private String breed;

    public Dog(String name, String breed) {
        super(name);  // 调用父类的构造器
        this.breed = breed;
    }
}
```

## 方法重写

**方法重写**：子类重写父类的方法
`@Override public <返回类型> <方法名>(<参数>) { ... }`
```java
public class Animal {
    public void makeSound() {
        System.out.println("Animal makes sound.");
    }
}

public class Cat extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Cat meows.");
    }
}

public class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Dog barks.");
    }
}
```

---

## 多态

**多态**：父类引用指向子类对象
`<父类> <变量> = new <子类>();`
```java
Animal animal1 = new Cat();
animal1.makeSound();  // 输出: Cat meows.

Animal animal2 = new Dog();
animal2.makeSound();  // 输出: Dog barks.
```

## this 关键字

**this 区分变量**：区分成员变量和局部变量
`this.<成员变量> = <参数>;`
```java
public class Student {
    private String name;

    public void setName(String name) {
        this.name = name;  // this.name 指成员变量，name 指局部变量
    }
}
```

---

**this 调用构造器**：调用本类的其他构造器
`this(<参数>);`
```java
public class Student {
    private String name;
    private int age;

    public Student() {
        this("Unknown", 0);  // 调用带两个参数的构造器
    }

    public Student(String name) {
        this(name, 0);  // 调用带两个参数的构造器
    }

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

---

**this 返回当前对象**：支持链式调用
`return this;`
```java
public class Student {
    private String name;
    private int age;

    public Student setName(String name) {
        this.name = name;
        return this;  // 返回当前对象
    }

    public Student setAge(int age) {
        this.age = age;
        return this;
    }
}

// 链式调用
Student student = new Student().setName("Alice").setAge(20);
```
