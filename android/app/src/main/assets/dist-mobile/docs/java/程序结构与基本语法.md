# Java 程序结构与基本语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 源文件结构

**包声明**：指定源文件所属的包
`package <包名>;`
```java
// 包声明
package com.example;
```

---

**导入语句**：导入需要使用的类
`import <全限定类名>;`
```java
// 导入语句
import java.util.Scanner;
import java.util.Date;
```

---

**类定义**：定义一个或多个类
`<修饰符> class <类名> { ... }`
```java
// 类定义
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

---

## 类的结构

**完整类定义**：包含继承、接口实现、成员变量、构造方法、成员方法
`<修饰符> class <类名> extends <父类> implements <接口> { ... }`
```java
// 完整类结构示例
public class Student extends Person implements Serializable {
    private String studentId;  // 成员变量
    private String major;

    public Student() {  // 无参构造方法
    }

    public Student(String name, int age, String studentId, String major) {  // 全参构造方法
        super(name, age);
        this.studentId = studentId;
        this.major = major;
    }

    public String getStudentId() {  // 成员方法
        return studentId;
    }

    public void setStudentId(String studentId) {
        this.studentId = studentId;
    }
}
```

---

## 主方法

**程序入口**：Java 程序的入口点
`public static void main(String[] args) { ... }`
```java
// 主方法示例
public static void main(String[] args) {
    System.out.println("Hello, World!");
    // 处理命令行参数
    for (int i = 0; i < args.length; i++) {
        System.out.println("Argument " + i + ": " + args[i]);
    }
}
```

## 注释规范

**单行注释**：解释单行代码
`// <注释内容>`
```java
// 这是一个单行注释
int age = 18;  // 定义年龄变量
```

---

**多行注释**：解释代码块
`/* <注释内容> */`
```java
/*
 * 这是一个多行注释
 * 可以跨越多行
 */
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i;
}
```

---

**文档注释**：生成 API 文档
`/** <注释内容> */`
```java
/**
 * 计算两个数的和
 * @param a 第一个加数
 * @param b 第二个加数
 * @return 两个数的和
 */
public int add(int a, int b) {
    return a + b;
}
```

## 标识符命名规范

**类名和接口名**：大驼峰命名法
`<UpperCamelCase>`
```java
// 类名示例
HelloWorld
StudentInfo
UserService
```

---

**方法名和变量名**：小驼峰命名法
`<lowerCamelCase>`
```java
// 方法名和变量名示例
getUserName
ageCount
calculateTotal
```

---

**包名**：全小写，使用点分隔
`<全小写.分隔>`
```java
// 包名示例
com.example.util
org.apache.commons.io
```

---

**常量名**：全大写，使用下划线分隔
`<UPPER_SNAKE_CASE>`
```java
// 常量名示例
MAX_VALUE
DEFAULT_TIMEOUT
PI
```

## 键盘录入

**Scanner 基本用法**：获取控制台输入
`Scanner <变量名> = new Scanner(System.in);`
```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int num = sc.nextInt();           // 读取整数
double d = sc.nextDouble();       // 读取浮点数
boolean b = sc.nextBoolean();     // 读取布尔值
String str1 = sc.next();          // 读取一个单词（遇到空格停止）
sc.nextLine();                    // 消耗换行符
String str2 = sc.nextLine();      // 读取整行
sc.close();                        // 关闭 Scanner
```

---

**Scanner 异常处理**：安全读取输入
`try { <读取操作> } catch (InputMismatchException e) { <处理> }`
```java
import java.util.InputMismatchException;
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int num = 0;
boolean valid = false;
while (!valid) {
    System.out.print("请输入整数: ");
    try {
        num = sc.nextInt();
        valid = true;
    } catch (InputMismatchException e) {
        System.out.println("输入错误，请重新输入整数!");
        sc.next();  // 消耗错误的输入
    }
}
sc.close();
```

## 代码风格

**K&R 风格**：左大括号放在行尾
`if (<条件>) { ... }`
```java
// 好的风格
if (condition) {
    // 代码块
}

// 不好的风格
if (condition)
{
    // 代码块
}
```

---

**try-with-resources**：自动关闭资源
`try (<资源声明>) { <使用资源> }`
```java
// 使用 try-with-resources
try (Scanner sc = new Scanner(System.in)) {
    // 使用 sc
}
```
