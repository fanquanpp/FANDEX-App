# Java 数据类型与类型转换

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 整数类型

**byte**：1 字节整数
`byte <变量名> = <值>;`
```java
byte b = 100;  // 范围 -128 ~ 127
```

---

**short**：2 字节整数
`short <变量名> = <值>;`
```java
short s = 1000;  // 范围 -32,768 ~ 32,767
```

---

**int**：4 字节整数（默认）
`int <变量名> = <值>;`
```java
int i = 100000;  // 范围 -2^31 ~ 2^31-1
```

---

**long**：8 字节整数
`long <变量名> = <值>L;`
```java
long l = 10000000000L;  // 必须加 L 后缀
```

---

## 浮点数类型

**float**：单精度浮点数
`float <变量名> = <值>F;`
```java
float f = 3.14f;  // 必须加 F 后缀
```

---

**double**：双精度浮点数（默认）
`double <变量名> = <值>;`
```java
double d = 3.1415926535;  // double 是默认的浮点数类型
```

---

## 字符类型

**char**：单个字符
`char <变量名> = '<字符>';`
```java
char c1 = 'A';        // 字符字面量
char c2 = 65;         // ASCII 码，对应 'A'
char c3 = '\u0041';   // Unicode 编码，对应 'A'
```

---

## 布尔类型

**boolean**：布尔值
`boolean <变量名> = <true|false>;`
```java
boolean flag = true;
boolean isReady = false;
```

---

## 引用数据类型

**类类型**：使用 class 定义的类型
`<类名> <变量名> = new <类名>();`
```java
String str = "Hello, Java!";              // 字符串对象
ArrayList<String> list = new ArrayList<>(); // 集合对象
```

---

**数组类型**：使用 [] 定义的类型
`<类型>[] <变量名> = { <元素> };`
```java
int[] numbers = {1, 2, 3, 4, 5};
String[] names = new String[3];
```

---

## 自动类型转换

**自动类型转换**：小容量类型向大容量类型转换
`<大类型> <变量名> = <小类型变量>;`
```java
byte b = 100;
short s = b;       // byte → short
int i = s;         // short → int
long l = i;        // int → long
float f = l;       // long → float
double d = f;      // float → double
char c = 'A';
int i2 = c;        // char → int，值为 65
```

---

## 强制类型转换

**强制类型转换**：大容量类型向小容量类型转换
`(<目标类型>) <表达式>`
```java
// 精度丢失
double pi = 3.14159;
int num = (int) pi;  // 结果为 3，小数部分被截断

// 数据溢出
int i = 130;
byte b = (byte) i;   // 结果为 -126，因为 130 超出了 byte 的范围

// 安全的强制转换
int i2 = 100;
byte b2 = (byte) i2;  // 结果为 100，在 byte 范围内
```

---

## 装箱与拆箱

**手动装箱**：基本类型转换为包装类
`<包装类> <变量名> = <包装类>.valueOf(<基本类型>);`
```java
int i = 100;
Integer iObj = Integer.valueOf(i);  // 手动装箱
Integer iObj2 = i;                  // 自动装箱（Java 5+）

boolean b = true;
Boolean bObj = Boolean.valueOf(b);  // 手动装箱
Boolean bObj2 = b;                  // 自动装箱
```

---

**手动拆箱**：包装类转换为基本类型
`<基本类型> <变量名> = <包装类变量>.<xxxValue>();`
```java
Integer iObj = 100;
int i = iObj.intValue();   // 手动拆箱
int i2 = iObj;             // 自动拆箱（Java 5+）

Boolean bObj = true;
boolean b = bObj.booleanValue();  // 手动拆箱
boolean b2 = bObj;                // 自动拆箱
```

## 字符串与基本类型转换

**基本类型转字符串**：使用 String.valueOf()
`String <变量名> = String.valueOf(<基本类型>);`
```java
int i = 100;
String s1 = String.valueOf(i);    // 方法 1
String s2 = i + "";               // 方法 2
String s3 = Integer.toString(i);   // 方法 3
```

---

**字符串转基本类型**：使用包装类的 parseXxx()
`<基本类型> <变量名> = <包装类>.parse<Xxx>(<字符串>);`
```java
String s1 = "100";
int i = Integer.parseInt(s1);

String s2 = "3.14";
double d = Double.parseDouble(s2);

String s3 = "true";
boolean b = Boolean.parseBoolean(s3);
```

---

## 整数默认类型

**整数字面量**：默认为 int 类型
`long <变量名> = <大数值>L;`
```java
int i = 100;                  // 正确
long l1 = 100;                // 正确，int 自动转换为 long
long l2 = 10000000000L;       // 必须加 L，否则超出 int 范围
```

---

## 浮点数默认类型

**浮点数字面量**：默认为 double 类型
`float <变量名> = <值>F;`
```java
double d = 3.14;          // 正确
float f1 = (float) 3.14;  // 强制转换
float f2 = 3.14f;         // 正确，加 F 后缀
```

---

## 运算中的类型提升

**类型提升**：byte、short、char 运算时提升为 int
`<结果类型> <变量名> = <操作数1> <运算符> <操作数2>;`
```java
byte b1 = 10;
byte b2 = 20;
// byte b3 = b1 + b2; // 错误：b1 + b2 结果为 int 类型
int i = b1 + b2;        // 正确

int i1 = 100;
double d1 = 3.14;
double d2 = i1 + d1;    // 结果为 double 类型
```

---

## 字符串拼接

**字符串拼接**：使用 + 运算符
`String <变量名> = <字符串> + <其他类型>;`
```java
int i = 10;
int j = 20;
String s = "Result: " + i + j;      // 结果为 "Result: 1020"
String s2 = "Result: " + (i + j);   // 结果为 "Result: 30"
String s3 = "Pi is " + 3.14;        // 结果为 "Pi is 3.14"
String s4 = 10 + 20 + " is the sum"; // 结果为 "30 is the sum"
```

---

## BigDecimal 精确计算

**BigDecimal**：用于精确计算
`BigDecimal <变量名> = new BigDecimal("<数值>");`
```java
import java.math.BigDecimal;

// 精度丢失问题
double d1 = 0.1;
double d2 = 0.2;
double sum = d1 + d2;  // 结果为 0.30000000000000004

// 使用 BigDecimal
BigDecimal bd1 = new BigDecimal("0.1");
BigDecimal bd2 = new BigDecimal("0.2");
BigDecimal bdSum = bd1.add(bd2);  // 结果为 0.3
```

## 范围检查

**类型范围检查**：使用包装类的 MIN_VALUE 和 MAX_VALUE
`if (<值> >= <包装类>.MIN_VALUE && <值> <= <包装类>.MAX_VALUE)`
```java
int i = 130;
if (i >= Byte.MIN_VALUE && i <= Byte.MAX_VALUE) {
    byte b = (byte) i;
    System.out.println("转换成功: " + b);
} else {
    System.out.println("转换失败：值超出 byte 范围");
}
```

## 字符串转换安全性

**字符串转换异常处理**：捕获 NumberFormatException
`try { <转换操作> } catch (NumberFormatException e) { <处理> }`
```java
String s = "123";
try {
    int i = Integer.parseInt(s);
    System.out.println("转换成功: " + i);
} catch (NumberFormatException e) {
    System.out.println("转换失败: " + e.getMessage());
}

// 空值检查
String s2 = null;
if (s2 != null) {
    int i2 = Integer.parseInt(s2);
} else {
    System.out.println("字符串为 null");
}
```
