# Java 控制流

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## if-else 语句

**基本 if-else**：条件分支
`if (<条件>) { ... } else if (<条件>) { ... } else { ... }`
```java
if (condition) {
    // 条件为真时执行
} else if (anotherCondition) {
    // 另一个条件为真时执行
} else {
    // 所有条件都为假时执行
}
```

---

**嵌套 if-else**：if 语句嵌套
`if (<条件>) { if (<条件>) { ... } else { ... } }`
```java
if (score >= 90) {
    if (score >= 95) {
        System.out.println("A+");
    } else {
        System.out.println("A");
    }
} else {
    System.out.println("B");
}
```

---

**卫语句**：提前返回，提高可读性
`if (<条件>) { return; }`
```java
public void processOrder(Order order) {
    if (order == null) {
        return;
    }
    if (!order.isValid()) {
        System.out.println("Invalid order");
        return;
    }
    // 处理订单逻辑
}
```

## switch 语句

**传统 switch**：多分支选择
`switch (<表达式>) { case <值>: ... break; default: ... }`
```java
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
    case 4:
    case 5:
        System.out.println("Weekday");
        break;
    case 6:
    case 7:
        System.out.println("Weekend");
        break;
    default:
        System.out.println("Invalid day");
}
```

---

**Java 12+ switch 表达式**：使用箭头语法
`switch (<表达式>) { case <值> -> <结果>; default -> <结果>; }`
```java
// 基本用法
String dayName = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3, 4, 5 -> "Weekday";
    case 6, 7 -> "Weekend";
    default -> "Invalid day";
};
```

---

**switch 表达式 yield**：返回复杂表达式
`switch (<表达式>) { case <值> -> { yield <结果>; } }`
```java
int result = switch (operation) {
    case "add" -> {
        yield a + b;
    }
    case "subtract" -> {
        yield a - b;
    }
    default -> {
        System.out.println("Invalid operation");
        yield 0;
    }
};
```

---

**Java 17+ 模式匹配**：类型匹配
`switch (<对象>) { case <类型> <变量名> -> <结果>; }`
```java
Object obj = "Hello";
String result = switch (obj) {
    case Integer i -> "Integer: " + i;
    case String s -> "String: " + s;
    case null -> "null";
    default -> "Other";
};
```

## for 循环

**标准 for 循环**：已知次数的循环
`for (<初始化>; <条件>; <更新>) { ... }`
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
```

---

**增强型 for 循环**：遍历数组、集合
`for (<类型> <变量> : <可迭代对象>) { ... }`
```java
// 遍历数组
int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println(number);
}

// 遍历集合
List<String> names = List.of("Alice", "Bob", "Charlie");
for (String name : names) {
    System.out.println(name);
}
```

---

**带标签的 for 循环**：控制多层循环
`<标签>: for (...) { <标签>: for (...) { continue <标签>; } }`
```java
outer: for (int i = 0; i < 3; i++) {
    inner: for (int j = 0; j < 3; j++) {
        if (j == 1) {
            continue outer;  // 跳过外层循环的当前迭代
        }
        System.out.println(i + ", " + j);
    }
}
```

---

## while 循环

**while 循环**：先判断后执行
`while (<条件>) { ... }`
```java
int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}
```

---

## do-while 循环

**do-while 循环**：先执行后判断，至少执行一次
`do { ... } while (<条件>);`
```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 10);
```

---

## 循环控制

**break 语句**：跳出当前循环
`break;`
```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // 跳出循环
    }
    System.out.println(i);
}
```

---

**带标签的 break**：跳出多层循环
`break <标签>;`
```java
outerLoop: for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
        if (i * j > 6) {
            System.out.println("Breaking outer loop");
            break outerLoop;
        }
        System.out.println(i + ", " + j);
    }
}
```

---

**continue 语句**：跳过当前迭代
`continue;`
```java
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // 跳过偶数
    }
    System.out.println(i);
}
```

---

## return 语句

**return 语句**：结束方法执行
`return <值>;` | `return;`
```java
public int add(int a, int b) {
    return a + b;  // 返回计算结果
}

public void validate(int value) {
    if (value < 0) {
        return;  // 提前结束方法
    }
    // 继续执行
}
```
