# Java 枚举与注解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 枚举基础

**枚举定义**：定义一组常量
`<修饰符> enum <枚举名> { <常量1>, <常量2>, ... }`
```java
public enum Status {
    ACTIVE, INACTIVE, PENDING
}

// 使用
Status status = Status.ACTIVE;
System.out.println(status);  // ACTIVE
```

---

## 带属性的枚举

**带属性的枚举**：枚举常量携带数据
`<修饰符> enum <枚举名> { <常量>(<值>); private final <类型> <属性>; <构造器> <getter> }`
```java
public enum Status {
    ACTIVE("活跃"), INACTIVE("停用"), PENDING("待审核");

    private final String description;

    Status(String description) {
        this.description = description;
    }

    public String getDescription() {
        return description;
    }
}

// 使用
Status status = Status.ACTIVE;
System.out.println(status.getDescription());  // 活跃
```

---

## 枚举方法

**枚举方法**：枚举中定义方法
`<修饰符> enum <枚举名> { <常量> { @Override <方法> }; public abstract <返回类型> <方法>(); }`
```java
public enum Operation {
    ADD {
        @Override
        public int apply(int a, int b) {
            return a + b;
        }
    },
    SUBTRACT {
        @Override
        public int apply(int a, int b) {
            return a - b;
        }
    };

    public abstract int apply(int a, int b);
}

// 使用
int result = Operation.ADD.apply(5, 3);  // 8
```

## 枚举常用方法

**values()**：获取所有枚举常量
`<枚举类>.values()`
```java
Status[] statuses = Status.values();
for (Status s : statuses) {
    System.out.println(s);
}
```

---

**valueOf()**：根据字符串获取枚举常量
`<枚举类>.valueOf(<字符串>)`
```java
Status status = Status.valueOf("ACTIVE");
System.out.println(status);  // ACTIVE
```

---

**ordinal()**：获取枚举常量的序号
`<枚举常量>.ordinal()`
```java
Status status = Status.INACTIVE;
System.out.println(status.ordinal());  // 1
```

---

**name()**：获取枚举常量的名称
`<枚举常量>.name()`
```java
Status status = Status.ACTIVE;
System.out.println(status.name());  // ACTIVE
```

## 枚举实现接口

**枚举实现接口**：枚举可以实现接口
`<修饰符> enum <枚举名> implements <接口> { ... }`
```java
public interface Describable {
    String describe();
}

public enum Color implements Describable {
    RED("红色"), GREEN("绿色"), BLUE("蓝色");

    private final String name;

    Color(String name) {
        this.name = name;
    }

    @Override
    public String describe() {
        return "Color: " + name;
    }
}
```

## 注解定义

**注解定义**：使用 @interface 关键字
`<修饰符> @interface <注解名> { <属性类型> <属性名>() [default <默认值>]; }`
```java
public @interface Cacheable {
    int ttl() default 3600;
    String key() default "";
}
```

---

## 元注解

**@Retention**：指定注解的保留策略
`@Retention(<RetentionPolicy>)`
```java
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;

@Retention(RetentionPolicy.RUNTIME)  // 运行时保留
public @interface MyAnnotation {
    String value();
}
```

---

**@Target**：指定注解的使用目标
`@Target(<ElementType>)`
```java
import java.lang.annotation.Target;
import java.lang.annotation.ElementType;

@Target(ElementType.METHOD)  // 只能用于方法
public @interface MyAnnotation {
    String value();
}
```

---

**@Documented**：包含在 Javadoc 中
`@Documented`
```java
import java.lang.annotation.Documented;

@Documented
public @interface MyAnnotation {
    String value();
}
```

---

**@Inherited**：子类继承注解
`@Inherited`
```java
import java.lang.annotation.Inherited;

@Inherited
public @interface MyAnnotation {
    String value();
}
```

---

## 注解使用

**注解使用**：在类、方法、字段上使用
`@<注解名>(<属性> = <值>)`
```java
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface Cacheable {
    int ttl() default 3600;
    String key() default "";
}

// 使用注解
@Cacheable(ttl = 600, key = "user:#id")
public User getUser(String id) {
    // 方法实现
}
```

## 内置注解

**@Override**：标记方法重写
`@Override`
```java
public class Animal {
    public void makeSound() {
        System.out.println("Animal sound");
    }
}

public class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Bark");
    }
}
```

---

**@Deprecated**：标记已过时
`@Deprecated`
```java
public class MyClass {
    @Deprecated
    public void oldMethod() {
        // 已过时的方法
    }

    public void newMethod() {
        // 新方法
    }
}
```

---

**@SuppressWarnings**：抑制警告
`@SuppressWarnings("<警告类型>")`
```java
@SuppressWarnings("unchecked")
public void method() {
    List list = new ArrayList();  // 原始类型，会有警告
    list.add("Hello");
}
```

---

**@FunctionalInterface**：标记函数式接口
`@FunctionalInterface`
```java
@FunctionalInterface
public interface Runnable {
    void run();
}
```
