# 变量与常量

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 变量赋值

**基本赋值：动态类型，无需声明**
`<变量> = <值>`

```python
# 基本赋值
x = 10
name = "Alice"
is_valid = True

# 动态类型改变
x = "Hello"  # 变量 x 现在指向字符串
```

---

**链式赋值：多个变量指向同一对象**
`<变量1> = <变量2> = <值>`

```python
# 链式赋值
a = b = c = 100
print(a, b, c)  # 输出: 100 100 100
```

---

**多重赋值（解包）：同时为多个变量赋值**
`<变量1>, <变量2> = <值1>, <值2>`

```python
# 多重赋值
x, y = 1, 2
print(x, y)  # 输出: 1 2

# 解包列表或元组
values = [3, 4, 5]
x, y, z = values
print(x, y, z)  # 输出: 3 4 5

# 使用 * 收集剩余值
first, *rest = [1, 2, 3, 4, 5]
print(first)  # 输出: 1
print(rest)   # 输出: [2, 3, 4, 5]
```

## 内存地址与引用

**查看内存地址：使用 `id()` 函数**
`id(<对象>)`

```python
# 查看变量的内存地址
x = 10
print(id(x))  # 输出: 内存地址

# 多个变量指向同一个对象
a = 42
b = a
print(id(a) == id(b))  # 输出: True
```

## 变量作用域（LEGB）

**局部作用域：函数内部定义的变量**
`def <函数>(): <局部变量> = <值>`

```python
def my_function():
    local_var = "local"
    print(local_var)  # 输出: local

my_function()
# print(local_var)  # 错误: NameError
```

---

**全局作用域：模块级别定义的变量**
`<全局变量> = <值>`

```python
global_var = "global"

def my_function():
    print(global_var)  # 输出: global

my_function()
```

---

**修改全局变量：使用 `global` 声明**
`global <变量名>`

```python
count = 0

def increment():
    global count  # 声明要修改全局变量
    count += 1
    print(count)  # 输出: 1

increment()
```

---

**修改嵌套作用域变量：使用 `nonlocal` 声明**
`nonlocal <变量名>`

```python
def outer_function():
    count = 0
    def inner_function():
        nonlocal count  # 声明要修改嵌套作用域变量
        count += 1
        print(count)  # 输出: 1
    inner_function()
    print(count)  # 输出: 1

outer_function()
```

## 引用计数与垃圾回收

**查看引用计数：使用 `sys.getrefcount()`**
`sys.getrefcount(<对象>)`

```python
import sys

x = [1, 2, 3]
print(sys.getrefcount(x))  # 输出: 2

y = x
print(sys.getrefcount(x))  # 输出: 3

y = None
print(sys.getrefcount(x))  # 输出: 2
```

## 实现真正的常量

**使用类实现常量：通过 `__setattr__` 禁止修改**
`class <常量类>: def __setattr__(self, name, value): raise AttributeError(...)`

```python
class Constants:
    """常量类"""
    MAX_CONNECTIONS = 100
    DEFAULT_TIMEOUT = 30
    PI = 3.14159265359

    def __setattr__(self, name, value):
        raise AttributeError("Constants cannot be modified")

const = Constants()
print(const.PI)  # 输出: 3.14159265359
# const.PI = 3.14  # 错误: AttributeError
```

## 枚举常量

**枚举：使用 `enum` 模块定义一组相关常量**
`class <枚举类>(Enum): <成员> = <值>`

```python
from enum import Enum, auto

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

class Direction(Enum):
    NORTH = auto()
    SOUTH = auto()
    EAST = auto()
    WEST = auto()

# 使用
print(Color.RED)        # 输出: Color.RED
print(Color.RED.value)  # 输出: 1

# 遍历枚举
for color in Color:
    print(color.name, color.value)
```

## 变量交换

**Pythonic 交换：使用元组解包**
`<变量1>, <变量2> = <变量2>, <变量1>`

```python
# 基本交换
a, b = 1, 2
a, b = b, a
print(a, b)  # 输出: 2 1

# 三个变量交换
x, y, z = 1, 2, 3
x, y, z = z, x, y
print(x, y, z)  # 输出: 3 1 2

# 列表元素交换
lst = [1, 2, 3, 4]
lst[0], lst[1] = lst[1], lst[0]
print(lst)  # 输出: [2, 1, 3, 4]
```
