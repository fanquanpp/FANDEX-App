## 条件分支

**if-elif-else 语句：根据条件执行不同代码块**
`if <条件>: <语句>`
`elif <条件>: <语句>`
`else: <语句>`

```python
# 基本用法
x = 7
if x > 10:
    print("Greater than 10")
elif x < 5:
    print("Less than 5")
else:
    print("Between 5 and 10")

# 多个 elif 条件
temperature = 25
if temperature < 0:
    print("Freezing")
elif 0 <= temperature < 10:
    print("Cold")
elif 10 <= temperature < 20:
    print("Mild")
elif 20 <= temperature < 30:
    print("Warm")
else:
    print("Hot")

# 嵌套 if 语句
a = 10
b = 5
if a > b:
    print("a is greater than b")
    if a > 20:
        print("a is also greater than 20")
    else:
        print("a is not greater than 20")
else:
    print("a is not greater than b")
```

## 三元表达式

**三元表达式：一行代码实现简单条件判断**
`<值1> if <条件> else <值2>`

```python
# 基本用法
score = 75
result = "Pass" if score >= 60 else "Fail"
print(result)  # Pass

# 嵌套三元表达式
temperature = 15
status = "Hot" if temperature > 30 else "Warm" if temperature > 20 else "Mild" if temperature > 10 else "Cold"
print(status)  # Mild

# 与函数结合
def get_grade(score):
    return "A" if score >= 90 else "B" if score >= 80 else "C" if score >= 70 else "D"

print(get_grade(85))  # B

# 用于列表推导式
numbers = [1, 2, 3, 4, 5]
even_odd = ["even" if num % 2 == 0 else "odd" for num in numbers]
print(even_odd)  # ['odd', 'even', 'odd', 'even', 'odd']
```

## match-case 语句

**模式匹配：Python 3.10+ 的 switch-case**
`match <值>:`
`    case <模式>: <语句>`
`    case _: <默认语句>`

```python
# 基本用法
status = 404
match status:
    case 200:
        print("OK")
    case 404:
        print("Not Found")
    case 500:
        print("Internal Server Error")
    case _:
        print("Unknown Status")

# 匹配不同类型
value = "hello"
match value:
    case int(x):
        print(f"Integer: {x}")
    case str(x):
        print(f"String: {x}")
    case list(x):
        print(f"List: {x}")
    case _:
        print("Other type")

# 匹配序列
point = (1, 2)
match point:
    case (0, 0):
        print("Origin")
    case (x, 0):
        print(f"On x-axis: {x}")
    case (0, y):
        print(f"On y-axis: {y}")
    case (x, y):
        print(f"Point: ({x}, {y})")

# 匹配字典
person = {"name": "Alice", "age": 30}
match person:
    case {"name": name, "age": age}:
        print(f"Name: {name}, Age: {age}")
    case {"name": name}:
        print(f"Name: {name}, Age unknown")
    case _:
        print("Invalid person data")

# 组合模式匹配
command = "quit"
match command:
    case "help" | "h" | "?":
        print("Show help")
    case "quit" | "q" | "exit":
        print("Exit program")
    case _:
        print("Unknown command")
```

## for 循环

**for 循环：遍历序列或可迭代对象**
`for <变量> in <可迭代对象>: <语句>`

```python
# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 遍历字符串
text = "Hello"
for char in text:
    print(char)

# 遍历元组
tuple_data = (1, 2, 3, 4, 5)
for num in tuple_data:
    print(num)

# 遍历字典
person = {"name": "Alice", "age": 30, "city": "New York"}

# 遍历键
for key in person:
    print(key)

# 遍历值
for value in person.values():
    print(value)

# 遍历键值对
for key, value in person.items():
    print(f"{key}: {value}")
```

**range() 函数：生成数值序列**
`range([<start>,] <stop> [, <step>])`

```python
# 基本用法
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# 指定起始值和结束值
for i in range(2, 7):
    print(i)  # 2, 3, 4, 5, 6

# 指定步长
for i in range(0, 10, 2):
    print(i)  # 0, 2, 4, 6, 8

# 倒序
for i in range(5, 0, -1):
    print(i)  # 5, 4, 3, 2, 1

# 遍历列表的索引
fruits = ["apple", "banana", "cherry"]
for i in range(len(fruits)):
    print(f"Index {i}: {fruits[i]}")
```

---

**enumerate() 函数：同时获取索引和值**
`enumerate(<可迭代对象> [, start=<起始索引>])`

```python
# 基本用法
fruits = ["apple", "banana", "cherry"]
for index, fruit in enumerate(fruits):
    print(f"Index {index}: {fruit}")

# 指定起始索引
for index, fruit in enumerate(fruits, start=1):
    print(f"Position {index}: {fruit}")

# 用于字符串
text = "Hello"
for index, char in enumerate(text):
    print(f"Character at {index}: {char}")
```

**zip() 函数：同时遍历多个序列**
`zip(<序列1>, <序列2>, ...)`

```python
# 基本用法
names = ["Alice", "Bob", "Charlie"]
ages = [30, 25, 35]
cities = ["New York", "London", "Paris"]
for name, age, city in zip(names, ages, cities):
    print(f"{name} is {age} years old from {city}")

# 处理不同长度的序列（只遍历到最短序列的长度）
short_list = [1, 2, 3]
long_list = [10, 20, 30, 40, 50]
for a, b in zip(short_list, long_list):
    print(f"{a} - {b}")

# 使用 zip(*) 解压缩
pairs = [(1, 10), (2, 20), (3, 30)]
a, b = zip(*pairs)
print(a)  # (1, 2, 3)
print(b)  # (10, 20, 30)
```

## while 循环

**while 循环：条件为真时重复执行**
`while <条件>: <语句>`

```python
# 基本用法
count = 0
while count < 5:
    print(count)
    count += 1

# 计算累加和
total = 0
number = 1
while number <= 10:
    total += number
    number += 1
print(f"Sum: {total}")  # 55

# 无限循环（需要 break 退出）
while True:
    user_input = input("Enter 'quit' to exit: ")
    if user_input == "quit":
        break
    print(f"You entered: {user_input}")

# 使用 else 子句
try_count = 0
max_tries = 3
while try_count < max_tries:
    print(f"Try {try_count + 1}")
    try_count += 1
else:
    print("Maximum tries reached")
```

## 循环控制语句

**break 语句：立即退出当前循环**
`break`

```python
# 在 for 循环中使用
fruits = ["apple", "banana", "cherry", "date"]
target = "cherry"
for fruit in fruits:
    if fruit == target:
        print(f"Found {target}!")
        break
    print(f"Checking {fruit}")

# 在 while 循环中使用
number = 0
while number < 10:
    print(number)
    if number == 5:
        break
    number += 1
```

---

**continue 语句：跳过本次循环，进入下一次迭代**
`continue`

```python
# 跳过偶数
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # 1, 3, 5, 7, 9

# 跳过空字符串
words = ["hello", "", "world", "", "python"]
for word in words:
    if not word:
        continue
    print(word)
```

## for-else 和 while-else

**循环 else 子句：循环正常结束时执行**
`for <变量> in <可迭代对象>: <语句>`
`else: <语句>`

```python
# for-else
fruits = ["apple", "banana", "cherry"]
target = "date"
for fruit in fruits:
    if fruit == target:
        print(f"Found {target}!")
        break
else:
    print(f"{target} not found")

# while-else
number = 0
target = 5
while number < 10:
    if number == target:
        print(f"Found {target}!")
        break
    number += 1
else:
    print(f"{target} not found in 0-9")

# 应用：查找素数
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    else:
        return True

print(is_prime(17))  # True
print(is_prime(18))  # False
```

## 异常处理

**try-except 语句：捕获和处理异常**
`try: <语句>`
`except <异常类型>: <处理>`
`else: <无异常时执行>`
`finally: <始终执行>`

```python
# 基本用法
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

# 捕获多种异常
try:
    number = int(input("Enter a number: "))
    result = 10 / number
except ValueError:
    print("Invalid input, please enter a number")
except ZeroDivisionError:
    print("Cannot divide by zero")

# 捕获所有异常
try:
    pass
except Exception as e:
    print(f"An error occurred: {e}")

# else 子句：当没有异常时执行
try:
    result = 10 / 2
except ZeroDivisionError:
    print("Cannot divide by zero")
else:
    print(f"Result: {result}")

# finally 子句：无论是否有异常都执行
try:
    file = open("example.txt", "r")
    content = file.read()
except FileNotFoundError:
    print("File not found")
finally:
    if 'file' in locals():
        file.close()
    print("File closed")

# 使用 with 语句（自动管理资源）
try:
    with open("example.txt", "r") as file:
        content = file.read()
        print(content)
except FileNotFoundError:
    print("File not found")
```
