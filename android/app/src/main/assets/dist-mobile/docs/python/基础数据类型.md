## 整数类型

**整数表示：支持十进制、二进制、八进制、十六进制**
`<十进制> = 123`
`<二进制> = 0b<二进制串>`
`<八进制> = 0o<八进制串>`
`<十六进制> = 0x<十六进制串>`

```python
# 十进制整数
x = 123

# 二进制整数（以 0b 或 0B 开头）
y = 0b1010  # 10

# 八进制整数（以 0o 或 0O 开头）
z = 0o123  # 83

# 十六进制整数（以 0x 或 0X 开头）
p = 0x1A  # 26
```

---

**整数运算：算术运算符**
`<运算符>: <表达式>`

```python
# 基本算术运算
addition = 10 + 5        # 15
subtraction = 10 - 5     # 5
multiplication = 10 * 5  # 50
division = 10 / 5        # 2.0（总是返回浮点数）
floor_division = 10 // 3 # 3（整除，向下取整）
modulo = 10 % 3          # 1（取模/余数）
power = 2 ** 3           # 8（幂运算）

# 复合赋值运算符
x = 10
x += 5  # x = x + 5 → 15
x -= 3  # x = x - 3 → 12
x *= 2  # x = x * 2 → 24
x /= 4  # x = x / 4 → 6.0
```

## 浮点数类型

**浮点数表示：普通小数与科学记数法**
`<浮点数> = <小数>`
`<科学记数法> = <尾数>e<指数>`

```python
# 普通浮点数
pi = 3.14159

# 科学记数法
avogadro = 6.022e23  # 6.022 × 10²³
small = 1.23e-4      # 1.23 × 10⁻⁴

# 浮点数运算
f1 = 3.14 + 2.71  # 5.85
f2 = 10.0 - 3.5  # 6.5
f3 = 2.5 * 4.0   # 10.0

# 浮点数精度问题
0.1 + 0.2  # 0.30000000000000004
```

## 复数类型

**复数：形如 `a + bj`**
`<复数> = <实部> + <虚部>j`
`complex(<实部>, <虚部>)`

```python
# 复数定义
c1 = 3 + 4j
c2 = complex(2, 5)  # 2 + 5j

# 复数运算
c3 = c1 + c2  # 5 + 9j
c4 = c1 * c2  # -14 + 23j

# 复数属性
print(c1.real)         # 3.0（实部）
print(c1.imag)         # 4.0（虚部）
print(c1.conjugate())  # 3 - 4j（共轭复数）
```

## 数学函数

**math 模块：提供数学运算函数**
`import math`
`math.<函数>(<参数>)`

```python
import math

# 常量
print(math.pi)  # π ≈ 3.141592653589793
print(math.e)   # e ≈ 2.718281828459045

# 三角函数
print(math.sin(math.pi/2))  # 1.0
print(math.cos(math.pi))    # -1.0

# 指数和对数
print(math.exp(1))       # e¹ ≈ 2.71828
print(math.log(math.e))  # 1.0
print(math.log10(100))   # 2.0

# 取整函数
print(math.floor(3.9))  # 3（向下取整）
print(math.ceil(3.1))   # 4（向上取整）
print(round(3.5))       # 4（四舍五入）

# 其他函数
print(math.sqrt(16))       # 4.0（平方根）
print(math.factorial(5))  # 120（阶乘）
```

## 字符串

**字符串定义：单引号、双引号、三引号、原始字符串**
`'<字符串>'`
`"<字符串>"`
`'''<多行字符串>'''`
`r'<原始字符串>'`
`b'<字节字符串>'`

```python
# 单引号字符串
s1 = 'Hello, World!'

# 双引号字符串
s2 = "Hello, World!"

# 三引号字符串（支持多行）
s3 = '''Hello,
World!'''

# 原始字符串（不转义）
s4 = r'C:\\path\\to\\file'  # 输出: C:\path\to\file

# 字节字符串
s5 = b'Hello'  # 类型为 bytes
```

---

**字符串拼接：使用 `+` 或 `*`**
`<字符串1> + <字符串2>`
`<字符串> * <次数>`

```python
# 使用 + 运算符
first_name = "Alice"
last_name = "Smith"
full_name = first_name + " " + last_name  # "Alice Smith"

# 使用 * 运算符重复
print("Hello" * 3)  # "HelloHelloHello"

# 使用 f-string
age = 30
message = f"My name is {full_name} and I am {age} years old."
```

---

**字符串切片：使用 `s[start:stop:step]`**
`<字符串>[<start>:<stop>:<step>]`

```python
s = "Hello, World!"

# 获取单个字符
print(s[0])    # 'H'
print(s[-1])   # '!'

# 切片操作
print(s[0:5])   # 'Hello'
print(s[7:])    # 'World!'
print(s[:5])    # 'Hello'
print(s[::2])   # 'Hlo ol!'
print(s[::-1])  # '!dlroW ,olleH'（反转字符串）
```

---

**字符串方法：常用操作**
`<字符串>.<方法>(<参数>)`

```python
s = "Hello, World!"

# 大小写转换
print(s.upper())       # 'HELLO, WORLD!'
print(s.lower())       # 'hello, world!'
print(s.title())       # 'Hello, World!'
print(s.capitalize())  # 'Hello, world!'

# 查找和替换
print(s.find("World"))           # 7
print(s.replace("World", "Python"))  # 'Hello, Python!'

# 分割和连接
print(s.split(", "))          # ['Hello', 'World!']
print(", ".join(["Hello", "Python"]))  # 'Hello, Python'

# 去除空白
print(" Hello ".strip())   # 'Hello'
print(" Hello ".lstrip())  # 'Hello '
print(" Hello ".rstrip())  # ' Hello'

# 检查方法
print(s.startswith("Hello"))  # True
print(s.endswith("!"))         # True
print("123".isdigit())        # True
```

## 字符串格式化

**f-string：Python 3.6+ 推荐方式**
`f"<文本>{<表达式>}"`

```python
name = "Alice"
age = 30

# 基本用法
print(f"My name is {name} and I am {age} years old.")

# 表达式
print(f"Next year I will be {age + 1} years old.")

# 格式化选项
pi = 3.14159
print(f"Pi is approximately {pi:.2f}")  # 3.14
print(f"Age: {age:03d}")                # 030
print(f"Name: {name:>10}")              # '     Alice'
print(f"Name: {name:<10}")              # 'Alice     '
print(f"Name: {name:^10}")              # '  Alice   '
```

---

**format() 方法：使用位置或关键字参数**
`"<文本>{}".format(<参数>)`

```python
# 基本用法
print("My name is {} and I am {} years old.".format("Alice", 30))

# 位置参数
print("{0} {1}".format("Hello", "World"))  # 'Hello World'
print("{1} {0}".format("Hello", "World"))  # 'World Hello'

# 关键字参数
print("My name is {name} and I am {age} years old.".format(name="Bob", age=25))

# 格式化选项
print("Pi is approximately {:.2f}".format(3.14159))  # 3.14
```

---

**% 运算符：C 风格格式化**
`"<格式>" % <值>`

```python
name = "Alice"
age = 30
pi = 3.14159

# 基本用法
print("My name is %s and I am %d years old." % (name, age))

# 格式化选项
print("Pi is approximately %.2f" % pi)  # 3.14
print("Age: %03d" % age)                # 030
print("Name: %10s" % name)              # '     Alice'
```

## 布尔类型

**布尔运算：逻辑运算符**
`<表达式> and <表达式>`
`<表达式> or <表达式>`
`not <表达式>`

```python
# 逻辑运算符
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# 比较运算符
print(5 > 3)    # True
print(5 == 3)   # False
print(5 != 3)   # True
print(5 >= 3)   # True
print(5 <= 3)   # False
```

---

**布尔上下文：所有对象都可在布尔上下文中使用**
`if <对象>: <语句>`

```python
# 布尔上下文中的评估
if "Hello":
    print("非空字符串为真")
if [1, 2, 3]:
    print("非空列表为真")
if 0:
    print("0 为假")
else:
    print("0 被视为 False")
if None:
    print("None 为假")
else:
    print("None 被视为 False")
```

## 空值 None

**None：表示「无」或「空」**
`<变量> = None`

```python
# 赋值
x = None
print(x)  # None

# 函数默认参数
def greet(name=None):
    if name is None:
        print("Hello, Guest!")
    else:
        print(f"Hello, {name}!")

greet()        # Hello, Guest!
greet("Alice") # Hello, Alice!

# 函数无显式返回值时返回 None
def no_return():
    pass

print(no_return())  # None

# 使用 is 运算符检查 None（推荐）
x = None
print(x is None)      # True
print(x is not None)  # False
```

## 类型转换

**基本类型转换：使用内置函数**
`int(<值>)`
`float(<值>)`
`str(<值>)`
`bool(<值>)`

```python
# 转换为整数
print(int("123"))   # 123
print(int(3.9))     # 3（向下取整）
print(int(True))    # 1
print(int(False))   # 0

# 转换为浮点数
print(float("3.14"))  # 3.14
print(float(10))       # 10.0
print(float(True))     # 1.0

# 转换为字符串
print(str(123))    # "123"
print(str(3.14))   # "3.14"
print(str(True))   # "True"
print(str(None))   # "None"

# 转换为布尔值
print(bool(1))       # True
print(bool(0))       # False
print(bool("Hello")) # True
print(bool(""))      # False
print(bool([]))      # False
print(bool(None))    # False
```

---

**容器类型转换：转换为列表、元组、集合、字典**
`list(<可迭代对象>)`
`tuple(<可迭代对象>)`
`set(<可迭代对象>)`
`dict(<键值对序列>)`

```python
# 转换为列表
print(list("Hello"))          # ['H', 'e', 'l', 'l', 'o']
print(list((1, 2, 3)))       # [1, 2, 3]

# 转换为元组
print(tuple([1, 2, 3]))      # (1, 2, 3)
print(tuple("Hello"))        # ('H', 'e', 'l', 'l', 'o')

# 转换为集合
print(set([1, 2, 3, 2, 1]))  # {1, 2, 3}（去重）

# 转换为字典
print(dict([("a", 1), ("b", 2)]))  # {'a': 1, 'b': 2}
```

## 类型检查

**type() 函数：获取对象类型**
`type(<对象>)`

```python
print(type(42))         # <class 'int'>
print(type(3.14))       # <class 'float'>
print(type("Hello"))    # <class 'str'>
print(type(True))       # <class 'bool'>
print(type(None))        # <class 'NoneType'>
print(type([1, 2, 3]))  # <class 'list'>
```

---

**isinstance() 函数：检查对象是否为指定类型**
`isinstance(<对象>, <类型>)`

```python
print(isinstance(42, int))          # True
print(isinstance(3.14, float))      # True
print(isinstance("Hello", str))     # True
print(isinstance([1, 2, 3], list)) # True

# 检查是否为多种类型之一
print(isinstance(42, (int, float))) # True
```
