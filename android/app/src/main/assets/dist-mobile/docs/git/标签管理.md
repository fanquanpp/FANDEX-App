# 标签管理

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 创建轻量标签

**在当前提交创建标签**：创建仅含提交引用的标签
`git tag <标签名>`

```bash
# 在当前提交创建 v1.0.0 标签
git tag v1.0.0;
```

**在指定提交创建标签**：为历史提交打标签
`git tag <标签名> <提交哈希>`

```bash
# 在 abc1234 提交创建 v0.9.0 标签
git tag v0.9.0 abc1234;
```

---

## 创建附注标签

**创建附注标签**：包含作者、日期、消息的标签
`git tag -a <标签名> -m "<标签消息>"`

```bash
# 创建附注标签 v1.0.0
git tag -a v1.0.0 -m "Release version 1.0.0";
```

**在指定提交创建附注标签**：为历史提交创建附注标签
`git tag -a <标签名> <提交哈希> -m "<标签消息>"`

```bash
# 在 abc1234 提交创建附注标签 v0.9.0
git tag -a v0.9.0 abc1234 -m "Release version 0.9.0";
```

---

## 语义化版本

**语义化版本格式**：标准版本号约定
`v<主版本号>.<次版本号>.<修订号>`

```text
# v1.2.3 含义
# 1 主版本号：不兼容的变更
# 2 次版本号：向后兼容的新功能
# 3 修订号：Bug 修复
v1.2.3
```

## 列出标签

**列出所有标签**：显示所有本地标签
`git tag`

```bash
# 列出所有标签
git tag;
```

**按模式过滤标签**：使用通配符筛选标签
`git tag -l "<模式>"`

```bash
# 列出 v1. 开头的标签
git tag -l "v1.*";
# 列出 v2.0 开头的标签
git tag -l "v2.0*";
```

**查看标签详情**：显示标签指向的提交信息
`git show <标签名>`

```bash
# 查看 v1.0.0 标签的详情
git show v1.0.0;
```

**查看标签对象内容**：显示标签对象的原始内容
`git cat-file -p <标签名>`

```bash
# 查看 v1.0.0 标签对象内容
git cat-file -p v1.0.0;
```

---

## 查看标签指向的提交

**获取标签指向的提交哈希**：解析标签引用
`git rev-parse <标签名>`

```bash
# 获取 v1.0.0 指向的提交哈希
git rev-parse v1.0.0;
```

**查看标签指向的提交日志**：显示标签对应的提交
`git log <标签名> -1`

```bash
# 查看 v1.0.0 标签指向的提交
git log v1.0.0 -1;
```

## 推送标签

**推送单个标签**：将指定标签推送到远程
`git push <远程仓库名> <标签名>`

```bash
# 推送 v1.0.0 标签到 origin
git push origin v1.0.0;
```

**推送所有标签**：将所有本地标签推送到远程
`git push <远程仓库名> --tags`

```bash
# 推送所有标签到 origin
git push origin --tags;
```

**只推送附注标签**：仅推送带注释的标签
`git push <远程仓库名> --follow-tags`

```bash
# 推送所有附注标签到 origin
git push origin --follow-tags;
```

## 删除标签

**删除本地标签**：移除本地仓库的标签
`git tag -d <标签名>`

```bash
# 删除本地 v1.0.0 标签
git tag -d v1.0.0;
```

**删除远程标签**：移除远程仓库的标签
`git push <远程仓库名> --delete <标签名>`

```bash
# 删除 origin 上的 v1.0.0 标签
git push origin --delete v1.0.0;
```

**删除远程标签（refs 写法）**：使用 refs 引用删除
`git push <远程仓库名> :refs/tags/<标签名>`

```bash
# 使用 refs 写法删除远程标签
git push origin :refs/tags/v1.0.0;
```

## 签名标签

**创建 GPG 签名标签**：使用 GPG 密钥签名标签
`git tag -s <标签名> -m "<标签消息>"`

```bash
# 创建 GPG 签名的 v1.0.0 标签
git tag -s v1.0.0 -m "Release v1.0.0";
```

**验证签名标签**：验证标签的 GPG 签名
`git tag -v <标签名>`

```bash
# 验证 v1.0.0 标签的签名
git tag -v v1.0.0;
```

---

## 配置 SSH 签名

**配置 SSH 签名格式**：使用 SSH 密钥进行签名
`git config --global gpg.format ssh`

```bash
# 配置使用 SSH 签名
git config --global gpg.format ssh;
```

**配置签名密钥**：指定用于签名的 SSH 密钥
`git config --global user.signingkey <密钥路径>`

```bash
# 指定 ed25519 密钥作为签名密钥
git config --global user.signingkey ~/.ssh/id_ed25519.pub;
```

## 检出标签

**检出到标签**：切换到标签对应的提交
`git checkout <标签名>`

```bash
# 切换到 v1.0.0 标签对应的提交
git checkout v1.0.0;
```
