# Go 泛型

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 泛型函数

**基本泛型函数**：使用类型参数
`func <函数名>[<类型参数> <约束>](<参数> <类型参数>) <返回值> { ... }`

```go
// 泛型函数，T 是类型参数
func Print[T any](v T) {
    fmt.Println(v);
}
```

**多类型参数**：多个类型参数
`func <函数名>[<T1>, <T2> <约束>](<参数>) <返回值>`

```go
// 多个类型参数
func Pair[K any, V any](k K, v V) (K, V) {
    return k, v;
}
```

## 类型约束

**any 约束**：无任何约束
`[T any]`

```go
// any 等价于 interface{}
func Identity[T any](v T) T {
    return v;
}
```

**comparable 约束**：支持 == 和 !=
`[T comparable]`

```go
// comparable 支持比较
func Contains[T comparable](slice []T, target T) bool {
    for _, v := range slice {
        if v == target {
            return true;
        }
    }
    return false;
}
```

**自定义约束**：定义类型集合
`type <约束名> interface { ~<类型1> | ~<类型2> }`

```go
// 自定义数值约束
type Number interface {
    ~int | ~int64 | ~float64;
}

func Sum[T Number](nums []T) T {
    var total T;
    for _, n := range nums {
        total += n;
    }
    return total;
}
```

**近似类型约束**：使用 ~ 包含底层类型
`~<类型>`

```go
// ~int 包含 int 及其自定义类型
type IntLike interface {
    ~int | ~int32 | ~int64;
}
```

## 泛型类型

**泛型切片**：参数化切片类型
`type <类型名>[T any] []T`

```go
// 泛型切片类型
type Stack[T any] []T;

func (s *Stack[T]) Push(v T) {
    *s = append(*s, v);
}

func (s *Stack[T]) Pop() (T, bool) {
    var zero T;
    if len(*s) == 0 {
        return zero, false;
    }
    v := (*s)[len(*s)-1];
    *s = (*s)[:len(*s)-1];
    return v, true;
}
```

**泛型 Map**：参数化 map 类型
`type <类型名>[K comparable, V any] map[K]V`

```go
// 泛型 map 类型
type Cache[K comparable, V any] map[K]V;

func (c Cache[K, V]) Get(k K) (V, bool) {
    v, ok := c[k];
    return v, ok;
}
```

**泛型结构体**：参数化结构体
`type <类型名>[T any] struct { ... }`

```go
// 泛型结构体
type Result[T any] struct {
    Value T;
    Err   error;
}

func Ok[T any](v T) Result[T] {
    return Result[T]{Value: v};
}
```

## 泛型方法

**泛型类型的方法**：方法使用类型参数
`func (<接收者> *<类型>[T]) <方法名>(<参数>) <返回值>`

```go
// 泛型类型的方法
type Queue[T any] struct {
    items []T;
}

func (q *Queue[T]) Enqueue(v T) {
    q.items = append(q.items, v);
}

func (q *Queue[T]) Dequeue() (T, bool) {
    var zero T;
    if len(q.items) == 0 {
        return zero, false;
    }
    v := q.items[0];
    q.items = q.items[1:];
    return v, true;
}
```

## 标准库约束

**constraints 包**：标准库约束
`[T constraints.Ordered]`

```go
// 使用标准库约束
import "golang.org/x/exp/constraints";

func Max[T constraints.Ordered](a, b T) T {
    if a > b {
        return a;
    }
    return b;
}
```

**cmp 包约束**：Go 1.21+ 内置
`[T cmp.Ordered]`

```go
// Go 1.21+ 使用 cmp 包
import "cmp";

func Min[T cmp.Ordered](a, b T) T {
    if a < b {
        return a;
    }
    return b;
}
```

## 泛型接口

**泛型接口**：参数化接口
`type <接口名>[T any] interface { <方法>(T) T }`

```go
// 泛型接口
type Container[T any] interface {
    Add(T);
    Get() (T, bool);
}

// 实现
type List[T any] struct {
    items []T;
}

func (l *List[T]) Add(v T) {
    l.items = append(l.items, v);
}

func (l *List[T]) Get() (T, bool) {
    var zero T;
    if len(l.items) == 0 {
        return zero, false;
    }
    return l.items[0], true;
}
```

## 类型推断

**自动类型推断**：调用时无需指定类型
`<函数名>(<参数>)`

```go
// 类型推断，无需显式指定 T
Print(42);           // T 推断为 int
Print("hello");      // T 推断为 string
Print(3.14);         // T 推断为 float64
```

**显式指定类型**：需要时显式指定
`<函数名>[<类型>](<参数>)`

```go
// 显式指定类型参数
Print[int](42);
Print[string]("hello");
```

## 泛型算法

**泛型过滤**：通用过滤函数
`func <函数名>[T any](<切片> []T, <函数> func(T) bool) []T`

```go
// 泛型过滤
func Filter[T any](slice []T, fn func(T) bool) []T {
    result := make([]T, 0);
    for _, v := range slice {
        if fn(v) {
            result = append(result, v);
        }
    }
    return result;
}
```

**泛型映射**：通用映射函数
`func <函数名>[T, U any](<切片> []T, <函数> func(T) U) []U`

```go
// 泛型映射
func Map[T, U any](slice []T, fn func(T) U) []U {
    result := make([]U, len(slice));
    for i, v := range slice {
        result[i] = fn(v);
    }
    return result;
}
```

**泛型归约**：通用归约函数
`func <函数名>[T, U any](<切片> []T, <初始值> U, <函数> func(U, T) U) U`

```go
// 泛型归约
func Reduce[T, U any](slice []T, initial U, fn func(U, T) U) U {
    result := initial;
    for _, v := range slice {
        result = fn(result, v);
    }
    return result;
}
```
