# Go 内存对齐

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本类型大小

**类型大小查询**：使用 unsafe.Sizeof
`unsafe.Sizeof(<变量>)`

```go
// 基本类型大小（64 位系统）
fmt.Println(unsafe.Sizeof(bool(false)));    // 1
fmt.Println(unsafe.Sizeof(int8(0)));        // 1
fmt.Println(unsafe.Sizeof(int16(0)));       // 2
fmt.Println(unsafe.Sizeof(int32(0)));       // 4
fmt.Println(unsafe.Sizeof(int64(0)));       // 8
fmt.Println(unsafe.Sizeof(int(0)));         // 8
fmt.Println(unsafe.Sizeof(float32(0)));     // 4
fmt.Println(unsafe.Sizeof(float64(0)));     // 8
fmt.Println(unsafe.Sizeof(string("")));     // 16
fmt.Println(unsafe.Sizeof([]int{}));         // 24
```

## 对齐系数

**类型对齐查询**：使用 unsafe.Alignof

```go
// 基本类型对齐系数（64 位系统）
fmt.Println(unsafe.Alignof(bool(false)));    // 1

fmt.Println(unsafe.Alignof(int64(0)));       // 8
fmt.Println(unsafe.Alignof(string("")));     // 8
fmt.Println(unsafe.Alignof([]int{}));         // 8
```

## 结构体内存布局

**字段顺序影响大小**：未优化布局
`type <类型> struct { ... }`

```go
// 未优化：24 字节
type Bad struct {
    A bool;    // 1 + 7 padding
    B int64;   // 8
    C int32;   // 4 + 4 padding
}
fmt.Println(unsafe.Sizeof(Bad{})); // 24
```

**优化后布局**：字段按大小降序排列
`type <类型> struct { ... }`

```go
// 优化后：16 字节
type Good struct {
    B int64;   // 8
    C int32;   // 4
    A bool;    // 1 + 3 padding
}
fmt.Println(unsafe.Sizeof(Good{})); // 16
```

## 字段偏移量

**获取字段偏移**：使用 unsafe.Offsetof
`unsafe.Offsetof(<结构体>.<字段>)`

```go
// 查看字段偏移量
type User struct {
    ID   int64;   // offset 0
    Name string;  // offset 8
    Age  int32;    // offset 24
    Active bool;  // offset 28
}
fmt.Println(unsafe.Offsetof(User{}.ID));     // 0
fmt.Println(unsafe.Offsetof(User{}.Name));   // 8
fmt.Println(unsafe.Offsetof(User{}.Age));     // 24
fmt.Println(unsafe.Offsetof(User{}.Active));  // 28
```

## Padding 计算

**Padding 规则**：字段起始地址必须是其对齐系数的倍数
`<字段对齐> - (<前字段结束> % <字段对齐>)`

```go
// Padding 计算示例
type Example struct {
    A int8;    // offset 0, size 1
    // padding 7 字节（对齐到 8）
    B int64;   // offset 8, size 8
    C int8;    // offset 16, size 1
    // padding 3 字节（结构体对齐 8）
}
// 总大小：24 字节
fmt.Println(unsafe.Sizeof(Example{})); // 24
```

## 结构体对齐规则

**结构体对齐**：取最大字段的对齐系数
`unsafe.Alignof(<结构体>{})`

```go
// 结构体对齐系数
type S struct {
    A int8;   // align 1
    B int64;  // align 8
    C int32;  // align 4
}
// 结构体对齐 = max(1, 8, 4) = 8
fmt.Println(unsafe.Alignof(S{})); // 8
```

**结构体大小计算**：最后一个字段结束 + padding 到对齐倍数
`<最后字段偏移> + <最后字段大小> + <padding>`

```go
// 结构体大小计算
type T struct {
    A int8;    // offset 0, size 1
    B int64;   // offset 8, size 8
}
// 最后字段结束：8 + 8 = 16
// 对齐到 8 的倍数：16（无需 padding）
fmt.Println(unsafe.Sizeof(T{})); // 16
```

## 切片和 Map 大小

**切片大小**：固定 24 字节（64 位）
`unsafe.Sizeof(<切片>)`

```go
// 切片大小固定（Data + Len + Cap）
s := make([]int, 100);
fmt.Println(unsafe.Sizeof(s)); // 24（不随元素数量变化）
```

**Map 大小**：固定 8 字节（指针）
`unsafe.Sizeof(<map>)`

```go
// Map 大小固定（仅指针）
m := make(map[string]int);
fmt.Println(unsafe.Sizeof(m)); // 8
```

**字符串大小**：固定 16 字节
`unsafe.Sizeof(<字符串>)`

```go
// 字符串大小固定（Data + Len）
s := "hello world";
fmt.Println(unsafe.Sizeof(s)); // 16
```

## 优化实践

**字段排列优化**：大字段在前
`type <类型> struct { <大字段>; <小字段> }`

```go
// 优化前：24 字节
type Before struct {
    Flag bool;     // 1 + 7 padding
    ID   int64;    // 8
    Age  int32;    // 4 + 4 padding
}

// 优化后：16 字节
type After struct {
    ID   int64;    // 8
    Age  int32;    // 4
    Flag bool;     // 1 + 3 padding
}
```

**相同类型字段聚集**：减少 padding
`type <类型> struct { <类型1字段们>; <类型2字段们> }`

```go
// 相同类型聚集减少 padding
type Optimized struct {
    A int64;
    B int64;
    C int32;
    D int32;
    E bool;
    F bool;
}
fmt.Println(unsafe.Sizeof(Optimized{})); // 28 -> 对齐到 32
```

## 64 位对齐

**64 位原子操作对齐**：需要 64 位对齐
`type <类型> struct { <字段> int64; ... }`

```go
// 32 位系统上原子操作需要 64 位对齐
type Counter struct {
    // 首字段为 int64 确保 64 位对齐
    Count int64;
    Name  string;
}

var c Counter;
atomic.AddInt64(&c.Count, 1);
```

**显式对齐**：使用空白字段
`_ [0]int64`

```go
// 强制 64 位对齐
type Aligned struct {
    _    [0]int64 // 强制对齐
    Data int64;
}
```

## 工具函数

**计算对齐后大小**：手动计算
`(<大小> + <对齐> - 1) &^ (<对齐> - 1)`

```go
// 对齐计算函数
func alignUp(n, a uintptr) uintptr {
    return (n + a - 1) &^ (a - 1);
}

// 使用
size := alignUp(13, 8); // 16
```
