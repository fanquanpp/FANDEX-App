# Go 错误处理

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## error 接口

**error 接口定义**：内置错误接口
`type error interface { Error() string }`

```go
// error 是内置接口
type error interface {
    Error() string;
}
```

---

## 基本错误处理

**函数返回错误**：标准错误返回模式
`func <函数名>(<参数>) (<返回值>, error) { ... }`

```go
// 返回结果和错误
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero");
    }
    return a / b, nil;
}
```

**检查错误**：if 语句检查 err
`if <err> != nil { ... }`

```go
// 标准错误检查
result, err := divide(10, 0);
if err != nil {
    log.Fatal(err);
}
fmt.Println(result);
```

## 创建错误

**errors.New**：创建简单错误
`errors.New("<消息>")`

```go
// 创建简单错误
err := errors.New("something went wrong");
```

**fmt.Errorf**：格式化错误消息
`fmt.Errorf("<格式>", <参数>)`

```go
// 格式化错误消息
err := fmt.Errorf("user %s not found", username);
```

## 自定义错误类型

**结构体错误**：实现 error 接口
`type <错误类型> struct { ... }`

```go
// 自定义错误类型
type ValidationError struct {
    Field   string;
    Message string;
}

func (e *ValidationError) Error() string {
    return fmt.Sprintf("validation error: %s - %s", e.Field, e.Message);
}
```

**带错误码的错误**：结构化错误信息
`type <错误类型> struct { Code int; Msg string }`

```go
// 带错误码的自定义错误
type AppError struct {
    Code int;
    Msg  string;
}

func (e *AppError) Error() string {
    return fmt.Sprintf("[%d] %s", e.Code, e.Msg);
}
```

## 错误包装与解包

**错误包装**：添加上下文信息
`fmt.Errorf("<上下文>: %w", <错误>)`

```go
// 包装错误添加上下文
if err := readFile(); err != nil {
    return fmt.Errorf("failed to load config: %w", err);
}
```

**错误解包**：获取被包装的错误
`errors.Unwrap(<错误>)`

```go
// 解包错误
inner := errors.Unwrap(wrappedErr);
```

**错误匹配**：检查错误是否为指定类型
`errors.Is(<错误>, <目标错误>)`

```go
// 检查错误链中是否包含特定错误
if errors.Is(err, os.ErrNotExist) {
    fmt.Println("file not found");
}
```

**错误类型匹配**：提取特定类型的错误
`errors.As(<错误>, &<目标>)`

```go
// 提取特定类型的错误
var pathErr *fs.PathError;
if errors.As(err, &pathErr) {
    fmt.Println("path:", pathErr.Path);
}
```

## panic 与 recover

**panic**：触发运行时错误
`panic(<值>)`

```go
// 触发 panic
func mustParse(s string) int {
    n, err := strconv.Atoi(s);
    if err != nil {
        panic(fmt.Sprintf("invalid number: %s", s));
    }
    return n;
}
```

**recover**：捕获 panic
`recover()`

```go
// 在 defer 中 recover
func safeDivide(a, b int) (result int, err error) {
    defer func() {
        if r := recover(); r != nil {
            err = fmt.Errorf("recovered: %v", r);
        }
    }();
    return a / b, nil;
}
```

**安全调用封装**：通用 recover 包装
`func <函数名>() { defer func() { recover() }() }`

```go
// 安全执行封装
func SafeGo(fn func()) {
    go func() {
        defer func() {
            if r := recover(); r != nil {
                log.Printf("recovered: %v", r);
            }
        }();
        fn();
    }();
}
```

## 错误处理模式

**哨兵错误**：预定义错误变量
`var Err<名称> = errors.New("<消息>")`

```go
// 哨兵错误
var ErrNotFound = errors.New("not found");
var ErrInvalid = errors.New("invalid input");

// 使用
if err := getUser(id); err != nil {
    if errors.Is(err, ErrNotFound) {
        // 处理未找到
    }
}
```

**错误类型断言**：检查具体错误类型
`if <变量>, ok := <错误>.(<类型>); ok { ... }`

```go
// 类型断言检查错误
if pathErr, ok := err.(*fs.PathError); ok {
    fmt.Println("path:", pathErr.Path);
}
```

## 错误聚合

**多错误合并**：使用 errors.Join
`errors.Join(<错误1>, <错误2>)`

```go
// 合并多个错误
err1 := validateName(name);
err2 := validateEmail(email);
err := errors.Join(err1, err2);
```

**错误迭代**：遍历聚合错误
`errors.Unwrap(<错误>)`

```go
// 迭代处理多个错误
for err := aggregatedErr; err != nil; err = errors.Unwrap(err) {
    fmt.Println(err);
}
```

## defer 错误处理

**defer 中处理错误**：确保资源释放
`defer func() { ... }()`

```go
// defer 中关闭并处理错误
func processFile(path string) error {
    file, err := os.Open(path);
    if err != nil {
        return err;
    }
    defer func() {
        if cerr := file.Close(); cerr != nil {
            log.Printf("close error: %v", cerr);
        }
    }();
    // 处理文件
    return nil;
}
```

## 错误处理最佳实践

**错误立即处理**：不要忽略错误
`if <err> != nil { return <err> }`

```go
// 立即返回错误
data, err := readFile(path);
if err != nil {
    return err;
}
```

**错误包装添加上下文**：保留原始错误
`fmt.Errorf("<上下文>: %w", <错误>)`

```go
// 包装错误保留原始信息
if err := db.Query(); err != nil {
    return fmt.Errorf("query users: %w", err);
}
```

**区分错误类型**：使用哨兵或类型
`var Err<名称> = errors.New("<消息>")`

```go
// 定义可检查的错误
var (
    ErrNotFound    = errors.New("not found");
    ErrUnauthorized = errors.New("unauthorized");
    ErrConflict    = errors.New("conflict");
);
```
