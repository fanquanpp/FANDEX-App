# Go 反射

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 反射三定律

**从接口到反射对象**：获取反射对象
`reflect.ValueOf(<值>)` / `reflect.TypeOf(<值>)`

```go
// 获取反射对象
var x float64 = 3.4;
v := reflect.ValueOf(x);
t := reflect.TypeOf(x);
fmt.Println("type:", t);        // float64
fmt.Println("value:", v);       // 3.4
```

**从反射对象到接口值**：还原为接口
`<反射对象>.Interface()`

```go
// 反射对象转回接口
v := reflect.ValueOf(42);
i := v.Interface();
n := i.(int);
```

**修改反射对象**：使用指针修改原值
`reflect.ValueOf(&<变量>).Elem()`

```go
// 通过指针修改原值
var x float64 = 3.4;
v := reflect.ValueOf(&x).Elem();
v.SetFloat(7.1);
```

## Type 操作

**获取类型信息**：类型名称和种类
`<Type>.Name()` / `<Type>.Kind()`

```go
// 获取类型名称和种类
type User struct{ Name string };
t := reflect.TypeOf(User{});
fmt.Println(t.Name()); // "User"
fmt.Println(t.Kind()); // struct
```

**Kind 枚举值**：类型种类
`reflect.<Kind>`

```go
// 常见 Kind 值
// reflect.Int / reflect.String / reflect.Bool
// reflect.Struct / reflect.Slice / reflect.Map
// reflect.Ptr / reflect.Func / reflect.Interface
```

## Value 操作

**获取值**：不同类型的获取方法
`<Value>.<类型>()`

```go
// 获取不同类型的值
v := reflect.ValueOf(42);
n := v.Int();      // int64

s := reflect.ValueOf("hello");
str := s.String(); // string

b := reflect.ValueOf(true);
flag := b.Bool();  // bool
```

**设置值**：通过指针修改
`<Value>.Set<类型>(<值>)`

```go
// 设置值（必须通过指针）
var x int = 10;
v := reflect.ValueOf(&x).Elem();
v.SetInt(100);
```

**CanSet 检查**：判断是否可修改
`<Value>.CanSet()`

```go
// 检查是否可设置
v := reflect.ValueOf(&x).Elem();
if v.CanSet() {
    v.SetInt(100);
}
```

## Struct 反射

**遍历字段**：获取结构体字段
`<Type>.NumField()` / `<Type>.Field(<索引>)`

```go
// 遍历结构体字段
type User struct {
    Name string `json:"name"`;
    Age  int    `json:"age"`;
}

t := reflect.TypeOf(User{});
for i := 0; i < t.NumField(); i++ {
    field := t.Field(i);
    fmt.Printf("%s %s %s\n", field.Name, field.Type, field.Tag);
}
```

**获取字段值**：通过 Value 获取
`<Value>.Field(<索引>)`

```go
// 获取字段值
u := User{Name: "Alice", Age: 30};
v := reflect.ValueOf(u);
name := v.Field(0).String();
age := v.Field(1).Int();
```

**设置字段值**：通过指针修改
`<Value>.Field(<索引>).Set<类型>(<值>)`

```go
// 设置字段值
u := &User{Name: "Alice", Age: 30};
v := reflect.ValueOf(u).Elem();
v.Field(0).SetString("Bob");
```

**读取标签**：获取字段标签
`<字段>.Tag.Get("<标签名>")`

```go
// 读取 json 标签
t := reflect.TypeOf(User{});
field, _ := t.FieldByName("Name");
jsonTag := field.Tag.Get("json");
```

## Method 反射

**获取方法数量**：统计方法数
`<Type>.NumMethod()`

```go
// 获取方法数量
type Calculator struct{};
func (c Calculator) Add(a, b int) int { return a + b }

t := reflect.TypeOf(Calculator{});
fmt.Println(t.NumMethod()); // 1
```

**调用方法**：通过反射调用
`<Value>.Method(<索引>).Call([]reflect.Value{ ... })`

```go
// 通过反射调用方法
c := Calculator{};
v := reflect.ValueOf(c);
method := v.Method(0);
result := method.Call([]reflect.Value{
    reflect.ValueOf(1),
    reflect.ValueOf(2),
});
fmt.Println(result[0].Int()); // 3
```

**按名称调用方法**：通过方法名调用
`<Value>.MethodByName("<方法名>")`

```go
// 按名称调用方法
method := v.MethodByName("Add");
result := method.Call([]reflect.Value{
    reflect.ValueOf(10),
    reflect.ValueOf(20),
});
```

## Slice 反射

**创建切片**：反射创建
`reflect.MakeSlice(<Type>, <长度>, <容量>)`

```go
// 反射创建切片
sliceType := reflect.TypeOf([]int{});
s := reflect.MakeSlice(sliceType, 0, 10);
```

**Append 元素**：反射追加
`reflect.Append(<切片>, <元素>)`

```go
// 反射追加元素
s := reflect.MakeSlice(reflect.TypeOf([]int{}), 0, 10);
s = reflect.Append(s, reflect.ValueOf(1));
s = reflect.Append(s, reflect.ValueOf(2), reflect.ValueOf(3));
```

**获取长度和容量**：反射查询
`<Value>.Len()` / `<Value>.Cap()`

```go
// 获取长度和容量
v := reflect.ValueOf([]int{1, 2, 3});
fmt.Println(v.Len()); // 3
fmt.Println(v.Cap()); // 3
```

## Map 反射

**创建 Map**：反射创建
`reflect.MakeMap(<Type>)` / `reflect.MakeMapWithSize(<Type>, <大小>)`

```go
// 反射创建 map
mapType := reflect.TypeOf(map[string]int{});
m := reflect.MakeMap(mapType);
```

**设置键值**：反射添加
`<Value>.SetMapIndex(<键>, <值>)`

```go
// 反射设置键值对
m.SetMapIndex(reflect.ValueOf("one"), reflect.ValueOf(1));
m.SetMapIndex(reflect.ValueOf("two"), reflect.ValueOf(2));
```

**获取值**：反射读取
`<Value>.MapIndex(<键>)`

```go
// 反射获取值
v := m.MapIndex(reflect.ValueOf("one"));
fmt.Println(v.Int()); // 1
```

**遍历 Map**：反射迭代
`<Value>.MapRange()`

```go
// 反射遍历 map
iter := m.MapRange();
for iter.Next() {
    k := iter.Key();
    v := iter.Value();
    fmt.Printf("%s: %d\n", k, v);
}
```

## Func 反射

**调用函数**：反射调用
`<Value>.Call([]reflect.Value{ ... })`

```go
// 反射调用函数
add := func(a, b int) int { return a + b };
v := reflect.ValueOf(add);
result := v.Call([]reflect.Value{
    reflect.ValueOf(1),
    reflect.ValueOf(2),
});
fmt.Println(result[0].Int()); // 3
```

## 反射创建实例

**通过类型创建实例**：反射实例化
`reflect.New(<Type>)`

```go
// 反射创建实例
t := reflect.TypeOf(User{});
v := reflect.New(t); // 返回 *User
instance := v.Interface().(*User);
```

## 反射判断

**IsNil 检查**：判断是否为 nil
`<Value>.IsNil()`

```go
// 检查是否为 nil
var p *int;
v := reflect.ValueOf(p);
if v.IsNil() {
    fmt.Println("nil pointer");
}
```

**IsValid 检查**：判断是否有效
`<Value>.IsValid()`

```go
// 检查是否有效
v := reflect.ValueOf(nil);
if !v.IsValid() {
    fmt.Println("invalid value");
}
```

**Zero 值**：获取类型的零值
`reflect.Zero(<Type>)`

```go
// 获取零值
t := reflect.TypeOf(0);
zero := reflect.Zero(t);
fmt.Println(zero.Int()); // 0
```
