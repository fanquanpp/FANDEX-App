# Go 基础语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 变量声明

**完整声明**：显式指定类型和初始值
`var <变量名> <类型> = <值>`

```go
// 显式声明变量类型和初始值
var name string = "Go";
var age = 15;
```

**批量声明**：同时声明多个变量
`var ( ... )`

```go
// 批量声明多个变量
var (
    x    int     = 10
    y    float64 = 3.14
    flag bool    = true
);
```

**短变量声明**：函数内最常用写法
`<变量名> := <值>`

```go
// 仅限函数内使用，自动推断类型
city := "Beijing";
count := 100;
```

**零值声明**：未初始化时使用零值
`var <变量名> <类型>`

```go
// 未初始化的变量使用零值
var score int;    // 0
var title string; // ""
var done bool;    // false
```

## 常量

**常量声明**：编译时确定的不可变值
`const <名称> = <值>`

```go
// 声明常量
const Pi = 3.14159;
const Language = "Go";
```

**批量声明常量**：同时声明多个常量
`const ( ... )`

```go
// 批量声明常量
const (
    StatusOK    = 200
    StatusError = 500
);
```

**iota 常量生成器**：自动递增的常量
`const ( <名称> = iota ... )`

```go
// iota 从 0 开始自动递增
const (
    Sunday    = iota // 0
    Monday           // 1
    Tuesday          // 2
);
```

**iota 位运算标志**：使用位运算生成标志位
`const ( <名称> = 1 << iota ... )`

```go
// 位运算生成权限标志
const (
    ReadPermission   = 1 << iota // 1  (001)
    WritePermission              // 2  (010)
    ExecutePermission            // 4  (100)
);
```

**iota 跳过值**：使用下划线跳过特定值
`_ = iota`

```go
// 跳过 0，从 1024 开始
const (
    _  = iota
    KB = 1 << (10 * iota) // 1024
    MB                    // 1048576
);
```

## 类型转换

**显式类型转换**：Go 不支持隐式转换
`<目标类型>(<值>)`

```go
// int 转 float64
var i int = 42;
var f float64 = float64(i);
```

**字符串与数值转换**：使用 strconv 包
`strconv.Itoa(<整数>)`

```go
// int 转 string
s := strconv.Itoa(42);
// string 转 int
n, err := strconv.Atoi("42");
```

**字符串与字节切片转换**：零拷贝转换
`[]byte(<字符串>)`

```go
// string 转 []byte
bytes := []byte("hello");
// []byte 转 string
str := string(bytes);
```

## 字符串操作

**字符串长度**：字节长度与字符数
`len(<字符串>)`

```go
// 字节数（UTF-8 编码）
fmt.Println(len("Hello, 世界"));                    // 13
fmt.Println(utf8.RuneCountInString("Hello, 世界")); // 9
```

**遍历字符串**：按字节或按 rune 遍历
`for i, r := range <字符串>`

```go
// 按 rune 遍历，正确处理 Unicode
for i, r := range "Hello" {
    fmt.Printf("%d:%c ", i, r);
}
```

**字符串查找**：使用 strings 包
`strings.Contains(<字符串>, <子串>)`

```go
// 查找与判断
strings.Contains("Hello, World", "World");  // true
strings.HasPrefix("Hello, World", "Hello");  // true
strings.HasSuffix("Hello, World", "World");  // true
strings.Index("Hello, World", "World");      // 7
```

**字符串变换**：大小写与替换
`strings.ToUpper(<字符串>)`

```go
// 字符串变换
strings.ToUpper("Hello");              // "HELLO"
strings.ToLower("Hello");              // "hello"
strings.TrimSpace("  hi  ");           // "hi"
strings.Replace("Hello", "l", "L", 1); // "HeLlo"
```

**字符串拆分与合并**：分隔与连接
`strings.Split(<字符串>, <分隔符>)`

```go
// 拆分字符串
parts := strings.Split("a,b,c", ",");
// 合并字符串
joined := strings.Join(parts, "-");
```

**字符串构建**：避免频繁拼接
`strings.Builder`

```go
// 使用 Builder 高效构建字符串
var b strings.Builder;
b.WriteString("Hello");
b.WriteString(", World");
result := b.String();
```

**原始字符串**：反引号包围不处理转义
`` `<内容> ``

```go
// 反引号包围的原始字符串
raw := `C:\Users\name\file.txt`;
```

## 指针

**取地址与解引用**：获取变量地址和值
`&<变量>` / `*<指针>`

```go
// 取地址和解引用
x := 42;
p := &x;          // p 是 *int 类型
fmt.Println(*p);   // 42（解引用）
*p = 100;         // 通过指针修改值
```

**指针零值检查**：nil 指针安全检查
`if <指针> != nil`

```go
// nil 指针检查
var p *int;
if p != nil {
    fmt.Println(*p);
}
```

**指针传递**：函数内修改影响外部
`func <函数名>(<参数> *<类型>)`

```go
// 指针传递修改外部变量
func doublePtr(n *int) {
    *n *= 2;
}
```

**new 函数**：分配零值内存并返回指针
`new(<类型>)`

```go
// new 分配零值内存
p := new(int);
*p = 42;
```

## if 语句

**基本 if 语句**：条件不需要括号
`if <条件> { ... }`

```go
// 基本条件判断
if x > 0 {
    fmt.Println("positive");
} else if x < 0 {
    fmt.Println("negative");
} else {
    fmt.Println("zero");
}
```

**带初始化的 if**：变量作用域限定在 if 块内
`if <初始化>; <条件> { ... }`

```go
// 初始化语句中的变量仅在此块可见
if err := doSomething(); err != nil {
    fmt.Println("Error:", err);
}
```

## for 循环

**经典三段式**：初始化、条件、后置
`for <初始化>; <条件>; <后置> { ... }`

```go
// 经典 for 循环
for i := 0; i < 10; i++ {
    fmt.Println(i);
}
```

**while 风格**：省略初始化和后置
`for <条件> { ... }`

```go
// while 风格循环
n := 1;
for n < 100 {
    n *= 2;
}
```

**无限循环**：省略所有部分
`for { ... }`

```go
// 无限循环
for {
    if shouldBreak() {
        break;
    }
}
```

**for-range 遍历**：遍历集合
`for <索引>, <值> := range <集合> { ... }`

```go
// 遍历切片
nums := []int{1, 2, 3};
for i, v := range nums {
    fmt.Printf("index=%d value=%d\n", i, v);
}
```

**Go 1.22+ for-range 整数**：直接遍历整数
`for i := range <整数> { ... }`

```go
// 遍历 0 到 4
for i := range 5 {
    fmt.Println(i);
}
```

## switch 语句

**基本 switch**：自动 break 不穿透
`switch <表达式> { case ... }`

```go
// 基本 switch 语句
switch day {
case "Monday":
    fmt.Println("周一");
case "Tuesday":
    fmt.Println("周二");
default:
    fmt.Println("其他");
}
```

**多值匹配**：一个 case 匹配多个值
`case <值1>, <值2>, <值3>:`

```go
// 多值匹配
switch color {
case "red", "green", "blue":
    fmt.Println("基础颜色");
}
```

**fallthrough 穿透**：强制执行下一个 case
`fallthrough`

```go
// fallthrough 继续执行下一个 case
switch n := 2; n {
case 1:
    fmt.Println("一");
    fallthrough;
case 2:
    fmt.Println("二");
    fallthrough;
case 3:
    fmt.Println("三");
}
```

**无条件 switch**：替代 if-else 链
`switch { case <条件>: ... }`

```go
// 无条件 switch
score := 85;
switch {
case score >= 90:
    fmt.Println("A");
case score >= 80:
    fmt.Println("B");
default:
    fmt.Println("D");
}
```

## break 与 continue

**continue 跳过**：跳过本次迭代
`continue`

```go
// 跳过当前迭代
for i := 0; i < 10; i++ {
    if i == 3 {
        continue;
    }
    fmt.Println(i);
}
```

**break 退出**：退出循环
`break`

```go
// 退出循环
for i := 0; i < 10; i++ {
    if i == 7 {
        break;
    }
    fmt.Println(i);
}
```

**标签跳转**：跳出外层循环
`break <标签>`

```go
// 标签跳转跳出外层循环
outer:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i == 1 && j == 1 {
            break outer;
        }
    }
}
```

## defer 语句

**基本 defer**：推迟到函数返回前执行
`defer <函数调用>`

```go
// 确保文件关闭
func readFile(path string) {
    file, err := os.Open(path);
    if err != nil {
        return;
    }
    defer file.Close();
}
```

**defer 执行顺序**：后进先出（LIFO）
`defer <函数调用>`

```go
// 多个 defer 按后进先出执行
defer fmt.Println("第一");  // 最后执行
defer fmt.Println("第二");  // 第二执行
defer fmt.Println("第三");  // 最先执行
```

**defer 参数求值**：参数在声明时立即求值
`defer <函数>(<参数>)`

```go
// 参数在声明时求值
x := 10;
defer fmt.Println(x); // 输出 10
x = 20;
```

**defer 修改命名返回值**：defer 可修改返回值
`defer func() { ... }()`

```go
// defer 修改命名返回值
func double() (result int) {
    defer func() {
        result *= 2;
    }();
    return 5;
}
```
