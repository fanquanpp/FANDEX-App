# Go 并发编程

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## Goroutine

**启动 goroutine**：使用 go 关键字
`go <函数>(<参数>)`

```go
// 启动 goroutine
go sayHello("World");

// 匿名函数 goroutine
go func(msg string) {
    fmt.Println(msg);
}("hello");
```

## Channel 创建

**无缓冲通道**：同步通信
`make(chan <类型>)`

```go
// 无缓冲通道，发送和接收必须同时就绪
ch := make(chan int);
```

**有缓冲通道**：异步通信
`make(chan <类型>, <容量>)`

```go
// 有缓冲通道，容量为 3
ch := make(chan string, 3);
```

---

## Channel 操作

**发送数据**：向通道发送值
`<通道> <- <值>`

```go
// 发送数据到通道
ch <- 42;
```

**接收数据**：从通道接收值
`<-<通道>`

```go
// 接收数据
v := <-ch;
```

**关闭通道**：关闭后不可发送但可接收
`close(<通道>)`

```go
// 关闭通道
close(ch);
```

**检查通道是否关闭**：带 ok 的接收
`<值>, <ok> := <-<通道>`

```go
// 检查通道是否关闭
v, ok := <-ch;
if !ok {
    fmt.Println("channel closed");
}
```

**遍历通道**：持续接收直到关闭
`for <值> := range <通道> { ... }`

```go
// 遍历通道直到关闭
for v := range ch {
    fmt.Println(v);
}
```

## select 语句

**基本 select**：多路复用
`select { case ... }`

```go
// select 多路复用
select {
case v := <-ch1:
    fmt.Println("from ch1:", v);
case v := <-ch2:
    fmt.Println("from ch2:", v);
case ch3 <- 42:
    fmt.Println("sent to ch3");
}
```

**带超时的 select**：使用 time.After
`case <-time.After(<时长>):`

```go
// 超时控制
select {
case v := <-ch:
    fmt.Println("received:", v);
case <-time.After(3 * time.Second):
    fmt.Println("timeout");
}
```

**带 default 的 select**：非阻塞操作
`default:`

```go
// 非阻塞接收
select {
case v := <-ch:
    fmt.Println("received:", v);
default:
    fmt.Println("no data");
}
```

## sync.WaitGroup

**WaitGroup 基本用法**：等待一组 goroutine 完成
`var <变量> sync.WaitGroup`

```go
// 使用 WaitGroup 等待所有 goroutine
var wg sync.WaitGroup;
for i := 0; i < 5; i++ {
    wg.Add(1);
    go func(n int) {
        defer wg.Done();
        fmt.Println("worker", n);
    }(i);
}
wg.Wait();
```

**Add/Done/Wait**：核心方法
`<wg>.Add(<数量>)` / `<wg>.Done()` / `<wg>.Wait()`

```go
// 计数器操作
wg.Add(1);    // 计数器 +1
go func() {
    defer wg.Done();  // 计数器 -1
    // 执行任务
}();
wg.Wait();    // 阻塞直到计数器归零
```

## sync.Mutex

**互斥锁**：保护共享资源
`var <变量> sync.Mutex`

```go
// 互斥锁保护共享变量
var (
    counter int;
    mu      sync.Mutex;
);

func increment() {
    mu.Lock();
    defer mu.Unlock();
    counter++;
}
```

**读写锁**：区分读和写
`var <变量> sync.RWMutex`

```go
// 读写锁，允许多个读或单个写
var (
    data map[string]int;
    rw   sync.RWMutex;
);

func read(key string) int {
    rw.RLock();
    defer rw.RUnlock();
    return data[key];
}

func write(key string, val int) {
    rw.Lock();
    defer rw.Unlock();
    data[key] = val;
}
```

## sync.Once

**单次执行**：确保函数只执行一次
`var <变量> sync.Once`

```go
// 单例模式初始化
var (
    instance *Config;
    once     sync.Once;
);

func GetConfig() *Config {
    once.Do(func() {
        instance = loadConfig();
    });
    return instance;
}
```

## sync.Cond

**条件变量**：等待条件满足
`sync.NewCond(<锁>)`

```go
// 条件变量等待通知
var (
    ready bool;
    mu    sync.Mutex;
    cond  = sync.NewCond(&mu);
);

func waitReady() {
    mu.Lock();
    for !ready {
        cond.Wait();
    }
    mu.Unlock();
}

func notifyReady() {
    mu.Lock();
    ready = true;
    cond.Broadcast();
    mu.Unlock();
}
```

## sync.Pool

**对象池**：复用对象减少 GC
`sync.Pool{ New: func() any { ... } }`

```go
// 对象池复用
var bufPool = sync.Pool{
    New: func() any {
        return new(bytes.Buffer);
    },
};

func process(data []byte) string {
    buf := bufPool.Get().(*bytes.Buffer);
    defer bufPool.Put(buf);
    buf.Reset();
    buf.Write(data);
    return buf.String();
}
```

## context 包

**创建根 context**：顶层 context
`context.Background()`

```go
// 根 context
ctx := context.Background();
```

**带取消的 context**：手动取消
`context.WithCancel(<父context>)`

```go
// 带取消的 context
ctx, cancel := context.WithCancel(context.Background());
defer cancel();

go func() {
    <-ctx.Done();
    fmt.Println("cancelled");
}();
```

**带超时的 context**：自动超时
`context.WithTimeout(<父context>, <时长>)`

```go
// 带超时的 context
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second);
defer cancel();

select {
case <-ctx.Done():
    fmt.Println("timeout:", ctx.Err());
}
```

**带截止时间的 context**：指定截止时刻
`context.WithDeadline(<父context>, <时间>)`

```go
// 带截止时间的 context
deadline := time.Now().Add(10 * time.Second);
ctx, cancel := context.WithDeadline(context.Background(), deadline);
defer cancel();
```

**带值的 context**：传递请求作用域数据
`context.WithValue(<父context>, <键>, <值>)`

```go
// 带值的 context
type key string;
ctx := context.WithValue(context.Background(), key("userID"), 123);

// 获取值
if v := ctx.Value(key("userID")); v != nil {
    fmt.Println(v);
}
```

## 并发模式

**Worker Pool**：固定数量的 worker
`for i := 0; i < <数量>; i++ { go <worker>() }`

```go
// Worker Pool 模式
func worker(id int, jobs <-chan int, results chan<- int) {
    for j := range jobs {
        results <- j * 2;
    }
}

jobs := make(chan int, 100);
results := make(chan int, 100);
for w := 0; w < 3; w++ {
    go worker(w, jobs, results);
}
for j := 1; j <= 5; j++ {
    jobs <- j;
}
close(jobs);
```

**Fan-out/Fan-in**：分散收集
`go <生产者>(); for i := 0; i < <数量>; i++ { go <消费者>() }`

```go
// Fan-out/Fan-in 模式
func fanOut(input <-chan int, n int) []<-chan int {
    outputs := make([]<-chan int, n);
    for i := 0; i < n; i++ {
        outputs[i] = process(input);
    }
    return outputs;
}
```

**Pipeline**：流水线处理
`<阶段1> -> <阶段2> -> <阶段3>`

```go
// Pipeline 模式
func generate(nums ...int) <-chan int {
    out := make(chan int);
    go func() {
        defer close(out);
        for _, n := range nums {
            out <- n;
        }
    }();
    return out;
}

func square(in <-chan int) <-chan int {
    out := make(chan int);
    go func() {
        defer close(out);
        for n := range in {
            out <- n * n;
        }
    }();
    return out;
}
```

## 原子操作

**原子加载**：原子读取
`atomic.Load<类型>(&<变量>)`

```go
// 原子加载
var count int64;
v := atomic.LoadInt64(&count);
```

**原子存储**：原子写入
`atomic.Store<类型>(&<变量>, <值>)`

```go
// 原子存储
atomic.StoreInt64(&count, 100);
```

**原子加减**：原子递增
`atomic.Add<类型>(&<变量>, <值>)`

```go
// 原子加法
atomic.AddInt64(&count, 1);
```

**原子比较交换**：CAS 操作
`atomic.CompareAndSwap<类型>(&<变量>, <旧值>, <新值>)`

```go
// CAS 操作
old := atomic.LoadInt64(&count);
swapped := atomic.CompareAndSwapInt64(&count, old, old+1);
```
