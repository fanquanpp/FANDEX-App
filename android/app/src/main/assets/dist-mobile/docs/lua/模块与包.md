# Lua 模块与包速查

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 模块定义

**表模块模式**
`local <M> = {} function <M>.<func>(<params>) <body> end return <M>`
```lua
-- 表模块模式
local M = {};
-- 私有变量
local privateVar = "私有变量";
-- 私有函数
local function privateFunction()
    return "私有函数";
end;
-- 公共函数
function M.publicFunction()
    return "公共函数";
end;
function M.accessPrivate()
    return privateVar .. ", " .. privateFunction();
end;
return M;
```

**环境模块模式**
`local <M> = {} local _ENV = <M> ... return <M>`
```lua
-- 环境模块模式（Lua 5.2+）
local M = {};
local _ENV = M;
-- 私有变量
local privateVar = "私有变量";
-- 公共函数（自动加入 M）
function add(a, b)
    return a + b;
end;
function multiply(a, b)
    return a * b;
end;
return M;
```

## 加载模块

**require 加载**
`local <module> = require("<module_name>")`
```lua
-- 加载模块
local MyModule = require("mymodule");
-- 使用模块函数
print(MyModule.add(1, 2));       -- 3
print(MyModule.multiply(3, 4));  -- 12
```

## 包搜索路径

**package.path**
`package.path = package.path .. ";<path>/?.lua"`
```lua
-- 查看包搜索路径
print(package.path);
-- 添加自定义搜索路径
package.path = package.path .. ";/path/to/modules/?.lua";
```

## 模块缓存

**require 缓存**
`package.loaded["<module>"]`
```lua
-- require 会缓存已加载的模块
local M1 = require("mymodule");
local M2 = require("mymodule");
print(M1 == M2);  -- true（同一个实例）
```

**模块重载**
`package.loaded["<module>"] = nil`
```lua
-- 清除缓存以重新加载模块
package.loaded["mymodule"] = nil;
local M = require("mymodule");  -- 重新执行模块代码
```

## 模块继承

**模块继承**
`local <Derived> = setmetatable({}, {__index = <Base>})`
```lua
-- 基础模块
local BaseModule = {
    baseMethod = function(self)
        return "基础方法";
    end
};
-- 派生模块
local DerivedModule = setmetatable({}, {__index = BaseModule});
function DerivedModule.derivedMethod(self)
    return "派生方法";
end;
return DerivedModule;
```

## 实用模块示例

**数学工具模块**
`local <M> = {} function <M>.<op>(<a>, <b>) return <expr> end return <M>`
```lua
-- 数学工具模块
local M = {};
function M.add(a, b)
    return a + b;
end;
function M.subtract(a, b)
    return a - b;
end;
function M.multiply(a, b)
    return a * b;
end;
function M.divide(a, b)
    if b == 0 then
        error("除数不能为零");
    end;
    return a / b;
end;
function M.pow(a, b)
    return a ^ b;
end;
function M.sqrt(a)
    return math.sqrt(a);
end;
return M;
```

**配置模块**
`local <M> = {} local <config> = {...} function <M>.<method>(<args>) <body> end return <M>`
```lua
-- 配置模块
local M = {};
-- 默认配置
local defaultConfig = {
    host = "localhost",
    port = 8080,
    timeout = 30,
    debug = false
};
-- 加载配置
function M.load(config)
    for k, v in pairs(config or {}) do
        defaultConfig[k] = v;
    end;
    return defaultConfig;
end;
-- 获取配置项
function M.get(key)
    return defaultConfig[key];
end;
-- 设置配置项
function M.set(key, value)
    defaultConfig[key] = value;
end;
return M;
```

**事件系统模块**
`local <M> = {} local <listeners> = {} function <M>.<method>(<args>) <body> end return <M>`
```lua
-- 事件系统模块
local M = {};
local listeners = {};
-- 注册事件监听器
function M.on(event, callback)
    if not listeners[event] then
        listeners[event] = {};
    end;
    table.insert(listeners[event], callback);
end;
-- 移除事件监听器
function M.off(event, callback)
    if listeners[event] then
        for i, listener in ipairs(listeners[event]) do
            if listener == callback then
                table.remove(listeners[event], i);
                break;
            end;
        end;
    end;
end;
-- 触发事件
function M.emit(event, ...)
    if listeners[event] then
        for _, callback in ipairs(listeners[event]) do
            callback(...);
        end;
    end;
end;
return M;
```

## LuaRocks 包管理

**LuaRocks 命令**
`luarocks [install|remove|list] <package>`
```bash
# 安装 LuaRocks（Ubuntu）
sudo apt install luarocks;
# 安装包
luarocks install luasocket;
# 卸载包
luarocks remove luasocket;
# 列出已安装的包
luarocks list;
```
