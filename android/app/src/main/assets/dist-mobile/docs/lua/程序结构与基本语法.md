# Lua 程序结构与基本语法速查

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 变量与赋值

**局部变量声明**
`local <name> = <value>`
```lua
-- 局部变量
local name = "Lua";
local version = 5.4;
-- 多重赋值
local a, b = 10, 20;
print(a, b);  -- 10 20
-- 交换变量
local x, y = 1, 2;
x, y = y, x;
print(x, y);  -- 2 1
-- 忽略返回值
local _, value = string.find("Hello Lua", "Lua");
print(value);  -- 7
```

**全局变量**
`<name> = <value>`
```lua
-- 全局变量（默认存储在 _G 表中）
global_var = "global";
function test()
    local local_var = "local";
    print(local_var);   -- local
    print(global_var);  -- global
end;
test();
print(global_var);  -- global
```

**块作用域**
`do local <name> = <value> end`
```lua
-- 块作用域
local x = 10;
print(x);  -- 10
do
    local x = 20;
    print(x);  -- 20
end;
print(x);  -- 10
```

## 注释

**单行注释**
`-- <comment>`
```lua
-- 单行注释
local x = 10;  -- 行尾注释
```

**多行注释**
`--[[ <content> ]]`
```lua
--[[
多行注释
可以跨越多行
]];
```

**文档注释**
`--- <description>`
```lua
--- 计算两个数的和
--- @param a number 第一个数
--- @param b number 第二个数
--- @return number 两个数的和
function add(a, b)
    return a + b;
end;
```

## 控制流

**if 语句**
`if <cond> then <body> [elseif <cond> then <body>] [else <body>] end`
```lua
local score = 85;
if score >= 90 then
    print("优秀");
elseif score >= 80 then
    print("良好");
elseif score >= 60 then
    print("及格");
else
    print("不及格");
end;
```

**嵌套 if**
`if <cond> then if <cond2> then <body> else <body> end else <body> end`
```lua
local age = 25;
local gender = "male";
if age >= 18 then
    if gender == "male" then
        print("成年男性");
    else
        print("成年女性");
    end;
else
    print("未成年人");
end;
```

**while 循环**
`while <cond> do <body> end`
```lua
local i = 1;
while i <= 5 do
    print(i);
    i = i + 1;
end;
-- 输出 1 2 3 4 5
```

**repeat-until 循环**
`repeat <body> until <cond>`
```lua
local i = 1;
repeat
    print(i);
    i = i + 1;
until i > 5;
-- 输出 1 2 3 4 5
```

**数值 for 循环**
`for <var> = <start>, <stop>[, <step>] do <body> end`
```lua
-- 递增
for i = 1, 5 do
    print(i);
end;
-- 递减
for i = 5, 1, -1 do
    print(i);
end;
-- 带步长
for i = 1, 10, 2 do
    print(i);
end;
```

**泛型 for 循环**
`for <k>, <v> in <iterator>(<table>) do <body> end`
```lua
-- ipairs：遍历数组
local fruits = {"apple", "banana", "orange"};
for index, value in ipairs(fruits) do
    print(index, value);
end;
-- pairs：遍历字典
local person = {name = "John", age = 30, city = "New York"};
for key, value in pairs(person) do
    print(key, value);
end;
```

## 流程控制语句

**break 语句**
`break`
```lua
for i = 1, 10 do
    if i == 5 then
        break;  -- 跳出循环
    end;
    print(i);
end;
```

**goto 语句**
`::<label>:: goto <label>`
```lua
-- goto 语句（Lua 5.2+）
::start::;
local i = 1;
while i <= 3 do
    print(i);
    i = i + 1;
    if i > 3 then
        goto end_loop;
    end;
end;
::end_loop::;
print("循环结束");
```

## 运算符

**算术运算符**
`<a> [+|-|*|/|^|%] <b>`
```lua
print(1 + 2);   -- 3 加法
print(5 - 3);   -- 2 减法
print(2 * 3);   -- 6 乘法
print(10 / 2);  -- 5 除法
print(2 ^ 3);   -- 8 幂运算
print(10 % 3);  -- 1 取模
print(-5);      -- -5 负号
```

**关系运算符**
`<a> [==|~=|<|>|<=|>=] <b>`
```lua
print(1 == 1);  -- true 等于
print(1 ~= 2);  -- true 不等于
print(1 < 2);   -- true 小于
print(2 > 1);   -- true 大于
print(1 <= 1);  -- true 小于等于
print(2 >= 1);  -- true 大于等于
```

**逻辑运算符**
`<a> [and|or] <b> | not <a>`
```lua
-- 注意：0 和 "" 在 Lua 中都视为 true
print(nil and "hello");    -- nil
print(false and "hello");  -- false
print("hello" and "world"); -- world
print(nil or "hello");     -- hello
print(false or "hello");   -- hello
print("hello" or "world"); -- hello
print(not true);           -- false
-- 短路求值设置默认值
local name = input_name or "Guest";
```

**字符串连接运算符**
`<str1> .. <str2>`
```lua
local str1 = "Hello";
local str2 = "Lua";
local result = str1 .. " " .. str2;
print(result);  -- Hello Lua
```

**表访问运算符**
`<table>.<key> | <table>["<key>"]`
```lua
local person = {name = "John", age = 30};
print(person.name);      -- John
print(person["age"]);    -- 30
local key = "name";
print(person[key]);      -- John
```

## 程序结构

**模块文件结构**
`local <module> = {} ... return <module>`
```lua
-- 模块文件结构
local module = {};
-- 私有变量
local private_var = "private";
-- 公共函数
function module.public_function()
    -- 函数体
end;
-- 私有函数
local function private_function()
    -- 函数体
end;
return module;
```

**主程序结构**
`require("<module>") function main() <body> end`
```lua
-- 主程序结构
local math = require("math");
-- 全局常量
APP_VERSION = "1.0.0";
-- 主函数
function main()
    print("Hello, Lua!");
end;
-- 入口检测
if arg and arg[0] == debug.getinfo(1, "S").source:sub(2) then
    main();
end;
```

## 错误处理

**error 抛出错误**
`error(<message>[, <level>])`
```lua
-- 抛出错误
error("发生错误");
-- 条件抛出
if not value then
    error("值不能为空");
end;
```

**pcall 保护调用**
`pcall(<func>[, <args>...])`
```lua
-- pcall 捕获错误
local success, result = pcall(function()
    return 10 / 0;
end);
if success then
    print("成功:", result);
else
    print("错误:", result);
end;
```

**xpcall 带错误处理函数**
`xpcall(<func>, <handler>[, <args>...])`
```lua
-- xpcall 带错误处理函数
local function error_handler(err)
    return "错误处理: " .. err;
end;
local success, result = xpcall(function()
    return 10 / 0;
end, error_handler);
print(success, result);
```

## 性能优化

**局部变量缓存**
`local <name> = <global>`
```lua
-- 缓存常用全局函数到局部变量
local print = print;
local math = math;
local table = table;
```

**字符串连接优化**
`table.concat(<table>, <sep>)`
```lua
-- 大量字符串连接使用 table.concat
local parts = {};
for i = 1, 1000 do
    parts[#parts + 1] = tostring(i);
end;
local result = table.concat(parts, ", ");
```
