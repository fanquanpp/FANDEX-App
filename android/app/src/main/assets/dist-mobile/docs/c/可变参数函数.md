# 可变参数函数

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 基本可变参数函数

**可变参数函数定义**
`#include <stdarg.h> / return_type <func>(<fixed_params>, ...) { ... }`
```c
#include <stdarg.h>
// 计算多个整数的和
int sum(int count, ...) {
    va_list args;
    va_start(args, count);          // 初始化参数列表
    int total = 0;
    for (int i = 0; i < count; i++) {
        total += va_arg(args, int); // 获取下一个参数
    }
    va_end(args);                   // 结束参数处理
    return total;
}
// 使用
int result = sum(3, 10, 20, 30);    // 60
int result2 = sum(5, 1, 2, 3, 4, 5); // 15
```

---

## va_list 类型与宏

**va_list：参数列表类型**
`va_list <name>;`
```c
va_list args;    // 声明参数列表变量
```

---

**va_start：初始化参数列表**
`void va_start(va_list <ap>, <last_named_param>);`
```c
va_list args;
va_start(args, count);    // count 是最后一个命名参数
```

---

**va_arg：获取下一个参数**
`<type> va_arg(va_list <ap>, <type>);`
```c
int value = va_arg(args, int);       // 获取 int 参数
double d = va_arg(args, double);     // 获取 double 参数
char *str = va_arg(args, char *);    // 获取字符串参数
```

---

**va_end：结束参数处理**
`void va_end(va_list <ap>);`
```c
va_end(args);    // 释放资源
```

---

**va_copy：复制参数列表（C99+）**
`void va_copy(va_list <dest>, va_list <src>);`
```c
va_list args, args_copy;
va_start(args, count);
va_copy(args_copy, args);    // 复制参数列表
// 可以独立遍历 args_copy
va_end(args);
va_end(args_copy);
```

## 可变参数函数示例

**printf 风格的日志函数**
`void <log_func>(const char *<format>, ...) { ... }`
```c
#include <stdarg.h>
#include <stdio.h>
#include <time.h>
void log_message(const char *level, const char *format, ...) {
    time_t now = time(NULL);
    char time_str[20];
    strftime(time_str, sizeof(time_str), "%H:%M:%S", localtime(&now));
    printf("[%s] %s: ", time_str, level);
    va_list args;
    va_start(args, format);
    vprintf(format, args);    // 使用 vprintf 处理可变参数
    va_end(args);
    printf("\n");
}
// 使用
log_message("INFO", "User %s logged in", "Alice");
log_message("ERROR", "Failed to open file: %s (code: %d)", "data.txt", 2);
log_message("DEBUG", "Value: %d, Pi: %.2f", 42, 3.14);
```

---

**可变参数求平均值**
`double <average>(int <count>, ...) { ... }`
```c
double average(int count, ...) {
    if (count <= 0) return 0.0;
    va_list args;
    va_start(args, count);
    double sum = 0.0;
    for (int i = 0; i < count; i++) {
        sum += va_arg(args, double);    // 注意：使用 double
    }
    va_end(args);
    return sum / count;
}
// 使用
double avg = average(3, 10.0, 20.0, 30.0);    // 20.0
```

---

**可变参数最大值**
`<type> <max>(int <count>, ...) { ... }`
```c
int max_int(int count, ...) {
    va_list args;
    va_start(args, count);
    int max_val = va_arg(args, int);
    for (int i = 1; i < count; i++) {
        int val = va_arg(args, int);
        if (val > max_val) {
            max_val = val;
        }
    }
    va_end(args);
    return max_val;
}
// 使用
int m = max_int(4, 10, 25, 8, 15);    // 25
```

## vprintf 系列函数

**vprintf：输出到 stdout**
`int vprintf(const char *<format>, va_list <ap>);`
```c
void custom_printf(const char *format, ...) {
    va_list args;
    va_start(args, format);
    vprintf(format, args);    // 使用 va_list 输出
    va_end(args);
}
```

---

**vfprintf：输出到文件**
`int vfprintf(FILE *<fp>, const char *<format>, va_list <ap>);`
```c
void log_to_file(FILE *fp, const char *format, ...) {
    va_list args;
    va_start(args, format);
    vfprintf(fp, format, args);
    va_end(args);
}
```

---

**vsprintf：输出到字符串**
`int vsprintf(char *<str>, const char *<format>, va_list <ap>);`
```c
void format_string(char *buffer, size_t size, const char *format, ...) {
    va_list args;
    va_start(args, format);
    vsnprintf(buffer, size, format, args);    // 使用 vsnprintf 更安全
    va_end(args);
}
```

---

**vsnprintf：安全输出到字符串**
`int vsnprintf(char *<str>, size_t <size>, const char *<format>, va_list <ap>);`
```c
void safe_format(char *buffer, size_t size, const char *format, ...) {
    va_list args;
    va_start(args, format);
    vsnprintf(buffer, size, format, args);    // 限制写入长度
    va_end(args);
}
```

## 可变参数宏（C99+）

**可变参数宏定义**
`#define <NAME>(...) <expr>(__VA_ARGS__)`
```c
#define LOG(...) printf(__VA_ARGS__)
#define DEBUG(fmt, ...) printf("[DEBUG] " fmt, ##__VA_ARGS__)
// 使用
LOG("Hello\n");
DEBUG("Value: %d\n", 42);
DEBUG("No args\n");    // ## 使其可以省略可变参数
```

---

**带日志级别的可变参数宏**
`#define <NAME>(level, fmt, ...) <log_func>(level, fmt, ##__VA_ARGS__)`
```c
#define LOG_INFO(fmt, ...)    log_message("INFO", fmt, ##__VA_ARGS__)
#define LOG_ERROR(fmt, ...)   log_message("ERROR", fmt, ##__VA_ARGS__)
#define LOG_DEBUG(fmt, ...)   log_message("DEBUG", fmt, ##__VA_ARGS__)
// 使用
LOG_INFO("Process started");
LOG_ERROR("File not found: %s", filename);
LOG_DEBUG("x=%d, y=%d, z=%d", x, y, z);
```

## 可变参数函数注意事项

**参数类型提升**
`va_arg(<ap>, <promoted_type>)`
```c
// char/short 自动提升为 int
// float 自动提升为 double
void func(int count, ...) {
    va_list args;
    va_start(args, count);
    char c = (char)va_arg(args, int);       // char 提升为 int
    short s = (short)va_arg(args, int);     // short 提升为 int
    float f = (float)va_arg(args, double);  // float 提升为 double
    va_end(args);
}
```

---

**使用哨兵值终止**
`while ((val = va_arg(<ap>, <type>)) != <sentinel>) { ... }`
```c
// 使用 NULL 作为字符串列表的结束标志
void print_strings(const char *first, ...) {
    va_list args;
    va_start(args, first);
    const char *str = first;
    while (str != NULL) {
        printf("%s\n", str);
        str = va_arg(args, const char *);
    }
    va_end(args);
}
// 使用
print_strings("Hello", "World", "C", NULL);    // 必须以 NULL 结尾
```
