# 程序结构与基本语法

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 源文件结构

**完整程序结构：包含头文件、宏定义、类型定义、全局变量、函数原型、主函数、函数实现**
`#include <header> / #define NAME value / typedef type name; / type global_var; / return_type func(params); / int main() { ... }`
```c
// 包含头文件
#include <stdio.h>
// 宏定义
#define PI 3.14159
// 类型定义
typedef unsigned int uint;
// 全局变量
int global_count = 0;
// 函数原型
void print_hello();
// 主函数入口
int main() {
    int local_val = 10;
    printf("Value: %d\n", local_val);
    print_hello();
    return 0;
}
// 函数实现
void print_hello() {
    printf("Hello!\n");
}
```

---

## 头文件结构

**头文件保护：防止重复包含**
`#ifndef HEADER_H / #define HEADER_H / ... / #endif`
```c
// 防止重复包含
#ifndef MY_HEADER_H
#define MY_HEADER_H
// 包含其他头文件
#include <stdio.h>
// 类型定义
typedef struct {
    int id;
    char name[50];
} Student;
// 函数原型
void init_student(Student *s, int id, const char *name);
#endif /* MY_HEADER_H */
```

## 注释

**单行注释：使用 `//`**
`// 注释内容`
```c
// 这是一个单行注释
int x = 10; // 行尾注释
```

---

**多行注释：使用 `/* ... */`**
`/* 注释内容 */`
```c
/*
 * 这是一个多行注释
 * 可以跨越多行
 */
/* 注释掉的代码
int y = 20;
*/
```

---

**文档注释：Doxygen 格式**
`/** @brief 描述 @param 参数 @return 返回值 */`
```c
/**
 * @brief 计算圆的面积
 * @param radius 圆的半径
 * @return 圆的面积
 */
double calculate_area(double radius) {
    return PI * radius * radius;
}
```

---

## 主函数

**无参数形式**
`int main() { ... return 0; }`
```c
// 无参数形式
int main() {
    // 代码
    return 0;
}
```

---

**带命令行参数形式**
`int main(int argc, char *argv[]) { ... }`
```c
// argc: 参数个数, argv: 参数数组
int main(int argc, char *argv[]) {
    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    return 0;
}
```

---

## 程序终止

**正常终止**
`return 0;` 或 `exit(0);`
```c
// 正常终止
return 0;   // 返回到调用者
exit(0);     // 直接终止整个程序
```

---

**异常终止**
`exit(n);` 其中 n ≠ 0
```c
// 异常终止
exit(1);     // 非零表示异常
```

## 编译命令

**单文件编译**
`gcc <source.c> -o <output>`
```bash
# 一步完成所有步骤
gcc hello.c -o hello
# 启用优化
gcc -O2 hello.c -o hello
# 生成调试信息
gcc -g hello.c -o hello
```

---

**多文件编译**
`gcc <file1.c> <file2.c> -o <output>`
```bash
# 编译多个源文件
gcc file1.c file2.c -o program
# 分别编译，然后链接
gcc -c file1.c -o file1.o
gcc -c file2.c -o file2.o
gcc file1.o file2.o -o program
```
