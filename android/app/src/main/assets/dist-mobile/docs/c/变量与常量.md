# 变量与常量

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 变量定义与初始化

**声明变量**
`type <var_name>;`
```c
// 声明变量（分配内存但未初始化）
int a;          // 局部变量，值为随机值
```

---

**声明并初始化**
`type <var_name> = <value>;`
```c
// 声明变量并初始化
int b = 20;    // 声明变量 b 并初始化为 20
```

---

**多变量声明**
`type <var1>, <var2> = <value>;`
```c
// 多变量声明
int x, y = 5;              // y 被初始化为 5，x 未初始化
int m = 10, n = 20, p = 30; // 同时声明并初始化多个变量
```

---

**赋值操作**
`<var_name> = <value>;`
```c
// 先声明后赋值
int a;
a = 10;
```

---

## 存储类

**auto 存储类：默认局部变量**
`auto type <var_name> = <value>;`
```c
// 显式声明为 auto（通常省略）
void function() {
    auto int x = 10;   // 显式 auto
    int y = 20;        // 隐式 auto
}
```

---

**static 存储类：保持值，限制作用域**
`static type <var_name> = <value>;`
```c
// 局部静态变量：函数结束后不销毁
void counter() {
    static int count = 0;  // 仅首次调用时初始化
    count++;
    printf("Count: %d\n", count);
}
// 全局静态变量：仅限当前文件访问
static int file_static = 100;
```

---

**extern 存储类：引用外部变量**
`extern type <var_name>;`
```c
// file1.c
extern int global_var;  // 声明外部变量
void print_global() {
    printf("Global var: %d\n", global_var);
}
// file2.c
int global_var = 50;    // 定义全局变量
```

---

**register 存储类：建议存储在寄存器**
`register type <var_name>;`
```c
// 建议将变量存储在寄存器中
void fast_calculation() {
    register int i;
    for (i = 0; i < 1000000; i++) {
        // 大量计算
    }
}
```

## 字面常量

**整数常量**
`<decimal> | 0x<hex> | 0<octal> | 0b<binary>[后缀]`
```c
100      // 十进制
0x1F     // 十六进制 (31)
010      // 八进制 (8)
0b1010   // 二进制 (10，C99+)
100U     // 无符号整数
100L     // 长整数
100UL    // 无符号长整数
100LL    // 长 long 整数
```

---

**浮点常量**
`<digits>.<digits>[后缀] | <digits>e<exp>`
```c
3.14     // 双精度浮点数
3.14f    // 单精度浮点数
3.14L    // 长双精度浮点数
2.5e-3   // 科学记数法 (0.0025)
```

---

**字符常量**
`'<char>' | '<escape>'`
```c
'A'      // 普通字符
'\n'     // 换行符
'\t'     // 制表符
'\\'     // 反斜杠
'\0'     // 空字符
```

---

**字符串常量**
`"<string>"`
```c
"Hello C"           // 字符串常量
"Line 1\nLine 2"   // 多行字符串
```

## 宏定义常量

**无参宏：定义常量**
`#define <NAME> <value>`
```c
// 定义常量
#define MAX_BUFFER 1024
#define PI 3.14159
#define MESSAGE "Hello World"
// 使用常量
int buffer[MAX_BUFFER];
double area = PI * radius * radius;
printf("%s\n", MESSAGE);
```

---

**带参宏：定义函数式宏**
`#define <NAME>(<params>) <expression>`
```c
// 定义带参数的宏
#define MAX(a, b) ((a) > (b) ? (a) : (b))
#define SQUARE(x) ((x) * (x))
// 使用宏
int max_val = MAX(10, 20);    // 结果: 20
int square_val = SQUARE(5);   // 结果: 25
```

## const 常量

**定义 const 常量**
`const type <NAME> = <value>;`
```c
// 定义 const 常量
const int DAYS_IN_WEEK = 7;
const double GRAVITY = 9.8;
const char* COMPANY_NAME = "Tech Corp";
```

---

**const 与指针**
`const type* <ptr> | type* const <ptr> = &<var> | const type* const <ptr> = &<var>`
```c
// 指向常量的指针（不能通过指针修改所指向的值）
const int* p1;
int const* p2;              // 与上面等价
// 常量指针（指针本身不能改变）
int* const p3 = &x;
// 指向常量的常量指针（既不能修改值，也不能修改指针）
const int* const p4 = &x;
```

## 枚举常量

**基本枚举：默认从 0 开始递增**
`enum <Name> { <MEM1>, <MEM2>, ... };`
```c
// 定义枚举
enum Days {
    SUN,    // 默认为 0
    MON,    // 默认为 1
    TUE,    // 默认为 2
    WED     // 默认为 3
};
// 使用枚举
enum Days today = WED;
printf("Today is day %d\n", today);  // 输出: 3
```

---

**自定义枚举值**
`enum <Name> { <MEM1> = <val>, <MEM2>, ... };`
```c
// 自定义枚举值
enum Months {
    JAN = 1,    // 1
    FEB,        // 2
    MAR,        // 3
    APR,        // 4
    MAY,        // 5
    JUN         // 6
};
// 不连续的枚举值
enum Status {
    SUCCESS = 0,
    WARNING = 100,
    ERROR = 200
};
```

## 左值与右值

**左值：指向内存位置的表达式**
`<lvalue> = <rvalue>;`
```c
int x = 10;    // x 是左值，10 是右值
x = 20;        // x 是左值，20 是右值
int y = x;     // y 是左值，x 是右值（x 的值被读取）
```
