# 函数详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 函数声明与定义

**函数声明（原型）**
`return_type <func_name>(<parameter_list>);`
```c
// 函数声明（可省略参数名）
int add(int, int);           // 省略参数名
int subtract(int a, int b);  // 包含参数名
void print_message(void);    // 无参数
```

---

**函数定义**
`return_type <func_name>(<parameter_list>) { ... return <expr>; }`
```c
// 函数定义
int add(int a, int b) {
    return a + b;
}
void print_message(void) {
    printf("Hello, Function!\n");
    // 无 return 语句
}
```

---

## 参数传递

**传值调用：复制实参的值**
`return_type <func>(<type> <param>) { ... }`
```c
// 修改形参不影响实参
void increment(int x) {
    x++;
    printf("Inside: %d\n", x);    // 输出 11
}
int main() {
    int a = 10;
    increment(a);
    printf("Outside: %d\n", a);   // 输出 10，实参未被修改
    return 0;
}
```

---

**传址调用：通过指针修改实参**
`return_type <func>(<type> *<param>) { ... }`
```c
// 通过指针修改实参
void increment_by_address(int *x) {
    (*x)++;
    printf("Inside: %d\n", *x);   // 输出 11
}
int main() {
    int a = 10;
    increment_by_address(&a);
    printf("Outside: %d\n", a);    // 输出 11，实参被修改
    return 0;
}
```

---

**数组作为参数：退化为指针**
`return_type <func>(<type> <arr>[], int <size>) { ... }`
```c
// 传递数组首地址和大小
void print_array(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}
int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    print_array(numbers, size);
    return 0;
}
```

## 返回值

**返回基本类型**
`return <expression>;`
```c
int max(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}
```

---

**返回指针**
`<type> *<func>(<params>) { ... return <ptr>; }`
```c
// 正确：返回动态分配的内存
int *create_dynamic_array(int size) {
    int *arr = (int *)malloc(size * sizeof(int));
    return arr;
}
// 错误：不要返回局部变量的指针
// int *bad_func() { int arr[5]; return arr; }  // 危险！
```

---

**无返回值（void）**
`void <func>(<params>) { ... [return;] }`
```c
void print_hello() {
    printf("Hello!\n");
    // 无 return 语句
}
void check_number(int n) {
    if (n < 0) {
        printf("Negative!\n");
        return;    // 提前返回
    }
    printf("Non-negative: %d\n", n);
}
```

## 递归

**阶乘计算**
`return_type <func>(<type> <param>) { if (<base>) return ...; return <recursive_call>; }`
```c
long factorial(int n) {
    if (n <= 1) return 1;              // 基准情况
    return n * factorial(n - 1);       // 递归调用
}
```

---

**斐波那契数列**
`return_type <func>(<type> <param>) { if (<base>) return ...; return <recursive1> + <recursive2>; }`
```c
int fibonacci(int n) {
    if (n <= 1) return n;                          // 基准情况
    return fibonacci(n - 1) + fibonacci(n - 2);    // 递归调用
}
```

---

**二分查找**
`int <search>(<type> <arr>[], int <low>, int <high>, <type> <target>) { ... }`
```c
int binary_search(int arr[], int low, int high, int target) {
    if (low > high) return -1;    // 基准情况：未找到
    int mid = low + (high - low) / 2;
    if (arr[mid] == target) return mid;
    else if (arr[mid] > target) return binary_search(arr, low, mid - 1, target);
    else return binary_search(arr, mid + 1, high, target);
}
```

## 存储类与作用域

**静态局部变量**
`static <type> <var> = <value>;`
```c
void counter() {
    static int count = 0;    // 仅首次调用时初始化
    count++;
    printf("Count: %d\n", count);
}
// 调用三次：输出 1, 2, 3
```

---

**extern 外部变量**
`extern <type> <var>;`
```c
// file1.c
extern int global_var;    // 声明外部变量
void function() {
    printf("Global: %d\n", global_var);
}
// file2.c
int global_var = 100;     // 定义全局变量
```

## 函数指针

**函数指针定义与赋值**
`return_type (*<ptr_name>)(<parameter_list>);`
```c
int add(int a, int b) { return a + b; }
// 定义函数指针
int (*add_ptr)(int, int);
// 赋值
add_ptr = add;
// 或直接初始化
int (*add_ptr)(int, int) = add;
```

---

**通过函数指针调用**
`<result> = <func_ptr>(<args>);` 或 `<result> = (*<func_ptr>)(<args>);`
```c
int result = add_ptr(10, 20);         // 调用
int result2 = (*add_ptr)(10, 20);     // 更明确的写法
```

---

**回调函数**
`typedef return_type (*<CallbackName>)(<params>);`
```c
// 定义回调函数类型
typedef void (*Callback)(int);
// 执行回调的函数
void process_array(int arr[], int size, Callback callback) {
    for (int i = 0; i < size; i++) {
        callback(arr[i]);
    }
}
// 具体的回调函数
void print_number(int num) { printf("%d ", num); }
void square_number(int num) { printf("%d ", num * num); }
```

---

**函数指针数组**
`return_type (*<array_name>[])(<params>) = { <func1>, <func2>, ... };`
```c
int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }
int multiply(int a, int b) { return a * b; }
int divide(int a, int b) { return b != 0 ? a / b : 0; }
// 函数指针数组
int (*operations[])(int, int) = {add, subtract, multiply, divide};
// 使用
int result = operations[0](10, 5);    // 调用 add
```

## 可变参数函数

**基本可变参数函数**
`#include <stdarg.h> / return_type <func>(<fixed_params>, ...) { ... }`
```c
#include <stdarg.h>
// 计算多个整数的和
double sum(int count, ...) {
    va_list valist;
    double sum = 0.0;
    va_start(valist, count);          // 初始化参数列表
    for (int i = 0; i < count; i++) {
        sum += va_arg(valist, int);   // 获取参数
    }
    va_end(valist);                   // 结束参数处理
    return sum;
}
```

## 内联函数

**内联函数定义**
`inline <type> <func>(<params>) { ... }`
```c
inline int max(int a, int b) {
    return a > b ? a : b;
}
int main() {
    int result = max(10, 20);    // 可能被内联展开
    return 0;
}
```
