# 预处理器与宏

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 文件包含

**包含系统头文件**
`#include <header_name>`
```c
#include <stdio.h>      // 标准输入输出
#include <stdlib.h>     // 标准库
#include <string.h>     // 字符串操作
#include <math.h>       // 数学函数
```

---

**包含用户头文件**
`#include "header_name"`
```c
#include "my_header.h"          // 当前目录
#include "utils/math_utils.h"   // 相对路径
#include "../config.h"          // 上级目录
```

---

## 宏定义

**无参宏：定义常量**
`#define <NAME> <value>`
```c
#define PI 3.14159
#define MAX_BUFFER 1024
#define VERSION "1.0.0"
#define NEWLINE '\n'
// 使用
double area = PI * radius * radius;
char buffer[MAX_BUFFER];
printf("Version: %s\n", VERSION);
```

---

**带参宏：定义函数式宏**
`#define <NAME>(<params>) <expression>`
```c
#define MAX(a, b) ((a) > (b) ? (a) : (b))
#define MIN(a, b) ((a) < (b) ? (a) : (b))
#define SQUARE(x) ((x) * (x))
#define ABS(x) ((x) >= 0 ? (x) : -(x))
// 使用
int max_val = MAX(10, 20);     // 20
int abs_val = ABS(-5);          // 5
double sq = SQUARE(3.0);        // 9.0
```

---

**多行宏定义**
`#define <NAME> <line1> \ <line2> \ ...`
```c
#define SWAP(a, b) do { \
    typeof(a) temp = a; \
    a = b; \
    b = temp; \
} while(0)
// 使用
int x = 10, y = 20;
SWAP(x, y);
```

---

**字符串化运算符**
`#<param>`
```c
#define PRINT_INT(x) printf(#x " = %d\n", x)
// 使用
int value = 42;
PRINT_INT(value);    // 输出: value = 42
```

---

**标记粘贴运算符**
`<param1>##<param2>`
```c
#define CONCAT(a, b) a##b
#define DEFINE_VAR(name) int CONCAT(var_, name) = 0
// 使用
DEFINE_VAR(count);    // 展开为: int var_count = 0;
int CONCAT(foo, bar) = 10;    // 展开为: int foobar = 10;
```

---

**可变参数宏**
`#define <NAME>(...) <expr>(__VA_ARGS__)`
```c
#define LOG(...) printf(__VA_ARGS__)
#define DEBUG(fmt, ...) printf("[DEBUG] " fmt, ##__VA_ARGS__)
// 使用
LOG("Hello\n");
DEBUG("Value: %d\n", 42);
DEBUG("No args\n");    // ## 使其可以省略可变参数
```

## 条件编译

**#ifdef：如果定义了宏**
`#ifdef <NAME> ... [#else] ... #endif`
```c
#ifdef DEBUG
    printf("Debug mode\n");
#endif
// #else 分支
#ifdef DEBUG
    printf("Debug mode\n");
#else
    printf("Release mode\n");
#endif
```

---

**#ifndef：如果未定义宏**
`#ifndef <NAME> ... #define <NAME> ... #endif`
```c
// 头文件保护
#ifndef MY_HEADER_H
#define MY_HEADER_H
// 头文件内容
void my_function();
#endif /* MY_HEADER_H */
```

---

**#if：基于常量表达式**
`#if <expression> ... [#elif <expr>] ... [#else] ... #endif`
```c
#define VERSION 2
#if VERSION == 1
    #define FEATURE_NAME "Version 1"
#elif VERSION == 2
    #define FEATURE_NAME "Version 2"
#else
    #define FEATURE_NAME "Unknown"
#endif
```

---

**defined 运算符**
`#if defined(<NAME>) [&& defined(<NAME2>)]`
```c
#if defined(DEBUG) && defined(VERBOSE)
    printf("Debug and verbose mode\n");
#elif defined(DEBUG)
    printf("Debug mode\n");
#endif
// 否定形式
#if !defined(RELEASE)
    #define TEST_MODE
#endif
```

## 预定义宏

**标准预定义宏**
`__FILE__ | __LINE__ | __DATE__ | __TIME__ | __func__`
```c
printf("File: %s\n", __FILE__);           // 当前文件名
printf("Line: %d\n", __LINE__);           // 当前行号
printf("Date: %s\n", __DATE__);          // 编译日期
printf("Time: %s\n", __TIME__);          // 编译时间
printf("Func: %s\n", __func__);          // 当前函数名
printf("Standard: %ld\n", __STDC_VERSION__);  // C 标准版本
```

## #pragma 指令

**#pragma once：头文件只包含一次**
`#pragma once`
```c
// 替代头文件保护宏
#pragma once
// 头文件内容
void my_function();
```

---

**#pragma pack：控制内存对齐**
`#pragma pack(<n>) ... #pragma pack()`
```c
#pragma pack(1)
struct PackedData {
    char c;
    int i;
};
#pragma pack()
```

---

**#pragma message：编译时输出消息**
`#pragma message("<message>")`
```c
#pragma message("Compiling " __FILE__)
#if defined(_WIN32)
    #pragma message("Windows platform")
#endif
```

---

## #error 和 #warning

**#error：编译错误**
`#error "<message>"`
```c
#if !defined(VERSION) || VERSION < 2
    #error "Requires VERSION >= 2"
#endif
```

---

**#warning：编译警告**
`#warning "<message>"`
```c
#warning "This code is deprecated"
```

---

## #undef

**取消宏定义**
`#undef <NAME>`
```c
#define MAX_SIZE 100
// 使用 MAX_SIZE
#undef MAX_SIZE
// 此后 MAX_SIZE 不再定义
```

---

## 预处理器运算符

**行延续**
`<line> \`
```c
#define LONG_MACRO(x) \
    do { \
        printf("%d\n", x); \
        x++; \
    } while(0)
```

---

**括号确保优先级**
`((<expr>))`
```c
#define SQUARE(x) ((x) * (x))
#define ADD(a, b) ((a) + (b))
// 避免宏展开时的优先级问题
int result = SQUARE(2 + 3);    // ((2+3) * (2+3)) = 25
```
