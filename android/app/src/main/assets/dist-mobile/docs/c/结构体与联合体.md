# 结构体与联合体

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 结构体定义

**基本结构体定义**
`struct <Name> { <type> <member>; ... };`
```c
struct Person {
    char name[50];
    int age;
    float height;
};
```

---

**同时定义结构体变量**
`struct <Name> { <members> } <var1>, <var2>;`
```c
struct Person {
    char name[50];
    int age;
} person1, person2;
```

---

**匿名结构体**
`struct { <members> } <var>;`
```c
// 只能在定义时声明变量
struct {
    int x;
    int y;
} point;
```

---

## 结构体初始化

**静态初始化**
`struct <Name> <var> = {<values>};`
```c
// 按顺序初始化
struct Person p1 = {"Alice", 25, 1.65};
// 部分初始化（未初始化的成员为 0）
struct Person p2 = {"Bob"};
// C99+：指定成员初始化
struct Person p3 = {
    .name = "Charlie",
    .age = 30
};
```

---

**动态初始化**
`<var>.<member> = <value>;`
```c
struct Person p4;
strcpy(p4.name, "David");
p4.age = 35;
p4.height = 1.75;
```

---

## 结构体成员访问

**直接访问（点运算符）**
`<var>.<member>`
```c
printf("%s\n", p1.name);
printf("%d\n", p1.age);
printf("%.2f\n", p1.height);
```

---

**通过指针访问（箭头运算符）**
`<ptr>-><member>` 或 `(*<ptr>).<member>`
```c
struct Person *ptr = &p1;
printf("%s\n", ptr->name);
printf("%d\n", ptr->age);
// 等价写法
printf("%s\n", (*ptr).name);
```

---

## 结构体作为函数参数

**传值调用**
`return_type <func>(struct <Name> <param>) { ... }`
```c
void print_person(struct Person p) {
    printf("Name: %s\n", p.name);
    printf("Age: %d\n", p.age);
}
// 调用
print_person(p1);
```

---

**传址调用（推荐）**
`return_type <func>(struct <Name> *<param>) { ... }`
```c
void update_person(struct Person *p, int new_age) {
    p->age = new_age;
}
// 调用
update_person(&p1, 26);
```

---

## 结构体数组

**结构体数组定义与初始化**
`struct <Name> <array_name>[<size>] = { {<values>}, ... };`
```c
struct Person people[3] = {
    {"Alice", 25, 1.65},
    {"Bob", 30, 1.75},
    {"Charlie", 35, 1.80}
};
// 访问数组元素
for (int i = 0; i < 3; i++) {
    printf("%s, %d\n", people[i].name, people[i].age);
}
```

## 嵌套结构体

**结构体包含另一个结构体**
`struct <Outer> { ... struct <Inner> <member>; ... };`
```c
struct Date {
    int day;
    int month;
    int year;
};
struct Person {
    char name[50];
    int age;
    struct Date birthday;    // 嵌套结构体
};
// 初始化
struct Person p = {
    "Alice",
    25,
    {15, 5, 1999}
};
// 访问嵌套结构体成员
printf("Birthday: %d/%d/%d\n",
    p.birthday.day, p.birthday.month, p.birthday.year);
```

## 内存对齐

**控制内存对齐**
`#pragma pack(<n>) ... #pragma pack()`
```c
// 设置对齐字节数为 1
#pragma pack(1)
struct PackedExample {
    char c;
    int i;
    double d;
};
#pragma pack()    // 恢复默认对齐
// 使用 packed 属性
struct __attribute__((packed)) PackedStruct {
    char c;
    int i;
    double d;
};
```

## 联合体

**联合体定义与使用**
`union <Name> { <type> <member>; ... };`
```c
union Data {
    int i;
    float f;
    char str[20];
};
// 使用（所有成员共享同一块内存）
union Data data;
data.i = 100;
printf("data.i = %d\n", data.i);
data.f = 3.14;
printf("data.f = %f\n", data.f);    // 修改 f 后 i 的值已改变
```

---

**判别式联合（Tagged Union）**
`struct <Name> { enum <Type> <tag>; union { <members> } <data>; };`
```c
enum DataType { INT, FLOAT, STRING };
struct TaggedUnion {
    enum DataType type;
    union {
        int i;
        float f;
        char str[50];
    } data;
};
void print_data(struct TaggedUnion tu) {
    switch (tu.type) {
        case INT:    printf("Int: %d\n", tu.data.i); break;
        case FLOAT:  printf("Float: %f\n", tu.data.f); break;
        case STRING: printf("String: %s\n", tu.data.str); break;
    }
}
```

---

**联合体与位域配合**
`union <Name> { struct <Flags> <flags>; <type> <value>; };`
```c
struct Flags {
    unsigned int is_active : 1;
    unsigned int is_admin : 1;
    unsigned int level : 3;
};
union FlagUnion {
    struct Flags flags;
    unsigned char value;
};
// 使用
union FlagUnion fu;
fu.value = 0;
fu.flags.is_active = 1;
fu.flags.level = 3;
printf("Value: 0x%X\n", fu.value);
```

## typedef 类型别名

**为基本类型创建别名**
`typedef <existing_type> <new_name>;`
```c
typedef unsigned int uint;
typedef long long int64;
typedef double real;
// 使用
uint count = 100;
int64 large_number = 9999999999;
```

---

**为结构体创建别名**
`typedef struct { <members> } <Name>;`
```c
// 方式1：先定义再别名
struct Person { char name[50]; int age; };
typedef struct Person Person;
// 方式2：定义同时创建别名
typedef struct {
    char name[50];
    int age;
} Person;
// 使用
Person p = {"Alice", 25};
```

---

**为指针类型创建别名**
`typedef <type> *<PtrName>;`
```c
typedef int *IntPtr;
typedef char *StrPtr;
// 使用
IntPtr p1, p2;    // 相当于 int *p1, *p2;
```

---

**为函数指针创建别名**
`typedef return_type (*<Name>)(<params>);`
```c
typedef int (*CompareFunc)(int, int);
int ascending(int a, int b) { return a - b; }
// 使用
CompareFunc cmp = ascending;
int result = cmp(5, 3);
```
