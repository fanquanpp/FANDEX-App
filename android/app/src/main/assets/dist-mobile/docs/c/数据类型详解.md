# 数据类型详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 整型

**基本整型声明**
`<signed|unsigned> <char|short|int|long|long long> <var_name>;`
```c
char c = 'A';              // 1 字节，-128 到 127
short s = 1000;            // 2 字节
int i = 100000;            // 4 字节
long l = 100000L;          // 4 或 8 字节
long long ll = 100000LL;   // 8 字节
unsigned int u = 100U;     // 无符号整型
```

---

## 浮点型

**浮点型声明**
`<float|double|long double> <var_name> = <value>;`
```c
float f = 3.14f;           // 4 字节，6-7 位有效数字
double d = 3.14159;        // 8 字节，15-16 位有效数字
long double ld = 3.14L;    // 8-16 字节，18-19 位有效数字
```

---

## 布尔型

**布尔型声明（C99+）**
`#include <stdbool.h> / bool <var_name> = <true|false>;`
```c
#include <stdbool.h>
bool is_valid = true;
bool is_done = false;
if (is_valid) {
    printf("Valid\n");
}
```

---

## 类型修饰符

**signed：有符号数**
`signed <type> <var_name>;`
```c
signed int x = -10;    // 可以存储负数（默认）
```

---

**unsigned：无符号数**
`unsigned <type> <var_name>;`
```c
unsigned int y = 10;    // 只能存储非负数，正数范围翻倍
```

---

**const：只读变量**
`const <type> <var_name> = <value>;`
```c
const int MAX_VALUE = 100;    // 常量，不可修改
```

---

**volatile：防止编译器优化**
`volatile <type> <var_name>;`
```c
volatile int sensor_value;    // 防止编译器优化，用于硬件访问或多线程
```

---

**register：建议存储在寄存器**
`register <type> <var_name>;`
```c
register int counter;    // 建议存储在寄存器中
```

---

## sizeof 运算符

**获取类型或变量大小**
`sizeof(<type|var>)`
```c
printf("char: %zu\n", sizeof(char));        // 1
printf("int: %zu\n", sizeof(int));          // 4
printf("double: %zu\n", sizeof(double));   // 8
int arr[10];
printf("arr count: %zu\n", sizeof(arr) / sizeof(arr[0]));  // 10
```

---

## 数组

**一维数组声明**
`<type> <array_name>[<size>];`
```c
int numbers[5] = {1, 2, 3, 4, 5};    // 完全初始化
int arr[5] = {1, 2, 3};              // 部分初始化，其余为 0
int arr2[] = {10, 20, 30};           // 自动推断大小为 3
char message[] = "Hello, C!";         // 字符数组
```

---

**二维数组声明**
`<type> <array_name>[<rows>][<cols>];`
```c
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

---

## 指针

**指针声明与初始化**
`<type> *<ptr_name> = &<var>;`
```c
int x = 10;
int *ptr = &x;    // ptr 指向 x 的地址
printf("值: %d\n", *ptr);      // 解引用，输出 10
printf("地址: %p\n", ptr);     // 输出地址
*ptr = 20;                    // 通过指针修改值
```

---

## 结构体

**结构体定义与使用**
`typedef struct { <members> } <Name>;`
```c
typedef struct {
    int id;
    char name[50];
    float salary;
} Employee;
// 使用
Employee emp1;
emp1.id = 101;
strcpy(emp1.name, "John Doe");
emp1.salary = 5000.0;
// 初始化
Employee emp2 = {102, "Jane Smith", 6000.0};
```

---

## 联合体

**联合体定义与使用**
`union <Name> { <members> };`
```c
union Data {
    int i;
    float f;
    char str[20];
};
// 使用（所有成员共享同一块内存）
union Data data;
data.i = 10;
printf("data.i: %d\n", data.i);
data.f = 3.14;
printf("data.f: %f\n", data.f);    // 修改 f 后 i 的值已改变
```

---

## 枚举

**枚举定义**
`enum <Name> { <MEM1>[=<val>], <MEM2>, ... };`
```c
enum Weekday {
    MONDAY,    // 0
    TUESDAY,   // 1
    WEDNESDAY  // 2
};
enum Weekday today = WEDNESDAY;
// 显式指定值
enum Color {
    RED = 1,
    GREEN = 2,
    BLUE = 4
};
```

## 空类型

**void 函数返回类型**
`void <func_name>(<params>) { ... }`
```c
void print_hello() {
    printf("Hello!\n");
    // 无 return 语句
}
```

---

**void 函数参数**
`<type> <func_name>(void) { ... }`
```c
int main(void) {
    // 代码
    return 0;
}
```

---

**void 通用指针**
`void *<ptr_name>;`
```c
void *generic_ptr;
int x = 10;
generic_ptr = &x;           // 可以指向任何类型
char c = 'A';
generic_ptr = &c;           // 重新指向
```

---

## 类型转换

**隐式转换：自动完成**
`<type> <var> = <other_type_var>;`
```c
int x = 10;
double y = x;    // 隐式转换：int → double
printf("y = %f\n", y);    // 输出 10.000000
```

---

**显式转换：强制类型转换**
`(<target_type>)<expression>`
```c
double pi = 3.14159;
int rounded_pi = (int)pi;    // 显式转换：double → int
printf("rounded_pi = %d\n", rounded_pi);    // 输出 3
// 指针类型转换
int x = 100;
void *ptr = &x;
int *int_ptr = (int *)ptr;
printf("*int_ptr = %d\n", *int_ptr);    // 输出 100
```

---

## typedef 类型别名

**为基本类型创建别名**
`typedef <existing_type> <new_name>;`
```c
typedef unsigned int uint;
typedef long long int64;
// 使用别名
uint count = 100;
int64 large_number = 9999999999;
```

---

**为结构体创建别名**
`typedef struct { <members> } <Name>;`
```c
typedef struct {
    int x;
    int y;
} Point;
// 使用别名
Point p = {10, 20};
```

---

## 标准固定宽度整数

**stdint.h 固定宽度类型**
`#include <stdint.h> / <int8_t|uint8_t|int16_t|...> <var>;`
```c
#include <stdint.h>
int8_t s8;        // 有符号 8 位整数
uint8_t u8;       // 无符号 8 位整数
int16_t s16;      // 有符号 16 位整数
uint16_t u16;     // 无符号 16 位整数
int32_t s32;      // 有符号 32 位整数
uint32_t u32;     // 无符号 32 位整数
int64_t s64;      // 有符号 64 位整数
uint64_t u64;     // 无符号 64 位整数
```
