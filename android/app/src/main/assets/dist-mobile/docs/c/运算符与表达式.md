# 运算符与表达式

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 算术运算符

**基本算术运算**
`<expr> <+|-|*|/|%> <expr>`
```c
int a = 10, b = 3;
printf("%d\n", a + b);    // 13 加法
printf("%d\n", a - b);    // 7  减法
printf("%d\n", a * b);    // 30 乘法
printf("%d\n", a / b);    // 3  整数除法（舍去小数）
printf("%d\n", a % b);    // 1  取模
```

---

**自增自减**
`<var>++ | ++<var> | <var>-- | --<var>`
```c
int c = 5;
printf("%d\n", c++);    // 5  后置自增，返回原值
printf("%d\n", c);      // 6  c 已自增
printf("%d\n", ++c);    // 7  前置自增，返回新值
printf("%d\n", c--);    // 7  后置自减
printf("%d\n", --c);    // 5  前置自减
```

---

## 关系运算符

**比较运算**
`<expr> <==|!=|>|<|>=|<=> <expr>`
```c
int a = 10, b = 3;
printf("%d\n", a == b);    // 0 等于
printf("%d\n", a != b);    // 1 不等于
printf("%d\n", a > b);     // 1 大于
printf("%d\n", a < b);     // 0 小于
printf("%d\n", a >= b);    // 1 大于等于
printf("%d\n", a <= b);    // 0 小于等于
```

---

## 逻辑运算符

**逻辑与、或、非**
`<expr> <&&|||> <expr> | !<expr>`
```c
int a = 10, b = 0;
printf("%d\n", (a > 0) && (b > 0));    // 0 逻辑与（短路：左为假时右不执行）
printf("%d\n", (a > 0) || (b > 0));    // 1 逻辑或（短路：左为真时右不执行）
printf("%d\n", !(a > 0));              // 0 逻辑非
```

## 位运算符

**按位运算**
`<expr> <&|||^|~|<<|>> <expr>`
```c
int a = 6;    // 0110
int b = 3;    // 0011
printf("%d\n", a & b);     // 2  按位与
printf("%d\n", a | b);     // 7  按位或
printf("%d\n", a ^ b);     // 5  按位异或
printf("%d\n", ~a);        // -7 按位取反
printf("%d\n", a << 1);    // 12 左移
printf("%d\n", a >> 1);    // 3  右移
```

---

**位操作宏**
`#define <NAME>(x, pos) ((x) <op> (1 << (pos)))`
```c
// 检查某一位是否为 1
#define CHECK_BIT(x, pos)  ((x) & (1 << (pos)))
// 设置某一位为 1
#define SET_BIT(x, pos)    ((x) |= (1 << (pos)))
// 清除某一位为 0
#define CLEAR_BIT(x, pos)  ((x) &= ~(1 << (pos)))
// 切换某一位的值
#define TOGGLE_BIT(x, pos) ((x) ^= (1 << (pos)))
// 使用
int x = 0;
SET_BIT(x, 2);        // 0100
TOGGLE_BIT(x, 1);     // 0110
if (CHECK_BIT(x, 2)) { /* bit 2 is set */ }
CLEAR_BIT(x, 2);     // 0010
```

## 赋值运算符

**复合赋值**
`<var> <=>|+=|-=|*=|/=|%=|<<=|>>=|&=|^=||= <expr>`
```c
int a = 10, b = 3;
a += b;    // a = a + b = 13
a -= b;    // a = a - b = 10
a *= b;    // a = a * b = 30
a /= b;    // a = a / b = 10
a %= b;    // a = a % b = 1
a <<= 1;   // a = a << 1
a >>= 1;   // a = a >> 1
a &= b;    // a = a & b
a ^= b;    // a = a ^ b
a |= b;    // a = a | b
```

## 其他运算符

**sizeof 运算符**
`sizeof(<type|var>)`
```c
printf("int: %zu\n", sizeof(int));       // 4
printf("char: %zu\n", sizeof(char));     // 1
int arr[10];
printf("count: %zu\n", sizeof(arr) / sizeof(arr[0]));    // 10
```

---

**取地址与解引用**
`&<var> | *<ptr>`
```c
int a = 10;
int *p = &a;       // 取地址
printf("%p\n", p); // 输出地址
printf("%d\n", *p); // 解引用，输出 10
*p = 20;           // 通过指针修改值
```

---

**条件运算符（三目运算符）**
`<condition> ? <expr1> : <expr2>`
```c
int a = 10, b = 3;
int max = (a > b) ? a : b;    // 找出最大值
int min = (a < b) ? a : b;    // 找出最小值
int result = (a % 2 == 0) ? 1 : 0;    // 判断奇偶
```

---

**逗号运算符**
`<expr1>, <expr2>, ..., <exprN>`
```c
int a, b, c;
c = (a = 5, b = 10, a + b);    // 从左到右执行，返回最后一个表达式的值
printf("%d\n", c);    // 15
// 在 for 循环中使用
for (int i = 0, j = 10; i < j; i++, j--) {
    printf("i=%d, j=%d\n", i, j);
}
```

## 表达式类型转换

**隐式类型转换**
`<type> <var> = <other_type_var>;`
```c
int a = 10;
float b = 3.14;
float result1 = a + b;        // int 转换为 float
int result2 = a + (int)b;    // float 转换为 int（截断小数）
```

---

**显式类型转换**
`(<target_type>)<expression>`
```c
double pi = 3.14159;
int radius = 5;
int area = (int)(pi * radius * radius);    // 显式转换
// 指针类型转换
int x = 100;
void *ptr = &x;
int *int_ptr = (int *)ptr;
printf("%d\n", *int_ptr);    // 100
```
