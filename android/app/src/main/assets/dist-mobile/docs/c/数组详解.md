# 数组详解

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## 一维数组

**数组声明**
`<type> <array_name>[<size>];`
```c
int numbers[5];       // 整型数组，大小为 5
float prices[10];     // 浮点型数组
char letters[26];     // 字符型数组
```

---

**数组初始化**
`<type> <array_name>[<size>] = {<values>};`
```c
int arr[5] = {1, 2, 3, 4, 5};    // 完全初始化
int arr[5] = {1, 2, 3};          // 部分初始化，其余为 0
int arr[] = {10, 20, 30};        // 自动推断大小为 3
int arr[10] = {0};               // 全部初始化为 0
```

---

**访问数组元素**
`<array_name>[<index>]`
```c
int arr[5] = {10, 20, 30, 40, 50};
printf("%d\n", arr[0]);    // 输出 10
printf("%d\n", arr[2]);    // 输出 30
```

---

**遍历数组**
`for (int i = 0; i < <size>; i++) { ... }`
```c
int arr[5] = {1, 2, 3, 4, 5};
for (int i = 0; i < 5; i++) {
    printf("arr[%d] = %d\n", i, arr[i]);
}
```

---

**计算数组大小**
`sizeof(<array>) / sizeof(<array>[0])`
```c
int arr[] = {1, 2, 3, 4, 5};
int size = sizeof(arr) / sizeof(arr[0]);
printf("Size: %d\n", size);    // 输出 5
```

## 多维数组

**二维数组声明与初始化**
`<type> <array_name>[<rows>][<cols>] = { {<values>}, ... };`
```c
// 完整初始化
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
// 部分初始化
int matrix[2][3] = {{1, 2}, {4}};        // 其余元素为 0
// 自动推断行数
int matrix[][3] = {{1, 2, 3}, {4, 5, 6}};
```

---

**访问二维数组元素**
`<array_name>[<row>][<col>]`
```c
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
printf("%d\n", matrix[0][0]);    // 输出 1
printf("%d\n", matrix[1][2]);    // 输出 6
```

---

**遍历二维数组**
`for (int i = 0; i < <rows>; i++) { for (int j = 0; j < <cols>; j++) { ... } }`
```c
int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        printf("%d ", matrix[i][j]);
    }
    printf("\n");
}
```

---

**三维数组**
`<type> <array_name>[<depth>][<rows>][<cols>];`
```c
int cube[2][2][2] = {
    {{1, 2}, {3, 4}},
    {{5, 6}, {7, 8}}
};
```

---

**多维数组作为函数参数**
`return_type <func>(<type> <arr>[][<cols>], int <rows>) { ... }`
```c
// 需要指定除第一维外的所有维度大小
void print_matrix(int matrix[][3], int rows) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
}
```

## 字符数组与字符串

**字符数组定义**
`char <array_name>[<size>] = {<chars>};`
```c
char chars[5] = {'H', 'e', 'l', 'l', 'o'};            // 普通字符数组
char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};         // 字符串（以 '\0' 结尾）
```

---

**字符串初始化**
`char <str>[] = "<string>";`
```c
char str[] = "Hello";      // 自动包含 '\0'，大小为 6
char str[10] = "Hello";    // 剩余空间填充 '\0'
```

---

**字符串操作函数**
`#include <string.h> / <func>(<args>);`
```c
#include <string.h>
char dest[50], src[] = "Hello";
// 计算长度
int len = strlen(src);           // 5
// 复制
strcpy(dest, src);
// 连接
strcat(dest, " World");
// 比较
int result = strcmp(str1, str2);  // 0 相等，<0 小于，>0 大于
// 指定长度操作
strncpy(dest, src, 3);           // 复制指定长度
strncat(dest, src, 3);           // 连接指定长度
strncmp(str1, str2, 3);          // 比较指定长度
```

---

**字符串输入输出**
`scanf("%s", <str>) | fgets(<str>, <size>, stdin) | printf("%s", <str>)`
```c
char str[100];
scanf("%s", str);                          // 遇到空格停止
fgets(str, sizeof(str), stdin);           // 读取一行（包括空格）
printf("%s\n", str);                       // 输出字符串
```

## 数组与指针

**数组名与指针的关系**
`<array_name>` 等同于 `&<array_name>[0]`
```c
int arr[5] = {1, 2, 3, 4, 5};
printf("%p\n", arr);        // 数组首元素地址
printf("%p\n", &arr[0]);    // 同上
printf("%d\n", *arr);       // 数组首元素值，输出 1
```

---

**指针算术访问数组**
`*(<ptr> + <n>)` 等同于 `<array>[<n>]`
```c
int arr[5] = {1, 2, 3, 4, 5};
int *p = arr;
printf("%d\n", *p);          // 输出 1
printf("%d\n", *(p + 1));    // 输出 2
printf("%d\n", p[2]);        // 输出 3
```

---

**指针数组：存储指针的数组**
`<type> *<array_name>[<size>];`
```c
int *ptr_array[3];
int a = 10, b = 20, c = 30;
ptr_array[0] = &a;
ptr_array[1] = &b;
ptr_array[2] = &c;
// 字符串数组
char *str_array[] = {"Hello", "World", "C Language"};
```

---

**数组指针：指向数组的指针**
`<type> (*<ptr_name>)[<size>];`
```c
int arr[5] = {1, 2, 3, 4, 5};
int (*p)[5] = &arr;          // 指向整个数组
printf("%d\n", *(*p));       // 输出 1
printf("%d\n", *(*p + 1));   // 输出 2
printf("%d\n", (*p)[2]);     // 输出 3
```

## 变长数组（VLA，C99+）

**运行时确定数组大小**
`<type> <array_name>[<variable>];`
```c
void func(int n) {
    int arr[n];    // 数组大小由参数 n 决定
    for (int i = 0; i < n; i++) {
        arr[i] = i + 1;
    }
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
}
```

## 动态数组

**使用 malloc 创建动态数组**
`<type> *<ptr> = (<type> *)malloc(<size> * sizeof(<type>));`
```c
#include <stdlib.h>
int size = 10;
int *arr = (int *)malloc(size * sizeof(int));
if (arr == NULL) {
    printf("Allocation failed!\n");
    return 1;
}
for (int i = 0; i < size; i++) {
    arr[i] = i + 1;
}
// 使用完毕后释放
free(arr);
arr = NULL;
```

---

**动态调整数组大小**
`<type> *<new_ptr> = (<type> *)realloc(<ptr>, <new_size> * sizeof(<type>));`
```c
int *new_arr = (int *)realloc(arr, new_size * sizeof(int));
if (new_arr == NULL) {
    printf("Reallocation failed!\n");
    free(arr);
    return 1;
}
arr = new_arr;
```

## 数组排序

**冒泡排序**
`void <sort>(<type> <arr>[], int <size>) { ... }`
```c
void bubble_sort(int arr[], int size) {
    for (int i = 0; i < size - 1; i++) {
        for (int j = 0; j < size - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
```

---

**二分查找**
`int <search>(<type> <arr>[], int <low>, int <high>, <type> <target>) { ... }`
```c
int binary_search(int arr[], int low, int high, int target) {
    if (low > high) return -1;
    int mid = low + (high - low) / 2;
    if (arr[mid] == target) return mid;
    else if (arr[mid] > target) return binary_search(arr, low, mid - 1, target);
    else return binary_search(arr, mid + 1, high, target);
}
```
