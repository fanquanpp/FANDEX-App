# 控制流

> **符号约定**：`< >` 必填参数 | `[ ]` 可选参数

---

## if-else 条件判断

**基本 if-else**
`if (<condition>) { ... } else if (<condition>) { ... } else { ... }`
```c
int score = 85;
if (score >= 90) {
    printf("Excellent\n");
} else if (score >= 80) {
    printf("Very Good\n");
} else if (score >= 60) {
    printf("Pass\n");
} else {
    printf("Fail\n");
}
```

---

**嵌套 if-else**
`if (<condition>) { if (<condition>) { ... } else { ... } } else { ... }`
```c
int age = 18;
int has_license = 1;
if (age >= 18) {
    if (has_license) {
        printf("You can drive\n");
    } else {
        printf("You need a license to drive\n");
    }
} else {
    printf("You are too young to drive\n");
}
```

---

**三目运算符简写**
`<condition> ? <expr1> : <expr2>`
```c
int a = 10, b = 20;
int max = (a > b) ? a : b;
printf("Result: %s\n", (a > b) ? "a is larger" : "b is larger");
```

## switch-case 多分支选择

**基本 switch-case**
`switch (<expr>) { case <val>: ... break; [default: ...] }`
```c
char grade = 'B';
switch (grade) {
    case 'A':
        printf("Great!\n");
        break;
    case 'B':
        printf("Good!\n");
        break;
    case 'C':
        printf("Average\n");
        break;
    default:
        printf("Unknown grade\n");
}
```

---

**switch 穿透现象**
`case <val>: case <val2>: ... break;`
```c
// 多个 case 共享同一代码块
int month = 2;
int days;
switch (month) {
    case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        days = 31;
        break;
    case 4: case 6: case 9: case 11:
        days = 30;
        break;
    case 2:
        days = 28;
        break;
    default:
        days = 0;
}
```

## for 循环

**基本 for 循环**
`for (<init>; <condition>; <update>) { ... }`
```c
// 基本 for 循环
for (int i = 0; i < 10; i++) {
    printf("%d ", i);
}
// 省略部分表达式
int j = 0;
for (; j < 10;) {
    printf("%d ", j);
    j++;
}
```

---

**嵌套 for 循环**
`for (...) { for (...) { ... } }`
```c
// 打印乘法表
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        printf("%d*%d=%d\t", j, i, i*j);
    }
    printf("\n");
}
```

---

**多变量 for 循环**
`for (<init1>, <init2>; <cond>; <update1>, <update2>) { ... }`
```c
// 使用多个循环变量
for (int i = 0, j = 10; i < j; i++, j--) {
    printf("i=%d, j=%d\n", i, j);
}
// 无限循环
// for (;;) { ... }
```

## while 循环

**基本 while 循环**
`while (<condition>) { ... }`
```c
int i = 0;
while (i < 10) {
    printf("%d ", i);
    i++;
}
```

---

**无限 while 循环**
`while (1) { ... if (<condition>) break; }`
```c
int count = 0;
while (1) {
    printf("Count: %d\n", count);
    count++;
    if (count >= 5) {
        break;
    }
}
```

---

## do-while 循环

**基本 do-while：至少执行一次**
`do { ... } while (<condition>);`
```c
int i = 10;
do {
    printf("Execute once\n");
    i--;
} while (i < 5);    // 条件为假，但已执行一次
```

---

**菜单驱动程序**
`do { ... } while (<condition>);`
```c
int choice;
do {
    printf("1. Option 1\n2. Option 2\n3. Exit\n");
    scanf("%d", &choice);
    switch (choice) {
        case 1: printf("Option 1\n"); break;
        case 2: printf("Option 2\n"); break;
        case 3: printf("Exiting...\n"); break;
    }
} while (choice != 3);
```

## 循环控制语句

**break：立即退出当前循环**
`break;`
```c
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;    // 当 i 等于 5 时退出循环
    }
    printf("%d ", i);
}
```

---

**continue：跳过本次循环剩余部分**
`continue;`
```c
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;    // 跳过偶数
    }
    printf("%d ", i);
}
```

---

**goto：无条件跳转**
`goto <label>; ... <label>:`
```c
// 基本用法
int i = 0;
start:
printf("i = %d\n", i);
i++;
if (i < 5) {
    goto start;
}
// 跳出多层循环
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            goto end_of_loops;
        }
    }
}
end_of_loops:
printf("Exited all loops\n");
```

## 状态机实现

**使用 switch 实现状态机**
`enum <State> { ... }; while (<state> != FINAL) { switch (<state>) { ... } }`
```c
enum State {
    STATE_START,
    STATE_READING,
    STATE_PROCESSING,
    STATE_FINISHED
};
enum State current_state = STATE_START;
while (current_state != STATE_FINISHED) {
    switch (current_state) {
        case STATE_START:
            current_state = STATE_READING;
            break;
        case STATE_READING:
            current_state = STATE_PROCESSING;
            break;
        case STATE_PROCESSING:
            current_state = STATE_FINISHED;
            break;
        case STATE_FINISHED:
            break;
    }
}
```
